Script to generate IAM tokens for service accounts
see https://clubs.at.yandex-team.ru/ycp/762 for more info.

How to run
~~~~~~~~~~

$ python3 -m venv .
$ source bin/activate
$ pip install -i https://pypi.yandex-team.ru/simple/ -r requirements.txt
$ python __main__.py path/to/account_data.json
