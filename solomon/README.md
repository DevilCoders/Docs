# Solomon

Solomon is a distributed and highly available monitoring service (last S in SaaS).

Key features: 
* **multi-dimensional** time-series data model  
* supports both **PULL** and **PUSH** metrics collecting model  
* rich built-in **graphing and dashboarding capabilities**
* wide range of **alerting** capabilities
