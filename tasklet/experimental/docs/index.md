# Tasklets 2.0

Эта документация описывает работу новых тасклетов,
которые разрабатываются на замену [первой версии](https://wiki.yandex-team.ru/sandbox/tasklets/).

Ключевые особенности новых тасклетов по сравнению с первой версией:
- Возможность использовать любой язык программирования, способный работать с протобуфами. Не только Python.
- Отсутствие привязки к Аркадии.

## Основные сущности доменной модели
### Namespace
Пространство имён, объединяет множество тасклетов.

### Tasklet
Задача, единица работы, выполняющая некоторое понятное действие.

### Build
Сборка, конкретная реализация тасклета. Включает программу и конкретное описание её интерфейса.

### Execution
Запуск тасклета с некоторыми входными данными, соответствующими схеме.

## Спецификация тасклета
Спецификация описывается в `yaml`-файле. По умолчанию ожидается имя `t.yaml`.

## Label
Метка, позволяющая при помощи строки указать конкретный Build. Например: `stable`/`testing`.
