# Что тут есть

Здесь лежат различные утилитки которые запускаются **по крону** в Bannerland YT:

+ `collector` - скрипт, собирающий таблицы в подневные/помесячные на основе конфига
+ `backup_yt_tables` - скрипт, делающий бекапы YT таблиц по конфигу
+ `cleaner` - очистка старых таблиц make banners и full state
