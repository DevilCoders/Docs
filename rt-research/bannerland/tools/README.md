# Что тут есть

Здесь лежат различные утилитки для ручного запуска в Bannerland YT.

bl-kit.py - скрипт для запуска различных шагов (в т.ч. железных) генерации баннеров

Бинарь `tools` в корне этой директории содержит мелкие утилиты различного назначения.

Бинари:
+ `yt_local` - тулза для загрузки и запуска локального YT
+ `upload_export_offers` - скрипт для заливки  `export_offers` файлов в YT (умеет в локальный YT)
+ `pocket` - работа с карманами
