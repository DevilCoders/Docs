OS Packages setup script
========================

Скрипт, который используется в sandbox таске **BUILD_PORTO_LAYER** для сборки слоя мониторинга.
В нем устанавливаются пакеты, которые нужны puppeteer для запуска chromium в headless режиме.

Пример таски в которой собирается наш слой [можно посмотреть тут](https://sandbox.yandex-team.ru/task/783310261/view).
