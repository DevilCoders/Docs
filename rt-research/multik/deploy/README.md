## Набор утилит для поднятия новых стендов MultiK

Основная документация по тому какие шаги надо сделать для того чтобы поднять новый стенд [лежит в wiki](https://wiki.yandex-team.ru/irt/multik/infrastructure/newstand/)

* deploy: Скрипт для поднятия стендов в Deploy. Поднимает проект, стейджи, etc.

* create_tables: Скрипт для заведения структуры таблиц в YT для нового стенда. Создает таблицы и проставляет симлинки.
