Файл mapping содержит данные с маппингом категорий Каталогии в категории Мультика.

Колонки в файле mapping:
direct_id_old
catalogia_name_old
direct_id_new
catalogia_name_new
multik_id
multik_name
virtual
label_type
comment

========
В колонке label_type лежат три значения: 
catalogia - лейбл для пустых категорий, которые не вошли в мультик
multik - категории, которые вошли в мультик
multik_virtual - виртуальные префиксы из старой каталогии.

В колонке virual лежат виртуальные префиксы, в которые маппятся категории. Например, если старая категория была "Опт _ телевизоры", то она маппится в два новых лейбла: в категорию "Телевизоры" и виртуальный префикс "Опт"

Если multik_name не соответствует catalogia_name_new, то категория была переименована. Именно это имя отображается в толоке. 

comment - пояснение к категории, подсказка для толокеров

multik_id формировался по принципу direct_id_new + 100000000. Виртуальным префиксам присвоены id, которые выглядят как 300100ХХХ, где Х - число [0;9].
