# Директория с артефактами миграции bmapi из qloud в YDeploy

Для миграции использовался [мигратор](https://wiki.yandex-team.ru/deploy/docs/migration/qloud/migrator/).
Команды миграции лежат в файлах migrate.sh
yaml-описание стейджей лежит в файлах deploy.yaml

## Отличия

1. Мигратор не мигрирует динамические ресурсы, их мы добавили руками. Так же схема работы с динамическими ресурсами в deploy значительно отличается от
схемы в qloud. Поэтому по договоренности с @modest решили оставить их статическими
1. YDeploy не умеет сам разархивировать ресурсы, поэтому добавлены init-команды, которые распаковывают ресурсы
