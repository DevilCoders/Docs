### get_error_addresses

Возвращает список адресов из фида заданного провайдера, которые не удалось геокодировать, в виде excel-файла.


#### Аргументы командной строки
- `--config` - путь к файлу с конфигом
- `--providers-info` - путь к файлу с дополнительной информацией о провайдерах
- `--name`, `-n` - имя провайдера
- `--out-dir`, `-o` - директория, куда мы будем сохранять результат (по умолчанию в текущую директорию)
- `--table`, `-t` - имя таблицы фидом, при геокодировании которого возникли ошибки (по умолчанию берется последний загруженный фид)
