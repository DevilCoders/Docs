### build_snippet

Используется для автоматической сборки сниппета для городов, указанных в конфиге. Строит вспомогательные таблицы, далее строит сниппет.


#### Аргументы командной строки
- `--config` - путь к файлу с конфигом
- `--providers-info` - путь к файлу с дополнительной информацией о провайдерах
- `--cleanup`, `-c` - необязательный параметр, при указании которого временные таблицы удаляются
