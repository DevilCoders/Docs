### build_provider_feed

Обрабатывает загруженные фиды от провайдеров: проводит обратное геокодирование адресов, чтобы получить их geocoder_id, проверяет результат и объединяет в выходную таблицу.


#### Аргументы командной строки
- `--config` - путь к файлу с конфигом
- `--providers-info` - путь к файлу с дополнительной информацией о провайдерах
