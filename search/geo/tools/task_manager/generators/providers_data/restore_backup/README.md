### restore_backup

Восстанавливает бэкап из ранее созданных таблиц. Можно указать название таблицы, а можно дату, когда был сделан бэкап. Если в выбранную дату было сделано несколько бэкапов, нужно дополнительно указать время.


#### Аргументы командной строки
- `--config` - путь к файлу с конфигом
- `--date, -d` - дата, когда был сделан бэкап
- `--time, -t` - время, когда был сделан бэкап
- `--name, -n` - название таблицы с бэкапом
- `--force, -f` - восстановить выбранный бэкап
- `--list, -l` - показать список доступных бэкапов
