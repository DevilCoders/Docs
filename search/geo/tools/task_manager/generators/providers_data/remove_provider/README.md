### remove_provider

Удаляет провайдера с заданным названием по адресу дома. Если не задан флаг `force`, то будет проведена только проверка заданного адреса, провайдер удален не будет. Можно удалить провайдера с заданным названием в целом регионе, задав его *geoid*. Если задать и *geoid*, и адрес, то удаление будет по адресу. Если не задано ни то, ни другое, то провайдер будет удален из всего сниппета.


#### Аргументы командной строки
- `--config` - путь к файлу с конфигом
- `--title, -t` - название удаляемого провайдера
- `--addr, -a` - адрес дома, из которого нужно удалить провайдера
- `--geoid, -g` - *geoid* региона
- `--force, -f` - удалить провайдера, не проверяя правильность адреса
