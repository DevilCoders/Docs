### get_provider_addresses

Позволяет найти в заданном городе адреса и *geocoder_id* зданий, где есть провайдер с заданным *id* или именем. Можно задать либо имя, либо *id*. Если задать и то, и другое, поиск будет вестись по имени.


#### Аргументы командной строки
- `--config` - путь к файлу с конфигом
- `--providers-info` - путь к файлу с дополнительной информацией о провайдерах
- `--city`, `-c` - название города, в котором ищем
- `--name`, `-n` - имя провайдера
- `--provider-id` - *id* провайдера
- `--limit`, `-l` - ограничение на размер результатов, по умолчанию `20`
