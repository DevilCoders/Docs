### build_logs

Делает выборку из логов крипты и выполняет геокодирование. Сохраняет результат в промежуточную таблицу, которая в дальнейшем используется для построения сниппета.


#### Аргументы командной строки
- `--config` - путь к файлу с конфигом
- `--providers-info` - путь к файлу с дополнительной информацией о провайдерах
