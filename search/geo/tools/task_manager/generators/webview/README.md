### webview

Сниппет webview содержит табы, имеющие *id*, *url* и, возможно, кнопки, которые также описываются своими *id* и *url*. Сейчас поддерживаются только табы, которые получаются из *booking*-урлов компаний с определенными *aref*'ами. Эта информация берется из экспорта Алтая, где организациям ставится в соответствие *url* и указывается провайдер этого *url*'a. На текущий момент запускается с помощью *nirvana*-[воркфлоу](https://nirvana.yandex-team.ru/flow/214b8d86-8893-4d96-b10e-cca63e2907c6/) и обновляется раз в час (см. *lama.yaml*).


#### Аргументы командной строки
- `--config` - путь к файлу с конфигом


#### Конфиг
Конфиг имеет формат *JSON*. Корневой объект, содержащий параметры, называется `webview_snippet`.

Общие параметры:
- `tab-ids` - соответствие между *id* провайдеров и *id* табов
- `cluster` - YT-кластер
- `source-table` - таблица экспорта Алтая с соответствием 
- `snippet-table` - таблица с итоговым сниппетом
