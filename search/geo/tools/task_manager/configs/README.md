## Конфиги Task Manager
- base_sync.json - конфиги для сниппетов заливаемых синхронно с релизом базы Геопоиска;
- fast_task_manager.json - конфиги для сниппетов заливаемых раз в час-полтора ([шедулер](https://sandbox.yandex-team.ru/scheduler/10119/view))
- task_manager.json - конфиги для сниппетов заливаемых раз в сутки ([шедулер](https://sandbox.yandex-team.ru/scheduler/9839/view))
- prepare_realty_task_manager.json - подготовка данных для сниппета realty/1.x ([шедулер](https://sandbox.yandex-team.ru/scheduler/12401/view))
- realty_task_manager.json - варка/заливка сниппета  realty/1.x ([шедулер](https://sandbox.yandex-team.ru/scheduler/12403/view))
- slow_task_manager.json - варка сниппета related_places/1.x ([шедулер](https://sandbox.yandex-team.ru/scheduler/13210/view))
- discovery.json - конфиг для сниппета discovery_events/1.x ([шедулер](https://sandbox.yandex-team.ru/scheduler/22166/view), [воркфлоу](https://nirvana.yandex-team.ru/flow/aae5ba01-941d-47e4-8b98-3e95b101229d/d1d14a4c-a884-4f2e-a2d6-e127a4716a9b/graph), аплоудит сниппет самостоятельно, запускаться должен по готовности входной таблицы)
