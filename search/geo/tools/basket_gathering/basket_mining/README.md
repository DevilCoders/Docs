Пример расчёта в nirvana:

[https://nirvana.yandex-team.ru](https://nirvana.yandex-team.ru/process/d4962860-a176-4a1d-87f0-f16cef7db1c1/graph)

Почитать описание параметров для запуска в ней можно на вики в гайде по запуску всего пайплайна:

[https://wiki.yandex-team.ru](https://wiki.yandex-team.ru/users/spirin-egor/pipelineguide/)

Так же можно запустить локально, но необходим запуск внутри репозиториев Статистики. Прочитать как это можно делать можно тут:

[Интеграция Nile с инфраструктурой Статистики](https://clubs.at.yandex-team.ru/statistics/1143)

При этом в запуск надо будет добавить дополнительные параметры: `result_table` (тег названия таблички), `mode` (сервис, для которого нужен майнинг), `toponym` (флаг, указывается, если майнинг для топонимического саджеста), `job_root` (папка на кластере, куда сохранятся результаты).
