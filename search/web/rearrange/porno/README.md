Переранжирование Porno:
1) вычисляет значение классификатора 'pl' (Porno Level); pl – matrixnet-классификатор наличия порно-интента в запросе; классификатор используют выдачу и флаги визарда;
2) удаляет неуместный контент (порно, мат) в зависимости от значения pl;
3) реализует фильтрацию в семейном поиске;
4) дофильтровывает получившуюся выдачу с помощью фикслистов, данные которых получает из кликовых источников.

Общее описание переранжирования: https://wiki.yandex-team.ru/iskandar/PornoRearrangeRule
Основная схема фильтрации переранжирования: https://wiki.yandex-team.ru/jandekspoisk/porno/Sxemapereranzhirovanijapornoijul2013
