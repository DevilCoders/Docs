Иногда в группу попадают несколько документов с одинаковым (или с похожим) URL, но которые имеют разные языки (и, соответственно, сниппеты).
Правило переранжирования TGroupRepresentativeRule в такой ситуации менят порядок документов _внутри_ группы: из каждого множества похожих URL'ов выбирается представитель, язык которого вероятно более знаком пользователю, и оставляется в группировке.
