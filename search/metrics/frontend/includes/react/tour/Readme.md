# MxTour
Небольшая библиотека с туром по проекту, использующая под капотом [Shepherd.js](https://shepherdjs.dev/docs/)

### Как это все работает
В `localStorage` хранится знание о просмотренных `step` пользователя в виде ключ - значение. Ключ - это уникальный `id` шага. Страничный тур формируется из шагов, которые не были показаны пользователю. Также уникальный `id` проставляется как атрибут у того элемента, где конкретный шаг тура должен быть показан.

### Почему Shepherd.js?
- Очень простой api;
- MIT лицензия (intro.js, например, предлагает заплатить за пользование);
- При необходимости можно отказаться и заменить на самописное решение.
