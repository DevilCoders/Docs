## Общие Bootstrap скрипты

Здесь хранятся скрипты, которые можно подключить в `etnry point` приложения.

Например, в `/bootstrap/lego` подключаются стили из `@yandex-lego/components`, конфигурируется тема, переопределяются переменные. Чтобы начать использовать `@yandex-lego/components` в новом проекте достаточно просто импортировать файл в корне приложения

```(js)
import '@metrics/react/bootstrap/lego';,
``` 

В идеальном мире вся общая часть по конфигурции проекта должна находиться в этой папке, а не непосредственно в каждом проекте:
```(js)
// в руте каждого проекта
import '@metrics/react/bootstrap';,
import App from './app;
``` 
Но пока есть старая кодовая база, приходится обходиться таким решением. По зависимостям каждого проекта видно, что нужно сделать, чтобы прийти в общему решению.
