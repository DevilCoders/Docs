Фильтрация по лейблам с помощью логического выражения. Поддерживаются операторы AND, OR, XOR. <a href="https://wiki.yandex-team.ru/metrics/docs/qgaas/korziny-filtry/#vyrazhenie" target="_blank" rel="noopener">Подробнее</a> про составление выражения.

Можно добавить лейбл, кликнув на него в списке запросов (каждый следующий лейбл добавляется через оператор AND).
