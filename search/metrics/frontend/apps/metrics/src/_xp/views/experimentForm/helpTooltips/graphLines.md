Настройка графиков.

Для скачиваемых хостов считаются метрики, они запоминаются для каждой скачки, и в итоге получаются графики этих метрик.
Чтобы график строился необходимо указать хотя бы один фильтр. <a href="https://wiki.yandex-team.ru/metrics/docs/graph/#kakpostroitgrafik" target="_blank" rel="noopener">Подробнее</a> про настройку графиков.
