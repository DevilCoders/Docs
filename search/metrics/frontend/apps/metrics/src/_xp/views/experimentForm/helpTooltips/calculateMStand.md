Если установлен параметр «After each enrichment», то по окончанию каждого запуска обогащений будет выполнен перерасчет mstand метрик.

В случае выбранного параметра «Once» вычисление mstand-метрик будет запущено после завершения всех обогащений, от которых зависит соответствующий набор mstand-метрик.
