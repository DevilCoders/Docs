# Оценка расширений

При передаче на базовые поиски слова и словосочетания запроса расширяются синонимами. Таким образом обеспечивается расширение области поиска и, как следствие, увеличение количества релевантных документов в выдаче.

При формировании сниппетов слова, входящие в поисковый запрос, выделяются (жирным шрифтом). Слова, являющиеся расширениями, также необходимо выделять.

Для оценки качества дополнения синонимами используется шкала расширений. При выставлении оценок анализируются сниппеты документов поисковой выдачи.

Список оценок, выставляемых для оценки качества расширений, приведен в таблице ниже

### Оценки расширений
Оценка | Описание
----- | -----
Расширение | Выделенное в сниппете слово/словосочетание является по отношению к слову/словосочетанию из запроса:<br/><br/>- Альтернативной формой.<br/>- Синонимом или близким по значению словом.<br/>- Результатом транслитерации.<br/>- Расшифровкой аббревиатуры.<br/>- Аббревиатурой.<br/>- Исправлением (от опечаток).<br/>- Переводом с русского/английского языков.
Разнообразие | Выделенное в сниппете слово не является синонимом к содержащемуся в запросе слову/словосочетанию, но расширяет прагматику запроса. Такие слова используются для привлечения внимания пользователя к содержимому документа (например, слово «скачать» в результате поиска по запросу, содержащему имя исполнителя).
Другое | Неверное расширение поискового запроса (в том числе — расширение слов из устойчивых выражений и стихов/текстов песен) или неверная лемма.
