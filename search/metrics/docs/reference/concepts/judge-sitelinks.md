# Оценка релевантности сайтлинков

Сайтлинки — это дополнительные ссылки в сниппете, ведущие на разделы сайта, к которому относится найденный документ. Позволяют переходить к требуемому разделу сайта непосредственно со страницы поисковой выдачи.

Так как сайтлинки выделяются для сайта и могут совпадать для разных поисковых запросов, необходимо собирать информацию об уместности сайтлинка для конкретного поискового запроса.

Правила оценки релевантности сайтлинков совпадает с правилами оценки [релевантности документов](judgeDocs.md).
