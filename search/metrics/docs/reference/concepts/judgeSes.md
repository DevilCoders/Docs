# Оценка запросов сессионного поиска

Задания асессоров составляются из запросов реальных пользователей. Пользователь может не найти требуемой информации в результатах поисковой выдачи. В этом случае запрос может быть изменен или уточнен. Данное поведение может также быть вызвано сведениями, полученными пользователем из сниппета или документа поисковой выдачи. В подобных ситуациях интересна оценка качества поиска в контексте пользовательской сессии  — т.е., оценка качества для конкретного пользователя. Для задач данного типа используется оценка запросов сессионного поиска.

Оценка документов в заданиях сессионного поиска выполняется по [шкале релевантности](judgeDocs.md). При этом поисковый запрос рассматривается в рамках конкретной пользовательской сессии. Таким образом, в большинстве случаев при оценке запросов сессионного поиска предметная область запроса сужается.
