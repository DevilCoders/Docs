# Качество поиска

Основной задачей поисковой системы является предоставление пользователю результатов, максимально соответствующих интенту запроса. Чем большая часть аудитории удовлетворена ответами поисковой системы, тем выше качество поиска.

Оценка удовлетворенности пользователей результатами поиска является сложной задачей. Это обусловлено значительным количеством (несколько десятков миллионов) ежедневных обращений к поисковой системе Яндекса и отсутствием надежного способа сбора оценок пользователей. Таким образом, упрощение оценки качества поиска сводится к ограничению количества оцениваемых запросов за счет формирования репрезентативной выборки реальных запросов пользователей и моделированию их поведения.

Для моделирования аудитории используется [служба асессоров](asessorsDept.md), основной задачей которой является наполнение базы оцененных запросов. Данная база используется при подборе алгоритма ранжирования, а также при оценке того, насколько хорошо работает алгоритм по сравнению с конкурентами.

Для оценки качества работы алгоритма используются метрики. Метрикой называется число, отражающее некоторую характеристику поискового ответа (далее — серпа). Такой характеристикой может быть, например, количество корневых страниц, найденных в первых десяти результатах поиска.

Совокупность метрик позволяет оценить различные аспекты качества поиска.
