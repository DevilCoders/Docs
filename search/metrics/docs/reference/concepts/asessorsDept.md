# Служба асессоров

Яндекс обладает собственной службой асессоров, задачей которой является анализ поисковых запросов, их оценка и обработка в соответствии с различными методиками и технологиями.

Сотрудники службы асессоров расположены в различных городах. Это позволяет распределять запросы таким образом, чтобы запрос обрабатывался асессором из наиболее близкого к пользователю региона.

Ежедневно в службу асессоров приходит на оценку несколько десятков запросов, соответствующие этим запросам «обобщенные ответы» нескольких поисковых систем (см. раздел [Общая схема оценки качества Поиска](generalDevelScheme.md)) и сопутствующая информация.

Для каждого запроса формируется задание. В каждое задание случайным образом включается несколько URL из «обобщенного ответа». Данные задания направляются асессорам. Одно задание выполняется одним или двумя асессорами в зависимости от настроек вероятности передачи задания второму асессору. После выполнения заданий для каждого URL вычисляется средняя оценка релевантности.

Оценки формируют информационную базу, которая используется для разработки и оценки качества алгоритмов поиска. В зависимости от решаемой задачи работа асессоров регламентируется соответствующим [набором инструкций](https://wiki.yandex-team.ru/JandeksPoisk/OcenkaKachestvaPoiska/InstrukciiAsessoram). Данные инструкции составляются менеджерами, заказывающими определенные типы оценок.

Служба асессоров занимается решением следующих задач, связанных с оценкой результатов поисковых выдач:
- [Оценка документов](judgeDocs.md).
- [Оценка релевантности изображений](judgeImages.md).
- [Оценка видео](judgeVideo.md).
- [Оценка мультиобъектных запросов](judgeMulti.md).
- [Оценка запросов сессионного поиска](judgeSes.md).
- [Оценка серых запросов](judgePorn.md).
- [Оценка зарекламленности выдачи](judgeAdv.md).
- Оценка сниппетов (см. страницу [Новая репрезентативность сниппетов](http://ang2.yandex-team.ru/wiki/-view?page=newrepresent) в [Вики](https://wiki.yandex-team.ru)).
- [Разметка спама](markupSpam.md).
