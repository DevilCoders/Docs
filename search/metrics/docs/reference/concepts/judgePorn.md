# Оценка серых запросов

Серыми являются запросы, которые могут иметь порно-интент, но при этом не являются явно порнографическими. Примерами таких запросов являются «школьницы», «азиатки», «медсестра».

Так как серые запросы не подразумевают явного порнографического интента, одной из целей Поиска является исключение порнографических результатов для данной группы запросов. Серые запросы формируют поток «Порно».

Оценки, выставляемые для запросов данного потока, используются для расчета метрик, отражающих степень загруженности результатов поиска порнографией и эротикой.

Список оценок, выставляемых документам по серым запросам, представлен в таблице.

### Оценки серых запросов
Оценка | Описание
----- | -----
Нет порно | Документ не содержит материалов эротического и порнографического характера (в том числе рекламы).
Порно-контент | Документ содержит материалы (текст / видео / изображения) эротического или порнографического характера или предложение услуг.
На грани | Документ содержит ненормативную лексику, материалы желтой прессы.
Порно-реклама | Документ содержит рекламу порно-продукции или интим-услуг.
