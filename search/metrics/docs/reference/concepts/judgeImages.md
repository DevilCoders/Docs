# Оценка релевантности изображений

Для оценки релевантности изображений используются запросы из лог-файлов поиска по картинкам.

Список оценок релевантности изображений представлен в таблице ниже.

### Оценки релевантности изображений
Оценка | ID | Описание
----- | ----- | -----
Релевантный+ | R+ | Объект запроса присутствует на изображении и находится на переднем плане. Качество изображения высокое, картинка не размытая. Если запрос имеет несколько интентов, то оценка выставляется каждому из интентов при условии качественного изображения.
Релевантный- | R- | Объект запроса плохо различим и не является первостепенным на изображении, отражена только одна из составляющих многословного запроса. Картинка содержит надпись (например, копирайт) или является результатом творчества (например, какие-то объекты дорисованы).
Нерелевантный | IR | Объект запроса отсутствует на изображении или является второстепенным. Картинка содержит результат творчества, не характеризующий объект запроса (например, пейзаж авторства профессора по химии).
Не открывается | _404 | Доступ к изображению невозможен (изображение не существует, не открывается, требует пароль).
