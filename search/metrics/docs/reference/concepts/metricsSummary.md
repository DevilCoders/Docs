# Метрики

Для оценки качества поиска используются метрики. Метрикой называется число, отражающее определенную характеристику набора поисковых ответов.

Метрики рассчитываются по формулам, переменными в которых могут быть такие параметры поисковой выдачи, как количество и позиции документов, имеющих определенную оценку релевантности, количество документов, которым выставлены оценки асессоров и т.п. Таким образом, источниками данных для расчета метрик являются результаты поисковой выдачи и оценки асессоров.

Для получения результатов поисковой выдачи производится набор запросов к поисковой системе. В случае расчета метрик, для которых требуются предварительно оцененные асессорами документы, оценки по запросу ставятся в соответствие документам скаченной поисковой выдачи.

Даже в том случае, если метрика считается по оцененным запросам, не все документы, попадающие в поисковые выдачи предварительно оценены асессорами. Причина заключается в том, что выдача по запросу изменяется со временем в связи с изменением массива документов и алгоритмов поиска.

Возможны два подхода к решению проблемы неоцененных документов:

- использование [классических и judged-метрик](judgedNnotjudged.md);
- [дооценка](metricsEval.md).

Важным аспектом оценки качества поиска является анализ динамики метрик. Для анализа динамики [используется исторический график](history-graph-types.md).
