# Оценка мультиобъектных запросов

Мультиобъектными являются запросы, по которым возможны несколько интентов. Примерами таких запросов являются: «ягуар» (марка автомобиля, животное, напиток, фильм), «Наполеон» (торт, император).

Оценки качества по мультиобъектному поиску выставляются в соответствии с правилами оценки [релевантности документов](judgeDocs.md) для каждого из интентов запроса.

При расчете метрик мультиобъектного поиска используются сведения о популярности каждого интента.

Для определения популярности интента среди пользователей используется вес — число в диапазоне от 0 до 1. Вес отражает долю пользователей, которым интересен соответствующий интенту смысл запроса. Для расчета веса используются данные от пользователей, выполняющих переформулировки запросов. Сведения о переформулировках сохраняются в поисковых лог-файлах. По этим данным строится граф, на основании которого каждому интенту присваивается вес.

Веса используются для расчета метрик, отражающих качество поиска по разным интентам мультиобъектного запроса.
