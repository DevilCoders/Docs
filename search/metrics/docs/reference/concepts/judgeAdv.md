# Оценка зарекламленности выдачи

Одним из факторов, определяющих качество поиска, является преобладание в поисковой выдаче документов, не содержащих избыточного количества рекламы.

Реклама может быть разделена на две группы:

- Всплывающая реклама. Оценивается в аспекте [Попап](#popup).
- Реклама, отображаемая на страницах сайта. Оценивается в аспекте [Зарекламленность](#adv).


## Попап {#popup}

Одной из распространенных форм рекламы в интернете является всплывающая реклама. Особенностью данного вида рекламы является открытие нового окна при нажатии в любой области документа (кликандер) либо при при попытке закрыть рекламный слой (попандер).

Список оценок, выставляемых при оценке документов на предмет всплывающей рекламы, приведен в таблице ниже.


### Оценки загруженности всплывающей рекламой {#popup}
Оценка | Описание
----- | -----
Попандер | Присутствует реклама типа «попандер» (открывается новое окно при попытке закрыть рекламный слой).
Кликандер | Присутствует реклама типа «кликандер» (открывается новое окно при нажатии в любой области документа).
Оба | Присутствует реклама обоих типов.
Ничего | В документе отсутствует всплывающая реклама.



## Зарекламленность {#adv}

Реклама является одной из основных статей дохода любого сайта. При этом качественный документ, ориентированный на пользователя, не должен содержать агрессивную рекламу, отвлекающую внимание от основного содержимого. Для оценки качества интеграции рекламы используется шкала зарекламленности.

Список оценок, выставляемых для оценки документов на зарекламленность, приведен в таблице ниже.


### Оценки зарекламленности выдачи {#adv}
Оценка | Описание
----- | -----
Чисто | Реклама аккуратно интегрирована в дизайн документа или отсутствует. Не мешает просмотру содержимого и не преобладает над контентом.
Годится | Реклама выделяется на фоне содержимого документа, но не мешает доступу к основному содержимому.
Мешает | Реклама мешает навигации и/или закрывает содержимое документа.
Невозможно | Реклама сильно затрудняет доступ к содержимому документа (который, возможно, создан для показа рекламы).
