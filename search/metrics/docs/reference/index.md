# Оценка качества поиска

**Справочник содержит** основные сведения об оценке качества поиска, принципах работы службы асессоров и типах метрик.

**Целевой аудиторией** являются менеджеры, аналитики и разработчики поиска, а также прочие сотрудники, которым требуется информация по оценке качества поиска и типам используемых метрик.

**Справочник не содержит** описания метрик.

## Структура документа {#structure}

Документ состоит из следующих разделов:

1. [Качество поиска](concepts/searchQuality.md): общее описание принципов оценки качества поиска, сведения об источниках данных и способах их обработки.
1. [Служба асессоров](concepts/asessorsDept.md): общие сведения о службе асессоров, описание типов и принципов выставления оценок.
1. [Метрики](concepts/metricsSummary.md): классификация и описание общих принципов расчета метрик, сведения об оцененности выдачи и исторических графиках.
1. [Приложения](reference/ElementsWeights.md): сведения об условных весах документов, экспериментальные данные о зависимости устойчивости метрик от оцененности, описание фильтров, применяемых при ранжировании.

Информация о релизах документа представлена в разделе [Релизы](concepts/releases.md).
