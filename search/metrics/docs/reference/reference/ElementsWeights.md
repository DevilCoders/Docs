# Условные и вероятностные веса оцененных документов

Для использования оцененных элементов в некоторых метриках были рассчитаны вероятностные веса, которые отражают степень релевантности документа. Вероятностные веса отличаются для разных потоков.

- [Релевантность документов](#doc-weights).
- [Релевантность картинок](#imagesweights).
- [Релевантность видео](#video).
