# Зависимость устойчивости метрик от оцененности

## Цели эксперимента {#aims}

Для корректной оценки показателей метрик необходимо понимать влияние неполной оцененности выдачи на стабильность измерений. Эксперимент необходим для изучения зависимости устойчивости некоторых метрик от оцененности.

## Описание эксперимента {#description}

На первом этапе были рассмотрены запросы с полной оцененностью. На них были рассчитаны средние значения метрик.

На втором этапе из них исключали по одной случайным образом выбранной оценке и подсчитывали среднее по всем запросам с девятью из десяти оцененных запросов (по всем запросам, у которых один документ не оценен и тем, что были оценены полностью, но с учетом выкидывания по одной оценке из каждой выдачи).

На каждой из следующих итераций выкидывалось еще по одной оценке. Таким образом, множество усреднения с каждым шагом уменьшалось.

Следует обратить особое внимание на то, что данный график описывает идеальное соотношение. В реальности оценка качества поиска требует большой точности, ведь даже хороший новый алгоритм не дает, как правило, прирост качества больше, чем в один процент. Реальные соотношения могут, т. о., незначительно отличаться. Тем не менее данный график отражает общие тенденции.

## Вывод {#conclusion}

C увеличением неоцененности среднее значение уменьшается, но изменение значения очень сильно зависит от метрики. Несложно заметить, что класс _judged_ метрик наиболее устойчив к падению оценнености.
