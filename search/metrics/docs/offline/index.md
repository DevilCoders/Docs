# Введение

**Документ содержит** описание офлайн-метрик (далее — <q>метрик</q>) — количественных показателей, отражающих определенные характеристики набора поисковых ответов. Офлайн-метрики рассчитываются в сервисе в [Metrics](http://metrics.yandex-team.ru) и применяются для оценки качества поиска Яндекса.

**Документ предназначен** для менеджеров, аналитиков и разработчиков Поиска, а также для сотрудников других подразделений, которым требуется информация по оценке качества поиска и применяемым метрикам.

## Структура документа {#structure}

Документ состоит из следующих разделов:

1. [Общие сведения](./concepts/basicinfo.md): краткая информация об области применения и типах метрик, правилах расчета и сервисе по работе с метриками.
    
1. [Формат именования метрик](./concepts/naming.md): описание формата именования офлайн-метрик в интерфейсе [Metrics](http://metrics.yandex-team.ru).
    
1. Разделы [A](./concepts/a.md) —[_404](./concepts/_404.md): описание метрик, сгруппированное по первому символу названия (без учета компонента характеристики <q>judged</q>).
    
1. [Архив](./concepts/archive.md): описание устаревших метрик, которые более не используются для оценки качества поиска.
    
1. [Терминология](./reference/term.md): список терминов и понятий, используемых в документе.
    

Информация о выпущенных версиях документа представлена в разделе [Релизы](./concepts/releases.md).
