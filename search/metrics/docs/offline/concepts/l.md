## link {#link}

Доля документов, имеющих метку <q>найден по ссылке</q>, в первых $n$ результатах поисковой выдачи. 

Метка <q>найден по ссылке</q> выставляется для документов, не содержащих слов запроса, но на которые ссылается большое количество документов, как на содержащие информацию по теме запроса. 

### Расчет метрики {#link-calc}

Значение метрики равно отношению количества страниц, имеющих метку <q>найден по ссылке</q>, к общему числу документов ($n$).
 $link = \frac{n_{link}}{n} { ,}$
 
где $n_{link}$ — количество документов, имеющих метку <q>найден по ссылке</q>, в первых $n$ результатах поиска.
