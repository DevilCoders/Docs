## useful {#useful}

Позиция полезного документа среди первых n результатов выдачи.

Метрика вычисляется для запросов, в результатах поиска по которым присутствуют полезные документы.

Если по запросу находится более одного полезного документа, для расчета значения метрики используется только один из них, расположенный наиболее близко к первой позиции выдачи.

### Расчет метрики {#useful-calc}

Значение метрики рассчитывается по формуле:

$useful = /left/{ /begin{array}{ll} 1 - \frac{n_{usf}}{n} {,} & n_{usf} < n// 0 {,}& n_{usf} /geq n /end{array} /right. { ,}$

где:

* $n_{usf}$ — номер позиции полезного документа;
* $n$ — рассматриваемая глубина поисковой выдачи.

Позиция считается с нуля, т. е. для полезного документа на первой позиции $n_{usf}=0$, на второй — $n_{usf}=1$ и т. д.
