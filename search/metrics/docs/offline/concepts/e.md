# E

В данном разделе приведено описание метрик, первой буквой в названии которой является <q>E</q> (без учета [оцененности](naming.md#judged)). {#list}

## empty-serp {#empty-serp}

Отсутствие результатов поиска по запросу. 

### Расчет метрики {#empty-serp-calc}

Для запроса является бинарной и принимает значения:

- <q>0</q> — результаты поиска присутствуют;
- <q>1</q> — результаты поиска отсутствуют.

Для потока отражает долю запросов, для которых отсутствуют результаты поиска.

## eng-wide-pfound {#eng-wide-pfound}

Общая оценка качества поиска англоязычных документов в латиническом потоке. При расчете метрики оценки документов на языках, отличных от английского, обнуляются.

### Расчет метрики {#eng-wide-pfound-calc}

Метрика рассчитывается по формуле [pfound](p.md#pfound), при этом вес документов на языках, отличных от английского, принимается за «0».
 $eng{-}wide{-}pfound = \sum\limits_{i=1}^{n} pLook_{i} * {pRel_{i}} { ,}$
 
где:

1. $pLook_{i}$ — это вероятность просмотра $i$-го результата выдачи;
1. $pRel_{i}$ — вероятностный вес элемента.

Вероятность просмотра `i`-го результата $pLook_{i}$ равна вероятности просмотра предыдущего элемента выдачи $pLook_{i-1}$ с поправкой на вероятностный вес предыдущего элемента и вероятность того, что пользователь дойдет до этого места:

$pLook_{i} = pLook_{i-1} * (1 - pRel_{i-1}) * (1 - pBreak) { ,}$
 
где:

1. $pRel_{i-1}$ — вероятностный вес ($i-1$)-го элемента выдачи;
1. $pBreak$ — вероятность того, что пользователь дойдет до $i$-го места (данная вероятность является константой: $pBreak=0,15$).

Вероятность просмотра первого элемента $pLook_{1}$ равна единице.
