# Общие сведения

Офлайн-метрикой (далее — <q>метрика</q>) называется число, отражающее определенную характеристику набора поисковых ответов.

Метрики применяются для оценки [качества поиска](http://doc.yandex-team.ru/analytics/metrics/concepts/searchQuality.xml) по следующим сервисам:

- Поиск.
    
- Видео.
    
- Картинки.
    
- Геопоиск.

- Маркет.
    

В зависимости от оцениваемых аспектов поисковой выдачи выделяются следующие группы метрик:

- Релевантность — соответствие выдачи поисковому запросу.

- Зарекламленность — наличие избыточного количества рекламы.

- Попап — наличие всплывающей рекламы.

- Порно — наличие материалов порнографического характера и ненормативной лексики.

- Спам — наличие и категории спама.

- Свежесть — наличие свежих документов в поисковой выдаче.

- Зарубеж — оценка качества поиска в латиническом потоке.

- Дубликаты — наличие дубликатов в поисковой выдаче.

- Мобильный поиск — оценка качества мобильного поиска.

В рамках оцениваемого аспекта метрики группируются по назначению:

- Технические (ALERT) — предназначены для получения информации о техническом состоянии SERP, например количество найденных документов. {#alert}
    
- Оценочные (JUDGMENT) — предназначены для получения общей оценки качества поиска по сервису, например мера отклонения выдачи от идеального ответа. {#judge}
    
    Как правило, из метрик данного типа формируется перечень основных метрик, применяемых для принятия решений об эффективности внедрения доработок сервиса.
    
- Информационные (INFORMATIONAL) — предназначены для получения данных об отдельной группе результатов поиска с заданной оценкой асессоров, например доля релевантных документов в первых пяти результатах выдачи. {#info}
    

Метрики рассчитываются по формулам, переменными в которых могут быть такие параметры поисковой выдачи, как количество и позиции документов, имеющих определенную оценку релевантности, количество документов, которым выставлены оценки асессоров и т. п. Таким образом, источниками данных для расчета метрик являются результаты поисковой выдачи и оценки асессоров.

Все метрики, приведенные в данном документе, рассчитываются в инструменте [Metrics](http://metrics.yandex-team.ru).

Общие правила расчета метрики:

1. Определяются сравниваемые поисковые системы и группа запросов.
1. Скачивается SERP в соответствии с заданными в [Metrics](http://metrics.yandex-team.ru) параметрами.
1. Выполняется расчет метрики по каждому запросу для выбранных систем.
1. Производится усреднение значений метрики для каждой системы:
    - по всем запросам — сумма значений метрики по каждому запросу делится на общее количество запросов;
    - по пересекающимся запросам — сумма значений метрики по запросам, присутствующим в каждом SERP делится на количество пересекающихся запросов, для которых метрика валидна.
    
1. Рассчитывается разница усредненных значений метрики сравниваемых SERP.
