# \_404

Доля не открывающихся видеороликов в первых n результатах выдачи по видеопоиску (с оценкой <q>\_404</q>).

## Расчет метрики {calc}

Значение метрики равно отношению количества документов, имеющих оценку «\_404», к общему количеству документов (n):

$\_404 = \frac{n_{\_404-5}}{n} { ,}$
 
где $n_{\_404-5}$ — количество документов в первых n результатах выдачи, имеющих оценку <q>\_404</q>.
