# B

В данном разделе приведено описание метрик, первой буквой в названии которой является <q>B</q> (без учета [оцененности](naming.md#judged)).

## bad-count {#bad-count}

Доля документов в первых n результатах поисковой выдачи, которым выставлена оценка <q>Мешает</q>. {#list}

### Расчет метрики {#bad-count-calc}

Значение метрики равно отношению количества документов, которым выставлена оценка <q>Мешает</q>, к общему числу документов (n):

$bad{-}count = \frac{n_{bad}}{n} { ,}$
 
где
 $n_{bad}$ — количество документов в первых n результатах поиска, имеющих оценку <q>Мешает</q>.

Общие сведения об оценке зарекламленности выдачи приведены в разделе [Зарекламленность](http://doc.yandex-team.ru/analytics/metrics/concepts/judgeAdv.xml#adv) документа [Оценка качества поиска](http://doc.yandex-team.ru/analytics/metrics/concepts/About.xml).
