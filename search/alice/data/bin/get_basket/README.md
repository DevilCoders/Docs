# Корзиновый сборщик

Скрипт подготовки корзин запросов в [таком формате](https://wiki.yandex-team.ru/otbet/format-korziny/).

Скрипт стоит запускать **2 раза**. 

При первом запуске укажите небольшие значения выборок, например `--n1 1000 --n2 1000`.
А выходе последнего кубика будут достаточные значения размеров выборок,  
которые необходимо передать в качестве параметров для основного запуска сбора больших корзин.
