Утилита для поиска записей, встречающиеся только в одной из двух таблиц.
Аналогом является exclusion join в yql, применительно ко всем полям таблицы.
Работает в виде MapReduce операции по всем хэшируемым полям таблицы, с редьюсом, который
обращаем внимание на кол-во строк в группе - если их больше одной, то на выходе ничего не будет.
Большая часть времени операции - стадия partitioning.

# Использование

./yt_diff \
	-f //path/to/first_table \
	-s //path/to/second_table \
	-o //path/to/output_table \
	-c yt_cluster \
	-t yt_token
