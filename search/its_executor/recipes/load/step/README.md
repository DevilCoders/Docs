### Последующие шаги прогрузки

`location`.yaml — наливает дополнительно x% трафика в локации `location` каждой вертикали.
Значения x для каждой вертикали, а также секции каждой вертикали, участвующие в прогрузке,
находятся в config.py.

C помощью параметра `vertical` можно изменить значение x для данной вертикали.
