### Рецепты для прогрузки вертикалей

+ `defaults` — вернуть дефолт
+ `first_step` — первый шаг прогрузки
+ `step` — последующие шаги прогрузки

Все рецепты работают для локаций `SAS`, `MAN` и `VLA`!
Локации типа `DEVNULL`, `PUMPKIN`, `YP_SAS`, `NEW_MAN` не затрагиваются!

В файле config.py находится список вертикалей для прогрузки.
Для каждой вертикали задан список секций, участвующих в прогрузке, а также
дефолтное значение шага прогрузки для этой вертикали.
