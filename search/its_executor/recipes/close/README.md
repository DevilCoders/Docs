## Рецепты для увода трафика из локации

`location`/`vertical`.yaml — рецепт для вертикали `vertical` в локации `location`

`location`/all.yaml — рецепт для одновременно всех вертикалей в локации

По умолчанию рецепт отливает 10% из данной локации, равномерно разливая по остальным. Этот процент можно менять, изменяя параметр `step`.

В разливании трафика участвуют только локации `SAS`, `MAN` и `VLA`! Локации типа `DEVNULL`, `PUMPKIN`, `YP_SAS`, `NEW_MAN` не затрагиваются!

Для быстрого закрытия локации достаточно поставить `step=100`.

Список вертикалей, для которых применяется рецепт, можно найти в начале файла generate.py.
