### JAR repack

Это очень простенькая утилита на python, которая перепаковывает `.jar`-файлы перед публикацией артефактов. Нужно использовать в комбинации с [UBERJAR()](https://wiki.yandex-team.ru/yatool/java/#uberjar). Перепаковка заключается в том, что из архива удаляются все классы, название которых не начинается с префикса `ru.yandex.` — чтобы опубликованный артефакт содержал в себе только классы собственно из Аркадии, но не содержал внешние зависимости из `contrib/`.
