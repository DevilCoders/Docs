# Стабильность в Яндексе

**9999** — кодовое название активности по повышению надежности.  

У многих сервисов уже есть голсы про «четыре девятки». В них формулируются цели, которые мы хотим достичь в следующем полугодии, чтобы Яндекс стал еще надежнее: ущерб для пользователей уменьшился, количество рецидивов инцидентов в продакшене было меньше, скорость их устранения возросла и повысилась скоординированность действий различных команд.
