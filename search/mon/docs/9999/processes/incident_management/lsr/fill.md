# Что нужно для того, чтобы провести LSR

 - [Очередь LSR тикетов](https://st.yandex-team.ru/LSR/)
 - [Дашборд по LSR тикетам](https://st.yandex-team.ru/agile/board/409/sprints)

## Формат

 - Ко встрече надо предварительно завести LSR-тикет в очереди LSR по [шаблону](https://st.yandex-team.ru/createTicket?template=418&queue=LSR).
 - В наблюдатели добавляются все присутствующие на встрече
 - Исполнителем ставится основной докладчик по инциденту
 - Action Items выделенные на LSR размечаются как LSR AI и дополнительно трекаются в warden (обязательный закоммит на их выполнение)
