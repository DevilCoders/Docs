# Кто такой Morty?
Это система шедулинга и автоматической выкатки поисковых релизов.

Таким образом Марти не требуется катить релиз классическим способом, но необходимо подтвердить выкатку и проследить за отсутствием ошибок.
## Apphost
**Что это?**
Apphost - общая шина поиска.
**Что может взорвать?**
Поиск целиком, не только web.

**Когда обычно приезжает в очередь?**
В Морти загружается с вечера, время выкатки обычно по будням в 8 утра.

**Когда обычно катим?**
Желательно по расписанию, если нет других факторов, например не успели еще выехать базы поискового индекса.

**Сколько катится?**
1ч - 1,5ч

**Кого призывать при проблемах?**
При проблемах на Apphost: дежурные от сервиса [Apphost](https://warden.yandex-team.ru/components/apphost)
При проблемах с Morty: дежурные от сервиса [Morty](https://warden.yandex-team.ru/components/spi-tools/s/morty)

**Как откатывать?**
Через дежурных от сервиса [Apphost](https://warden.yandex-team.ru/components/apphost)
