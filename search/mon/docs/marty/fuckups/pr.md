## Общая информация
* Если Марти узнал о проблеме, которая заслуживает внимания PR/SMM, привлекаются дежурные от этих сервисов, или информация дублируется в кризисный PR чат.
* Марти привлекает дежурных от PR/SMM в красных протоколах.
* При необходимости привлекается руководитель PR @igrabovskiy.
* При необходимости используется [бот SMM](https://t.me/smm_alarm_bot) для сбора ссылок из соц.сетей.
## Кризисный чат
* Марти следит за чатами созданными в Кризисном PR чате [командой PR взаимодействия](https://warden.z.yandex-team.ru/components/pr).
    * Этот чат предназначен для оперативного реагирования на PR-кризисы.
    * Участники: все ответственные от команд PR, SMM, саппорт и дежурных
    * Задачи чата: 
        * информирование коллег о проблемах/кризисах, 
        * уточнение у коллег инфо о наличии открытых Протоколов кризисов и  вводные по инцидентам. 
    * Марти привлекает всех необходимых дежурных от нужных вертикалей, следит за ходом координации PR инцидентам
