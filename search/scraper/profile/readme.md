# Профили Scraper

Если добавляется новый профиль или удаляется старый необходимо отредактировать файл ya.make в этой директории. Каждый профиль описывается в разделе RESOURCE в следующем формате.
```
...
<profile_name>.json /profile/<profile_name>
...
```
Где `<profile_name>` имя профиля (часть имени файла до `.json`). Каждый профиль описывается отдельной строкой. При создании профиля необходимо добавить соответствующую строчку 
в вышеуказанный файл. При удалении профиля необходимо удалить и соответствующую строку в файле.

Небольшая дока по профилям: https://wiki.yandex-team.ru/jandekspoisk/metrics/scraper/profiles/
