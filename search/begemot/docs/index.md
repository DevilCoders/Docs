# О библиотеке

Библиотека  позволяет писать и исполнять регулярные выражения, где в качестве элементарной единицы выступает не символ, а слово ( в терминах библиотеки). Поддерживаются стандартные операции регулярного языка: `\* ? + | ^ $`. Основным преимуществом библиотеки является то, что можно в правилах отмечать именованные захватывающие выражения и получать соответствующие захваченные цепочки текста в результате.

Также в рамках общей библиотеки  Remorph есть под-библиотеки TokenLogic и Char. TokenLogic позволяет писать правила матчинга токенов без учета порядка их следования, а Char — правила для обычного посимвольного матчинга с учетом границ.

В Аркадии библиотека расположена в директории [kernel/remorph](https://arcadia.yandex.ru/wsvn/arc/trunk/arcadia/kernel/remorph).

Для использования  вне Аркадии существует библиотека libyandex-remorph.so, интерфейс которой не использует каких-либо специфичных для Аркадии типов. Она доступна в составе релизов.
