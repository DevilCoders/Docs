## Бегемот

Это сервис для получения переколдовки пользовательского запроса.

Серп → репорт → бегемот → поиск.
Результат работы бегемота — qtree запроса, ответы правил, rearr- и relev- параметры.

Более подробная документация — тут: https://wiki.yandex-team.ru/Begemot/

## Что где лежит

- Правила: `rules/`, каждое в своей папке. Пучки правил сгруппированы вместе, например `rules/qtree`.
- Описания шардов и данные бегемота: `data/`. Данные, используемые совместно с визардом, — `/search/wizard/data/wizard`.
- Инфраструктурный код бегемота — в остальных папках тут.
- Запускаемый демон: `/web/daemons/begemot`.
