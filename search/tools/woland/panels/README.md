== Что здесь?
Тут хранятся .yaml описания панелей Woland. Всё что лежит не в директориях `_snippets`, считается панелью и должно корректно рендериться Воландом.
Сниппеты -- это кусочки описания панелей, которые сами панелями не являются по причине недостаточности описания. Кубики из которых можно быстро сделать панель.
Такие кубики нужно складывать в директории `_snippets` на том уровне вложенности, где это кажется правильным, например: если у вас совсем common-кубик, сложите его в
`panels/_snippets`, кубики касающиеся только Apphost в `panels/apphost/snippets`, Begemot в `_panels/begemot/snippets` и т.д.
