Скрипт для добавления/удаления к пачке сервисов какого-либо ресурса одного
типа по его локальному имени.

Делалось в рамках задачи "Пропатчить всем сервисам instancectl.conf"
для настройки "Трёх корочек".

Все файлы в сервис надо привозить в виде sandbox-ресурсов.

Дефекты
=======
Не работает с static files / url files. Все файлы должны привозиться
в виде sandbox-ресурсов (потому что так у них хотя бы версионирование
заводится хоть в каком-то виде).
