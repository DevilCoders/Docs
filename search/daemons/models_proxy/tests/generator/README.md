Генератор запросов и ответов для теста search/daemons/models_proxy/tests.

Возможно, вам не нужно запускать его руками; регулярный процесс
https://a.yandex-team.ru/projects/mmeta-web/ci/releases/timeline?dir=search%2Fprod_resources&id=get-models-proxy-queries
запускает генератор самостоятельно и публикует результаты как ресурсы в sandbox,
откуда их в свою очередь подцепляет автообновление.

Требует собранных бинарников search/daemons/models_proxy и search/tools/evlogdump.
Также требует либо запросов к среднему, либо OAuth-токена для похода в sandbox,
откуда умеет доставать такие запросы из приёмки среднего.

Минимальный вариант запуска:
`ya make -r && OAUTH_TOKEN=``cat sandbox-token`` ./generator -m <путь-к-models_proxy> -e <путь-к-evlogdump>.`
Такой запуск:
* сходит в nanny за инстансами среднего в хамстере, выберет случайный, который ответит на запрос конфига;
* выкачает с выбранного инстанса конфиг со списком источников в файл mmeta.cfg;
* сходит в sandbox за приёмочными запросами к среднему, выкачает их в файл с именем из ресурса
  (которое определяет собиралка этих запросов, сейчас по схеме mmeta_prod_queries_cgi_YYYYMMDD.txt);
* поднимет над выкачанным конфигом models_proxy на указанном порту localhost
  в папке models_proxy_dir для конфигов и eventlog;
* выстрелит в выбранный инстанс среднего первой 1000 из выкачанных запросов,
  к которым допишет переопределение &metahost2=MODELS_PROXY:... на локально поднятую models_proxy,
  флаги для прокси, под которыми она пишет в свой eventlog все запросы и ответы к себе и подысточникам,
  и &reqinfo=models_proxy_tests_generator на всякий случай. Стреляет в один поток, чтобы не перегрузить
  средний левой нагрузкой, для 1000 запросов ожидаемое время обстрела в районе 10 минут;
* распарсит eventlog от models_proxy с помощью evlogdump, выделив из лога запросы и ответы в удобном для тестов формате;
  потом на полученные файлы стоит натравить gzip.

Файл со списком запросов можно указать уже готовый, тогда sandbox-токен не нужен.
При желании также можно переопределить инстанс среднего, конфиг среднего, число запросов.
