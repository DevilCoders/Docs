Генератор yasm-панелей Noapacheupper (верхний метапоиск)
========================================================

Ответственный: mvel@

Ссылки на все панельки имеются на production-странице Noapacheupper:
https://wiki.yandex-team.ru/jandekspoisk/kachestvopoiska/upper/production/


Как использовать
================

    ya make -t && ./noapache_yasm

После этого сгенерированные панельки надо обновить руками через интерфейс Yasm.
Для неручного обновления была также написана Sandbox-задача
search/garden/sandbox-tasks/projects/UpdateNoapacheupperYasmPanels
но она в данный момент не используется.
