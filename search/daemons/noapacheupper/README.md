Noapacheupper (noapache, upper)
===============================

Основной "смеситель" выдачи источников поисковой выдачи.

Исторически выделен из apache bundle
в виде отдельного сервиса после восстановления секрета сборки бандла ВМП (верхнего метапоиска).
Так как apache в этом сервисе был уже не нужен, программу назвали noapacheupper.

Содержит два основных блока

1) Сетевая часть (UPPER_INT) -- neh-базированная запчасть, которая запрашивает источники
2) BLENDER - функциональная запчасть, содержащая блендер и переранжирования.
