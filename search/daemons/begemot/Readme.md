Это разные демоны бегемота.

Каждая папка содержит `ya.make` следующего вида:

```
PEERDIR(
    search/begemot/rules/...        # все нужные правила
    search/daemons/begemot/main     # int main()
)
```

Для участия в тестах на замкнутость шарда каждая папка должна содержать `lib/ya.make`, в котором есть все правила и который используется из `search/begemot/data/ut/своя-папка`.
