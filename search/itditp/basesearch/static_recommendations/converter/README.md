### Для чего нужно

Утилита создана для объединения урлов в один прото файл, который потом передается базовому поиску  

##### Типы файлов:

Текстовый вид файлов - урлы в текстовом формате, разделенные переносом строки  
Бинарный вид файлов - `TVector<TString> urls`, сохраненный через `Save(&file, urls)`  

### Как пользоваться

##### В 99% случаев:

1. Прописать новый файл в `default_config.txt`  
2. Запустить бинарь без флагов: `./converter`  
3. Перетащить файл `statc_recommendations_binary` в `../data/` (полный путь ниже)  

##### Подробная инструкция:  

Если нужно сконвертировать файл с урлами в текстовом виде `converter -i file_with_urls.txt`  
Если нужно сконвертировать файл в бинарном виде `converter -b binary_file_with_urls`  
Если нужно сконвертировать много файлов:  

* первый вариант (только для файлов в текстовом виде): записать имена файлов по строчкам в файл `config.txt`, запустить `converter -c config.txt`  
* второй вариант: пишем все файлы через опцию -i или -b: `converter -i a.txt -i b.txt -b c`  

В результате работы, получится файл `static_recommendations_binary`, который нужно положить по пути `arcadia/search/itditp/basesearch/static_recommendations/data/` (aka `../data/`)  
А затем перебилдить поиск
