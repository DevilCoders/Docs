
### Получение текущего телефона пользователя

| Адрес              | HTTP метод |
|--------------------|------------|
| `/v1/auth/phone`   | `GET`      |

| Заголовок         | Тип                | Обязательный | Описание                  |
|-------------------|--------------------|--------------|---------------------------|
| Authorization     | OAuth $auth_token | Да            | Токен из АккаунтМенеджера |
| Lang              | String            | Нет           | Локаль пользователя       |
| Origin            | String            | Нет           | Источник запросов         |

| Параметр  | Тип           | Обязательный | Описание                   |
|-----------|---------------|--------------|----------------------------|
| Lang      | String        | Нет          | Локаль пользователя        |
| Origin    | String        | Нет          | Источник запросов          |


### Форматы ответов:

#### Успех
Код: 200

```json
{
  "phone": "+7945*****55", // Обязательное. Привязанный телефон из базы, либо телефон из паспорта
  "phoneToUnbind": "+7945*****55", //Оциональное поле. если телефон у нас отличается от паспортного
}
```

#### Нет телефона в паспорте
Код: 404

```json
{
    "code": "noBoundPhone"
}
```

#### Токен, с которым пользователь пришел в паспорт почему-то невалиден. Никогда не должно происходить в аппе, но мало ли
Код: 401

```json
{
    "code": "noRights"
}
```
