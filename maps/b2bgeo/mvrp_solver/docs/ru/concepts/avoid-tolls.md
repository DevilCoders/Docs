# Объезд платных дорог

По умолчанию, при маршрутизации не учитывается дополнительная стоимость за проезд по платным дорогам. Например, маршрут в аэропорт Шереметьево вероятнее всего пройдет по такой дороге - объезд по бесплатным дорогам возможен, но займет больше времени, а значит, с точки зрения Маршрутизатора он будет дороже. Однако, курьеру придется заплатить за проезд, что может привести к лишним затратам.

Чтобы избежать возможных расходов на платные дороги, используйте опцию `options.avoid_tolls` = `true`. В этом случае Маршрутизатор будет строить маршрут без использования платной дороги, если это возможно.

{% note warning "Внимание." %}

При планировании с доставкой из или на точки, находящиеся на платных дорогах и использовании опции `options.avoid_tolls` может быть построен неоптимальный маршрут. Если планируется доставка в такие места, рекомендуется установить `options.avoid_tolls` = `false`.

{% endnote %}


**Пример 1**

В этом примере маршрут построен до аэропорта Шереметьево и обратно на склад. Маршрут проходит по платной трассе М-11, и курьер будет обязан оплатить проезд по ней. В стоимости маршрута эти расходы не учитываются.
