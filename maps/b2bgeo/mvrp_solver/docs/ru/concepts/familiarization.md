# Шаг 1. Ознакомление с&nbsp;продуктом

Сначала рекомендуется получить общее представление о возможностях продукта и выполнить планирование на своих данных. (Если этот шаг уже был выполнен ранее, его можно пропустить.)

1. Ознакомьтесь с материалами [промосайта](https://www.yandex.ru/routing).

1. Яндекс.Маршрутизация состоит из двух сервисов: Планирование и Мониторинг. Информация о мониторинге приведена в [документации сервиса](https://yandex.ru/routing/doc/delivery/concepts/about.html).

1. Для регистрации в продукте воспользуйтесь разделом Быстрый старт.

1. Первый тест сервиса рекомендуется начинать с шаблона Excel.

   Это будет полезно и для разработчиков, потому что структура данных в Excel практически повторяет запрос к API. Более того, из задачи, запущенной через шаблон, можно будет получить готовый JSON-запрос к API.

   Выполнить тестовое планирование можно непосредственно в интерфейсе системы. Файлы примеров доступны там же и в разделе [Пример планирования](https://yandex.ru/routing/doc/vrp/concepts/example.html).

1. Сопоставьте данные, которые уже сейчас есть в вашей системе, с теми, которые нужно внести в шаблон Excel. Чтобы узнать, как заполнять файл, ознакомьтесь с [видео-уроками](https://yandex.ru/routing/support#video_section) и [инструкцией](https://yandex.ru/routing/doc/vrp/concepts/excel-fill-guide.html).

1. Введите в шаблон Excel свои данные за некоторый прошедший день. Для получения первых результатов планирования заполните минимум полей. Остальные поля можно оставить пустыми (подробнее см. в разделе [Инструкция по заполнению Excel](https://yandex.ru/routing/doc/vrp/concepts/excel-fill-guide.html)).

   - Если у вас нет координат, внесите в шаблон сведения об адресах. В интерфейсе их можно геокодировать (найти координаты по адресу).
   - Если вы не знаете, какие значения ставить в поля со стоимостью и штрафами, оставьте их пустыми (параметры примут значения по умолчанию). Главное, не ставьте 0.

1. Загрузите шаблон Excel в интерфейс. Обратите внимание на ошибки и предупреждения, которые возникли в процессе. Нужно, чтобы ваш шаблон корректно запускался на планирование. Для этого исправьте ошибки (либо в Excel, либо непосредственно в интерфейсе импорта). Без этого начать планирование не получится.

   С предупреждениями желательно ознакомиться. При их наличии планирование можно запустить.

1. Получите первый результат оптимизации. Он может не соответствовать вашим ожиданиям, но, как правило, большинство вопросов решаются корректировкой исходных данных и настроек. Чтобы изменить их, переходите к [Шагу 2](testing.md).
