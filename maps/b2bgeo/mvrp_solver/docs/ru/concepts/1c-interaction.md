# Схема взаимодействия модуля с сервисами Яндекс

![Схема взаимодействия](https://courier.yandex.ru/vrs-doc/1C-interaction.png "Схема взаимодействия")

Модуль взаимодействует с API сервисов:

- [Геокодер](https://yandex.ru/dev/maps/geocoder/)

  Модуль передает данные по адресам, получает информацию по координатам и их [точности](https://yandex.ru/dev/maps/geocoder/doc/desc/reference/precision.html).

- [Построение оптимальных маршрутов](https://yandex.ru/routing/doc/vrp/ref/MultipleVehiclesRouting.html)

  Модуль передает данные для построения маршрутов (склад, заказы, транспортные средства, опции), получает ответ — ID задачи на маршрутизацию. Далее по этому ID модуль получает ответ в виде распределения заказов по транспортным средствам и последовательности посещения заказов.

- [Мониторинг](https://yandex.ru/routing/doc/delivery/)

  Модуль передает в Мониторинг данные по маршрутам и заказам, которые нужно отслеживать. Эти маршруты затем можно открыть в мобильном приложении Яндекс.Курьер.
