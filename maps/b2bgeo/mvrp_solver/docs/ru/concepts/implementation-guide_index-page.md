# Порядок внедрения

Внедрение сервиса планирования маршрутов состоит из следующих шагов:

1.	[Ознакомление с продуктом](familiarization.md)
2.	[Тестирование продукта](testing.md)
3.	[Проезд по маршрутам](ride.md)
4.	[Интеграция](integration.md)
5.	[Опытная эксплуатация](trial-operation.md)
6.	[Промышленная эксплуатация (тиражирование)](commercial-operation.md)

Это рекомендованный набор шагов: не во всех случаях его можно выполнить в указанном порядке. (Например, в некоторых случаях тестирование продукта или проезд по маршрутам невозможны без интеграции.). Детализированные рекомендации внутри каждого шага носят обобщенный характер: нужно оценивать их применимость в своей ситуации. В простых случаях некоторые описанные действия можно не выполнять.
