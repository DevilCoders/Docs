# Первый запуск через Excel

Для запуска первого планирования:

1.	Получите доступ в [интерфейс планирования](https://yandex.ru/courier/). Для регистрации рекомендуется завести отдельный аккаунт на [Яндекс.Паспорт](https://passport.yandex.ru/). Данный логин в дальнейшем может быть использован в качестве учетной записи администратора, также к ней будет привязан ключ доступа к запуску сервиса маршрутизации.

2.	В интерфейсе планирования скачайте пример заполненного файла Excel. Пример доступен в разделе **Планирование** => **Начать планирование** => **Примеры файлов**.

  <img class="step--img" src="https://courier.yandex.ru/vrs-doc/quickstart-1.png" />

  Также скачать пример файла Excel можно в разделе [Примеры](https://yandex.ru/routing/doc/vrp/concepts/example.html). Для начала рекомендуется скачать простой файл, без большого количества параметров.

3.	Для первого планирования в файле Excel достаточно заполнить следующие данные (остальные поля можно оставить пустыми):

  **Данные по складу**:

  *	Название склада (можно просто «Склад»)
  *	Координаты склада
  *	Время работы склада (можно поставить максимально широко – с 00:00-23:59 -на текущие сутки планирования или с 00:00-1.23:59 – на текущие и следующие сутки планирования)

  **Данные по заказам**:

  *	Номер заказа
  *	Координаты и\или адрес заказа (геокодировать адрес можно будет непосредственно в интерфейсе)
  *	Временное окно заказа
  *	Объемно-весовые характеристики заказа (вес и\или объем и\или количество грузовых мест)

  **Данные по машинам\курьерам**:

  *	Номер транспортного средства\курьера
  *	Вместимость транспортного средства\курьера
  *	Время его работы

  При необходимости ознакомьтесь с [инструкцией по заполнению файла Excel](excel-fill-guide.md).

4.	Заполните файл Excel. Загрузите файл в интерфейс планирования.

  После загрузки могут возникнуть <img class="step--img" src="https://courier.yandex.ru/vrs-doc/error.png"/> ошибки и <img class="step--img" src="https://courier.yandex.ru/vrs-doc/warn.png"/> предупреждения. Ошибки необходимо исправить, с предупреждениями можно планировать дальше.

  Если в заказах были указаны адреса, а не координаты, то необходимо их геокодировать в интерфейсе.

  При планировании выберите следующие опции:

  *	Дата маршрута - **текущая дата**;
  *	Часовой пояс - **часовой пояс, в котором указаны все временные окна**;
  *	Метод передвижения - укажите **метод передвижения** (при необходимости его можно указать для каждого курьера);
  *	Качество планирования - выберите **Оптимально**.

5. Дождитесь завершения планирования и оцените результаты в интерфейсе. Результаты также можно выгрузить в Excel.

Для дальнейшей настройки параметров маршрутизации ознакомьтесь с возможностями [настроек параметров маршрутизации](reference.md).
