# Планирование

В разделе **Планирование** вы можете указать исходные данные для построения маршрутов и посмотреть полученное решение.

Чтобы загрузить файл с исходными данными, нажмите кнопку **Выберите Excel файл для загрузки** или перетащите документ в окно браузера.

Также вы можете сделать это на следующем шаге — в окне импорта. Чтобы перейти к нему, нажмите кнопку **Начать планирование**.
