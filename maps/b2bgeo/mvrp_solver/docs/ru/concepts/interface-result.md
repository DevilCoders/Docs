# Работа с полученным решением

После завершения автоматического построения маршрутов вы можете посмотреть и отредактировать результат в разделе **Планирование**.

Каждое решение имеет свой уникальный идентификатор (`task_id`). Он фигурирует в составе адреса:

`https://yandex.ru/courier/companies/<company_id>/depots/all/mvrp/<task_id>`,

где `<company_id>` — идентификатор вашей компании.

Сохраните содержимое адресной строки браузера, чтобы вернуться к полученному решению в дальнейшем.

Чтобы запустить другую задачу маршрутизации, нажмите **Новое планирование**.

## Элементы страницы

На странице работы с полученным решением представлены блоки:

- **Карта** — географическое представление построенных маршрутов и нераспределенных заказов. Число на точке означает ее порядковый номер в последовательности объезда пунктов маршрута.

- **Маршруты** — метрики маршрутов и итоговые показатели всего решения.

- **Заказы** — заказы, которые входят в выбранный маршрут.

- **Нераспределенные заказы** — заказы, которые не удалось включить в маршруты.

Вы можете изменить отображение блоков:

**Настроить местоположение**

Наведите указатель на левый верхний угол блока, нажмите левую кнопку мыши и перетащите в нужное место.

**Настроить размер**

Используйте управляющий элемент в правом нижнем углу блока.

**Изменить ширину столбцов**

Передвиньте границы заголовка столбца, чтобы изменить его ширину.

**Изменить порядок и видимость столбцов**

Нажмите ![](https://courier.yandex.ru/vrs-doc/table-settings.png "") в правом верхнем углу блока. Чтобы скрыть столбец, отключите его. Чтобы изменить последовательность столбцов, перетащите название колонки на нужную позицию в списке.  Для возврата к стандартным настройкам нажмите **Сбросить**.

## Просмотр решения

Чтобы перейти к заказу или маршруту, выберите нужный объект на карте или в таблице. В результате на указанном заказе или маршруте будет установлен фокус во всех возможных блоках страницы.

Для перехода к конкретному объекту также служит поле **Поиск**. Выберите заказ или маршрут в выпадающем списке, и он будет выделен в блоках.

Нажмите значок ![](https://courier.yandex.ru/vrs-doc/result-vis.png "") в строке маршрута в таблице, чтобы не показывать его на карте. Чтобы скрыть все маршруты, кроме выделенного, нажмите ![](https://courier.yandex.ru/vrs-doc/result-vis.png "") в заголовке таблицы.

Если к маршруту или заказу есть замечания, в строке таблицы выводится значок ![](https://courier.yandex.ru/vrs-doc/warn.png ""). Наведите на него указатель, чтобы увидеть пояснение.

## Редактирование решения

Вы можете изменить результат планирования:

**Исключить заказ из маршрута**

   Перетащите строку из таблицы **Заказы** в таблицу **Нераспределённые заказы**.

**Включить нераспределенный заказ в маршрут**

- Используя таблицы: нажмите на целевой маршрут в таблице или на карте. Перетащите строку из таблицы **Нераспределённые заказы** в таблицу **Заказы**.

- Используя карту: перетащите значок нераспределенного заказа на карте на линию целевого маршрута.

**Назначить маршрут для другой машины**

   Выделите машину в разделе **Маршруты** и нажмите **Заменить маршину**. В открывшейся таблице выберите необходимый автомобиль.

**Перенести заказ из одного маршрута в другой**

- Используя таблицы: перетащите объект из таблицы **Заказы** в таблицу **Маршруты** на строку целевого маршрута.

- Используя карту: перетащите точку с заказом на линию целевого маршрута.

Последовательность объезда точек отредактировать нельзя.

Чтобы выполнить перетаскивание сразу нескольких элементов, сначала выделите их с нажатой клавишей **Ctrl**.

После каждой корректировки система пересчитывает итоговые показатели. Изменение метрик отражается в таблице **Маршруты**.  Разница по сравнению с первоначальным результатом планирования в сторону улучшения показателя выводится зеленым цветом, в сторону ухудшения — красным.

При каждом изменении результатов генерируется новый идентификатор решения `task_id`. Сохраните адрес страницы, чтобы в дальнейшем вернуться к работе с отредактированным вариантом. Первая версия решения при корректировке не изменяется и продолжает быть доступной по своей ссылке.

## Экспорт решения

Чтобы скачать итоговое решение, нажмите кнопку **Экспортировать**. Возможные варианты экспорта:

- **Выгрузка в Excel**

   Сохранение результатов в Excel-файл.

- **Выгрузка в JSON**

   Решение можно получить в виде файла или ссылки на данные в формате JSON.

- **Внутренняя система**

  Связь отредактированного решения с исходным будет сохранена в сервисе.  Для получения скорректированного решения в дальнейшем можно использовать запрос [/children](https://yandex.ru/routing/doc/vrp/ref/ChildTasks/getChildTasks.html) с ID оригинальной задачи.

- **Отправка в Мониторинг**

   Система проверит возможность выгрузки информации в сервис [Мониторинг](https://yandex.ru/routing/doc/delivery/concepts/about.html). При наличии ошибок их необходимо исправить и повторить попытку экспорта.

   Если проблем нет, выберите статус, который будет присвоен всем выгружаемым заказам в Мониторинге, и подтвердите экспорт.
