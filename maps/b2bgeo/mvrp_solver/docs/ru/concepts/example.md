# Пример планирования

В этом разделе вы можете скачать примеры шаблонов Excel, которые можно использовать для планирования маршрутов в веб-интерфейсе:

- **[Скачать простой пример](https://yandex.ru/courier/s3/import-mvrp-examples/ru/Простой%20пример.xlsx)** (минимальный набор параметров)

- **[Скачать расширенный пример](https://yandex.ru/courier/s3/import-mvrp-examples/ru/Пример%20с%20расширенными%20параметрами.xlsx)** (основные параметры, подходят для большинства случаев)

- **[Скачать полный пример](https://yandex.ru/courier/s3/import-mvrp-examples/ru/Пример%20со%20всеми%20возможными%20параметрами.xlsx)** (все возможные параметры)



{% note %}


Предварительно сконфигурированные автомобили, заказы и склад вы найдете на соответствующих листах. Название каждого столбца содержит ссылку на раздел документации, где описан соответствующий параметр. Также некоторые столбы содержат дополнительные комментарии. Заголовки столбцов в файле окрашены в зависимости от важности параметра:

- Красным обозначены обязательные или крайне рекомендованные для всех запросов параметры.
- Черным обозначены параметры, используемые для точной настройки под модель доставки заказов.
- Серым обозначены параметры для опытных пользователей. С помощью этих параметров можно добиться тонкой настройки задач планирования.
