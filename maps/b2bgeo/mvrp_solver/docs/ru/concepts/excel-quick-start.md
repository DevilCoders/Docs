# Первый запуск через Excel

В этой инструкции описано как быстро начать работу с интерфейсом Яндекс.Маршрутизации и планированием маршрутов через Excel.

## Шаг 1.	Получите доступ

Чтобы получить доступ в интерфейс планирования, зайдите на сайт [Яндекс Маршрутизации](https://yandex.ru/courier/) и заполните форму регистрации. Для регистрации рекомендуется завести отдельный аккаунт на [Яндекс.Паспорт](https://passport.yandex.ru/).

Данный логин в дальнейшем может быть использован в качестве учетной записи администратора, также к ней будет привязан ключ доступа к запуску сервиса маршрутизации.

## Шаг 2. Скачайте пример заполненного файла Excel

Пример доступен в разделе *Планирование* => *Начать планирование* => *Примеры файлов*

##### (вставить скриншот)

Также примеры доступны в разделе [Пример планирования](https://yandex.ru/routing/doc/vrp/concepts/example.html).

Для начала рекомендуется скачать простой пример, без большого количества параметров.

## Шаг 3. Заполните Excel

Для первого планирования укажите следующую информацию:

*Данные по складу*
•	Название склада (можно просто «Склад»);
•	Координаты склада;
•	Время работы склада (можно поставить максимально широко – с 00:00-23:59 -на текущие сутки планирования или с 00:00-1.23:59 – на текущие и следующие сутки планирования).

*Данные по заказам*

•	Номер заказа;
•	Координаты и\или адрес заказа (геокодировать адрес можно будет непосредственно в интерфейсе);
•	Временное окно заказа;
•	Объемно-весовые характеристики заказа (вес и\или объем и\или количество грузовых мест).

*Данные по машинам и курьерам*
•	Номер транспортного средства или курьера;
•	Вместимость транспортного средства или курьера;
•	Время его работы.

При необходимости ознакомьтесь с [инструкцией по заполнению файла Excel](./excel-fill-guide.md).

## Шаг 4. Загрузите файл в интерфейс планирования	и запустите планирование

Заполните файл Excel и загрузите его в интерфейс планирования.
После загрузки могут возникнуть ошибки и предупреждения. Перед планированием все ошибки необходимо исправить. Предупреждениям исправлсять не обязательно, но рекомендуется  можно планировать дальше.

Если в заказах были указаны адреса, а не координаты, то необходимо их геокодировать в интерфейсе (подробнее тут – вставить ссылку после https://st.yandex-team.ru/DOCAPI-10728)

	При планировании выберите следующие опции:
•	Дата маршрута = текущая дата
•	Часовой пояс = часовой пояс, в котором указаны все временные окна
•	Метод передвижения = укажите метод передвижения (при необходимости его можно указать для каждого курьера – подробнее тут)
•	Качество планирования = выберите Оптимально

Дождитесь завершения планирования и оцените результаты в интерфейсе. Результаты также можно выгрузить в Excel (подробнее тут).
