#  Выгрузка заказов из 1С

Яндекс.Маршрутизация предоставляет модуль для экспорта заказов из 1С в формат, подготовленный для импорта в Маршрутизатор. Модуль представляет собой внешнюю обработку 1С формата EPF.

Перед началом работы, убедитесь что вы скачали [файл внешней обработки](https://courier.yandex.ru/vrs-doc/Yandex_Routing_version_02.04.2020.epf).

## 1. Запуск обработки

Запуск обработки происходит обычным для внешних файлов способом через пункты меню: **Файл** ⟶ **Открыть…**. Для открытия также можно воспользоваться горячими клавишами Ctrl+O.

   <img class="step--img" src="https://courier.yandex.ru/vrs-doc/1c-guide-step-1.png"/>

{% note %}

В зависимости от конфигурации и ее версии внешний вид диалога открытия файла может отличаться от представленного.

{% endnote %}

## 2. Выбор периода

Выберите период, за который необходимо собрать данные по документам с помощью кнопки **Изменить период**.


   <img class="step--img" src="https://courier.yandex.ru/vrs-doc/1c-guide-step-2-2.png"/>

Откроется окно выбора периода.

   <img class="step--img" src="https://courier.yandex.ru/vrs-doc/1c-guide-step-2-3.png"/>

После того, как необходимый период отмечен, нажмите кнопку **Выбрать**.

   <img class="step--img" src="https://courier.yandex.ru/vrs-doc/1c-guide-step-2-4.png"/>

## 3. Заполнение и редактирование списка

После выбора периода нажмите кнопку **Заполнить список**.


   <img class="step--img" src="https://courier.yandex.ru/vrs-doc/1c-guide-step-3-2.png"/>

Модуль собирает данные за указанный пользователем период по документам (зависит от конфигурации), формируя таблицу определенной структуры.

   <img class="step--img" src="https://courier.yandex.ru/vrs-doc/1c-guide-step-3-3.png"/>

При необходимости данные в таблице могут быть отредактированы на форме модуля.

   <img class="step--img" src="https://courier.yandex.ru/vrs-doc/1c-guide-step-3-4.png" />

Для сохранения данных после редактирования нажмите кнопку **Выгрузить в эксель**.


   <img class="step--img" src="https://courier.yandex.ru/vrs-doc/1c-guide-step-3-6.png"/>

После нажатия на эту кнопку откроется стандартный диалог сохранения файла. Место сохранения выбирается пользователем самостоятельно.

   <img class="step--img" src="https://courier.yandex.ru/vrs-doc/1c-guide-step-3-7.png"/>

## Интерфейс Яндекс.Маршрутизация в 1С

На форме модуля есть кнопка **Яндекс.Маршрутизация**.

   <img class="step--img" src="https://courier.yandex.ru/vrs-doc/1c-guide-routing-2.png"/>

При нажатии кнопки **Яндекс.Маршрутизация** открывается выпадающий список возможных опций.

   <img class="step--img" src="https://courier.yandex.ru/vrs-doc/1c-guide-routing-3.png"/>

* При выборе пункта **Открыть в браузере по умолчанию** открывается cтраница сервиса Яндекс.Маршрутизация в браузере.
* При выборе пункта **Открыть в 1С** сервис Яндекс.Маршрутизация открывается непосредственно в 1С.

   <img class="step--img" src="https://courier.yandex.ru/vrs-doc/1c-guide-routing-4.png"/>
