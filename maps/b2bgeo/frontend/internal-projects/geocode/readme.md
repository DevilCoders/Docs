## Обратное геокодирование

![image](https://jing.yandex-team.ru/files/losnikitos/2020-04-12_16-38-39.png)

1. Идем на geojson.io, рисуем полилилинию
2. Копируем JSON в data.json
3. Запускаем node .
4. На выходе будет
