# GeoJSON layer
Схема формата стилей, который позволяет стилизовать пользовательские тайловые слои на карте.

Концепция:
Пользователь добавляет на карту слой(оверлей) объектов с атрибутами.
Среди атрибутов есть 'tags', который определяет набор тегов объекта.

С помощью селекторов по тегам и типам пользователь определяет набор объектов, к которым применяются те или иные стили.
Для иконок пользователь указывает image id, который используется рендерером карты для того, чтобы запросить у пользователя соответствующую иконку.
