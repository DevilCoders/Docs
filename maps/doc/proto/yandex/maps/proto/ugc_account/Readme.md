## Описание

### Терминология:
**Maps UGC Account** - сервис, который предоставляет единую точку доступа к контенту, генерируемого силами пользователей Карт. Логически состоит из двух частей:
* **Assignments**: Предоставляет доступ к общей ленте заданий релевантных конкретному пользователю.
* **Contributions**: Предоставляет доступ к ленте событий ассоциированных со вкладом конкретного пользователя.
Технически тоже из двух частей:
* [**Бэкофис Maps UGC Account**][Backoffice] - часть функциональности, предназначенная для межсервисного взаимодействия: из пользовательского ui запросы сюда не приходят.
* [**Фронтофис Maps UGC Account**][Frontoffice] - часть **Maps UGC Account**, в которую приходят пользовательские запросы из ui. Описан [здесь](https://a.yandex-team.ru/arc/trunk/arcadia/maps/doc/proto/yandex/maps/proto/ugc_account/Api.md).

##### Зона ответственности **Maps UGC Account**
* отдавать данные для отображения ленты заданий для пользователя; задания в ленте отранжированы по релевантности
* удалять задание по запросу пользователя (кнопка _"Не знаю"_/_"Пропустить"_)
* фильтровать ленту заданий:
  - по статусу: активное/отложено/готово
  - по типу задания: добавить фото/добавить зеркала/сообщить адрес/добавить отзыв и т.п.
* отдавать данные для отображения ленты вклада:
  - фильтровать ленту по типу вклада

Для реализации этого [**Бэкофис Maps UGC Account**][Backoffice] должен уметь следующее:
* принимать новые задания от **Сервиса UGC**
* принимать информацию о выполнении задания
* удалять задание по запросу от **Сервиса UGC**
* принимать информацию о создании нового события от **Сервиса UGC**
* редактировать информацию о случившихся событиях: например, удалять фото из превью фото, если пользователь удалил фотографию


**Сервис UGC** - внешний сервис, который обслуживает некоторую предметную область (фотки, панорамы, зеркала, фидбек и т.п.). Сервис UGC генерирует задания и публикует их в **Assignments**. Таких сервисов много.

##### Зона ответственности **Сервиса UGC**:
* **Сервис UGC** генерирует задания и публикует их в **Assignments**
* **Сервис UGC** предоставляет программный интерфейс для управления вкладом:
  - позволяет показывать детальную информацию о вкладе
  - позволяет редактировать вклад при необходимости
* **Сервис UGC** сообщает сервису **Maps UGC Account** краткую информацию о событиях, которые происходит со вкладом пользователя
