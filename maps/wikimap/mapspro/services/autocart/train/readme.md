#### Скрипты для обучения сетей
* **bld_edges** - обучение сети для детектирования ребер полигонов зданий
* **bld_edges_verts** - обучение сети для детектирования ребер и вершин полигонов зданий
* **bld_segm** - обучение сети для семантической сегментации

##### Настройка окружения
Для запуска на локальной машине необходимо правильно настроить окружение.

Для запуска необходимы следующие библиотеки:
* numpy
* opencv
* tensorflow

Для настройки можно воспользоваться virtualenv и выполнить следующие команды:
> virtualenv --system-site-packages train_env 
> source ~/train_env/bin/activate 
> pip install --upgrade numpy 
> pip install --upgrade opencv-python 
> pip install --upgrade tensorflow

Для выхода из активированного virtualenv можно воспользоваться командой:
> deactivate

##### Настройка параметров обучения
**Запуск на локальной машине**
Для настройки параметров обучения при локальном запуске необходимо задать соответствующим переменным окружения необходимые параметры. Проще всего указать эти значения в файле **run_local_train.sh** и выполнить команду **./run_local_train.sh** для запуска обучения.

**Запуск в нирване**
Все парметры обучение можно задать через графический интерфейс нирваны.
