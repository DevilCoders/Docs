# Библиотека для проверки geolock-ов

Geolock - это полигон карте, в котором запрещены правки.

Для каждого geolock-а задан branch, в котором он действует.

## Кто создаёт geolock-и?

Обычно geolock создаётся автоматически при выполнении масштабной операции на карте.
Например: подвижка по спутнику.

После того как операция выполнилась, она удалит созданный geolock.

Иногда geolock создают руками в базе.
Такой geolock называется `Manual`.
И отличается значением `0` в поле `commit_id`.

## Кто проверяет geolock-и?

Geolock-и проверяет editor при выполнении операций модификации данных на карте.

В редаторе НК на UI отправляются данные о наличии geolock-а в видимой области карты.
Если geolock есть - появляется специальный значок, который говорит "Сейчас здесь нельзя редактировать".
