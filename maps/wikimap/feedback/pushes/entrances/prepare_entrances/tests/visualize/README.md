## Test data visualization

Здесь лежит визуализация данных, которые используются в тестах `prepare_entrances`. Для её просмотра нужен собранный [`easyview`](https://a.yandex-team.ru/arc/trunk/arcadia/maps/tools/easyview).
Запустить визуализацию можно довольно просто:
```bash
./visualize | easyview
```
или так:
```bash
./visualize --resfile result
easyview -f result
```
