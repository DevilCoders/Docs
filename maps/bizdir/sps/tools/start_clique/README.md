Инструмент для старта chyt клики `*maps_sps_clique`, которая используется для
скачивания данных об организациях из экспорта Справочника в компоненте
`sprav_loader`.

[Дашборд](https://solomon.yandex-team.ru/?project=yt&cluster=hahn&service=clickhouse&cookie=Aggr&dashboard=chyt_v2&l.operation_alias=maps_sps_clique)

Пример использования:
```
./start_clique --cluster hahn --restart
```
