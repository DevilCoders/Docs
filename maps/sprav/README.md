# Справочник организаций

Документация: <https://docs.yandex-team.ru/sprav/>

В этой папке лежат новые проекты Справочника. Основной код лежат в [sprav](https://a.yandex-team.ru/arc_vcs/sprav).

## Содержимое подпапок

* `callcenter` - бэкенд коллцентра и фоновые процессы.
* `mining` - процессы парсинга сайтов.
* `pedestrians` - генерация заданий для пешеходов и генерация фида по результатам обходов пешеходами.
* `sprav_duty` - общий конфиг Седема с настройками дежурств.
