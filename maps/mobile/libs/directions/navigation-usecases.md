# Usecase нового навигационного API

### 1: Построение и ведение по маршруту
1. Отображается карта.
1. Приложение строит маршрут между N точками, координаты точек получает из нажатий накарту.
1. Добавляет навигационный слой на карту.
1. Пользователь выбирет маршрут нажатием на карту.
1. Начинается ведение. В процесее ведения:
  + пробки по маршруту обновляются;
  + кривая маршрута откусывается;
  + блидайший маршрут сопровождается балуном.
