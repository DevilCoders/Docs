# buildWhatWhereQueries #

Функция предназначена для построения запросов вида "что" + "где". Перед
разбиением из полного запроса выбрасываются все заранее известные
предлоги (т.е. "на", "в" и т.д.). Оставшийся запрос разбивается на две
части `words[:i]` и `words[i:]`, где `i` принимает значения от `1`
до `words.size() - 1`. Одна часть становится запросом "что",
другая – запросом "где".

Например, для запроса "кафе шоколадница на арбате" мы получим следующие
разбиения:

    WhatWhereQuery{whereQuery="кафе шоколадница", whatQuery="арбате"}
    WhatWhereQuery{whereQuery="шоколадница арбате", whatQuery="кафе"}
    WhatWhereQuery{whereQuery="кафе", whatQuery="шоколадница арбате"}
    WhatWhereQuery{whereQuery="арбате", whatQuery="кафе шоколадница"}

Запросы упорядочиваются по количеству слов в части "где".

Чтобы не возвращать большое число запросов "что" + "где" для очень длинных
пользовательских запросов, максимальное число в пользовательском запросе (за
вычетом предлогов) ограничено константой `MAX_WHAT_WHERE_QUERY_WORD_COUNT`.
Если число слов в запросе превышает эту константу, то запросы "что" + "где"
не генерируются.
