# Воркер генерации заданий на обход индоров

Воркер tasks-generator запускается каждые 2 часа, и создаёт задания на
сбор данных для радиокарт.

Задания создаются для следующих индоров:

-   индор входит в список активных индоров
-   для него нет созданных заданий со статусом available/pending/aquired
-   с момента последнего завершённого задания прошло более 30 дней

Активные индоры - это индоры, для которых задания генерируются автоматически.
Их список находится в табличке `ugc.active_indoors`.

Задание на обход создаётся с использованием текущих маршрутов обхода из НЯК.
Если для активного индора не удалось получить маршруты, таска не создаётся.

Задания создаются в статусе `pending`.
В статус `active` они переводятся `task-activator`-воркером.
