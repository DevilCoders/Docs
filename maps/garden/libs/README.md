# Общие библиотеки

* `auth` - библиотека для аутентификации при работе с API Огорода.
* `click_utils` - вспомогательные классы для библиотеки `click`.
* `garden_client` - библиотека для вызовов API Огорода.
* `jns_client` - обёртка для API JNS для рассылки нотификаций.

Остальные библиотеки содержат код, который переиспользуется между небольшим числом огородных модулей.
