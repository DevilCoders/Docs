# Огород

Огород — система публикации геоданных.

Документация: https://docs.yandex-team.ru/garden/

## Содержимое папок

### Разработчикам огородных модулей:

* `modules` - код огородных модулей.
* `sdk` - SDK для разработки огородных модулей.
* `libs` - библиотеки, которые переиспользуются между небольшим числом модулей.
* `conf` - частично устаревшие конфиги модулей.

### Разработчикам Огорода

* `server` - сервер Огорода - реализует API.
* `scheduler` - шедьюлер Огорода - запускает таски.
* `tools` - вспомогательные тулы.
* `sandbox` - таски в Sandbox.
* `libs_server` - библиотеки, которые переиспользуются сервером, шедьюлером и тулами.
* `common_tests` - интеграционные тесты, тесты MyPy
* `docs` - исходники документации Огорода.
* `rfc` - документы для обсуждения идей по развитию Огорода (Request For Comments).
