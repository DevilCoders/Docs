# Garden CLI
Приложение предназначено для работы с Огородом из консоли.

# Сборка и релиз
По коммиту в директорию с тулом CI автоматически запускает сборку тула.

CI: https://a.yandex-team.ru/projects/maps-core-garden/ci/releases/timeline?dir=maps/garden/tools/garden_cli&id=garden-cli-release

Собранный тул хранится в Sandbox с типом ресурса [MAPS_GARDEN_TOOLS_GARDEN_CLI](https://sandbox.yandex-team.ru/resources?type=MAPS_GARDEN_TOOLS_GARDEN_CLI&limit=20).

Чтобы новая версия тула стала доступна для пользователей, нужно зайти в CI и нажать на кнопку релиза в `stable`.
