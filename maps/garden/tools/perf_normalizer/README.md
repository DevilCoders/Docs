# Perf Normalizer

Тул `perf_normalizer` периодически вычитывает записи про таски из Монго-коллекции `task_log` и перекладывает их в коллекции `task_statistics` и `resource_statistics`.

Данные из этих коллекций используются в ручке сервера `/modules/<module_name>/builds/<int:build_id>/full_info/` для отображения списка тасок на странице билда в UI (Perf Analyzer).

Тул работает в виде демона под управлением supervisor в докер-контейнере с шедьюлером Огорода.

Если тул падает, то запущенные таски перестают отображаться в UI. Это не является критической поломкой. Чинить тул ночью и в выходные нет необходимости.
