# Garden CLI Launcher
Приложение работает как специальная обертка-лончер над `Garden CLI`.

Предполагается, что тул `Garden CLI` будет обновляться достаточно часто. Лончер один раз в день автоматически проверяет наличие свежей версии тула в Sandbox.

Для Ubuntu приложение-лончер поставляется через deb-пакет `yandex-maps-garden-cli-launcher`.

При изменении кода самого лончера, нужно попросить разработчиков огородных модулей вручную обновить deb-пакет на своих машинах. Предполагается, что обновление лончера будет происходить сильно реже, чем обновление `Garden CLI`.

# Сборка и релиз
При изменении файла `package/debian/changelog` CI автоматически запускает сборку deb-пакета.

CI: https://a.yandex-team.ru/projects/maps-core-garden/ci/releases/timeline?dir=maps/garden/tools/garden_cli_launcher&id=garden-cli-launcher-release

После сборки пакет автоматически будет залит в `unstable` deb-репозиторий.
Для релиза пакета в `stable` deb-репозиторий нужно нужно зайти в CI и нажать на кнопку релиза в `stable`.
Перед релизом в `stable` необходимо протестировать `unstable` версию пакета и лончера.
