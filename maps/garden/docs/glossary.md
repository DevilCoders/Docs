# Глоссарий

* `Билд` - процесс и артефакты исполнения модуля
* `Граф задач` - представление зависимостей и порядка исполнения задач в виде графа
* `Граф модуля` - подграф общего графа для задач модуля
* `Задача` - объект, создающий внутренние ресурсы Огорода, получая на вход другие ресурсы.
* `Модуль` - набор задач, объединенных в один исполняемый бинарный файл.
* `Ресурс` - сущность, от которой зависит или которую производит модуль.
* `Таска` - синоним термина `Задача`
* `Контур` - сущность, которая владеет билдами и ресурсами. Контуры отличаются версиями модулей, которые в них работают.
