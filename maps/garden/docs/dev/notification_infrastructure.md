# Инфраструктура оповещений.

## Почта

Рассылка почтовых уведомлений происходит в классе `EmailNotifier`.
Шаблоны находятся [тут](https://a.yandex-team.ru/arc/trunk/arcadia/maps/garden/scheduler/templates).
Письма присылаются автору билда при его завершении. Если билд был запущен автостартером, то письма не отсылаются.

Почта отправляется с общеяндексовского сервера `outbound-relay.yandex.net` с адреса `noreply@yandex-team.ru`.

## Инфра

Про инфру есть [отдельная](infra.md) страница.

## Стартрек

При неудачном завершении билда создаётся тикет в [Стартреке](https://st.yandex-team.ru/MAPSGARDENBUILD)
в очереди `MAPSGARDENBUILD`.

Шаблоны для тикетов находятся [тут](https://a.yandex-team.ru/arc/trunk/arcadia/maps/garden/scheduler/templates).

Ответственным для билдов из стабильного и datatesting контуров назначается мейнтейнер модуля.
Так же назначаются наблюдатели из abc-группы модуля в седеме.

Для билдов из тестового Огорода тикеты создаются в очереди `TEST` и ответственные и наблюдатели не назначаются.

## Телеграм

Оповещения в телеграм реализованы с помощью [JNS](https://jns.yandex-team.ru/projects/Maps-Core-Garden).

Общие оповещения от Огорода отправляются в чат `Разработка Огорода`.

## Оповещения из тасок

Оповещения из тасок так же реализованы с помощью JNS.
Для этого используется отдельный [шаблон](https://jns.yandex-team.ru/projects/Maps-Core-Garden/templates/garden-task-message) оповещений.

Каналы для оповещений задают авторы модулей самостоятельно по [этой](notification.md) инструкции.
