# Билд

Билд (`build`) - процесс и артефакты исполнения [модуля](module.md).

Новый билд можно запустить, если для него готовы входные данные. Это можно сделать несколькими способами:

1. Вручную через UI.
2. Автоматически по завершению другого билда. Подробности на странице [описания автостартеров](autostart.md).
3. Автоматически через API. На практике используется обычно для обнаружения и регистрации внешних ресурсов.

По умолчанию количество билдов не ограничено, что может приводить к превышению используемой квоты для хранения ресурсов.
Для решения этой проблемы необходимы специальные настройки модуля. Подробности на [странице](build_limits.md).
