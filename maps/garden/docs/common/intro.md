# Обзор системы

Огород - это платформа для выполнения задач, аналогичная Sandbox, Nirvana.
Отличие Огорода в том, что это [dataflow](https://en.wikipedia.org/wiki/Dataflow) система: задачи запускаются автоматически по мере появления входных данных.

Огород предназначен для оркестрации процесса публикации геоданных.
Огород каждый день принимает на вход данные из разных источников (Народная Карта, Справочник и другие) и производит данные для конечных сервисов (рендерер, роутер, геокодер и другие).

Разработчики могут добавлять свои задачи для подготовки данных для любых карточных сервисов.

Огород использует YT для выполнения задачи в виде vanilla-операций, которые в свою очередь могут запускать другие MR операции.
