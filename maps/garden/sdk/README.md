# Garden SDK

Содержит библиотеки, которые используются для разработки огородных модулей.

* `core` - содержит главные классы предметной области: `Task`, `Resource`, `TaskGraphBuilder`.
* `extensions` - расширяет либу `core` удобными вспомогательными классами и обёртками: `EasyBuilder`, `EnsureEqualProperties`, `ResourceNamer` и другими.

---

* `resources` - классы огородных ресурсов `FileResource`, `DirResource`, `PythonResource` и другие.
* `yt` - ресурсы и задачи для работы с YT.
* `sandbox` - ресурсы и задачи для работы с Sandbox.
* `ecstatic` - ресурсы и задачи для работы с Ecstatic.

---

* `module_autostart` - типовые автостартеры и классы для написания собственных автостартеров.
* `module_rpc` - отвечает за обмен сообщениями с шедьюлером Огорода, а также обёртка для запуска огородного модуля.
* `module_traits` - классы для работы с настройками модуля, которые хранятся в `module_traits.json`.

---

* `cpp` - классы для использования в C++ коде: `GardenError`, `AutotestsFailedError`, `DataValidationWarning`.
* `cython` - обёртка для вызова плюсовых функций из Питона, которая правильно конвертирует классы исключений.

---

* `utils` - несколько простых вспомогательных функций.
* `test_utils` - вспомогательные функции для написания тестов огородных модулей.
