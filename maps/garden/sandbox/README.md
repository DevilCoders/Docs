# Sandbox

Бинарные таски, которые запускаются в Sandbox.

* `build_task` - таска, которая запускается по коммиту в Аркадию и собирает бинарник огородного модуля.
* `osm_downloader` - таска для скачивания данных OpenStreetMap для модуля `osm_src`. Запускается по расписанию каждую ночь.
* `osm_statistics_postprocessing` - таска для постобработки таблиц со статистикой OpenStreetMap
* `server_autotests` - таска для запуска автотестов всего Огорода. Запускается по расписанию каждую ночь.
* `yt_stats_collector` - таска, которая парсит логи шедьюлера YT и пишет YT-таблицы со статистикой работы огородных тасок. Запускается по расписанию каждую ночь.
