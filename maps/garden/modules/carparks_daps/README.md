# carparks_daps

## Module description

This modules extracts driving arrival points that were manually set for some toponyms in NMAPS by editors.

NMAPS snippets will be used instead of mined in [dap_snippets tool](https://a.yandex-team.ru/arc/trunk/arcadia/maps/carparks/tools/dap_snippets).
