# carparks_validator
Огородный модуль автоматической валидации датасета парковок. Сравнивает тайлы
в указазанных областях, генерируемые рендерерами парковок; между собой
сравниваются рендереры, использующие датасеты из stable (референсный) и
testing (валидируемый) бранчей экстатика.

Посылает отчет о результатах валидации на почту. Если валидация считается
непройденной, то билд модуля завершается с ошибкой.
Валидация считается непройденной, если:
1) не удалось загрузить хотя бы один тайл;
2) число изменившихся тайлов больше порога;
3) изменение числа атрибутов или геометрических примитивов больше порога.

Непройденная автоматическая валидация еще не обозначает, что датасет
невалидный, просто большое число изменений требует ручной проверки и активации
нового слоя парковок.
