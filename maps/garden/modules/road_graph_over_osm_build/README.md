# road_graph_over_osm_build

## Garden module description
This Garden module is the [road_graph_build](https://a.yandex-team.ru/arc/trunk/arcadia/maps/garden/road_graph_build) module' counterpart but uses [ymapsdf_osm](https://a.yandex-team.ru/arc/trunk/arcadia/maps/garden/modules/ymapsdf_osm) module as a source.
The module uploads built files to ecstatic storages.
