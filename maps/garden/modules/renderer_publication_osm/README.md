# renderer_publication_osm

## Module description

This module compiles maps using data from Open Street Maps.

The module obtains:
- OSM data from module `renderer_denormalization_dump_osm`.
- designs from module [renderer_map_stable_bundle](../renderer_map_stable_bundle/README.md) and related modules.

The result is uploaded as ecstatic dataset [yandex-maps-renderer-compiled-osm](http://core-ecstatic-coordinator.maps.yandex.net/pkg/yandex-maps-renderer-compiled-osm/versions).
