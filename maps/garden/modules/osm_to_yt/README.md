# osm_src

## Общее описание модуля

Map-модуль, вырезает из OSM данных планеты данные для заданного региона, и конвертирует их в набор YT-таблиц.

## Старт модуля и входные данные

На вход модулю подаются:
* RemoteDirResource со ссылкой на файл OSM данных планеты в формате PBF
* YtFileResource c coverage-файлом для регионов

## Выходные данные

Результатом работы программы являются таблицы на YT:
* nodes (точки)
* nodes_tags (теги к точкам)
* ways (линии)
* ways_nodes (составные для линии точки)
* ways_tags (теги к линиям)
* relations (объединения элементов)
* relations_tags (теги к объединениям)
* relations_members (элементы, входящие в объединения)
