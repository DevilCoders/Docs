# renderer_navi_stable_bundle

## Module description
[Design testing workflow](https://wiki.yandex-team.ru/users/vbenkevich/Design-testing-garden-part/)

This module copies stylesheets and icons resources from stylerepo_navi_design_src (navigator design). Created resources will be used during next renderer_publication module executions. The idea of this module to make new design available for renderer_publication only after it will be manually checked.
