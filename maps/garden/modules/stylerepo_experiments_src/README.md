# stylerepo_experiments_src

## Module description
[Design testing workflow](https://wiki.yandex-team.ru/users/vbenkevich/Design-testing-garden-part/)

This module obtains map designs and resources (icons, patterns) for active experiments (maps, navi) from cartograph [tool](http://cartograph.maps.yandex-team.ru/) using stylerepo [api](https://wiki.yandex-team.ru/users/avlasyuk/Stylerepo-bundle-release/). Module is executed by cartograph when new version of design is ready. Created resources will be used during next module renderer_design_testing executions.
