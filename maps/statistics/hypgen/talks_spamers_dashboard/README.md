Скрипт для расчета дашборда спамеров в разговорчиках:
https://datalens.yandex-team.ru/vkz7urqvg6loq-talks-dash?state=f8b6f0c3138

По умолчанию скрипт запускается ежедневно и делает расчет за вчерашний день.

Скрипт считает всех пользователей, которые оставили более 5 показанных сообщений 
или всего оставили более 10, и хотя бы одно из них было показано.

Параметры запуска:

`--create` - создает табличку для дешборда. Нужно только при первом запуске

`--remove` - Стирает текущую таблицу с данными дешборда (использовать с осторожностью!)

`--date` - задает дату расчета. Поддерживает значение `yesterday`
