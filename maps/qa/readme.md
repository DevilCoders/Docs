# Metrics

Расчитываем различные показатели по таскам Трекера.

Данные берём из [YT](https://yt.yandex-team.ru/hahn/navigation?path=//home/startrek/tables/prod/yandex-team/queue/maps).

Результатом являются 
[отчёты в statface](https://stat.yandex-team.ru/Maps_Plus_Beta/QA/Metrics)
и таблички с подробностями по отдельным таскам на [YT](https://yt.yandex-team.ru/hahn/navigation?path=//home/maps-qa/metrics&).

[Подробности на wiki](https://wiki.yandex-team.ru/AleksandrStrashnov/qa/#realizacija). 

[Из каких тасок получились такие числа?](https://wiki.yandex-team.ru/maps/qa/metrics/#kaksmotretpodrobnostiotchjotov)


## coverage-quality

 - Локальный перерасчёт за день занял 30 минут.
 - Локальный перерасчёт за год занял 40 минут.


## time-to-fix

 - Локальный перерасчёт за день занял 30 минут.
 - Локальный перерасчёт за год занял 125 минут.

## time-to-result

 - Локальный перерасчёт за день занял 35 минут.
 - Локальный перерасчёт за год занял 117 минут.

## time-to-lsr

 - Метрика времени разбора LSR тикетов тест-лидами

## geo-devices

 - Метрики обеспечения устройств гео сервисов.

## passed_by_assessors

 - Метрики перевода кейсов в ассесоров.
