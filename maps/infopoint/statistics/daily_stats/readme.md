## About
Утилита собирает статистику по [events-log](https://yt.yandex-team.ru/hahn/navigation?path=//home/logfeller/logs/maps-core-jams-infopoints-stable-events-log/1d) таблицам. По этим данным строятся графики в DataLens:

Это нужно:
- Для анализа динамики количества дорожных событий/комментариев, генерируемых пользователями 
- Для анализа времени, затрачиваемого на модерацию
- Для анализа результатов модерации
