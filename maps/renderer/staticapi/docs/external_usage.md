# Внешние пользователи

В бесплатной версии API нужно соблюдать [условия бесплатного использования](https://yandex.ru/dev/maps/jsapi/doc/2.1/terms/index.html). Бесплатный доступ осуществляется без ключа API. Часть пользователей используют старые бесплатные ключи API, которые раньше выдавались.

Если проект не соответствует условиям бесплатного использования API, нужно использовать [коммерческую версию](https://yandex.ru/dev/maps/commercial/doc/concepts/about-enterprise.html).
