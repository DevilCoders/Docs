Генератор патронов для navi-stopwatch
===

После вкоммиченных изменений необходимо собрать новый ресурс при помощи таски, и обновить ID ресурса [`binary_version.json`](binary_version.json).<br>
Для этого можно открыть ресурс на сандбоксе и перейти по ссылке на создавший его таск (например, ресурс https://sandbox.yandex-team.ru/resource/1939371734/view и таска https://sandbox.yandex-team.ru/task/873108539/view).
