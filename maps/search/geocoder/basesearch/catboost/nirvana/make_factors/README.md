В этой папке находятся yql-запросы, используемые в графе нирваны в кубиках Geocoder formula: make_factors (composite).

Этот кубик преобразует данные в формате preprocessed_logs с помощью списка факторов factor_names в таблицу, в которой находятся пары с факторами для катбуста и допольнительными колонками для последующего анализа.

Если что-то непонятно, можно смотреть на
[вики-страничку формулы](https://wiki.yandex-team.ru/vladimirzajjcev/cheats/services/formula/) и туда же в словарик, а также в граф нирваны (ссылка есть на вики).
