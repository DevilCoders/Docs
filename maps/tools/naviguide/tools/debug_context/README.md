Вывод отладочного контекста
===

Симуляция [`simulate`](/arc/trunk/arcadia/maps/tools/naviguide/tools/simulate) выдаёт файл `debug`, который содержит подробную информацию о кандидатах и вероятностях переходов. Как правило, файл получается достаточно большой, и не переваривается [`easyview`](/arc/trunk/maps/tools/easyview).<br>
Данная тулза позволяет оставить информацию о кандидатах только возле интересных событий, которые помечены на таймлайне (перестроения/пользовательские проблемы/etc.) с помощью `_timeline_mark`.

Использование:
```bash
$ cat debug result | ./debug_context --context 5 | easyview
```
