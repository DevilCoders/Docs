###Что тут лежит?

Все для того, чтобы найти интересные треки, разметить их с помощью толокеров и загрузить полученный результат в YT

Скрипт для поиска треков `maps/tools/track_selection/find_tracks`
Библиотека для работы с толокой `maps/tools/track_selection/toloka`


##find_tracks

Утилита для поиска треков, которые матчер разметил неправильно
В результате получится таблица с найденными треками, на которые достаточно просто смотреть в easyview

##toloka
Скрипт который берет таблицу, полученную в результате `find_tracks` и делает из нее файл, который можно загрузить в толоку и набор скриптов для обработки размеченных толокерами треков

##Pipeline

1) Ищем интересные треки с помощью `find_track`

2) Из полученной таблицы `INTERESTING_TRACKS` и скрипта `toloka/from_yql_to_tsv` делаем файл `toloka.tsv`

3) Делаем контрольные/обучающие задания. Это удобно делать так: загрузить `toloka.tsv` в толоку, разметить там, выгрузить, с помощью скрипта `toloka/split_tasks` отделить основные/контрольные/обучающие задания

4) Загружаем все это в толоку, ждем пока толокеры сделают свое дело, выгружаем 

4.5) Если хочется посмотреть, как толокеры поразмечали треки, можно воспользоваться `toloka/results_converter`

5) С помощью export_from_toloka загружаем полученные треки в YT
