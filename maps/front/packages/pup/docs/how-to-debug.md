# Отладка теста

Поместите оператор `debugger;` в нужное место вашего теста и запустите `pup` с флагом `--debug`.
Чтобы начать отладку в браузере на базе Chromium откройте браузер и перейдите в `chrome://inspect` (возможно `browser://inspect`) и нажмите кнопку "Open Dedicated DevTools for Node", которая отобразит список процессов к которым можно подключиться.
