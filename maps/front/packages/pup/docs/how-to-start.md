# Quick Start

[PUP](https://github.yandex-team.ru/denis-vs/pup) позволяет создавать и запускать функциональные тесты. Следуя этой инструкции, вы сможете быстро подключить тесты в вашем проекте. Подробно изучить как работает [Jest](https://jestjs.io/docs/en/api) и [Puppeteer](https://github.com/puppeteer/) можно в официальной документации.
Pup запускает тесты в одном окружении. Это необходимо для снижения зависимости теста от платформы. Например, отображение шрифтов может незначительно отличаться в зависимости от операционной системы, поэтому Pup использует Docker, чтобы результат не зависел от того в какой ОС выполняется проверка. Для работы потребуется установить [Docker](https://docs.docker.com/install/), а также ознакомиться с документацией.

Pup автоматически создаст дирректорию в вашем проекте для функциональных тестов с базовым примером. Ознакомиться как писать тесты, можно в документации puppeteer.

## Установка пакета
#### Для работы нужен предустановленный Докер.

```
npm i --registry http://npm.yandex-team.ru --save-dev @yandex-int/pup
```

## Запуск тестов
```
npx pup
```

## Запуск тестов в ржиме записи
Для записи моков запустите `pup` с флагом `--mode=record`.
```
npx pup --mode=record
```
