# О справочнике

Справочник содержит описание программного интерфейса JavaScript API версии 3.0.

Целевой аудиторией справочника являются разработчики сайтов, которые хотят использовать интерактивные
Яндекс.Карты на своих веб-страницах.

Справочник содержит описания открытых классов и методов API и рассчитан на разработчиков,
знакомых с JavaScript и сервисом Яндекс.Карты. Классы перечислены в алфавитном порядке.
