# Глоссарий

## Карта

Формирование карты на странице сводится к описанию входящих в нее _источников данных_ и _слоев_.

Карта может сообщать пользователю о _событиях_, которые возникают в процессе пользовательских или программных взаимодействий.

Пользовательские взаимодействия с картой осуществляются с помощью _поведений_, которые можно включать и отключать.

## Источники данных

Источники данных хранят в себе данные, которые можно отобразить на карте с помощью _слоев_.

Существуют 3 типа источников данных.

### `YMapTileDataSource`

Описывает источник данных для тайлового слоя. Слой может быть растровым или векторным.

### `YMapFeatureDataSource`

Описывает источник геометрических данных описанных в формате `geojson`.

Описание данных для данного источника осуществляется с помощью компонента `YMapFeature`.

### `YMapMarkerDataSource`

Описывает источник точечных данных, которые могут быть отображены в виде DOM-элементов.

Описание и визуальное представление данных для данного источника описывается с помощью компонента `YMapMarker`.

## Слои

Слой представляет собой визуальное отображение _источника данных_.

Порядок и визуальное перекрытие слоев осуществляется с помощью _zIndex_.

## Поведения

Поведения позволяют пользователю производить манипуляции с картой, используя различные устройства ввода.

Например, с помощью колеса мышки можно изменять масштабирование карты.

## События

События позволяют программно определять текущее состояние карты и ее реакцию на взаимодействия пользователя.

Например, событие `onClick` может сообщить нам, что пользователь кликнул по какому-то объекту на слое.
