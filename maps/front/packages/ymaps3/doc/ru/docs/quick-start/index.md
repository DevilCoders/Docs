# Быстрый старт

{% note alert "Внимание." %}

В настоящий момент JS API 3.0 доступна в режиме бета-тестирования. Это означает, что вы можете в числе первых пользователей опробовать новые функции в деле.

Бета-версия также означает, что интерфейсы методов и поведение в API может поменяться. В связи с этим мы не рекомендуем использовать бета-версию в продакшене.

{% endnote %}

Ниже приведены инструкции, как показать на странице простую карту.

1. [Получите API-ключ](#get-api-key)
1. [Подключите API](#load-api)
1. [Создайте контейнер для карты](#create-map-container)
1. [Создайте карту](#create-map)

## Шаг 1. Получите API-ключ { #get-api-key }

Зайдите на страницу [Кабинета Разработчика](https://developer.tech.yandex.ru/) и нажмите кнопку **Получить ключ**.
Во всплывающем окне выберите сервис «JavaScript API и HTTP Геокодер».

После заполнения формы появится надпись «Сервис успешно подключен».
Созданный ключ будет доступен в разделе «Ключи». Его необходимо использовать в дальнейшем при подключении API.

{% note info "Примечание." %}

JS API 3.0 работает только с ключами, у которых заполнено поле "Ограничение по HTTP Referer".

{% endnote %}

{% note info "Примечание." %}

Ключ будет активирован в течение 15 минут после получения.

{% endnote %}

## Шаг 2. Подключите API { #load-api }

Добавьте в заголовок `head` HTML-страницы строку следующего вида:

```html
<head>
  <script src="https://api-maps.yandex.ru/3.0-beta/?apikey=<ваш API-ключ>&lang=ru_RU" type="text/javascript"></script>
</head>
```

[Подробнее о подключении API](../dg/concepts/load.md)

## Шаг 3. Создайте контейнер для карты { #create-map-container }

Создайте видимый контейнер **ненулевого** размера, в котором будет размещена карта.
В качестве контейнера может использоваться любой HTML-элемент блочного типа (например, элемент `div`).
Карта заполнит этот элемент полностью.

```html
<body>
  <div id="map" style="width: 600px; height: 400px"></div>
</body>
```

Уникальный идентификатор контейнера (в примере это `id="map"`) будет использоваться в следующем шаге для получения
указателя на контейнер карты.

> Это не обязательное условие, для инициализации карты требуется `HTMLElement`,
> вы можете получить его любым доступным методом.

## Шаг 4. Создайте карту { #create-map }

В JavaScript-коде создайте экземпляр карты. Конструктору нужно передать:

- идентификатор HTML-контейнера;
- центр карты;
- коэффициент масштабирования.

Пример:

```html
<script type="text/javascript">
  // Промис `ymaps3.ready` будет зарезолвлен, когда
  // загрузятся все компоненты API.
  ymaps3.ready.then(init);
  function init() {
    // Создание карты.
    const myMap = new ymaps3.YMap(document.getElementById('map'), {
      location {
          // Координаты центра карты.
          // Порядок по умолчанию: «долгота, широта».
          center: [37.64, 55.76],

          // Уровень масштабирования. Допустимые значения:
          // от 0 (весь мир) до 19.
          zoom: 7
      },
    });
    // Добавляем слой для отображения схематической карты Яндекса.
    myMap.addChild(new ymaps3.YMapDefaultSchemeLayer());
  }
</script>
```

{% note alert "Внимание." %}

Создавать карту следует после того, как веб-страница загрузится целиком и будет создан контейнер с id.
Это нужно, чтобы к нему можно обращаться по `id`.
Чтобы инициализировать карту после загрузки страницы, можно воспользоваться промисом [`ymaps3.ready`](../dg/concepts/load.md#ready).
Он резолвится тогда, когда API будет загружен и готов к работе.

{% endnote %}

[Подробнее о создании карты](../dg/concepts/map.md)

## Результат

Полный текст примера:

```html
<!DOCTYPE html>
<html xmlns="http://www.w3.org/1999/xhtml">
  <head>
    <title>Быстрый старт. Размещение интерактивной карты на странице</title>
    <meta http-equiv="Content-Type" content="text/html; charset=utf-8" />
    <script src="https://api-maps.yandex.ru/3.0-beta/?apikey=<ваш API-ключ>&lang=ru_RU" type="text/javascript"></script>
    <script type="text/javascript">
      ymaps3.ready.then(init);
      function init() {
        const myMap = new ymaps3.YMap(document.getElementById('map'), {
          location: {
            center: [37.64, 55.76],
            zoom: 7
          }
        });
      }
    </script>
  </head>

  <body>
    <div id="map" style="width: 600px; height: 400px"></div>
  </body>
</html>
```
