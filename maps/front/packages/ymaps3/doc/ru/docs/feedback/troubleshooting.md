# Техническая поддержка

Если при работе с API у вас возникли трудности или замечания, пожалуйста, сообщите нам об этом. Выберите нужную категорию для обращения:

- [Вопросы по использованию API](#troubleshooting__code-debug)

- [Предложения по улучшению API](#troubleshooting__new-feature)

## Вопросы по использованию API { #troubleshooting\_\_code-debug }

Если ваш код не работает или вы не знаете, как решить свою задачу, поищите ответ в сообществе API Карт на [Stack Overflow](https://ru.stackoverflow.com/questions/tagged/yandex-maps-api) (сообщество на русском). При поиске используйте метки:

- [yandex-maps-jsapi](https://ru.stackoverflow.com/questions/tagged/yandex-maps-jsapi) – JavaScript API;
- [yandex-geocoding-api](https://ru.stackoverflow.com/questions/tagged/yandex-geocoding-api) – HTTP API Геокодера;
- [yandex-places-api](https://ru.stackoverflow.com/questions/tagged/yandex-places-api) – HTTP Поиска по организациям;
- [yandex-static-maps-api](https://ru.stackoverflow.com/questions/tagged/yandex-static-maps-api) – Static API;
- [yandex-maps-embed](https://ru.stackoverflow.com/questions/tagged/yandex-maps-embed) – общая метка для Конструктора и Виджета Яндекс.Карт.

Если вы не нашли свой вопрос на Stack Overflow, задайте его там.
Прежде чем публиковать вопрос, внимательно ознакомьтесь со статьей [Как задавать хорошие вопросы](https://ru.stackoverflow.com/help/how-to-ask).

{% note info "Примечание." %}

Также вы можете поискать ответ в [Клубе API Карт](https://yandex.ru/blog/mapsapi).
Клуб доступен только для чтения — все технические вопросы следует задавать на [Stack Overflow](https://ru.stackoverflow.com/questions/tagged/yandex-maps-api).

{% endnote %}

## Ошибка в API или в документации { #troubleshooting\_\_doc-error }

Если вы обнаружили ошибки в API, пожалуйста, сообщите об этом нам через [форму обратной связи](#feedback).
По возможности приведите фрагмент кода, в котором возникает ошибка.
Для этого загрузите код на [CodeSandbox](https://codesandbox.io) и опубликуйте его.
Полученную ссылку укажите в описании проблемы. В качестве основы можно взять [шаблон](https://codesandbox.io/s/fervent-mendel-rfd4xv).

Если вы нашли ошибку в документации, напишите об этом нам также через [форму обратной связи](#feedback).
В теме сообщения выберите пункт «Другое». По возможности приложите скриншот.
