# sprav-protos

В пакете собранны схемы справочика в protobuf формате.
Все схемы выкачиваются из [аркадии](https://a.yandex-team.ru/arc/trunk/arcadia/sprav/protos).

## Обновления схем

Для получения свежей версии схем из аркадии необходимо выполнить команду:

```npm run update-schemas```
## Сборка
Чтобы пересобрать текущую версию схем, указанную в файле [revision](revision), выполните:
```npm run build```
