# maps-transliteration

Транслитерация текста повторяющая функциональность модуля [arcadia/maps/libs/stringutils/transliteration](https://a.yandex-team.ru/arc/trunk/arcadia/maps/libs/stringutils/transliteration.cpp).
Под капотом находится C++ библиотека [ICU](https://unicode-org.github.io/icu/userguide/transforms/general/#overview) - International Components for Unicode и [node-биндинг](https://github.com/longnow/node-icu-transliterator) к ней.
