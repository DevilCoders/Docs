## Скрипт для генерации QR кодов

Раз в 30 минут, запускается скрипт, который:
- берет все тикеты экспериментов
    - созданные после 01.09.2021 
    - у которых есть тэг начинающийся с `mob_maps` 
- берет все выборки, для каждой из них
    - берет каждый сервис (`MAPS_UI`, `MAPS_GEOSEARCH`, ...)
    - для каждого сервиса отдельно генерирует `QR code`
- прикладывает одним сообщением в тикет эксперимента все `QR` коды
- добавляет к тикету тэг `maps-ab_qr-code`
- если какая-либо выборка поменялась, то робот обновит предыдущий комментарий новыми `QR` кодами
