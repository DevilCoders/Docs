# Структура файлов

[Аркадия](https://wiki.yandex-team.ru/arcadia/) накладывает ряд ограничений на файловую структуру.

> Подробнее об ограничениях читайте в разделе [Требования к структуре файлов](https://a.yandex-team.ru/arc_vcs/frontend/docs/faq/requirements.md).

## Файловая структура

* `packages` — код npm-пакетов
* `services` — код Node.js-сервисов
