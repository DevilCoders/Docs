# company-card-with-map

Карточка компании с картой и кнопками «Моя организация» и «Рекламировать».
Используется в блоке `company-snippet` в результатах поиска и списке похожих компаний при создании организации.

В качестве данных принимает нормализованный объект компании.

```js
{
    block: 'company-card-with-map',
    company: {
        id: 42,
        names: [],
        address: {},
        emails: [],
        phones: [],
        urls: [],
        rubrics: []
    }
}
```
