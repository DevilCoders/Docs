# page

Блок отвечает за генерацию обвязки страницы.

Структура любой страницы следующая:
```
page__content
    header
    navigation
    main
    footer
    yandex-metrica
```

## page__bound

Элемент отвечает за сетку.

Общая ширина (левый отступ под лого + контент) 960px.

Отступ под кириллическое лого — 235px, под латинское — 206px. Исчезает на узких экранах (например, на планшетах в вертикальной ориентации).

Под контент и правый отступ остается 960 - 235 = 725 либо 960 - 206 = 754.

Под контент отводится три варианта:
* 560px — основной
* 725/754px — съедает отступ справа, используется для широких колонок
* 960px — съедает всю доступную ширину, включая отступ слева. Используется для табличной раскладки (страница Мои заказы), требует паддинг.

### page__bound_type_list-page

Модификатор элемента bound для страниц со списком (e.g список организаций, список заявок)

Минимальная ширина 750px вместо обычного 560px
