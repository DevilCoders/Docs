# show-more

Блок для показа дополнительных элементов по клику на Еще

## Обзор блока

### Модификаторы блока

| Модификатор | Допустимые значения | Способы использования | Описание |
| ----------- | ------------------- | -------------------- | -------- |

### Специализированные поля блока

| Поле | Тип | Описание |
| ---- | --- | -------- |

### Элементы блока
| Элемент | Описание |
| ------- | -------- |
| item | элемент который необходимо скрыть/показать. Имеет модификатор `hidden: true`. |
| more | элемент по которому происходит клик |

## Описание блока

Блок для показа дополнительных элементов по клику на Еще

```js
{
    block: 'show-more',
    js: true,
    content: [
        ['a','b','c'].map((item, index) => ({
            block: 'my-block',
            mix: { block: 'show-more', elem: 'item', elemMods: { hidden: index > 1 } },
            item
        })),
        ['a','b','c'].length > 1 && {
            block: 'button',
            mods: { theme: 'islands', size: 's' },
            mix: { block: 'show-more', elem: 'more' },
            text: i18n(block, 'more-companies', { count: companies.length - COMPANIES_LIMIT })
        }
    ]
}
```
