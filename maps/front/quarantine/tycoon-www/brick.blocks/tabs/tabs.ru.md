# Блок tabs

Простой переключатель табиков.

## Модификаторы

| Модификатор | Допустимые значения | Способы использования | Описание |
| ----------- | ------------------- | -------------------- | -------- |
| type | <code>'button'</code>, <code>'line'</code> | <code>BEMJSON</code> | Тип группы радиопереключателей. |
| theme | <code>'islands'</code> | <code>BEMJSON</code> | Стилевое оформление. |
| size | <code>'m'</code>, <code>'l'</code>  | <code>BEMJSON</code> | Размер радиопереключателей в группе. Используется только для радиопереключателей с модификатором theme в значении islands. |

## Элементы
### tabs-group

Элемент, содержащий radio-group для переключения табов.

### pane

Переключаемое содержимое.

Модификатор *active* указывает на выбранный в данный момент.

## Поля
### tabs

Поле, отвечающее за отображение самих табов в переключателе.
В него передается массив объектов с полями:
*   `<String> text` - текст на кнопке
*   `<Boolean> active` - флаг, указывающий, на выбранный таб
*   `<String> url` - строка, содержащая ссылку или относительный путь, которые будет устанавливаться при переключении на таб через `history.pushState`

## Использование

```javascript
{
    block: 'tabs',
    tabs: [
        {
            text: 'tab1',
            active: true,
            url: '/myurl'
        },
        {
            text: 'tab2',
            url: '/mysecondurl'
        }
    ],
    content: [
        {
            elem: 'pane',
            elemMods: { active: true },
            content: 'my tab content'
        },
        {
            elem: 'pane',
            content: 'my second tab'
        }
    ]
}
```
