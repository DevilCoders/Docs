# devops

Набор вебхуков/сервисов, облегчающих [дежурство](https://wiki.yandex-team.ru/maps/dev/ui/deploy/duty) (self service).

## Вебхук для трекера
При создании тикета в очереди [GEOMONITORINGS](https://st.yandex-team.ru/GEOMONITORINGS) вызывается хук `/devops/webhook/startrek`, в результате чего в тикет добавляется комментарий с информацией, облегчающей выяснения причины срабатывания мониторинга:

* Список графиков в графите для каждого хоста, на котором сработала проверка.
* Документация по проекту, а также ссылки из соответствующего проекта ABC с типом `monitoring`.
* Дежурные и системные администраторы из соответствующего проекта ABC добавляются в followers.
