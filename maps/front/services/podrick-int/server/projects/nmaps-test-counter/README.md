# Счетчик для обеспечения параллелизации функциональных автотестов в НК

## Что это и для чего нужно
Для возможности параллельного запуска функциональных тестов в НК, необходимо разграничить зоны, где происходит редактирование данных.
Перед запуском автотестов запрашивается значение этого счетчика, позволяющее затем выбрать нужный спан на карте.
