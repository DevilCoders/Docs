# livereload

Наша реализация livereload основана на https://github.com/mklabs/tiny-lr.

## Принцип работы
1. Шаблон инжектит на страницу [скрипт](https://github.com/livereload/livereload-js), который раздается с помощью `tiny-lr` middleware.
2. Этот скрипт соединяется по WS с `tiny-lr` сервером.
3. Сервер с помощью `chokidar` следит за изменениями собранных файлов и отправляет по WS команды на обновление стилей или перезагрузку страницы.

Шаблон попадает на страницу по зависимостям от блока `page`.
Сервер `tiny-lr` поднимается только для dev-режима.
