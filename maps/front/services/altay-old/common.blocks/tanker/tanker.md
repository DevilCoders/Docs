tanker
------

Хранит служебные переводы-функции для упрощения работы со склоняемыми ключами.

`plural` используется с числительными и возвращает индекс нужной формы в массиве с переводами вида `['one', 'some', 'many', 'none']` по параметру `{ count: N }`, где N — произвольное целое число.

Пример использования:

```js
this.i18n('b1', 'key1', { count: 1 }); // ключ
this.i18n('b1', 'key1', { count: 3 }); // ключа
this.i18n('b1', 'key1', { count: 5 }); // ключей
this.i18n('b1', 'key1', { count: 0 }); // ключей
```
