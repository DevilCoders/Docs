# radio

Доопределяет `radio` из `bem-components`.

Повторяем нативное поведение радио-кнопок, когда они могут быть связаны по одинаковому полю name.
https://github.com/bem/bem-components/issues/1759

```js
{
    block: 'radio',
    mods: { theme: 'islands', size: 'm' },
    name: 'tmpl',
    val: 'BEMHTML',
    text: 'Use BEMHTML'
}
```
