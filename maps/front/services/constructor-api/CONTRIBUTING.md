## Технологии
 * Серверная часть работает на NodeJS 8
 * Для формирования страницы используется express с шаблонизатором [mustache](https://www.npmjs.com/package/mustache).
 * Для сборки статики используется [ymb](https://www.npmjs.com/package/ymb).
 * Для проверки ошибок и оформления кода используется eslint. Автоматически запускает перед коммитом.
 * Для написание серверной части используется JS ES2016, а для сборщика используется JS ES2015.

## Быстрый старт
```
make deps
make dev.setup
make dev
```

## Тестирование
В проекте две системы unit тестов. Обе автоматически запускается при каждом коммите.
Одна проверяет серверный код (папка /server/), а второй клиентский код.
Первая располагается в папке /tests/server/. Запускается командой make test-server.
Для тестирования используется mocha.

Вторая находится в папке /tests/client. Запускается командой make test-client.
Для тестирования используется mocha-chrome + expect.js.
Файлы теста лежат вместе с исходного файлами проекта.

Юнит тесты запускаются локально.

## Важные команды
```
make install - установка npm зависимостей
make dev - Поднятие dev-стенда. Билдер автоматически следит за изменениями в папке src/
make validate - Запуск валидации и тестов
make test - Запуск всех тестов
```
