# parking-int
The service integrates the APIs of parking and payment providers and exposes an Intranet-only HTTP API for Yandex client applications.

Supported payment providers:
- [VTB mutlicarta](./payment-providers/multicarta.md)

Supported parking providers:
- Moscow's Public Parking
