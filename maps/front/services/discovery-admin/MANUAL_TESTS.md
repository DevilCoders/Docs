# Ручной список тестов

Пока автомеханизмы не настроены. Делаем ручное тестирование по договоренности.

## Шаги:
- Проверку перед PR выполнять небольшую по собственным ощущениям.
- После утверждения и всех необходимых approved в PR выложить на тестинг из собственной ветки, проверить тесты по подходящему уровню ниже.
- Сделать merge в мастер с сообщением о выполненном условии проверки (Тест: Без проверки).

## Уровни проверки:
1. **Без проверки.** Совсем мелкие изменения, которое точно не должно ничего поломать + изменения описаний, не проверяются.
2. **Проверка конкретных правок.** Небольшие изменения, которые не должны задеть общей логики. Проверяем только конкретный раздел, где были изменения.

3. **Общая проверка.** Небольшие изменения в общей логике. Основное проверить тесты выделенные жирным.

4. **Глубокая общая проверка.** Существенный рефакторинг. Проверять все тесты целиком.


## Список тестов

### Раздел страниц
1. **Список всех страниц должен загружаться корректно.**
2. **Проверить создание страницы.**
3. Все фильтры на списке страниц должны работать. Также если подразумевается ввод текста помимо выбора из списка (селекты), проверить, что все вводится корректно

###  Редактирование страницы
3. **Нажать на кнопку обновить.  Должно показываться успешное сообщение. Любое сообщение успеха исчезает через 5 секунд.**
4. **Нажать на кнопку опубликовать. Должно показываться успешное сообщение**.
5. **Нажать на кнопку снять с публикации. Должно показаться сообщение успеха**.
6. **Поменять что-то в основной информации и в блоке организации. Обновить. Перезагрузить страницу, проверить что все обновления появились.**
7. Проверить, что селект Гео регион, Партнер, Рубрики и Теги связанных подборок подгружаются, попробовать выбрать что-то другое. Также выбор через печать.
8. Проверить загружается ли библиотека изображений в блоке организации.
9. В организации по нажатию на применить, должна быть прогрузка значений.
10. Проверить добавление всех возможных блоков. Проверить удаление блоков.
11. Проверить сортировку блоков.
12. Проверить сортировку фич и картинок в галерее.
13. Сделать ошибочные данные: URL адрес через пробел. Сохранить, должно появиться сообщение об ошибки и проверить исчезание через 10 секунд.

### Раздел партнеров
11. **Список всех партнеров должен загружаться корректно.**
12. **Проверить создание партнера.**

### Редактирование партнера
13. **При нажатии на кнопку обновить. Должно появиться успешное сообщение.**
14. **Проверить возможность изменения разных полей. Обновить и перезагрузить страницу, проверить, что данные поменялись.**
15. Добавить партнерские ссылки не менее 2-х штук + проверить их сортировку.

### Раздел тегов
16. **Список тегов должен отображаться и загружаться корректно.**
17. **Проверить создание тега.**
18. **Проверить обновление тега.**
19. Проверить работу фильтров в заголовках на общей странице списков тегов.

### Раздел рубрик
20. **Проверить корректность загрузки.**
21. Проверить работу фильтра по заголовку.

### Регионы
22. **Таблица должна загружаться корректно.**
23. **Проверить создание региона.**

### Редактирование региона
24. **Проверить обновление региона. Должно быть сообщение успеха.**
25. **Проверить публикацию региона. Должно быть сообщение успеха.**
26. **Проверить снятие с публикации. Должно быть сообщение успеха.**
27. **Проверить клонирование.**
28. **Поменять информацию в заголовках. Обновить. Перезагрузить. Должно все обновиться.**
29. Добавить поисковую подсказку. Проверить что все корректно.
30. Проверить сортировку поисковых подсказок.
31. Проверить категории. Поменять тексты основные и в тегах. Добавить новый тег. Удалить старый. Отсортировать.
32. Проверить работу карты. Добавить, удалить область. Поменять значения мин, макс.
