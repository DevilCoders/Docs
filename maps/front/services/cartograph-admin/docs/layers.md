## Типы слоев

* ```border``` - границы административно-территориального деления (АДР)
* ```Building``` - здания
* ```building_number``` - адресные точки
* ```country_label``` - подписи стран
* ```hydro_a``` - полигоны воды
* ```Hydro_a_label``` - подписи для всех полигонов воды (можно выделять подписи для разных типов: например, для озер, для морей)
* ```hydro_l``` - линейная гидрография
* ```Hydro_overlay``` - болота и ледники
* ```landuse_a``` - кварталы, промышленная застройка,
* ```locality_a``` - полигоны Административных границ города
* ```place_label``` - подписи городов и районов Населенных пунктов
* ```Poi``` - организации
* ```region_label``` - подписи областей, районов, штатов
* ```relief_p``` - подписи островов
* ```Road``` - дороги
* ```road_label``` - тоже дороги, но слой предназначен для рисования подписей дорог
* ```roadnet_a``` - полигоны мостов и пешеходных мостов
* ```Roadnet_p``` - светофоры, ворота, калитки.
* ```transport_p``` - остановки общественного транспорта, аэропорты, вокзалы, гавани, порты, пристани, метро метро
* ```transport_l``` - линии метро (он пока не добавлен), (то, что не имеет физического отображения)
* ```Vegetation``` - лес
* ```Vegetation_planned``` - парки, национальные парки, газоны, сады.
