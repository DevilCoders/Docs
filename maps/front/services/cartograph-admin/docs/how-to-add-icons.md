## Как добавлять иконки

Если на карте для иконки предусмотрено дневное и ночное отображение, то:
- Нажать Загрузить. При загрузке имя файла должно иметь постфикс ```day``` или ```night```
- Имя файла для ночной иконки и дневной должно совпадать (за исключением постфикса)
- Если выбрать файл с постфиксом, то картограф сообщит, что не хватает второго файла для ночного или дневного отображения.
- Если выбрать файл без постфикса, то картограф применит этот ресурс для обоих режимов: дневного и ночного.

Иконки грузятся в бекенд, бекенд возвращает идентификатор, сам ресурс по http скачивается по указанному url шаблону, в котором подставлен id иконки.

Рендерер и бек оперирует шаблоном: ```rc:{id иконки}```.
