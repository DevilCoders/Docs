# Sprite Icon Component

Компонент принимает урл до файла `svg`

На `componentDidMount` компонент подгрузит svg файл и вставит его в спрайт (при отсутствии спрайта, он его создаст), при рендере еще одного такого же компонента запрос и вставка в спрайт производиться не будет.

Что бы иметь возможность управлять цветом из css после импорта файла из макетов замените цвет из атрибутов на переменную `currentColor`, теперь цветом компонента можно управлять через правило `color`.

```
  <svg xmlns="http://www.w3.org/2000/svg" width="36" height="36" version="1" viewBox="0 0 36 36">
-  <path fill="#000" d="M28.146 7.146L18 17.293 7.854 7.146l-.708.708L17.293 18 7.146 28.146l.708.708L18 18.707l10.146 10.147.708-.708L18.707 18 28.854 7.854z"/>
+  <path fill="currentColor" d="M28.146 7.146L18 17.293 7.854 7.146l-.708.708L17.293 18 7.146 28.146l.708.708L18 18.707l10.146 10.147.708-.708L18.707 18 28.854 7.854z"/>
  </svg>
```
