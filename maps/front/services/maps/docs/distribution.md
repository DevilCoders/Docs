# Дистрибуции в СНГ

конфиг для дистрибуции

-   [тестовый](https://bunker.yandex-team.ru/maps-www/testing/distribution?view=raw)
-   [продовый](https://bunker.yandex-team.ru/maps-www/production/distribution?view=raw)

структура конфига:

-   верхний уровень - опции из директа ('orgpage','transport','traffic' и тд)
-   у каждой опции есть разделение по доменам (com, tr, by, ua, kz, uz)

Для изменения параметров у опции, нужно конкретно менять параметры для каждого из доменов

параметры:

-   title - титульный текст,
-   backgroundImage - фоновое изображение,
-   buttonText - текст кнопки для открытия приложения,
-   logo - логотип карт (если логотип не указан, берется дефолтный логотип в зависимости от локали)

После изменений надо нажать "Сохранить и опубликовать"

## Добавление картинок и логотипов

картинки лежат [здесь](https://bunker.yandex-team.ru/maps-www/files/distribution?view=raw)

если ссылка на картинку уже есть, то картинку заливать не надо

Добавление картинки:

-   нажать плюсик (+)
-   в появившемся окне выбрать upload
-   загрузить картинку
-   ссылку на картинку взять из href
