# theme
## "theme" - эта папка содержит макеты, они же layout'ы.

## Почему `src/theme/islands`
Изначально планировалась возможность делать темы оформления для API.
Одна из тем называлась так же как портальные компоненты и система шаблонизации ~2013 года: "острова".
Отсюда `theme/islands`.

Параллельно с islands в картах была разработана своя бибилиотека компонентов и система шаблонизации названная "islets".
В то время апи делал элементы управления для Яндекс.Карт.
Для них была создана своя тема "islets".
Часть из них используется до сих пор внутри апи.

```
theme
|__islands - основные layout-ы для API
    |__*
    |__islets.* - специальные layout-ы, созданные когда-то для Maps
```
