# Огород

## Запуск
```bash
# Настроить локальную машину для работы сайта по https
# Команда создаёт ssl-сертификат ~/.ssl/garden.ssl.crt
npm run local-setup
# После её выполнения добавить файл в KeyChain.app (кликнуть по файлу дважды) и в интерфейсе KeyChain.app:
# 1. Найти сертификат <your-login>.garden.dev.yandex-team.ru
# 2. Выбрать для него "Всегда доверять"

# Установить зависимости
npm install

# Старт
npm start
```


В терминале появится сообщение:
```bash
[nodemon] starting `node server/app.js`
13:34:03 - info: Приложение запущено на https://<your-login>.garden.dev.yandex-team.ru:8080
```
Идём по этому урлу в браузере.

Если браузер ругается на небезопасное соединение и не открывает сайт, убедитесь, что выполнили цель `npm run
local-setup` и добавили руками созданный сертифкат в `KeyChain.app` (см. инструкцию выше)


## Мониторинги

Если меняем настройки мониторингов (поднять/опустить timings), то после изменений то не нужно перевыкатывать образ, нужно
просто выполнить команду `qtools alerts push -f`.

Ели ты хочешь добавить каких то новых графиков или переменовать или изменить их параметры (именно графиков, не
алертов), тогда нужно перевыкатить образ.
