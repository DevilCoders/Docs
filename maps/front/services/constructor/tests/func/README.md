# Autotests for Constructor
[Hermione](https://github.com/gemini-testing/hermione/)

About testing [here](./doc/testing.md)

## Getting started
```
cd tests/func
```

Install the packages
```
make install
```

## Run tests
```
make test
```

## Run smoke tests
```
make test-smoke
```

## Run reporter gui
```
make test-reporter-gui
```
