All tickets are generated with [tvmknife](https://wiki.yandex-team.ru/passport/tvm2/debug/#tvmknife) utility in `unittest` mode.

For easy install `tvmknife` you can use [docker image](https://github.yandex-team.ru/ikokostya/docker-tvmknife).

Also you can use [tvmknife bin](https://wiki.yandex-team.ru/passport/tvm2/debug/#tvmknife)
