# Qloud CPU usage

Скрипт который подсчитывыет usage CPU по каждому окружению и формирует готовый отчет (для вставки в СтарТрек)

## Установка
```
$ pip install -i https://pypi.yandex-team.ru/simple/ yasmapi
```
Добавить в окружение переменную QTOOLS_TOKEN
```
export QTOOLS_TOKEN="AQAD-<bla-bla-bla>"
```

## Использованием
```
$ python app.py
```
Операция может занимать до 10 минут, так как на каждое окружением идет поход в Головам и агрегированием данных
