# Нарезка панорамы на фотографии в перспективной проекции

Запуск:
```bash
./bin/undistort <OpenSfM dataset>
```

## Входные данные
Папка с OpenSfM-датасетом, содержащая папку `images/` с файлами вида `<panorama id>.jpg`

Опционально может присутствовать файл `exif_overrides.json`, содержащий координаты панорам.

## Выходные данные
В исходном датасете в подпапке `undistort` создаётся нарезанный OpenSfM-датасет.
Создаётся каталог `undistort/images/` с нарезанными фотографиями.

Также создаются:
* каталог `images` с нарезанными фотографиями;
* файл `exif_overrides.json`, содержащий копию метаданных панорам, и идентификатор перспективной камеры;
* файл `camera_models_overrides.json`, содержащий описание перспективной камеры (параметры проекции).
