# Слияние фичей нарезанных фотографий в фичи исходных панорам

Запуск:
```bash
./bin/merge_features <OpenSfM dataset>
```

## Входные данные
Папка с OpenSfM-датасетом, содержащая папку `images/` с файлами вида `<panorama id>.jpg`
Внутри датасета подпапка `undistort/`, содержащая нарезанный на фотографии датасет, созданный с помощью утилиты `undistort`.
В папке `undistort/features` должны находиться фичи фотографий, сохранённые с помощью OpenSfM.

## Выходные данные
Фичи фотографий из `undistort/features` объединяются и записываются в фичи панорам в папку `features`.
