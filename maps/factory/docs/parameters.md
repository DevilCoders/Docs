### Стандартные параметры

#### `filter`

Ручки API, предполагающие массовые операции с мозаиками и исходниками мозаик (например, рендеринг и хотспотирование), принимают параметр `filter`. Этот параметр служит для фильтрации множества объектов, к которым относится выполняемая операция. `filter`  задаётся набором токенов с разделителями. Токен имеет формат: `<type>:<value>`, где `type` может принимать значения:

* `aoi` — по поставке. В качестве `value` нужно передавать идентификаторы AOI через запятую.
* `order` — по заказу. В качестве `value` нужно передавать идентификаторы заказов через запятую.
* `status` — по статусу мозаики. В качестве `value` нужно передавать текстовые значения статусов через запятую.
* `mosaic-source` — по исходникам мозаик. В качестве `value` нужно передавать идентификаторы MosaicSource через запятую.

Части параметра `filter` задают СДНФ (дизъюнкцию конъюнкций фильтров, задаваемых отдельным токеном):

* значения внутри токена соединяются через `,`,
* токены внутри конъюнкции соединяются через `*`,
* конъюнкции внутри дизъюнкции соединяются через `|`.

Пустое значение `filter` соответствует пустому набору конъюнкций, который для всех мозаик возвращает `false`.
