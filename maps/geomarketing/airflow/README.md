### Запуск
1. Добавить файл secrets.yml в основую директорию.
2. Выполнить env.py ```python3 env.py```
3. Проверить установку Docker ```docker --version``` и Docker-Compose ```docker-compose --version``` - в ответе должна быть версия (возможно, перед командами надо будет использовать ```sudo```). Если версию не показывает, выполнить установку [Docker](https://docs.docker.com/engine/install/) и [Docker-Compose](https://docs.docker.com/compose/install/).
