# Данные из Вгляда и их матчинг к данным об отправке

На этом этапе производится выгрузка данных из Взгляда.
Описание данных: https://wiki.yandex-team.ru/pythia/manual/#rezultatyoprosanaytcdannymikripty
### [Nirvana](https://nirvana.yandex-team.ru/flow/30061906-7d25-4d35-b251-850143fe0ea8/bb2795ed-0e76-4deb-8273-a0214bf25bd3/graph)


### Структура
1. Сырые результаты опросов по всем slug из лога сегментов:
   * **answers** - ответы;
   * **interviews** - данные о прохождении опроса;
   * **questions** - последние значения названий вопросов для каждого опроса;
   * **options** - "конфиг" ответов для их объединения.
