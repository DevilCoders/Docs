# Дополнительные процессы по Дешёвому трекингу

В этой дирректории собраны дополнительные процессы.
У каждого процесса должна быть своя директория с описанием процесса.
Если процесс связан с одним определённым сегментом, то в названии лучше использовать
id сегмента. В Nirvana каждый отдельный процесс выделать областью.
