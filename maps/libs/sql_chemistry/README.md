# SqlChemistry: Simple C++ ORM library for PostgreSQL

Library features:

* Object-Relational Mapping between PostgreSQL and C++.
* Serialization and deserialization for all common types.
* Create, read, update and delete functions.
* Collection of filters to construct WHERE conditions.
* ORDER BY, OFFSET and LIMIT. 
* INNER JOIN.

For more information [visit wiki page](https://wiki.yandex-team.ru/maps/dev/core/sqlchemistry).
