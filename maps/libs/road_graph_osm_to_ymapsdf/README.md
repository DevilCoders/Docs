# Исходные данные: .osm.pbf файлы
Планету целиком можно скачать по [ссылке](https://planet.openstreetmap.org/pbf/planet-latest.osm.pbf) либо через [торрент](https://planet.openstreetmap.org/pbf/planet-latest.osm.pbf.torrent).

C сайта [geofabrik](https://download.geofabrik.de/) также можно скачать отдельно континенты, страны, районы. Например, чтобы скачать Калининградскую область, надо выполнить:
```bash
wget https://download.geofabrik.de/russia/kaliningrad-latest.osm.pbf
```

Также скачать файл с Европой и всю планету целиком можно с одного из зеркал:
```bash
wget https://ftp5.gwdg.de/pub/misc/openstreetmap/download.geofabrik.de/europe-latest.osm.pbf
wget https://ftp5.gwdg.de/pub/misc/openstreetmap/planet.openstreetmap.org/pbf/planet-210510.osm.pbf
```

Альтернативный источник данных в формате OSM:
