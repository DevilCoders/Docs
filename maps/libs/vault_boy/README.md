# Vault boy

В этом разделе находится описание наших практик по работе с секретными данными, используя секретницу.
 
Здесь содержатся:

*   Примеры работы с секретницей.
*   Небольшая библиотека с шаблонизатором. Она служит для облегчения загрузки секретов из файлов в конфиги из ресурсов.

Подробрая документация [находится на wiki](https://wiki.yandex-team.ru/maps/dev/core/vaultboy/).
