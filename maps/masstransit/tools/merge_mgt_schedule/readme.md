Эта утилита подклеивает расписания из данных мосгортранса к нашим данным yandex-maps-masstransit-data-rasp. Если коротко, то утилита для каждой нашей нитки с помощью некоторых эвристик пытается найти соответствующую нитку в данных МГТ. Для ниток, у которых нашлось соответствие, утилита заменяет наше расписание на расписание из данных МГТ.

Утилита принимает на вход папку с нашими данными, папку с данными мосгортранса, и папку, в которую утилита сложит новые файлы с расписаниями в нашем формате. Кроме того, утилите можно указать черный список наших ниток, к которым не нужно подклеивать расписание. Этим стоит пользоваться в случаях, когда утилита находит неправильное соответствие для нитки и подклеивает неправильное расписание. Еще утилита может отправлять статистику о подклеенных расписаниях в статфейс - не надо этого делать при разработке. Предполагается, что статистика будет отправляться только при сборке данных в огороде.

Подробнее про расписания МГТ в картах можно прочитать на вики: https://wiki.yandex-team.ru/users/bugsbunny/MGT/
