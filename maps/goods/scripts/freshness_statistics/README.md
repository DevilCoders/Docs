# Freshness Statistics

These scripts are used to compute Funduk data age statistics. They are needed for this [dashboard](https://datalens.yandex-team.ru/ii428fzx1g00c-dashbord-po-vozrastu-dannyh-funduka).

Scripts are regulary started inside of this [Nirvana workflow](https://nirvana.yandex-team.ru/flow/c59129cb-84fa-4266-8bf3-0760c389921a) by the following [Reactor reaction](https://reactor.yandex-team.ru/browse?selected=8095555).
