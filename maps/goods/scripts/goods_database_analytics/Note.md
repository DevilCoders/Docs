# Goods Database Analytics

These scripts are used to calculate statistics for [Goods Database Analytics Dashboard](https://datalens.yandex-team.ru/3g2bc23id74ow-analitika-tovarnoy-bazy).

Usage example:

```
./statistics_calculator goods-analytics --aggregated-statistics-path //home/maps/geoapp/goods/stable/statistics/aggregated_statistics
```
