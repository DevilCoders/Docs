#### Атрибуты задачи:
- permalink - нужен правильного расчета размера очереди
- previous_id - id задачи из PRICE_SEARCH
- original_id - mds_file_name из PRICE_SEARCH.result_json.mds_file_names
- source_json
  - url - PRICE_SEARCH.source_json.url
  - source_mdsfilekey - равно original_id
  - image_mdsfilekey - вычисляемое при разрезании картинок
  - subimage_mdsfilekey - вычисляемое при разрезании картинок
- result_json
  - items - элементы прайса
    - category
    - category_status 
    - name
    - name_status
    - price
    - price_status
    - volume
    - volume_status
  - missed_items - количество пропущенных позиций
  - precision - точность по итогу приемки
  - recall - полнота по итогу приемки
  - is_valid - правильность расшифровки по результату приемки (в экспорт попадает только true)
- context - берется из PRICE_SEARCH
  - rubric - обязательно для разделения по проектам
