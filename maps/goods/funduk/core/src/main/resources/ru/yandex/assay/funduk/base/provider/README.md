**Как добавить нового поставщика:**

- добавить описание поставщика в providers.json
- завести фид, если такого еще нет
  - пример заполнения полей для фида только с матчингом по original_id (https://paste.yandex-team.ru/534920/text)
  - созданный фид нужно добавить в конфиг (feeds), чтобы следить за тем, что он работает
- добавить в ./update запрос для сбора данных об услугах поставщика, и ссылку на него в конфиг
(update_query_path)
- при необходимости добавить
  - таблицы (tables), что следить за их актуальностью
  - дополнительные запросы (queries) для данного поставщика
