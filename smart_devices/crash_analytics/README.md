# Утилиты и инфраструктура работы с креш-дампами умных колонок.

Рекомендуемый способ запуска:
1. Перейти в директорию `smart_devices/crash_analytics/package`.
2. Запустить сборку `ya make`.
3. Запустить нужные утилиты.

Т.к. утилиты при работе запускают друг друга, код написан в предположении что все программы лежат в одной директории.

## Доступные утилиты

- `yql_minidump` - скачивает креш-дампы из логов и получает стектрейсы с помощью символов из прошивки.
- `minidump_analyzer` - группирует скачанные дампы по стектрейсам с ошибкой.
