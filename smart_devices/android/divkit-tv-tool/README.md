## divkit-tv-tool

Приложение для проверки дивных рендерилок (классов, которые для различных нужд генерируют divjson с помощью kotlin dsl)

Позволяет верстать на dsl и сразу проверять на девайсе запуская приложение. При старте генерируется divjson и тут же отображается.

### How-to

Для настройки окружения в MainActivity необходимо задать ```environment: Environment```

* ```TvMainEnvironment``` для эмуляции промоблока главной (с отступом и бэкграундом)
* ```PromoEnvironment```  для промика/фичабординга (без особенностей)

Внутри окружения также есть поле шаблона ```template: CardWithTemplates```, которое получается из классов типа ```GiftPromoTemplate```, ```TvMainTemplate```..., которые и есть рендерилки, что мы хотим отлаживать
