# QA tools (https://st.yandex-team.ru/TVANDROID-5099)

## Features:

- Show version of yandex apps
- Show properties (clids, fingerprint)
- Show crash logs
- Show ANR logs
- Clear logcat
- Reset channels (clear TV storage, home-app and com.yandex.tv.input.efir)
- Restart SUW
- Open Factory menu

## For build use: ./gradlew -Pproduct=tv -p tv/qa-tools assemble
