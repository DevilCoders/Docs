Centaur Test tool
=================

General usage:
`ya make && ./centaur_test --help`


Commands
========

Execute
-------

Allows to run __some__ directives. 

Best used for debugging `show_view` like `./centaur_test execute --name show_view --payload path/to/payload.json`. 

Example for this `payload.json`: https://proxy.sandbox.yandex-team.ru/2182854234
