# giftPromoDelayMs
**Type:** Long
**Default:** -1

**Related tickets**
[TVANDROID-5194](https://st.yandex-team.ru/TVANDROID-5194)

**Context**
```json
{
  "com.yandex.tv.home": {
    "giftPromoDelayMs": "ms"
  }
}
```

**Description**
Через сколько (количество миллисекунд) показать промо подарка после старта или после возвращения домой.
Если отсутствует или равно `-1`, то промо подарка не будет показано.
