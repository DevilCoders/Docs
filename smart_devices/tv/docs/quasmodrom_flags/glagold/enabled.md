# enabled
**Type:** Object
**Default:** -

**Related tickets**
None

**Context**
```json
{
  "glagold": {
    "enabled": true,
    "peersExchange": true
  }
}
```

**Description**
`enabled` - включает и выключает сервис glagold, необходимый для тандемности
`peersExchange` - поможет модулю сообщить о себе колонке, если он ее видит, а колонка его нет.
