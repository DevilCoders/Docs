# Доступы

Каждый сервис в L3 manager относится к одному из сервисов [ABC](https://abc.yandex-team.ru/). По умолчанию создавать и редактировать сервис может только сотрудник, у которого есть роль «Ответственный за L3» или «Системный администратор» в данном ABC-сервисе.

Вы также можете выдать ограниченный доступ к вашему сервису в L3 manager для сотрудников из других подразделений, если у них есть роль «Ответственный за L3» или «Системный администратор» в одном из ABC-сервисов.

Чтобы выдать доступ к вашему сервису в L3 manager:

1. Убедитесь, что сотрудник, которому вы выдаете доступ, имеет роль «Ответственный за L3» в своем ABC-сервисе.
    
1. На странице вашего сервиса в L3 manager нажмите кнопку **Manage roles**→**Add new role**. Будет открыта форма выдачи прав в вашем сервисе:

    ![image](../_assets/manage-roles.png =600x178)

1. - В поле **ABC** выберите ABC-сервис, сотруднику которого вы хотите выдать доступ к вашему сервису L3 manager.
    - В поле **Role** выберите права для сотрудника:
    
    - update RS groups — изменять список рилов.
    - deploy service — деплоить конфигурации.
    - edit VS for service — изменять настройки виртуальных серверов.
    
1. Сохраните изменения кнопкой **Add**.
