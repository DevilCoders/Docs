# Терминология

Анонс

: Объявление сервиса доступным в сети.

Виртуальный сервер

: Составной элемент сервиса, описывающий соответствие между:
- Внешними IP-адресом, портом и протоколом сервиса.
- Списком L3 балансеров.
- Рилами, на которых запущено приложение.

Деплой конфигурации

: Процесс доставки конфигурации сервиса до L3 балансеров.

Конфигурация

: Список виртуальных серверов и настроек L3 manager для доступа к этим серверам. В один момент времени у сервиса может быть только одна активная конфигурация.

Сервис

: Приложение, работающее на серверах Яндекса и принимающее пользовательский трафик. Основная сущность в L3 manager.

Рил

: Real server, сервера в одном из ДЦ Яндекса, на которых запущены копии вашего приложения. Рилы могут быть указаны адресами хостов, группами Кондуктора, Gencfg или RT.

Флапающий рил {#flapping-reel}

: Рил, который регулярно меняет состояние с рабочего на нерабочий и обратно. Такое поведение может быть вызвано, например, отказывающим диском. Флапающий рил может приводить к постоянному снятию с анонса и анонсированию сервиса.
