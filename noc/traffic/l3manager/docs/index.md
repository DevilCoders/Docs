# Введение

**Документ содержит** описание и инструкции по работе с L3 manager — сервисом для управления конфигурациями L3 балансеров.

**Целевая аудитория документа** — разработчики, которым необходимо управлять трафиком пользователей и обеспечивать доступность своих сервисов.

**Документ не содержит** описания [API L3 manager](https://l3.tt.yandex-team.ru/api.html).

Пожелания и замечания к документу пишите в тикет [DOC-4468](https://st.yandex-team.ru/DOC-4468).

## Структура документа {#section_h3k_kr5_scb}

- [Общие сведения](concepts/intro.md) — описание принципов работы L3 manager.
- [Доступы](concepts/accesses.md) — процесс получения и выдачи прав в L3 manager.
- Операции — инструкции по [созданию](concepts/quickstart.md) и [удалению](concepts/service-delete.md) сервисов.
- [Логи и мониторинг нагрузки](concepts/logs.md) — дополнительные инструменты для наблюдения за сервисами в L3.
- [Решение проблем](concepts/troubleshooting.md) — описание ошибок и способов борьбы с ними.
- [Терминология](concepts/terms.md) — глоссарий документа.

О выпущенных версиях документа читайте в разделе [Релизы](concepts/releases.md).
