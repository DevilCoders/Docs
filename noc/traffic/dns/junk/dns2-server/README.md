dns2-server - dns сервер, который может потреблять различные источники записей
например, статические из файлов конфигураций или json endpoint установленного
формата так чтобы отвечать ими. Опционально форвадить остальные запросы в указанный
upstream, выступая авторитетным источником для ответов. Предполагается использовать
в качестве одного из backend'ов конфигурации dnsdist для VIEW UA/WORLD?
