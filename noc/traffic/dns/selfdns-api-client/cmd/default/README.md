* selfdns-client использует в качестве default plugin для вычисления адресов
  ip4, ip6 а также hostname поставляемый с selfdns-client пакетом shell 
  скрипт (до версии 0.2.12). Этот shell скрипт имеет ряд проблем с проверкой
  условий, в которых он запускается, в частности, некоррктная работа по определению
  ip4 адресов, see [2] и связанные с ним.

  [2] https://st.yandex-team.ru/TRAFFIC-11604 

* эта версия selfdns-client default plugin пытается корректно вычислить все
  значения, в случае если какие-то условия нарушаются (например у сигнальных
  fqdn нет A/AAAA записей, или они недоступны для resolve'а и вычисления
  ip route'ов), default plugin заканчивает свою работу с ошибками и 
  сам selfdns-client при этом в сторону сервера ничего не отправляет.
