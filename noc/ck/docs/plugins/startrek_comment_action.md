# startrek_comment - добавить комментарий к задаче в startrek

## Параметры

- `issue` - задача в startrek
- `text` - текст комментария
- `summonees` - список логинов пользователей для призыва в тикет вместе с комментарием; поддерживает две дополнительные
  переменные шаблона `{{ creator }}` (автор задачи) и `{{ assignee }}` (исполнитель задачи); если в списке логинов для
  призыва указан исполнитель задачи `{{ assignee }}`, но при этом исполнитель задачи ещё не назначен, то (очевидно)
  этот элемент списка логинов будет проигнорирован

## Примеры

Пример с использованием переменных шаблонов:

```yaml
name: startrek-comment-example
actions:
  - startrek_comment:
      issue: "{{ startrek_issue }}"  # тут подставится та задача, с которой была создана джоба
      # В тексте комментария можно использовать все доступные переменные.
      # Можно использовать средства YAML для описания многострочного комментария
      text: |-
        Это комментарий к джобе на устройстве из ДЦ {{ device.dc_code }} ({{ device.datacenter_rt_tag }})
        Комментарий многострочный
      summonees:
        - "{{ creator }}"  # призвать автора задачи
        - "{{ assignee }}"  # призвать исполнителя задачи (если назначен)
```

Пример с явным указанием параметров:

```yaml
name: startrek-comment-example
actions:
  - startrek_comment:
      issue: NOCDEV-4898
      text: Обрати внимание на эту задачу
      summonees:
        - azryve
        - vladstar
        - azatkurbanov
```
