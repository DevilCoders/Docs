# Пресеты

Для удобства создания учений предусмотрена возможность создавать пресеты.

Пресет имеет следующие поля:

* ***Projects*** – список макросов проектов, которые мы собираемся закрывать
* ***Location*** – датацентры, на которых проходят учения
* ***Duration*** – длительность учений (в формате hh:mm на фронте и в секундах на сервере)

{% note info %}

На сервере ***duration*** интерпретируется в секундах

{% endnote %}

* ***Exclude*** – список макросов, хостнеймов или IP-адресов, исключённых из учений
* ***CloseFrom*** – список макросов, хостнеймов или IP-адресов, который если не пуст, то учения проходят только на хосты
  из списка
* ***Username*** – создатель пресета (устанавливается автоматически)

Для того чтобы с [главной страницы учений](https://firewall.yandex-team.ru/drills) перейти
на [страницу пресетов](https://firewall.yandex-team.ru/drills/presets), нужно нажать на кнопку ***View presets***
в правом верхнем углу.

![go_to_presets](../images/go_to_presets.png)

Чтобы вернуться назад на [главную страницу](https://firewall.yandex-team.ru/drills), нужно нажать на кнопку
***View drills***, появившуюся на месте предыдущей.

![go_to_drills](../images/go_to_drills.png)

На [странице пресетов](https://firewall.yandex-team.ru/drills/presets) доступна фильтрация по первым трём полям и
по создателю (по умолчанию Вам показываются только пресеты, созданные Вами). 
Также там можно отсортировать пресеты по любому полю, нажав на соответствующую клетку шапки таблицы.

В колонках ***Exclude*** и ***CloseFrom*** написано лишь количество макросов в соответствующих полях. Однако,
при наведении на них курсора, все макросы перечислены во всплывающем окне. Там же этот список можно и скопировать.

![macro_popup_preset](../images/preset_manage/macro_popup_preset.png)

{% note info %}

Во всплывающем окне макросы в целях экономии места разделяются запятой, однако при копировании они будут разделены
переносом строки, что обеспечит возможность потом их вставить в соответствующие поля форм создания/редактирования
в правильном формате

{% endnote %}
