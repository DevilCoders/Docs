### Тулзы для настойки мониторига

В будущем должны стать частью сервиса и выполнятся автоматически

#### alertmaker.py
Тулза для создания агрегатов в juggler по каждому проверяемому номеру.
Для работы требует в переменной окружения `JUGGLER_TOKEN` наличие OAuth токена
и установленного https://juggler-sdk.n.yandex-team.ru

#### solomon.py
Тулза для создания алертов в solomon. Попытка хранить код алертов в VCS.
Для изменения кода алерта нужно пробовать запускать с разными версиями конфига какждого типа алертов,
т.к. не умеет синхронизировать текущую версию из solomon
Для работы требует в переменной окружения `SOLOMON_TOKEN` наличие OAuth токена.
