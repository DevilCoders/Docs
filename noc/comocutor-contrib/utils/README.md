### soft_modernizer.py
Обновление софта на свитчах. Скачивает и устанавливает софт, но не перезагружает. В данный момент поддерживаются только некоторые хуавеи.
Так как считается что патч применять безопасно, то патч обновляется без перезагрузки.
Если софт или патч новее, то ничего не делаем.

Пример использования:
```bash
$ ./soft_modernizer.py '{$cn_ce6850-ei-test}'
ce6850-ei-test.yndx.net: firmware is actual
```
```bash
./soft_modernizer.py myt6-s15 myt6-s16 -p
myt6-s16: need to update from VRPVersion<V100 R5 C10> to VRPVersion<V200 R2 C50>
myt6-s15: startup firmware is actual. Reboot is needed.
myt6-s16:   6%|███▌                                                       | 6/100 [00:16<04:13]

```
Доступные опции:

```-n``` - не обновлять софт, а только показать где нужно обновить.

```-p``` - включение отображение шкалы загрузки.

```-d``` - включение отладочного режима.

В аргументах можно указывать предикат RT или список хостов.
