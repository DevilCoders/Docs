## Аутентификация и Авторизация
### Аутентификация
Для использования Александрии необходимо пройти процесс аутентификации.
* OAuth [token](https://oauth.yandex-team.ru/authorize?response_type=token&client_id=f64118e511274790a201e2b02a3687e9)
* SessionID (cookie при заходе на '.*yandex-team.ru')

Гранты для экземпляра web-сервера выданы только для сетей:
* `_NOCDEV_ALEXANDRIA_PROD_` 
* `_RT_TEST_NETS_`  

### Авторизация
В v1 права на запись выдаются только пользователям, перечисленным в настройках серверного приложения.
В v2 ролевая модель плоская, у всех одинаковые права.
