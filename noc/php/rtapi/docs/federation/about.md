# GraphQL Federation

GraphQL Federation - это композиция нескольких GraphQL API (subgraphs) в единый API Gateway (Supergraph Gateway).

![Высокоуровневое представление GraphQL Federation](_images/img1.png)

Команда Racktables планирует исследование этой темы и MVP в Q2-Q3 2022.
