# Быстрый старт

Начать пользоваться GraphQL достаточно просто.

Для начала перейдите по [ссылке]({{ urls.invapi-ro }}) в web-интерфейс read-only.

В левом верхнем окошке можно писать query:
```graphql
query {
  ...
}
```

Например, напишем запрос выдающий информацию о устройствах:
```graphql
query {
  object(code: "{многодэшка}") {
    id
    fqdn
    breed
  }
}
```

Вывод:
```
{
  "data": {
    "object": [
      {
        "id": "31474",
        "fqdn": "iva-e1.yndx.net",
        "breed": "vrp85"
      },
      {
          ...
      }
    ]
  }
}
```

Давайте разберем запрос:
- `object(code: "{многодэшка}")` - запрашиваем объект, с rackcode равным `{многодэшка}`. Также можно укать другие поля: id, ids, asset, serial, search.
- id, fqdn, breed - поля, выводимые в ответе

{% note info %}

Можно воспользоваться подсказками в этом поле, для этого:
- в Windows нажимаем `ctrl` и высвечиваются подсказки
- в MAC нажимаем `option + space`
- в Linux нажимаем... у кого Linux, тот поймет что жать

{% endnote %}
