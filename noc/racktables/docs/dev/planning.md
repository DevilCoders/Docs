# Планирование и Организация

## Дашборды в Startrek
Планирование текущих задач и сроков по стриму Inventory на дашбордах:  
**Racktables**: [Startrek Dashboard Racktables](https://st.yandex-team.ru/dashboard/38723)  
На этом дашбоде мы планируем все задачи Inventory по кварталам и месяцам, чекаем статусы текущих и просроченных задач  


**InvAPI**: [Startrek Dashboard InvAPI](https://st.yandex-team.ru/dashboard/66548)  
Тут все задачи только с компонентой `InvAPI`  


**Alexandria**: [Startrek Dashboard Alexandria](https://st.yandex-team.ru/dashboard/54201)  
Тут все задачи только с компонентой `Alexandria`  


**Racktables (Backlog Refinement)**: [Startrek Dashboard RT Backlog](https://st.yandex-team.ru/dashboard/61777)  
На этом дашборде просматриваем `Backlog` и новые запросы в `_project_rt`, приоритизируем задачи, раскидываем по приоритетам
