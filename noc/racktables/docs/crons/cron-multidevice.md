## Список мультидевайсных Кронов и из назначения

## Что это
Мультидевайсные кроны предназначены чтобы ходить по списку устройств (обычно предикат) и выполнять какую-то полезную работу, документирование или обновление.

### AutoSyncPorts
Предназначен для автоматического обновления портов в РТ  
Особенности:
  - Обновляет только физические порты
  - Документирует в РТ
  - работает по предикату `[работает auto-sync-ports]`

### AutoLLDPLink
Предназначен для автоматического документирования линковки по LLDP в РТ
Особенности:
  - работает по предикату `[работает auto-lldp-link]`
