### База знаний по диагностике и проблемам для дежурных NOCDEV

Каждый дежурный записывает в подходящую директорию продукта типовые проблемы с которыми сталкивается. База в процессе формирования и сейчас находится на стандии эксперимента. Документация архитектуры внутренних сервисов должна жить в собственных репозиториях.

Основной раздел документации NOC находится по [ссылке на портале NOCDOC](https://docs.yandex-team.ru/nocdoc/)
