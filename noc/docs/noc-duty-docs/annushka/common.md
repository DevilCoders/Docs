## Методические указания по поиску неисправностей
### Ошибка деплоя или получения конфига

Убедись что comocutor последней версии и не забудь приложить версию comocutor из ```pip show comocutor``` в тикет.

Для сбора дебага нужно запустить аннушку на один хост с дебагом:

```bash
ann --log-level debug2 ...
```
<span style="color:red">Перед отправкой дебага убедитесь что там нет вашего пароля!</span>

Дополнительный дебаг лежит в deploy/DATE/HOSTNAME_TS.
