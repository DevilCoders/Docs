# Процедура обновления генератора конфигураций

В этом разделе рассмотрим процедуру обновления генератора в ann.

Все генераторы располагаются в папке `annushka/generators` проекта annuska. 

В NOC было принято решение о внедрении правил по обновлению генераторов:
- Merge Request в master ветку проекта возможен только при наличии номера тикета в названии Merge Request. Допустимые варианты:
  - NOC-XXXXX
  - \[NOC-XXXXX\]
- Производится Review изменений сотрудником из списка приведенного в `owners.py`

{% note info "Причины появления правил" %}

Внесение дополнительных ужесточающих правил обновления генераторов связано с тем, что конечная цель - автоматизация внесения измененй на сети. Это приводит к дополнительным рискам "завалить" сразу большое число устройств. Мы защищаемся от этого путем тестирования и дополнительной валидации изменений.

Также связка с тикетом нужна для отслеживания "докатки" изменений на сети автором этих изменений. Это особенно актуально для unsafe diff.

{% endnote %}
