# 11. Условия жизни для сервисов во VLA2
Большинство изменений, которые мы планируем сделать во VLA2, относятся только к внутренней топологии и логике, поэтому напрямую сервисов это не коснется. Большинство пунктов постепенно внедряются последние годы и абсолютно не новы, с точки зрения потребителей:
- L3 на порт - по 4 адреса (2BB + 2FB) с Project\_ID на ToR
- Фабрика ДЦ только IPv6, отсутствие IPv4
- Ликвидация Vrf - все серверные сети внутри фабрики находятся в GRT
- Межсерверное взаимодействие - через HBF
- Скорость подключения серверов: 25 Gbps
- Отсутствие переподписки между модулями и между зданиями
- _В финальной конфигурации топологии ДЦ или при выполнении необходимых условий_
