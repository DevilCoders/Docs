# 3. Топология сети

Верхнеуровнево, в существующей сети возможно выделить 4 составных элемента:
1. Сеть датацентров. Служит для создания внутренней связности между различными сущностями в ДЦ
2. Магистральный сегмент сети (Backbone). Основной задачей узлов магистрального уровня является агрегирование и транспорт трафика между датацентрами и точками присутствия
3. Точки присутствия (Border). Необходимы для обеспечения внешней связности сервисов Яндекса
4. Офисный сегмент сети (не рассматривается в контексте документа)
