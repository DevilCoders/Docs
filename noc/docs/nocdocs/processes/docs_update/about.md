## Введение

Какие проблемы мы решить общим порталом документации:
- общее информационное пространство для сотрудников Отдела сетевой инфраструктуры
- поддержание документации в актуальном состоянии
- быстрее получать требуемую документацию
- обеспечить прозрачный процесс написания и обновления документации


## Правила ведения документации отдела

Текущие договоренности по ведени документации в Отделе сетевой инфраструктуры:

1. Процессы отдела, описания взаимодействий между группами отдела, межгрупповые инструкции размещаются на [портале документации NOC Docs](https://docs.yandex-team.ru/nocdoc).
1. Внутренние инструкции групп, личные записи, драфты документации можно вести как в docs, так и на wiki. При этом на общий портал документации необходимо добавить актуальную ссылку на кластер с внутренней документацией подразделения.
1. За актуализацию документацию подразделения отвечают руководители групп и подразделений. 
1. Для перевода задачи в очереди NOCTODO в статус "Закрыт" необходимо провести актуализацию соответствующей документации.

{% note warning %}

Проект с документацией Отдела сетевой инфраструктуры находится в репозитории в Акрадии по ссылке [https://a.yandex-team.ru/arc/trunk/arcadia/noc/docs/nocdocs](https://a.yandex-team.ru/arc/trunk/arcadia/noc/docs/nocdocs).
