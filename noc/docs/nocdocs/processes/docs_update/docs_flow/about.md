# Workflow ведения документации в docs

Всячески приветствуется добавлять документацию в общий репозиторий документации в [Аркадии](https://a.yandex-team.ru/arc/trunk/arcadia/noc/docs/nocdocs).

Если вы хотите что-то обновить или добавить на портал, но у вас не получается, то администраторами портала являются:
- [Воловик Вадим](https://staff.yandex-team.ru/vdvolovik)
- [Савин Андрей](https://staff.yandex-team.ru/korxif)

## Если вы хотите обновить документацию на портале

Вы можете сделать это самостоятельно используя либо web-интерфейс, либо через IDE. Соответствующие инструкции:
- [Обновление через web-интерфейс](https://docs.yandex-team.ru/nocdoc/processes/docs_updatehowto/howto/update_via_web/about)
- [Обновление через Arc](https://docs.yandex-team.ru/nocdoc/processes/docs_update/howto/update_via_arc/about)

Общий workflow следующий:
- Заводим задачу в ST в очереди [NOCDOC](https://st.yandex-team.ru/nocdoc/). Можно сразу через [форму](https://st.yandex-team.ru/createTicket?queue=nocdoc)
- Вносим изменения любым удобным для вас способом
- Делаем Pull Request, привязывая его к тикету NOCDOC
- Дожидаемся завершения Pull Request
- Успех!
