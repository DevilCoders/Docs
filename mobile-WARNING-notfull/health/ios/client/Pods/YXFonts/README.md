# YXFonts

Набор фирменных шрифтов Яндекса.
Шрифты поставляются podspec'ами. Например, можно подключить следующие YSText
```
  pod 'YXFonts/YSText/Regular'
  pod 'YXFonts/YSText/Medium'
  pod 'YXFonts/YSText/Bold'
  pod 'YXFonts/YSText/Light'
  pod 'YXFonts/YSText/RegularItalic'
  pod 'YXFonts/YSText/MediumItalic'
  pod 'YXFonts/YSText/BoldItalic'
  pod 'YXFonts/YSText/LightItalic'
```

**Замечание:** если подключить YSText целиком через `pod 'YXFonts/YSText'`, то подтянутся все 8 шрифтов
