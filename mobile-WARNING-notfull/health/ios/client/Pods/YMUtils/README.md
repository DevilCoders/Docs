# YMUtils

To run the example project, clone the repo, and navigate to "Example" folder and open YMUtils.xcworkspace

## Installation

YMUtils is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod ‘YMUtils’, '= 1.0.0'
```
