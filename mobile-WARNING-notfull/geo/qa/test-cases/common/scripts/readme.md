### Scripts
#### `isnstall_hooks.sh`
Устанавливает прекоммитные проверки для `git`.

#### `arc_config_setup.sh`
Устанавливает прекоммитные проверки для `arc`. 
Добавляет в arc алиасы:

##### Создание ветки для работы по тикету
`arc new MOBNAVITESTING-666` создаст на базе самого свежего trunk ветку `MOBNAVITESTING-666` и переключится на неё. Имена веток валидируются: должны являться номерами тикетов из очередей MOBNAVITESTING, MAPSMOBILETEST.

##### Создание PR со всеми изменениями
`arc review-all`

1. Добавит все изменения в индекс
2. Закоммитит изменения в текущую ветку
3. Сделает ребейз текущей ветки на самый свежий trunk
4. Запушит изменения на сервер
5. Создаст публичный PR из этой ветки в trunk

##### Доделывание правок по ревью
`arc update-all`

Сделает всё то же, что и `review-all`, только не будет создавать новый PR, а просто допушит изменения в текущую ветку.

##### Удаление лишних веток
`arc cleanup` удалит из локального репозитория все ветки которые были полностью замержены в trunk. 

##### Монтирование аркадии после перезагрузки
`arc remount` примонтирует Аркадию в `~/arcadia`

#### `npm_install.sh`
Максимально правильная установка Node.js для работы с palmsync.

- Установит `nvm` из homebrew
- Пропишет инициализацию `nvm` в `.bash_profile` и `.zshrc`
- Установит в `nvm` нужную версию ноды
- Поставит alias `nvm default` на нужную версию ноды
