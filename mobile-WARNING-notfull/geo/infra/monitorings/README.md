# Мобильные мониторинги Яндекс.Карт и Навигатора

Чатик в Slack **#mobile-backend-infra**.

* Сервак, который подготавливает данные для мониторингов и пушит их в Голован [deploy/README.md](./deploy/README.md)
* Конфиги дашбордов в Головане: [yasm/panels/README.md](./yasm/panels/README.md)
* Конфиги алертов в Головане: [yasm/alerts/README.md](./yasm/alerts/README.md)

## Общая схемма работы мобильных мониторингов

* На сервере крутятся таски, которые периодически выгружают данные из ClickHouse или AppMetrica и пушат их в отдельные сигналы в Головане.
* В Головане настроены дашборды и алерты, которые смотрят на сигналы.
* При применении конфига в Головане алерты из Голована прорастают в Juggler.
