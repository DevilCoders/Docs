# Как настроить?
Выполнить команду
`ln -s ${ПУТЬ_ДО_АРКАДИИ}/mobile/market/android/app-market/arc-hooks/arcconfig ~/.arcconfig` – создать сиблинк
или
`cp ${ПУТЬ_ДО_АРКАДИИ}/mobile/market/android/app-market/arc-hooks/arcconfig ~/.arcconfig`  – скопировать конфиг в домашнюю директорию

# Как закомитить в обход arc-hooks?
Использовать параметр --no-verify
