## Генератор патронов для droideka

Собирает логи балансера из `production` окружения `droideka` и генерирует файл патронов для Яндекс.Танка.

#### Параметры
`-d`, `--date` - дата (в формате `YYYY-MM-DD`) за которую брать логи для патронов, по-умолчанию берётся вчерашний день.

`-f`, `--file` - имя файла, в который сохрянать патроны, по-умолчанию `ammo-<date>.txt`

`-u`, `--upload` - флаг, при указании которого патроны будут автоматически залиты в [Lunapark](https://lunapark.yandex-team.ru/) под названием `Droideka ammo [<date>]`.

#### Пример запуска
```ya make && ./ammo_gen --date 2021-01-30 --file ammo.txt --upload```

или без параметров

```ya make && ./ammo_gen```

### Запуск нагрузочного тестирования
- Зайти на страницу [повтора стрельбы](https://lunapark.yandex-team.ru/2655496/repeat);
- выбрать залитый файл патронов по имени (или залить вручную, если `--upload` не был задан);
- задать нужное название, описание и схему нагрузки;
- нажать кнопку "Огонь".
