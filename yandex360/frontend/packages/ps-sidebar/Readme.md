##Пакет виджетов

Публикуется во внутрений NPM в пакет `@ps-int/ps-sidebar`. Публиковать пакет обязательно нужно командой
```npm run package <version>```

Если публикуете беты, то используйте команду `package-beta`:
```npm run package-beta <version>-beta.0```

Посмотреть компоненты можно, запустив локальную страницу с примерами командой `npm start`.

Компоненты разрабатываются на TS, для поддержки проектов на JS перед публикацией компилируются в отдельную папку `dist`.
