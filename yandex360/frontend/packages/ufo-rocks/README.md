# rocks
Общие компоненты для диска и DV

##Темизация
Тема для клиента, паблика и заметок отнаследована от темы лего, собирается тулзой [themekit](https://github.com/bem/themekit)
Пересобрать тему - `npm run themekit`. Дополнительные токены определены [тут](lib/components/lego-components/tokens/disk.tokens.yml)

##Страницы с примерами
```bash
npm install 
npm start
```

Деплой на gh-pages
```bash
npm run gh-pages
```
