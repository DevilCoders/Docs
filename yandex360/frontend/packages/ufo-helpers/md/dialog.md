## getClientResourceDialogUrl(dirname, path, dialog) ⇒ <code>string</code>
Возвращает url на указанный диалог для указанного ресурса (файла)

**Kind**: global function  

| Param | Type | Description |
| --- | --- | --- |
| dirname | <code>string</code> | путь до папки, в которой находится файл |
| path | <code>string</code> | путь до файла |
| dialog | <code>&#x27;select&#x27;</code> &#124; <code>&#x27;slider&#x27;</code> | название диалога |
