## shortenName(first, last, ellipsis) ⇒ <code>string</code>
сокращает строку в в формате ssss...ss

**Kind**: global function  
**Returns**: <code>string</code> - сокращенная строка  

| Param | Type | Description |
| --- | --- | --- |
| first | <code>number</code> | количество первых симоволов (4 по умолчанию) |
| last | <code>number</code> | количество последних символов (2 по умолчанию) |
| ellipsis | <code>string</code> | разделитель (по умолчанию ...) |
