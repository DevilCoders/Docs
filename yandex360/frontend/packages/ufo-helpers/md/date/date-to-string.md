## /lib/date/date-to-string ⇒ <code>String</code>
Форматирует переданное значение даты/времени в указанный формат.
Локализованные строковые значения должны быть переданы через аргументы.
Формат по умолчанию: DD.MM.YYYY HH:mm

| Токен | Результат |
| ----- | --------- |
| M     | 1 2 ... 11 12 |
| MM    | 01 02 ... 11 12 |
| MMMM  | январь февраль ... ноябрь декабрь (именительный падеж, массив имён должен быть передан 3 аргументом) |
| mmmm  | января февраля ... ноября декабря (родительный падеж, массив имён должен быть передан 4 аргументом) |
| D     | 1 2 ... 30 31 |
| DD    | 01 02 ... 30 31 |
| E     | Вс Пн Вт ... Сб |
| YY    | 01 02 .. 16 17   |
| YYYY  | 2001 2002 ... 2016 2017 |
| H     | 1 2 ... 22 23 |
| HH    | 01 02 ... 22 23 |
| h     | 1 ... 12 |
| p     | am/pm |
| m     | 1 2 ... 58 59 |
| mm    | 01 02 ... 58 59 |
| s     | 1 2 ... 58 59 |
| ss    | 01 02 ... 58 59 |

**Returns**: <code>String</code> - Форматированная строка даты  

| Param | Type | Default | Description |
| --- | --- | --- | --- |
| date | <code>Date</code> &#124; <code>Number</code> |  | Дата или число милисекунд, прошедших с 1 января 1970 года GMT+0 |
| [format] | <code>String</code> | <code>&#x27;DD.MM.YYYY HH:mm&#x27;</code> | Формат, в соответствии со схемой выше |
| [monthsNominative] | <code>Array</code> |  | Массив локализованных имён месяцев в именительном падеже |
| [monthsGenitive] | <code>Array</code> |  | Массив локализованных имён месяцев в родительном падеже |
| [weekDays] | <code>Array</code> |  | Массив локализованных имён дней недели |
