##Пакет с общими компонентами персональных сервисов

Публикуется во внутрений NPM в пакет `@ps-int/ps-components`

Посмотреть компоненты можно, запустив локальную страницу с примерами командой `npm start`.

Компоненты разрабатываются на TS, для поддержки проектов на JS перед публикацией компилируются в отдельную папку `dist`.

Также добавлена отдельная сборка в папку `dist_umd` - туда собираются компоненты, которые нужно использовать
на сервере проектов, у которых нет полноценного серверного рендеринга.
В эту сборку попадают только компоненты, вручную указанные в файле `webpack.config.js`.

###Содержание

####ps-header (`src/ps-header`)
Общая шапка персональных сервисов, объединённых в виртуальное рабочее место "Почта 360".

[Подробнее](./src/ps-header/Readme.md)

####share-dialog (`src/share-dialog`)
Компонент диалога поделения файлом с возможностью делиться документами на редактирование.

Переводы диалога можно обновить, выполнив команду `npm run tanker:share-dialog`

####notifications (`src/notifications`)
Redux-часть (reducers, actions, sagas) для очереди нотифаек

####confirm-dialog (`src/confirm-dialog`)
Простой диалог подтверждения - с 1 или 2 кнопками.
Умеет переводить фокус на себя при открытии
