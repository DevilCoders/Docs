# touch-components – общие компоненты для [QUINN](../../services/quinn) и [MOBMAIL2](../mobmail2).

## Внимание

Тут не работают [class fields](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Classes/Class_fields). Заставить их работать может помешать [mobmail2](../mobmail2)

Если нет интерета, ставить с `PUPPETEER_SKIP_CHROMIUM_DOWNLOAD=1`.
