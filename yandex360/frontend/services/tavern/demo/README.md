# Демостенд таверны
Сервис для тестирования интеграции таверны в хост-сервис. Это не сама таверна, а эмуляция хост-сервиса.


# Разработка
Собираем демо-приложение
```bash
npm i
```

[опционально] Линкуем портал (реакт-компонент для встраивания таверны) если хотим его разрабатывать.
```bash
npm i
```

Запускаем таверну
```bash
npm start
```
 - aдрес демостенда – `https://XXX.taverndemo-dev.mail.yandex.ru/`

Где `XXX` имя вашей qyp машины. Управлять тем с какого хоста грузится сама таверна можно тут – `./features/app/components/App/App.tsx`
