# Source files for Yandex.Mail's styles

## Lib

Any mixins, libraries and other stuff that is not rendered at all when included are placed at `lib` directory.

All the actual styles for actual pages are placed into `hubs` directory, divided into different hubs.

“Index” files, which are compiled to `.css` are placed into `styl` directory, each named after corresponding hub, like `boot.styl`, `common.styl`, etc.

Files for themes are placed into `themes` directory.
