**Как обновлять локали и добавлять новые кейсеты**

1) Добавить в конфиг .tanker.json новый кейсет
2) Добавить ключи в tanker.yandex-team.ru в проект mail-liza
3) В папке locales/ запустить `npm run update-locales`

**Соглашения по разработке**

При редактировании/создании новых react компонентов предпочтительно использовать
именно новый способ локализации через LangProvider + I18N компоненты.
Подробнее [здесь](https://github.yandex-team.ru/Daria/liza/wiki/Соглашения-по-разработке#Локализация-компонентов)
