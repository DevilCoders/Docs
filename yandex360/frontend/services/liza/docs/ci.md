# CI

Сейчас к репозиторию подключен Trendbox CI, который триггерится на различные события гитхаба.
Настройки в `.trendbox.yml`.

## Запуск сборки

TLDR: `npm run package 1.2.3 DARIA-12345`

Для запуска сборки используется скрипт, который специальным образом
готовит коммит с мета-информацией, которую сможет подхватить CI
и запустить сборку с нужными параметрами.

*Простой случай:*

Достаточно указать версию и тикет (порядок не важен).
Например, `npm run package 1.2.3 DARIA-12345`.

*Другие параметры:*

Если использовать дополнительные параметы, нужно добавить `--` при запуске скрипта, например, `npm run package -- -j 1.2.3 DARIA-12345`.

 * `-r` или `--rebase` — перед пушем поребейзиться от `origin/master`.
 *  `--unsafe` — не делать проверки на подмерженность мастера
 * `--unsafe-latest-release` — Не делать проверку на то, что последний релиз в мастере
 * `--unsafe-master` — не делать проверку на то, что мастер смёржен в текущую ветку
