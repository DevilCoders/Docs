# Поддерживаемые браузеры

Все сказанное относится к бете. Версии браузеров, которые поддерживают без префиксов.

## Список

* Chromium 29+ (Chrome 29+, Opera 17+, YandexBrowser 13.12+)
* Firefox 28+
* IE 11+
* Safari 8+ (все равно c префиксом)

## Что нам надо

### CSS calc

* Chrome 26+
* Firefox 16+
* IE 10+
* Opera 15+
* Safari 6.1+

### CSS flexbox

В том числе `flex-direction: reverse`
* Chrome 29+
* Firefox 28+
* IE 11+
* Opera 17+
* Safari 8+ (все равно в префиксом)
