# Установка и запуск тестов

## Установка

Для установки тестов, достаточно внутри проекта запустить команду:

```
npm install
```

Которая установит все необходимые зависимости, перечисленные в `package.json`


## Запуск

Для полной проверки, code style и тестов, необходимо выполнить:

```
npm test
```

Если вы хотите запустить только unit тесты необходимо, внутри проекта, выполнить команду:

```
npm run-script unit-test
```

## Примечание

Команды **test** и **unit-test** определены внутри `{PROJECT}/package.json`, секция **scripts**.
