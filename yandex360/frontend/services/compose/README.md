## Разработка

```bash
npm run prepare-dev
npm start
```

## Сборка

Нужен тикет в Daria.

```bash
npm ci # если ещё не
npm run package {{version}} {{ticket}}
```

Локально:
1. Обновляет версии `package.json`
2. Собирает и публикует пакет `@ps-int/mail-compose-client`
3. Ставит его в `server`
4. Коммитит все изменения
5. Пушит релизный тег

В ci:
1. Собирает и публикует sandbox-ресурс

Чтобы отдать в тестирование релиз, можно воспользоваться стендом на PR, который создаётся
на создание PR.
Однако, на нём нужно руками обновить ресурс композа (тип `MAILFRONT_COMPOSE`, в списке нужно выбрать
собранную версию пакета).
