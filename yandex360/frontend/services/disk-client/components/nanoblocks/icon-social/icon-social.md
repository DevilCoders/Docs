# Социальная иконка nb-icon-social

![](https://yadi.sk/i/gXu-GvdLZQeMP_XXL.jpg)

## Yate API

```
nb-icon-social({
    'icon': 'vkontakte'
    // xs - 16x16 | s — 25x25 | m — 28x28 | l - 32x32
    'size': 's'
    // 'default' | 'gray'
    'theme': 'gray'
    'rounded': true()
    'content': 'Привет!'
})
```

* 'icon'
    * 'vkontakte'
    * 'facebook'
    * 'odnoklassniki'
    * 'mail'
    * 'moimir'
    * 'instagram'
* 'type'
    * 'link' — иконка-ссылка
* 'size'
    * 's'  — 25x25 (по умолчанию)
    * 'm'  - 28x28
    * 'l'  — 32x32
    * 'xl' — 38x38
* 'theme'
    * 'default' (по умолчанию)
    * 'gray'
* 'rounded' — скругленная иконка, по умолчанию — нет.
* 'content'
