# icon-start-play

Рисует блок с иконкой плей, который растягивается на весь размер родителя, абсолютно поверх.
Содержит сложную логику ховера.

![](https://yadi.sk/i/HwAo5CsUdZWLb_XXL.jpg)

```
icon-start-play()
```

**Внимание**: родитель, поверх которого хочется растянуть блок с иконкой должен иметь значение `position` отличное от `static`.
