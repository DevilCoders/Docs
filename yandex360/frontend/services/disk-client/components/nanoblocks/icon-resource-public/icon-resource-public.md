## Иконки публичности ресурса

Публичности в определенном смысле может быть двух типов:

* published — есть ссылка на паблик

Блок отображает признак публичности ресурса в интерфейсе.

[Публичность](http://jing.yandex-team.ru/files/vitkarpov/2014-04-29_1657.png)

### Yate API

```yate
nb-icon-resource-public({
    published: true()
})
```

#### published

Указывает на публичность ресурса


### Stylus API

```
nb-icon-resource-public($sizeIcon)
```

#### $sizeIcon

Задает высоту и радиус скругления блока.
