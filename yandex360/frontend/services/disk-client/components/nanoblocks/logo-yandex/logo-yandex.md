## logo-yandex

Показывает наш замечательный логотип в SVG. XML вставляется прямо в DOM.

### Yate

```
nb-logo-yandex({
    'lang': 'ru' | 'en' | ..
    // логотип для ДДД
    'hostDDD': 'www.custom-host.ru'
})
```

Есть варианты русского и английского языков.
