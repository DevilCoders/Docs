# nb-welcome-screen-mobile

Рисует приветственные экран на мобильном, где промотируем установку мобильного приложения

```yate
nb-welcome-screen-mobile({
    'os': 'iOS' | 'Android'
    'locale': 'ru' | 'en' | 'uk' | 'tr'
    'urlDownloadApp': 'http://...'
    'urlDisk': 'https://disk.yandex.ru/'
})
```
