### Hermione JS

**Hermione тесты** - система тестирования верстки в режиме "черного ящика": в окружении, приближенному к реальному, запускается реальный браузер, открывается поднятая бета, выполняются какие-то пользовательские действия (нажать на кнопку, проскроллить страницу) и проверяется какое-то состояние (открылся попап, кнопка поменяла цвет).

Запустить тесты можно через консоль из нужного раздела:
```
TUS_TOKEN=<...> YANDEX_ENVIRONMENT=prestable ./node_modules/.bin/hermione -r plain --grep 'Авторизация' --retry 3 --base-url='https://disk8.dsp.yandex.ru'
```

Параметры:
- `TUS_TOKEN` - секретный токен для походов в [TUS](https://wiki.yandex-team.ru/test-user-service/), можно опустить если запускаются тесты с фиксированной авторизацией
- `YANDEX_ENVIRONMENT` - окружение, по дефолту `development`
- `r` - репортер тестов, по дефолту `flat` (отчет после всех тестов), `plain` - прогон в живом времени
- `retry` - ретраи, по дефолту их нет
- `base-url` - адрес стенда, дефолтные значения берутся из конфига гермионы, протокол указывать обязательно
- `grep` - если нужен конкретный тест/сьют

**Браузеры и платформы** для тестирования указаны в файле `@ps-int/ufo-hermione/browsers.js`
