Форму события можно предзаполнять get-параметрами, переданными при переходе на `/event`

| Параметр формы        | Параметр в URL            | Тип                     | Примечание |
| --------------------- |:-------------------------:| -----------------------:| -----------|
| Название встречи      | `name`                    | строка                  | urlencoded |
| Описание встречи      | `description`             | строка                  | urlencoded, не должно быть слишком большим во избежание переполнения url |
| Начало события        | `startTs`                 | ms с начала эпохи unix  | работает только совместно с `endTs` |
| Конец события         | `endTs`                   | ms с начала эпохи unix  | работает только совместно с `startTs` |
| Весь день             | `isAllDay`                | 0 _или_ 1               | |
| Видимость события     | `othersCanView`           | 0 _или_ 1               | 0 - видят только участники |
| Организатор           | `organizer`               | логин или email         | только на корпе |
| Участники             | `attendees` или `invited` | список email-ов или логинов | есть спецзначение `me()` для добавления себя |
| Переговорки           | `resources`               | список email-ов         | только на корпе |
