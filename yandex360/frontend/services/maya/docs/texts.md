Актуально для всех текстов в интерфейсе публичного и корпоративного календаря.

1. Ключи хранятся в Танкере в проекте [Calendar 2](https://tanker-beta.yandex-team.ru/project/calendar_2?branch=master).
Добавление и изменение ключей необходимо производить через интерфейс Танкера.
2. Затем отдать получившиеся тексты на русском - редакторам в очереди DOCHELP
([ссылка для создания тикета](https://st.yandex-team.ru/createTicket?queue=DOCHELP&_form=54613)), 
поставив исполнителем Катю Куненко (ke1e6@).

3. После согласования текстов редакторами - заказать переводы через [форму](https://st.yandex-team.ru/createTicket?queue=TRANSLATE&_form=13067) после того, как тексты на оригинальном языке прошли через редакторов.

Для выгрузки ключей из Танкера:
```bash
npm run tanker
```
Выгруженные изменения необходимо закоммитить (возможно, с использованием селективного коммита, поскольку коммитить стоит только ключи, относящиеся к текущей ветке).
