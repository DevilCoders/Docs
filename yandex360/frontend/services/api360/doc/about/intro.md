# Введение

[Яндекс 360 для бизнеса](https://admin.yandex.ru/) — это набор сервисов для работы, который включает корпоративные версии Яндекс Почты, Яндекс Диска и другие приложения.

Яндекс 360 для бизнеса позволяет управлять структурой организации посредством HTTP-запросов к REST API.

Документ описывает доступные операции API, форматы запросов и ответов. Чтобы сделать свой первый запрос, прочтите раздел [Доступ к API](access.md).

На данный момент API позволяет получать, добавлять и удалять информацию о следующих объектах:

- [подразделения](../ref/DepartmentService.html);

- [группы](../ref/GroupService.html);

- [сотрудники](../ref/UserService.html).
