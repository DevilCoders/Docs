# Версии API

Ниже приведены изменения API, от последних к первым.

### Июнь, 2022 {#june-2022}

- Появилась возможность удалять сотрудника из организации и группы.

- Исправлена ошибка с удалением всех сотрудников при редактировании группы.

### Март, 2022 {#version-1-0-0}

Добавлена возможность получать, добавлять и удалять информацию о следующих объектах:

- [подразделения](../ref/DepartmentService.html);

- [группы](../ref/GroupService.html);

- [сотрудники](../ref/UserService.html).
