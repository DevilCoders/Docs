# Husky hooks

Скрипт для запуска хуков husky в монорепе.

Husky установлен в качестве зависимости в корне монорепы и устанавливает git-хуки.
На данный момент подключены только `pre-commit` и `pre-push`, которые запускают
соответствующие скрипты в измененных пакетах:

* `pre-commit` — есть ли в изменения относительно HEAD.
* `pre-push` — есть ли дифф относительно апстрима или ветки `dev`, если апстрима нет, относящихся к сервису файлов.


## Поключение сервиса

* Устанавливать husky отдельно для сервиса не нужно.
* Пакет должен быть в списке пакетов лерны (можно проверить `lerna la`).
* В `package.json` сервиса добавить скрипты `pre-commit`, `pre-push`.
