# Автоматизация: юниты



_**Тут еще идет проработка и реализация**_

{% note info %}

trunk должен быть максимально стабильным и поэтому, код, который в него вливается, должен быть покрыт юнитами и гермиона-тестами

{% endnote %}

**Базовые положения по юнитам:**

* юниты пишутся разработчиками;
* отстутствие юнитов не блокирует передачу в тестирование;
* в CI настроена метрика, которая не позволит влить код в trunk без юнитов (еще прорабатывается)
