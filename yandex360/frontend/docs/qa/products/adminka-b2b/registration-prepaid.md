### Регистрация предоплатника на престейбле

* завести нового пользователя
* перейти по ссылке https://admin2.dsp.yandex.ru/ или другой актуальный стенд
* выбрать владельца будущей организации
* вручую ввести все реквизиты или предзаполнить данные (см. ниже)
* ввести данные карты и завершить регистрацию

{% note warning "Ручной ввод реквизитов" %}

* регистрировать организации можно, но важно указывать свои тестовые email'ы, номера телефонов и индексы
* тестирование проводим на престейбле. В проде проверяем только баги на предмет актуальности

{% endnote %}

{% list tabs %}

- Ручной ввод ревизитов

  **Как работает:**

    * вводим ИНН `1234567894` (для ИНН проверяется контрольная сумма, так что рандомный набор цифр может завершиться фэйлом)
    * вводим КПП `111111111` (все единицы)
    * вводим адрес регистрации `123444 тест` (набор цифр и текст может быть любым, но индекс должен быть равен 6 символам и после должен идти текст, иначе все сломается на валидации фронта)
    * вводим номер телефона `+71111111111`

- Предзаполнение реквизитов

  **Как работает:**

    * можно ввести реальный ИНН организации и сработает автозаполнение - это работает [kvSaaS](https://wiki.yandex-team.ru/jandekspoisk/saas/)
    * останется заполнить только телефон/email

## Создание 2+ организации

* из админки через селект выбора организации || на странице выбора организации (`/select-organization`) создать новую организацию
* новой созданной организации доступны **предоплатные тарифы**

{% note info "Про тарифы 2+ организаций предоплатников" %}

* если, хотя бы в одной организации владельца, включен постоплатный тариф, то все последующие созданные организации буду иметь постоплатные тарифы и не будет предоплатных
* в остальных случаях пользователю доступны предоплатные тарифы

{% endnote %}
