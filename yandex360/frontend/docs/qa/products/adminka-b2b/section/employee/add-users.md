# Добавление сотрудников

_Настоятельно рекомендуется сперва ознакомиться с [классификацией пользователей](https://docs.yandex-team.ru/psfront/qa/products/adminka-b2b/group-users)_


## Добавление портальными админами

Портальный админ может приглашать пользователей в организацию по ссылке-приглашению

**Как происходит процесс добавления:**

* админ генерирует ссылку-приглашение кликом на кнопку "+ Добавить сотрудника" в разделе "Сотрудники"
* копирует ссылку и отправляет ее тому, кого хочет видеть в данной организации (реципиенту)
* реципиент переходит по ссылке и принимает инвайт на вступление в организацию на соответствующем экране
* после принятия инвайта реципиента редиректит на экран выбора организации (`/select-organization`), а админ видит данного сотрудника в общем списке

{% note info "Важное про ссылку-приглашение:" %}

* ссылка-приглашение живет 10 минут. после истечения данного временного интервала, ссылка протухает и перестает принимать в организацию
* по одной ссылке можно добавить N-сотрудников - главное, чтобы они приняли инвайт в течение 10 минут
* новая ссылка генерируется на каждое новое открытие попапа добавления сотрудника, не нужно ждать протухания уже сгенерированной
* 2 раза (или более) один и тот же сотрудник не может вступить в организацию по одной или разными ссылками

{% endnote %}


## Добавление доменными админами

Они умеют не только добавлять пользователей по ссылке-приглашению (флоу полностью аналогичен), но и создавать новых пользователей вручную

**Как происходит процесс ручного добавления:**

* должен быть подтвержден основной домен в организации
* админ кликом на кнопку "+ Добавить сотрудника" вызывает форму с добавлением сотрудника
* в форме есть два таба - "создать аккаунт" и "пригласить" (это приглашение по ссылке)
* в форме "создать аккаунт" админ вводит все необходимые поля, как, например: ФИО (по отдельности), логин/пароль. Остальные поля опциональны
* после заполнения необходимыми данными пользователь жмет кнопку "Сохранить"
* пользователь создан в организации и под ним можно залогиниться в сервисах Яндекс 360
