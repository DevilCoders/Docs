# Оргструктура


## Общее описание

**Оргструктура** - это бесплатная фича, которая позволяет пользователям организовать иерархию своей организации с помощью распределения сотрудников по подразделениям и группам

Также, данная фича была критически важна для цели закрытия Коннекта, которая позволила безболезненно мигрировать пользователям из Коннекта в Яндекс 360, сохранив уже выстроенную иерархию

## Что может делать администратор

* добавлять пользователей как в общий пул сотрудников, так и сразу в подразделение
* мигрировать пользователей по подразделениям и группам
* создавать новые и редактировать текущие подразделения и группы
* назначать выбранных людей в подразделениях и группах руководителями соответствующей структуры
