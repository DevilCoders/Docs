# Редактирование сотрудников

_Настоятельно рекомендуется сперва ознакомиться с [классификацией пользователей](https://docs.yandex-team.ru/psfront/qa/products/adminka-b2b/group-users)_


## Редактирование портальных сотрудников

С **портальными сотрудниками** можно проделать следующие действия:

* сделать администратором организации / отобрать права администратора
* удалить из организации

{% note info "Доступ к интерфейсу админки" %}

* сотрудникам запрещен доступ к интерфейсу адмминки
* такая привилегия доступна только администраторам
* рубильником администратора текущий администратор как раз управляет доступом к интерфейсу определенным сотрудникам

{% endnote %}


## Редактирование доменных сотрудников

С **доменными сотрудниками** можно проделать следующие действия:

* проверить письма (тайник)
* редактировать
* изменить пароль
* редактировать алиасы
* сделать администратором организации / отобрать права администратора
* заблокировать
* удалить из организации
