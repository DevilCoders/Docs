# Структура карточки организации


Карточка организации состоит из:

* наименования организации
* если тариф подключен - отображается наименование тарифа + стоимость за мес/чел
* если тариф не подключен - отображается текст "Огранченная версия" + кнопка улучшить
* количество сотрудников в организации

{% note info "Важные тезисы про карточки" %}

* первой карточкой всегда отображается текущая организация
* отображаются только организации, где админ является плательщиком (например: пользователя пригласили админом в организацию - она не отображается на странице, если админ не находится в ней в текущий момент)
* не отображаются организации, где админ является рядовым сотрудником
* если админ не плательщик в текущей организации, то она отображается первой в списке организации + отображатся организации, где админ является плательщиком, если они есть

{% endnote %}
