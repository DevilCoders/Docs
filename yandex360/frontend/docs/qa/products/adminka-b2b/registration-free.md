# Флоу бесплатной организации


## Регистрация бесплатной организации без реквизитов

* завести нового пользователя
* перейти по ссылке https://admin2.dsp.yandex.ru/master-registration?productId=salespromo (ключевой тут параметр `productId`)
* нажать на кнопку "Продолжить"

Средиректились в админку, подключен бесплатный тариф

{% note info "Нюанс про беслпатный тариф" %}

* на самом деле, бесплатного тарифа, как отдельного продукта, не существует и billing360 не знает про такой тариф
* бесплатный тариф - это отсутствие любого другого подключенного платного тарифа

{% endnote %}
