# О сервисе


## Общее

**Админка b2b** - это продукт, который позволяет администраторам организаций управлять своими сотрудниками, заводить новые домены, покупать подписки, расширяющие функционал продукта и многое другое. Входит в состав `Яндекс 360 для бизнеса`

**Яндекс 360 для бизнеса** - это виртуальный офис для команды. В его состав входит корпоративная почта, календарь, облачное хранилище, редактор документов и сервисы для коммуникации
