# monkey-patching-extension
Расширение на хром которое должно искать перегруженные методы

## Установка
Чтобы установить расширение в chrome или Ya-browser зайдите на chrome://extensions в chrome или в browser://extensions
в Ya-browser, включить режим разработчика, а затем установить распакованное расширение и выбрать папку с этим экстеншеном.

## Использование
Зайти на интересующую страницу, нажать на иконку расширения, нажать на кнопку и наслаждаться результатом.
