# SST
## Server Side Tag
Серверный тег метрики. Что такое [серверный тег](https://developers.google.com/tag-platform/tag-manager/server-side)

## Установка
Для уставки и сбора программы нужен [nodejs](https://nodejs.org/en/)

В самом простом случае достаточно, находясь в корневой папке проекта, выполнить команду
```
npm ci
```

Если установлен [nvm](http://nvm.sh), можно использовать нужную версию ноды:
```
nvm install && nvm use 12
```
## Сборка
Сборкой тега является файл _build.tpl
Чтобы его получить после установки нужно выполнить команду
```
npm run build
```
