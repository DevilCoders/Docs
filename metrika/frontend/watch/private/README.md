# Как добавить приватную фитчу
1. В features.json прописать ее название в поле code, описание в поле desc, а так же указать в каких сборких следует ее использовать в поле include и самое главное указать папку / файл от куда нужно импортировать функцию инициализации относительно папки src.
1. В файле или папке нужно создать экспортируемую функцию initPrivateProvider. Эта функция будет выполнена 1 раз при подключении файла скрипта. Она должна вернуть функцию провайдер фитчи которая в свою очередь будет вызвана на каждый инстанс счетчика
