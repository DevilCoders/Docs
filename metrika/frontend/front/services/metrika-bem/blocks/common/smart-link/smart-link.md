## Блок для ссылок с "живыми" урлами

Проблема: у нас есть параметры, которые должны прокидываться в урле при переходе по страницам. Такие как filter, period, date1, date2, goal, ... . При этом параметры в урле могут меняться без перезагрузки страницы, соответственно, ссылки станут не актуальными.

Решение:

    // Ссылка на отчёт sites, все нужные параметры подставятся автоматически.
    // Предполагается, что в урле есть обязательные параметры, типа counter_id.
    {
        block: 'link',
        mix: {block: 'smart-link', js: {path: '/stat/sites'}},
        url: '#', // Ссылка будет генерироваться автоматически,
        content: 'Содержимое ссылки'
    }

    // Ссылка на целевой отчёт. Отчёт тот же, меняется только цель.
    {
        block: 'link',
        mix: {block: 'smart-link', js: {params: {goal: '100'}}},
        url: '#',
        content: ''
    }
