Скрипт autocreate.sh позволяет загрузить, выбрать бранч, собрать и запустить проект на сервере metrika-dev.

Скрипт принимает следующие аргументы:
./autocreate.sh [branch-name] [dest-dir]
 - branch-name - название (или часть названия) бранчи, которая будет собрана и запущена
 - dest-dir - каталог, в который будет загружен проект. По умолчанию - beta.
    !!! Каталог дял проекта устанавливается всегда относительно домашнего каталога текущего пользователя !!!

После запуска скрипта приложение доступно по адресу: http://beta.{user-name}.metrika-dev.yandex.ru/
