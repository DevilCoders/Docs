Многие почтовые клиенты позволяют стилизовать теги только инлайново, с помощью атрибута `style`. Но копипастить код не хочется. Поэтому можно писать html и css как обычно, а потом вызвать команду ниже, которая заинлайнит все стили
## Установка

    npm i // достаточно выполнить один раз
    npm start

Скрипт пройдётся по всем html файлам из текущей папки, заинлайнит стили и положит получившиеся файлы в папку `build`
