Скрипт сборки 503 страницы для отдачи в finnal-error-handler

Собирается в 2 вариантах:
1. Авторизованный вариант с заглушками для пользовательских данных. Можем использовать, когда у нас есть данные из blackbox
2. Неавторизованный вариант, который не содержит пользовательских данных.

Каждый вариант дублируется на всех языках.
