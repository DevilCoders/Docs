### Storybook
Подключен storybook, для разработки 
```npm run storybook:dev```
Про storybook:
[storybook-showcase](https://github.yandex-team.ru/search-interfaces/frontend/tree/master/packages/storybook-showcase)
[storybook](https://storybook.js.org/docs/guides/guide-react/)

Для добавления истории в компонент, историю следует описать в файле `<имя-истории>.story.tsx`, рядом с кодом компонента.

В ci сделана автоматическая сборка и выкладка на github pages.
