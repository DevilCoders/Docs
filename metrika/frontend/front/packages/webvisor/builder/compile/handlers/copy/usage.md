# Обработчик для копирования файлов

Умеет копировать файлы из папки в папку.

## Пример

Живой пример из таска `copy-extension-manifest`:

```javascript
exports.copy = [{
  src: [
    'dev/misc/extension/manifest.json',
    'dev/misc/extension/icon.png',
  ],
  dest: 'dev/misc/extension/build'
}]
```

Здесь неявно указан тип обработчика (`exports.copy = {}`). Можно указывать сразу несколько групп файлов с разными путями назначения, т.к. `copy` является массивом.

# Параметры настройки обработчика

Каждый элемент массива `copy` является объектом с двумя свойствами:

- src – строка или массив строк, может содержать как прямые пути до файлов, так и glob-паттерны
- dest – путь до директории назначения
