# Стандартный обработчик

Этот обработчик будет установлен для любого таска, где явно не указан конкретный обработчик.

Данны обработчик выведет сообщение `No handler assigned for %handler_name%` если указано имя несуществующего обработчика.

# Параметры настройки обработчика

Данный обработчик не имеет настроек
