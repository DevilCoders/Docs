# Обработчик очистки

Данный обработчик нужен для удаления файлов и директорий

Принимает список файлов или glob-паттерн через настройку `entry`.

## Пример

Например, нам необходимо очистить некую папку от всех jpeg изображений. Конфиг такого таска будет выглядеть следующим образом:

```javascript
exports.entry = [
  'path/to/dir/*.jpg',
  'path/to/dir/*.jpeg'
]

exports.type = 'clean'
```

Обратите внимание, что используется параметр `type` для указания типа обработчика. Это необходимо, так как настройки с именем `clean` нет и сборщик не может неявно определить тип используемого компилятора.

# Параметры настройки обработчика

Данный обработчик не имеет настроек
