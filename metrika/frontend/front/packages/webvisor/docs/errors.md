# Обработка ошибок от API Metrica

Плеер поддерживает отображение кастомных сообщений об ошибках. Для добавления новой ошибки нужно добавить ключи в Танкер

[Кейсет в танкере](https://tanker.yandex-team.ru/?project=metrika-bem&branch=master&keyset=blocks%3Acommon%3Awebvisor-proto)

Правила именования ключей следующие:

* Заголовок страницы с ошибкой: `error-message-[error_code]`
* Описание ошибки: `error-message-[error_code]-description`

Оба ключа являются опциональными

В случае отсутствия заголовока будет показано стандартное сообщение "Не удалось записать визит"

В случае отсутствия описания текст под заголовком будет скрыт.
