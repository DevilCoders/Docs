## Архитектура

Основна архитектуры нового плеера – рендереры (renderers), это микромодули, которые получают на вход данные и отрисовывают некое состояние.

Единственный источник данных для всего плеера MetrikaBridge – API модуль для общения с сервером. В продакшене MetrikaBridge предоставляется метрикой (репозиторий metrika-bem), т.к. только там известно, как общаться с сервером.

В локальном девелопе MetrikaBridge – апи для общения с локальной MongoDB
