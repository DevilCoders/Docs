Структура объекта DOM-мутации

```javascript
{
  // Идентификатор ноды внутри которой
  // произошла мутация
  target: 12
  
  // Таймстамп в миллисекундах, отсчет ведется
  // с момента начала записи
  stamp: 34465

  // Группа событий (фиксированное значение для всех мутаций)
  // Необходимо для сборки событий на уровне плеера
  group: 'mutation',
  
  // Расширенная информация о мутации
  // Если данных какого-то из типов нет, то и соответствующий
  // массив будет отсутствовать для экономии места
  meta: {
    index: 0,
    changes: {
      a: [],
      b: [],
      c: [],
      d: []
    }
  }
}
```

### Описание объектов в `meta.changes`

Удаленные ноды

```javascript
a: [
  {
    // идентификатор
    id        : 12,
    // идентификатор предыдущей ноды или null
    pr        : 123,
    // идентификатор следующей ноды или null
    nx        : 456
    // идентификатор родителя
    pa        : 123
  }
]
```

---

Добавленные ноды и их дети

```javascript
b: [
  {
    // идентификатор
    id : 18
    // имя ноды
    nm : "DIV"
    // идентификатор родителя
    pa : 12
    // пространство имен или null
    ns : "ns"
    // идентификатор предыдущего соседнего элемента или null
    pr : 45,
    // идентификатор следующего соседнего элемента или null
    nx : 91,
    // атрибуты ноды, для всех нод кроме #text и #comment
    at : {
      class   : "someClass",
      id      : "someID"
    }
    // текст, только для нод #text и #comment
    ct : "text node content"
  }
]
```

---

Список изменений атрибутов

```javascript
c: [
  {
    // идентификатор
    id : 10
    // список измененных атрибутов содержащий предыдущее и новое состояние
    at : {
      class       : {
        o: "oldValue",
        n: "newValue"
      }
    }
  }
]
```

---

Изменения тектовых нод

```javascript
d: [
  {
    // идентификатор
    id : 10
    // предыдущее и новое состояние текста ноды
    ct : {
      o: "old text",
      n: "new text"
    }
  }
]
```
