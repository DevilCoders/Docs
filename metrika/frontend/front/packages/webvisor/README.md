# Webvisor 2.0

Этот репозиторий содержит кодовую базу модулей Вебвизор 2.0.

### Состав репозитория

* Ядро – [core](src/core), библиотеки, использующиеся в проекте глобально
* Плеер – [player](src/player2), интерфейс плеера
* Рекордер – [recorder](src/recorder), библиотека, встраиваемая в счетчик, отвечает за запись данных
* [Локализация ошибок плеера](docs/errors.md)
