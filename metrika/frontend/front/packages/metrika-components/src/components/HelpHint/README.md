Тултип с подсказкой 
```(jsx)
<HelpHint>текст подсказки</HelpHint>
```

Имеет параметр `size` со стандартными размерами, прокидывается в компонент [Tooltip](https://lego.yandex-team.ru/libs/islands/v5.22.0/desktop/tooltip/)
```(jsx)
<HelpHint size="m">текст подсказки</HelpHint>
```

Может быть сразу открыт
```(jsx)
<HelpHint isOpen={true} size="m">текст подсказки</HelpHint>
```
