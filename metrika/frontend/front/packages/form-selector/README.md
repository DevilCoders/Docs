# Интерфейс выбора форм для создания целей

## Разработка

Интерфейс метрики открывает вкладку с сайтом, на котором подключен счетчик. Через
[RemoteControl](https://github.yandex-team.ru/Metrika/watch/blob/develop/src/RemoteControl.js) подключается скрипт с
интерфейсом захвата. Данные

![схема](./scheme.png)


## Разработка

Для разработки интерфейса отдельно от метрики/счетчика:

```bash
npm run start:button
npm run start:form
# Смотреть: http://localhost:3000
```

Для разработки вместе с метрикой/счетчиком:

Обычный запуск метрики в локальном окружении будет использовать скрипт с прода,
чтобы подменить его на локальный нужно использовать прокси кода счетчика c переменной окружения
`formSelectorPath`, в которую нужно указать путь до папки со `build` в пакете `form-selector`

Например:
```
formSelectorPath=/Users/semenova2kate/frontend/packages/form-selector/build/ BUNDLE=prod npm run proxy
```

Во всех остальных окружениях: беты/тестинг/престейбл/прод подгружаются соотвествующие версии из веток,
там ничего подменять не нужно.
