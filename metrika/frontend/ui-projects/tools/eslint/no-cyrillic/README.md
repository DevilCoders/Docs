# Папка с линтерами для i18n

Здесь лежат плагины:

-   no-cyrillic

## no-cyrillic

Линтер для проверки использования в коде русских символов. Позволяет использовать кириллицу только в комментариях и
вызовах функции `i18n`.

### Зависимости

Устанавливаем зависимости для тестов командой `npm ci`

### Тесты

Чтобы запустить тесты для проверки правила, используйте следующую команду, находясь в папке `linters`

`npm run test tests/no-cyrillic.js`
