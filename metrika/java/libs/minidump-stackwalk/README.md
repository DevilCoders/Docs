## minidump-stackwalk
JNI для получения модели Minidump со стектрейсами при помощи тулзы breakpad/minidump_stackwalk

В текущей версии не принимает на вход символы.

## IntelliJ IDEA
Сгенерировать файл проекта minidump-stackwalk.ipr можно так:

    ya make --checkout
    ya ide idea --project-root=.
