# cdp-essential

Корневой модуль для проекта CDP, от него зависят все остальные модули `cdp-*` и `metrika/java/api/segments`.

Содержит в себе минимальный набор классов, необходимых для `metrika/java/api/segments` и в первую очередь
создан для того, чтобы не тащить весь cdp-common и его зависимости в `metrika/java/api/segments`
