### Удаление

Окружение не может быть удалено, если хотя бы одно из следующих условий истинно:

- в системе есть конфиги, связанные с этим окружением
- в системе есть окружения, для которых это окружение является родителем
- в системе есть программы, связанные с этим окружением
- окружение является корневым
