### Создание

Описание полей формы создания окружения:

- `Name` - которкое имя окружения. Полное имя будет сформировано из этого имени и имени родителя: `{parent_name}.{name}`. Не может быть изменено после создания.
- `Parent` - родитель для окружения. Не может быть изменён после создания.


### Создание нескольких окружений

Описание полей формы создания нескольких окружений:
- `Name` - шаблон для генерации имён окружений.
- `Parent` - родитель для окружений
