### Клонирование

При клонировании создаётся новое окружение, с таким же набором переменных, как и в исходном. Тип переменных при этом не меняется (ссылки в исходном окружении создаются как ссылки в новом окружении)

Описание полей формы клонирования окружения:

- `Name` - см. создание окружения
- `Parent` - см. создание окружения
- `Go to clone` - перейти ли на страницу созданного окружения или остаться на исходной странице.
