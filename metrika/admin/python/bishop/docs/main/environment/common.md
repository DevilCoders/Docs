## Окружение (Environment)

Контейтер для хранения переменных, которые используются при рендеринге шаблонов.
Имя окружения уникально. Каждое окружение связано с другими окружениями отношениями родитель-потомок.
На странице дерева окружений можно увидеть графическое представление этих отношений.
Родитель может быть только один, потомков - много.
[Пример окружения](https://bishop.mtrs.yandex-team.ru/environment/metrika.playground.example)
