## Запрос на активацию (Activation Request)

Список запросов можно посмотреть вот [тут](https://bishop.mtrs.yandex-team.ru/activation_request)
Для выкладки новой версии конфига (сделать конфиг активным) нужно подтвердить запрос на активацию.
В запросе на активацию можно посмотреть дифф планируемых изменений между текущим активным конфигом и тем, который планируется активировать.
Запрос создаётся в момент создания новой версии конфига. Так же запрос можно создать руками на странице конкретного конфига.
При включенной автоматической активации, запрос на активацию деплоится автоматически.
