### Удаление

{% note alert %}

Так как в API всегда доступна только активная версия конфига, при её удалении пропадёт возможность получения конфига для пары программы/окружение. Даже при наличии других версий конфига для пары программа/окружения до активации одной из этих версий конфиг через API будет недоступен.

{% endnote %}

В UI есть возможность удалить до 20 конфигов за раз. Если необходимы более глобальные действия, нужно воспользоваться CLI.
