## Конфиг (Config)

Отрендеренный шаблон для конкретной пары программы+окружение.
Версия конфига уникальна в пределах системы и не привязана к паре программа+окружение.
В один момент времени активна только одна версия конфига для пары программа+окружение.

При изменении переменных в окружении или изменении шаблона создаётся новая версия конфига.
Через API доступна только активная версия конфига.

Активировать конфиг можно с помощью запроса на активацию конфига.
