### Создание

Описание полей формы создания шаблона:
- `Name` - имя шаблона. Должно быть уникально. Нельзя изменить после создания.
- `Format` - формат шаблона. Используется для валидации синтаксиcа при рендеринге конфига (один из: `XML`, `YAML`, `JSON`, `PLAINTEXT`). Для формата `PLAINTEXT` валидация синтаксиса отсутствует.
- `Text` - Jinja2 текст конфига
