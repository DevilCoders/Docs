### Изменение

Редактирование отличается от создания невозможностью ввода имени. Так же, после сохранения изменений, будут найдены все программы, на которые влияет данный шаблон (либо он напрямую используется какой-либо программой, либо какие-то шаблоны включают его в себя), и будут созданы новые версии конфигов.
