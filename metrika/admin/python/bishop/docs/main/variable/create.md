### Создание

Описание полей формы создания переменной:

- `Type` - тип переменной. Используется для конвертации значения в правильный тип.
- `Name` - имя пемененной. Уникально в пределах окружения.
- `Value` - значение переменной. В зависимости от значения поля типа, будет сконвертировано в один из типов.
