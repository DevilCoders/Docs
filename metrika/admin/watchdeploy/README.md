# Релизный процесс

watchdeploy попадает под требования Sox, поэтому релизы должны быть строго задокументированы.
Чтобы выложить очередной релиз, необходимо:
* Создать тикет в st/MTRSADMIN
* Прикрепить к нему PR (код-ревью должно быть пройдено)
* В комментарии отписать результаты тестирования
* Прикрепить к таску в st кондукторный тикет в stable
