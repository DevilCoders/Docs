## Ручной запуск

Для создания конфига и ручного запуска расчет есть утилитка, которая позволяет сварить конфиг запуска:
`create_daily_config.py <date> >config.pb.txt`

Далее в конфиге можно подменить выходные пути руками (CleanSessions, FraudsSessions, StaffSessions) на какие-нибудь локальные таблички (если запуск для тестирования).

Сам запуск
`./ymusic-session-maker --config config.pb.txt`

Также для ускорения тестирования можно строить, например, только стафф-сессии: они маленькие, но на посмотреть глазами должно хватить (нужно закомментировать код поcтроения clean и fraud сессий).

Далее над получившимеся табличками можно запустить расчет метрик, посмотреть глазами, etc.

*Важно: если вы запускаете руками расчет продовых сессий (дабы ускорить появление табличек), надо также руками создать выходные артефакты. См. выше ссылку на реактор*
