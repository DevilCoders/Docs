# Технологии проекта [Sandbox](https://sandbox.yandex-team.ru)

## Стек
Проект написан на [backbone v1.1.2](http://backbonejs.org/) c использованием [marionette v1.8.8](http://marionettejs.com/).
В качестве шаблонизатора используется [handlebars](http://handlebarsjs.com/).

За основу проекта был взят [JavaScript skeleton application](https://github.yandex-team.ru/ftdebugger/skeleton).

## Блоки
В качестве блоков проекта используется [bemmy v1.3.8](https://github.yandex-team.ru/bemmy/bemmy).

## Сборка
Сборка производится с помощью [gulp](http://gulpjs.com/).
