# Объекты страниц sandbox

Для каждой страницы созданы свои Объекты Страниц (`./path/to/root/spec/page-blocks/`).
Объекты страниц представляют собой массив хешей и специфичных классов элеметов страницы.
Используются для того, чтобы декларативно обращаться к блокам из тестов.

На данный момент существуют:

* `general.js` – Базовые блоки. Содержит элементы, которые есть на любой странице сервиса;
* `tasks.js` – Блоки, которые специфичны для странцы /tasks
