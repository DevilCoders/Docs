# Правила работы с TestPalm на проекте Sandbox

В [TestPalm](https://testpalm.yandex-team.ru) для проекта `sandbox` заведён проект
[Sandbox-frontend](https://testpalm.yandex-team.ru/sandbox-frontend) в организации
[Unity](https://testpalm.yandex-team.ru/organizations/unity).

В этом документе будут перечислены правила и best practices, которые должны соблюдаться при работе в TestPalm
с проектом [sandbox-ui](https://a.yandex-team.ru/arc/trunk/arcadia/sandbox/ui).
На данный момент нет интеграции тестов и тесткейсов из TestPalm, но планируется.
