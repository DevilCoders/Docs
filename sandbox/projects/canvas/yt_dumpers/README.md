Выгрузка данных в yt
===============================================================

См https://st.yandex-team.ru/CANVAS-140

Входные данные:
* База данных с canvas

Выходные данные:
* Таблички в YT

Запускается через sandbox schedulers.

По-умолчанию скрипт использует кластер hahn (hahn.yt.yandex.ru.[//home/canvas/])
Максимальное разрешение по времени - день. Если уже есть дамп за этот день, см. значение overwrite_existing.

Для тестовых целей лучше кастомный путь до таблиц в YT.

Проверить, что все хорошо можно в Web GUI YT: https://yt.yandex-team.ru/hahn/#page=navigation&path=//home/canvas
