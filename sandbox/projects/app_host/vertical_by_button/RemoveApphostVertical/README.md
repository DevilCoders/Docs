# RemoveApphostVertical

## Общее описание
Данная задача (`RemoveApphostVertical`) используется для удаления всех результатов работы задачи `CreateNewApphostVertical`
и ее подзадач - `CreateNannyServicesForApphostVertical` и `CreateNewRmForApphostVertical`, в т.ч.:
* Всех созданных няня-сервисов для новой вертикали
* Балансера в awacs'e для вертикали
* Сетевых макросов
* Конфига для вертикали apphost'a
* Конфига для компоненты в Release Machine
