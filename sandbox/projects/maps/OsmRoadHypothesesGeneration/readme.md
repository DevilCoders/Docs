## Генерация дорожных гипотез по OSM

В указанном bounding box скачиваем OSM карту. Из нее извлекаем все дорожные
сегменты с фильтрацией по типу. Затем каждый сегмент пробуем приматчить к нашему
графу, если не получается, генерируем гипотезу. Гипотезы которые находятся
слишком близко друг к другу объединяем вместе.

Для выгрузки из OSM используется osmnx. Для его использования необходим
кастомный lxc image.

Вспомогательный код таски распологается в
https://a.yandex-team.ru/arc/trunk/arcadia/maps/graph_quality/hypotheses/bin/osm/road_graph_diff .
