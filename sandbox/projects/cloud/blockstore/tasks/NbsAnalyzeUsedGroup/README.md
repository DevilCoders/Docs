# Задача для анализа нагрузки на группы

Задача собирает данные из таблиц метрик и строит графики для представления результатов.

Для запуска задачи из консоли ее нужно собрать:
```
ya make
```

После чего ее нужно опубликовать и запустить. Сделать это можно одной командой из консоли:
```
./NbsAnalyzeUsedGroup run --attr task_type=NBS_ANALYZE_USED_GROUP --attr release=stable --wait '{"custom_fields": [{"core_resource": "идентификатор ресурса core", "name": "ydb_table", "value": "список таблиц через пробел"}]}'
```

Данная команда опубликует бинарник задачи в sandbox и запустит его отдельной задачей.
Для просмотра результатов понадобится зайти по выведенной в консоль ссылке, результаты будут в ресурсах задачи.
