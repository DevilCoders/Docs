# Конфиги групп тестирования

## Использование конфигов

Конфиги используется:
* в звездолёте
* в трекерных санитайзерах.

### Конфиги в звездолёте

**Используются:**
* `testing_groups.yaml`

### Конфиги в трекерных санитайзерах

**Используются:**
* `testing_groups.yaml`
* `testing_groups_mobile.yaml`

**Особенности:**
* Трекерные санитайзеры используют из общего конфига только те группы,
у которых в поле `activity` (список строк) есть строка `tracker_sanitizers`.
* Трекерные санитайзеры поддерживают необязательное поле `startrek_queue`,
обозначающее стартречную очередь, к которой относятся компоненты из поля `components`.
Если поле не заполнено, то дефолтной очередью считается `BROWSER`.
* Трекерные санитайзеры поддерживают виртуальную компоненту `*` (звёздочка).
Она обозначает любую компоненту из соответствующей стартречной очереди, за исключением тех,
что были явно упомянуты в группах с такой же очередью.
