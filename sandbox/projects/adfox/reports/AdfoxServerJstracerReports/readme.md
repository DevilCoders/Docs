## Информация по генерации отчетов ##

1) В отчете только 1 итоговая таблица - #OUTPUT_TABLE#
2) Отчет считается за 1 день
3) В отчете обязательно одно поле `HourTime`, все остальные поля будут обработаны следующим образом:
* Поля вида `P95`, `P99` - как percentile; 
* Поле `Hits` - как количество запросов всего; 
* Остальные поля - как дополнительные `labels` в метриках;

В `labels` кроме полей таблицы также добавляется `tag` отчета.
