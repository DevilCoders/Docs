Команда deploy.sh позволяет запускать деплой препрода/автотестов из консоли.

Для запуска надо пробросить ключи по ssh(флаг -A) и нужна установленная настроенная команда yav.
Все секреты должны подставляться автоматически.

Для запуска деплоя автотестов:

`./deploy.sh autotest`

Для запуска деплоя стэнда ассесоров:

`./deploy.sh preprod`

Внимание!
Инициализация БД для ассесоров из консоли запускается отдельным скриптом!
Запускать инициализацию БД нужно дождавшись окончания деплоя стенда https://deploy.yandex-team.ru/stages/adfox-crowdtest
