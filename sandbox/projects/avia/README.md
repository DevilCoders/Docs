##Алгоритм добавления мониторинга работы в таск:
- Наследуемся от AviaBaseTask
- Добавляем в ya.make в PEERDIR sandbox/projects/avia/base
- Если в таске переопределен метод ``on_prepare``, то в нем нужно вызвать метод предка
- Также при переопределении методов ``on_success``, ``on_failure``, ``on_break`` нужно обязательно вызвать соответствующий метод предка
- ``on_execute``, свободен от таких требований, в нем вызывать метод предка нет необходимости
