
### Parameters

В модуле parameters определен класс параметров, зависящий от типа окружения. Тип окружения определяется параметром
environment_type (или backtesting_environment_type). В завасимости от значения от этого параметра
остальные параметры определены или в модуле consts, или задаются пользователем (при значении environment_type(backtesting_environment_type) равным "custom").<br>

Если в задаче нужны параметры, зависящие от окружения, нужно брать их из модуля parameters. При этом, в начале выполнения таски
нужно вызвать функцию <i>init_<класс_параметров>_params</i>. В результате ее работы в зависимости от типа окружения для каждого параметра <param_name> будет
проинициализирован атрибут таски <param_name> (т.е. в коде надо будет использовать не self.Parameters.<param_name>, а self.<param_name>)
