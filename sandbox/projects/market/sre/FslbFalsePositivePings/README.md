# FslbFalsePositivePings

Данный мониторинг находит все запросы в ручку /ping к FSLB балансерам, которые были обработаны дефолтным виртуальным хостом ping. 
Таким образом ответ на пинги будет "вечнозеленый" вне зависимости от живости пингуемого сервиса.
Если находится хотя бы один - отправляется крит в Джагглер.
