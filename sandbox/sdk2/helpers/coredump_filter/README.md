# Coredump Filter

Данное местоположение не является финальным для данной библиотеки,
она должна жить в `library/python`, но по некоторым особенностям sandbox
не может там жить. Поэтому документация для удобства индексации и поддержания
порядка вынесена в [правильное место](https://a.yandex-team.ru/arc/trunk/arcadia/library/python/coredump_filter/README.md),
а код пока что живёт тут до лучших времён.
