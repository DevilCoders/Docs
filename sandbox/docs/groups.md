# Группы

**Группа** — множество пользователей, объединенные одним проектом или сервисом.
Sandbox имеет возможность синхронизации содержимого группы с внешними источниками:
[ABC](https://abc.yandex-team.ru/), [Staff](https://staff.yandex-team.ru/) или [группами в Аркадии](https://a.yandex-team.ru/arc_vcs/groups).

Свойства и состав группы может редактироваться любым участником этой группы.

## Список групп { #list }

Полный список групп можно посмотреть на странице [Groups](https://sandbox.yandex-team.ru/admin/groups).

![Список групп](img/group-list.png "Список групп")

Для создания новой группы нажмите на **Create > Group**.
Редактирование новой или существующей группы выглядит следующим образом:

![Редактирование группы](img/group-edit.png "Редактирование группы")

В поле "ABC group handle" нужно указать название сервиса. Его можно взять из саджеста или из url со страницы группы в АБЦ. Например, для [Сервисов поискового портала](https://abc.yandex-team.ru/services/meta_search/) это будет "meta_search" а для [Инфраструктуры браузера](https://abc.yandex-team.ru/services/browserinfra/) -  "browserinfra"
Привязать группу к ABC сервису может только участник АБЦ сервиса с ролью "Руководитель" или "Менеджер аппаратных ресурсов".

Отметим, что:
* При использовании синхронизации со Staff в группу всегда включаются все сотрудники подразделения.
* При синхронизации с ABC возможно указание скоупа в сервисе через `/` в формате `<service>/<scope>`, например `sandbox/development`.
