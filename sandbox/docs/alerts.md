# Уведомления

## Уведомления о задачах

Sandbox умеет сообщать через почту, Telegram, Q и Juggler о том, что задача или планировщик перешли в определенный статус.
[Подробно об уведомлениях](dev/notification.md)

## Уведомления о квотах

Sandbox отгружает данные об API и MDS квотах в [соломон](https://solomon.yandex-team.ru/?project=sandbox&dashboard=sandbox_consumption).
Для отображения потребления и квоты пользователя на графике API квот задайте параметр `owner`.
Для отображения дисковых потребления и квоты пользователя на графике MDS квот задайте параметр `mds_bucket_name`. 

Для настройки алертов в соломоне на API и MDS квоты создайте в `своем` проекте сущность `Alert`, которая будет получать данные из этих графиков (проект `sandbox`, сервис и кластер `sandbox_consumption`).
Продробнее про алерты в Solomone можно почитать в [документации](https://docs.yandex-team.ru/solomon/concepts/alerting).
