# Доступы

Доступы заказываются в [https://puncher.yandex-team.ru/](https://puncher.yandex-team.ru/) — кнопка «Создать заявку на правило» в верхнем правом углу.

### Доступы, нужные людям
* `TCP 443` до `sandbox.yandex-team.ru`;
* `TCP 443` до `proxy.sandbox.yandex-team.ru`.

### Доступы, необходимые машине-клиенту Sandbox
* `TCP 443` до `sandbox.yandex-team.ru` — пользователи и локальные инсталяции Sandbox ходят в прод через балансер;

### Доступ агентов во внутреннюю сеть { #intranet }

Linux-агенты Sandbox описываются фаервольным макросом `_CMSEARCHNETS_`.
Проектные сети не поддержаны. Нет возможности использовать свои макросы.

MacOS-агенты общего назначения находятся в макросе `_C_SEARCH_INSTRUM_SANDBOX_MAC_`.
Также некоторые агенты могут находиться в составе выделенных сетевых макросов:
например, `_C_MAC_MOBILE_SANDBOX_` у хостов [соответствующей кондукторной группы](https://c.yandex-team.ru/groups/mac_mobile_sandbox).

### Доступ агентов во внешнюю сеть { #internet }
Доступ в интернет из задач разрешен, дополнительных действий для этого не требуется.
