# Авторизация

Для авторизации запросов необходимы:

* Идентификатор партнера **park_id**
* Идентификатор клиента **X-Client-ID**
* Секретный API-ключ **X-API-Key**

Чтобы получить их выберете в Диспетчерской пункт «Настройки» и перейдите в раздел API. Редактировать его может только сотрудник с ролью «Директор».


## Пример запроса

```bash
curl -X POST \
  https://fleet-api.taxi.yandex.net/v1/parks/driver-profiles/list \
  -H 'X-Client-ID: <Идентификатор клиента>' \
  -H 'X-Api-Key: <Секретный API-ключ>' \
  -d '{
	"query": {
		"park": {
			"id": "<Идентификатор партнера>"
		}
	}
}'
```
