# /monitoring-areas/{zone_name}
## Описание

Изменение существующей зоны мониторинга или создание новой.

Общий формат запроса к ручке:

    PUT /monitoring-areas/SVO
    {
        "name": "SVO",
        "geometry": { ... }, # геометрия как в GET /monitoring-areas
        "type": "airport" # произвольная строка (необязательное поле)
    }

### Ответы

Код:

    200 - всё ок
    400 - ошибка в формате 
    5хх - ошибка сервера
