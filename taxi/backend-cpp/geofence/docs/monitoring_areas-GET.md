# /monitoring-areas
## Описание

Получение списка зон мониторинга.

Общий формат запроса к ручке:

    GET /monitoring-areas?offset=10&limit=100

### Ответы

Код:

    200 - всё ок
    5хх - ошибка сервера

Тело:

    [{
        "geometry": { "type": "MultiPolygon", "coordinates": [
            [
                [[1,1], [1, -1], [-1, -1], [-1, 1]], # Внешняя оболочка
                ...  # Выколотые области
	    ]
	},
        "name": "test_area", # имя зоны
        "type": "airport" # тип зоны (необязательное поле)
    }]
