# /monitoring-areas/{zone_name}
## Описание

Получение информации об одной зоне мониторинга.

Общий формат запроса к ручке:

    GET /monitoring-areas/SVO

### Ответы

Код:

    200 - всё ок
    404 - нет такой зоны
    5хх - ошибка сервера

Тело:

    {
        "geometry": { ... } # геометрия как в GET /monitoring-areas
        "name": "test_area", # имя зоны
        "type": "airport" # тип зоны (необязательное поле)
    }
