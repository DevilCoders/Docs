## /monitoring-areas/{zone_name}/state
### Описание

Получение текущего слепка положения водителей внутри зон.

Общий формат запроса к ручке:

    GET /monitoring-areas/{zone_name}/state

### Ответы

Код:

    200 - всё ок
    5хх - ошибка сервера

Тело:

    {
        "events" : [
	    { 
	        "clid" : "300",
		"uuid" : "10",
		"arrival_time" : 1521547446, # Время въезда водителя в зону
		"last_update_time" : 1521547450,  # Время, когда последний раз водителя видели (в этой зоне / выезжающим из зоны)
		"event_time" : 1521547446,  # Время события
		"event_type" : "entered",
		"event_id" : "3"
	    }
	],
        "last_event_id": "15"
    }
