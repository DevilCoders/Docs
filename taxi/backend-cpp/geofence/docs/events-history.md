## /monitoring-areas/{zone_name}/events-history
### Описание

Получение списка эвентов для конкретной зоны мониторинга.

Общий формат запроса к ручке:

    POST /monitoring-areas/{zone_name}/events-history
    {
        "range": {
            "newer_than": <id последнего известного события>,
        },
        "limit": 10
    }

### Ответы

Код:

    200 - всё ок
    400 - неверный формат запроса
    5хх - ошибка сервера

Тело:

    {
	"last_event_id": "455",
	"unread": 378,  # Сколько событий не попало в ответ
	"events": [
    	    {
	        "clid" : "300",
		"uuid" : "10",
		"arrival_time" : 1521547446, # Время въезда водителя в зону
		"last_update_time" : 1521547450,  # Время, когда последний раз водителя видели (в этой зоне / выезжающим из зоны)
		"event_time" : 1521547446,  # Время события
		"event_type" : "entered",
		"event_id" : "1"
	    },
	    ...
	]
    }
