# Общий код для клиентов хэдхантинговых доскок объявлений


## Про хэдхантинговые доски объявлений

zarplata.ru, rabota.ru, и т.д. работают по одному принципу:
- авторизоваться (получить токен)
- загрузить список обновившихся резюме
- купить контакты заинтересовавших резюме

Есть рейтлимитер на все запросы.

У некоторых досок (zarplata.ru) есть идентификаторы соискателей, то есть
можно как-то понимать, что один и тот же человек составил два разных
резюме, но это неточно, потому что непонятно, насколько можно доверять
процедуре аутентификации соискателей.

## Про клиентов и общий код

Клиенты работают по одному паттерну: получить обновившиеся резюме и
купить контакты:
```
    # client: ZarplataruClient
    await client.authorize()
    latest_resumes = await client.get_latest_resumes()
    resume = await client.buy_full_resume(latest_resumes[-1].resume_id)
```

Общий код клиентов вынесен в данный пакет. Сами клиенты реализованы в
виде плагинов `plugins/clients/*`.
