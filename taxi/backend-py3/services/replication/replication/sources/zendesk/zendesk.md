## Zendesk

* `zendesk_id` - токен в секдисте, по которому хранятся данные для подключения клиента zendesk.
* `resource` - тип ресурса, который нужно получать из зендеска. Сейчас поддерживаются:
  * тикеты - `tickets` (репликация по полю `generated_timestamp`)
  * события тикетов - `ticket_events` (репликацию по полю `timestamp`)
