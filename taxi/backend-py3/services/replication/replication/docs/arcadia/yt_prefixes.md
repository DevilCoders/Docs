## Заведение YT аккаунта
Список уже доступных путей для записи в YT можно увидеть [здесь]({{ link_to_default_flaky_yaml }}).
Если у вас еще нет аккаунта в YT, либо квоты на динамические таблицы, сперва, ознакомьтесь с документацией YT.

Вам нужны 3 типа квот:
- [Аккаунт](https://yt.yandex-team.ru/docs/description/common/quotas#resources) (дает гарантию на место, чанки, ноды, таблеты)
- Префикс пути в YT - как правило 1 путь будет соответствовать одному аккаунту в YT и будет совпадать с `//home/{account}`. Но на самом деле любую папку может быть выставлен произвольный аккаунт
- [table cell bundle]({{ link_to_yt_docs_tablet_cell_bundles }}) - нужен чтобы динамические таблицы работали стабильно

Далее, вам нужно понять, откуда вы [получите квоту]({{ link_to_yt_docs_quotas }}), из какого ABC сервиса.

### Добавление новых YT префиксов

Когда аккаунт готов, для добавления новых префиксов в сервис, стоит проследовать следующей инструкции:

1. Добавить префиксы [здесь]({{ link_to_default_flaky_yaml }}).
Для прода стоит также указать кастомный [table cell bundle]({{ link_to_yt_docs_tablet_cell_bundles }}),
это нужно, чтобы запись в динамические таблицы была стабильной,
**default** квота не даёт гарантий.
Префикс для временных операций можно не указывать,
по умолчанию он считается как `{prefix}/tmp`.
2. Выдать доступы через [интерфейс YT]({{ link_to_yt_docs_acl }}) на **read/write/mount**
следующим роботам: {{ yt_replication_robots }}. А также, **use** на ваш аккаунт и бандл данным роботам.
3. Префикс будет доступен после обычного [деплоя](deploy.md) правил. Можно добавить новый префикс сразу вместе с новыми правилами.

Также, есть наши внутренние рекомендации, о том, какую лучше делать [структуру папок в YT]({{ link_to_yt_wiki_choose_path }}).

## Задание YT префиксов в правилах
YT префикс можно задать на уровне группы правил:
```
yt:
   prefix: taxi
```
[Пример в коде]({{ link_to_yt_flaky_yaml }}).
Также префикс можно переопределить для конкретного правила в секции:
```
target:
    prefix: taxi
```
Общие настройки YT при этом хранятся [здесь]({{ link_to_default_flaky_yaml }}),
здесь же можно найти необходимые префиксы из уже существующих, либо
добавить новые, а также посмотреть префикс по умолчанию.
