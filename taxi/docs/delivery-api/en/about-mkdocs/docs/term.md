# Terminology

Offer means a delivery option or a route selected according to the parameters specified in the order.

Order means a registered delivery request that contains information required for the provision of services (sender, list of services, recipient, shipment, and so on).

Label means an accompanying document, a sticker on the shipment that contains information about the sender and recipient, shipment number and weight, delivery method and has a barcode with the shipment number.

Delivery Receipt means an accompanying document that confirms the transfer of the goods during order pickup.
