# Терминология

Оффер — вариант доставки или маршрут, подобранный по заданным параметрам заказа.

Заказ — оформленное задание на доставку, содержащее информацию, необходимую для оказания услуг (например, отправитель, перечень услуг, получатель, отправление и т. д.).

Ярлык — сопроводительный документ, представляющий собой наклейку на отправлении с информацией об отправителе и получателе, номере и весе отправления, способе доставки, а также содержащий штрих-код с номером отправления.

Акт приема-передачи — сопроводительный документ, подтверждающий передачу товаров в рамках забора заказов.

ПВЗ — пункт выдачи заказов.
