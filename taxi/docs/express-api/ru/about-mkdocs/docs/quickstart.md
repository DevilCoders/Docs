# Начало работы

Перед началом работы с API Яндекс&#160;Доставки [получите](access.md) OAuth-токен.

## Заказ перевозки

После того, как вы получили токен, вы можете использовать API Яндекс&#160;Доставки для заказа доставки отправлений.

Процедура заказа выглядит следующим образом:

1. Создайте заявку (операция [Создание заявки с мультиточками](../ref/v2/claims/IntegrationV2ClaimsCreate-docpage)).<br/>
После попадания заявки в систему запустится процесс оценки заказа.
1. Узнайте результат оценки заказа (операция [Получение информации по заявкам с учетом мультиточек](../ref/v2/claims/IntegrationV2ClaimsInfo-docpage/)).<br/>
В случае успешной оценки вы получите стоимость для клиента.<br/>
Иначе ответ будет содержать причины, по которым заявка не может быть исполнена.
1. При необходимости отредактируйте заявку (операция [Редактирование заявки с мультиточками](../ref/v2/claims/IntegrationV2ClaimsEdit-docpage)).<br/>
До подтверждения заявки её можно редактировать.
1. Подтвердите заказ (операция [Подтверждение заявки](../ref/v1/claims/IntegrationV1ClaimsAccept-docpage)).<br/>
В случае успешной оценки заказ можно подтвердить, после чего запустится процесс поиска исполнителя.
1. Узнайте состояние заявки (операция [Получение информации по заявкам с учетом мультиточек](../ref/v2/claims/IntegrationV2ClaimsInfo-docpage)).
1. При необходимости отмените заказ (операция [Отмена заявки](../ref/v1/claims/IntegrationV1ClaimsCancelC-docpage)).
