# Оплата при получении

При получении груза пользователю приходит ссылка на оплату через [Яндекс&#160;Кассу](https://kassa.yandex.ru/).

Оплата при получении работает в одном из двух режимов:

- Без фискализации. 
- С фискализацией через Яндекс&#160;Кассу.

Подробности про создание заявок с оплатой при получении можно узнать в операции [Создание заявки с мультиточками](../ref/basic/IntegrationV2ClaimsCreate-docpage) и в [API Яндекс&#160;Кассы](https://kassa.yandex.ru/developers/api#intro). 

Режим работы определяется типом подключенной Яндекс&#160;Кассы и влияет на необходимые в запросах поля.

## Авторизация {#auth}

Для проведения операций необходимо выдать права OAuth-приложению логистики для этого:

1. Войдите в Кабинет Корпоративного Клиента Яндекс&#160;Доставки с помощью выданного вам логина и пароля.
2. На вкладке **Профиль** нажать кнопку **Яндекс&#160;Касса Подключить**. Кнопка доступна только если менеджер подключил данную возможность.
3. На открывшейся странице выбрать нужную учетную запись и дать разрешения.

Текущее состояние подключения и статус фискализации Яндекс&#160;Кассы можно узнать на вкладке **Профиль**.
