# Интеграция с 1C:БО

Интеграция с 1С:БО открывает возможности автоматизированной доставки первичных документов от клиента аутсорсеру. Экономьте свое время на заполнении данных и планируйте доставку на 3 дня вперед. Информация о курьере и текущем состояния заявки в режиме реального времени делает процесс передачи документов и других посылок прозрачным.

## Чем хороша интеграция

* Адреса точек отправки и доставки, контактные данные получателя и размеры посылки заполняются с помощью справочников.
* Повторная доставка по заявке с сохранением ранее внесенных данных и возможностью изменить их выполняется в один клик.
* Все статусы заявки от согласования цены до доставки посылки фиксируются во времени и сохраняются для отслеживания истории.
* Вы можете спланировать доставку документов к определенному времени на 3 дня вперед.
* После авторизации в личном кабинете Яндекс Go вы получаете быстрый доступ к нему напрямую из формы создания заявки.
* Интеграция разработана и поддерживается 1С:БухОбслуживание.

## Что нужно для интеграции с 1С:БО

* Подключите личный кабинет Яндекс Go в 1С:БухОбслуживание. Зарегистрируйте организацию в сервисе Яндекс Go по реферальной ссылке из настроек УФА, получите токен и проверьте корректность подключения.
* Создайте заявку. Заполните адреса точек отправки и доставки, укажите контактные данные получателя, выберите или задайте самостоятельно размеры посылки и отправьте заявку в работу.
* Наблюдайте за процессом доставки. Изменение статуса заявки в режиме реального времени делает процесс доставки прозрачным и понятным.
