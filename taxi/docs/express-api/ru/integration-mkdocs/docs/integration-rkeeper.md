# Интеграция с R_keeper

Установите веб-приложение Яндекс&#160;Доставки и отправляйте заказы в доставку прямо с терминала r_keeper.
Интеграция совместима с большинством доставочных модулей r_keeper, позволяет отправлять в доставку заказы из любых источников.

## Чем хорош модуль

* Вы можете отправить в доставку любой заказ из r_keeper, даже если у вас нет доставочного модуля.
* Доступен заказ на ближайшее время или отложенный заказ.
* Можно установить максимальную стоимость, при которой происходит автоматическое подтверждение заказа.
* Есть возможность отправлять в доставку заказы, по которым требуется оплата курьеру картой.
* Вы можете написать комментарий к адресу ресторана, указать особые требования (термосумка, автокурьер) и дополнительные опции (SMS-подтверждение).
* После отправки заявки вы можете проследить ее выполнение и связаться с исполнителем.
* Интеграция разработана и поддерживается Carbis.

## Что нужно для интеграции с R_keeper

Вы можете подключить интеграцию самостоятельно или обратиться за помощью к вашему дилеру.
Инструкция по работе интеграции и установочный файл находятся [здесь](https://wiki.carbis.ru/external/r_keeper_и_яндекс.доставка/1_установка_и_настройка).
