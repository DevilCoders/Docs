# Модуль для Opencart

Модуль Яндекс&#160;Доставки легко настроить и с ним вы можете вызвать курьера прямо со страницы заказов в Opencart. Модуль поддерживает все актуальные тарифы Яндекс&#160;Доставки.

## Чем хорош Модуль

* Позволяет показывать стоимость Экспресс Доставки Яндекс&#160;Доставки в корзине.
* Поля заявки на доставку заполняются автоматически из карточки заказа.
* Можно создать заявку на ближайшее время, или указать желаемое.
* Можно рассчитать стоимость услуги перед отправкой заказа.
* Можно выбрать подходящий тариф доставки, или рассчитать стоимость по умолчанию.
* В одну доставку можно отправить один или сразу несколько заказов.
* Доступна опция «Оплата курьеру картой».
* После создания заявки на доставку можно отслеживать статус доставки, а также посмотреть номер телефона курьера.

## Что нужно для работы с Модулем

Для использования Модуля:

* Необходимо иметь Кабинет Корпоративного Клиента Яндекс&#160;Доставки. Если у вас его нет, вы можете отправить заявку прямо из Модуля.
* Также понадобится [получить ключ](https://yandex.ru/dev/maps/geocoder/) API Геосервисов. В Модуле есть подробная инструкция, как это сделать. Ключ активируется от нескольких часов до нескольких суток.
