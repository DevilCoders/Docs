# Интеграция с iiko

Интеграция Яндекс&#160;Доставки c iiko позволяет отправлять в доставку заказы, полученные с любых платформ. И для этого вам не понадобится свой штат курьеров.

## Чем хорош модуль

* Вы можете отправить в доставку любой заказ из iiko при условии, что по нему принята оплата.
* В iikoWeb вы можете самостоятельно настроить интеграцию или изменить настройки.
* Вы можете указать спецтребования (термосумка, автокурьер) и дополнительные опции (SMS-подтверждение).
* Если указать в настройках интервал вызова курьера, то курьер приедет к моменту готовности блюда.
* После отправки заявки можно отследить ее выполнение и связаться с исполнителем.
* Интеграция разработана iiko. Задать вопросы по работе модуля вы можете в службу поддержки вашего дилера iiko.

## Системные требования

* Интеграция доступна на версии iiko не ниже 7.5.6.
* Интеграция доступна для тарифов iikoCloud и Life-time при наличии модуля iikoDelivery. Для тарифа Life-time требуется лицензия iikoAPI Cloud.
* Поддерживаются как одиночные рестораны, так и сети. Исключение — конфигурация «mini chain», когда в один RMS объединены несколько торговых предприятий. Необходимо соблюдать правило: 1 сервер RMS = 1 адрес ресторана.
* На терминале, где для доставки заказа будут назначать курьера Яндекс&#160;Доставки, должен работать сервис iikoTransport.
* На терминал, где для доставки заказа будут назначать курьеров, необходимо установить плагин интеграции Plugin.Front.DeliveryExternalCourierService.

## Что нужно для интеграции с iiko
