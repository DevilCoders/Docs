# Модуль для WooCommerce

Модуль Яндекс&#160;Доставки легко настраивается и позволяет вызвать курьера прямо со страницы заказов в WooCommerce.

## Чем хорош Модуль

* Позволяет показывать стоимость Экспресс Доставки Яндекс&#160;Доставки в корзине.
* Перед отправкой заказа вы можете рассчитать стоимость услуги.
* Вы можете отправить в доставку один заказ или сразу несколько.
* Доступна опция «Оплата курьеру картой».
* После создания заявки на доставку вы можете отслеживать статусы доставки, а при необходимости — посмотреть номер телефона курьера.
* Если у вас еще нет кабинета партнера Яндекс&#160;Доставки, вы можете отправить заявку прямо из Модуля.
* Интеграция разработана и поддерживается Creative Motion.
