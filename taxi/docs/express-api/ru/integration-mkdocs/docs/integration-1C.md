# Интеграция с 1C

Интеграция с Модулем Яндекс&#160;Доставки будет доступна в следующих типовых продуктах 1С:

* ERP: Управление предприятием и Комплексная автоматизация версии 2.4.13 и выше.
* Управление Торговлей версии 11.4.13 и выше.
* Управление Нашей Фирмой версии 1.6.25 и выше.

## Чем хороша интеграция

* Интеграция программных продуктов 1С c Яндекс&#160;Доставкой легко настраивается и позволяет вызвать курьера прямо из карточки клиентского заказа. 
* Интеграция поддерживает большинство тарифов Яндекс&#160;Доставки. Выберите подходящий по стоимости или срочности исполнения заказа.
* Поля заявки на доставку заполнятся автоматически из заказа. 
* Можно создать заявку на ближайшее время, или указать желаемое.
* Перед отправкой заявки можно рассчитать стоимость услуги.
* После создания заявки на доставку вы можете отследить статус доставки, а также посмотреть данные курьера — номер его автомобиля и телефона. 
* Интеграция разработана 1С. Задать вопросы по работе модуля вы можете в службу поддержки 1С [dostavka@1c.ru](mailto:dostavka@1c.ru).

## Что нужно для интеграции с 1С

* Для использования сервиса необходимо зарегистрироваться в 1С:Бизнес сеть. Это можно сделать бесплатно из программного продукта 1С. 
* Для работы с Яндекс&#160;Доставкой необходимо иметь Кабинет Корпоративного Клиента на портале этой службы. Если у вас его нет, вы можете перейти к заполнению заявления на подключение прямо из модуля интеграции.
* Также понадобится специальный ключ (токен) API. Его можно получить в Кабинете Корпоративного Клиента. На форме подключения перевозчиков в приложении 1С есть подробная инструкция, как это сделать.
* Чтобы подключить интеграцию, воспользуйтесь [инструкцией](https://its.1c.ru/video/1c_courier).
