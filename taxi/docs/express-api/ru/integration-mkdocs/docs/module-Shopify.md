# Модуль для Shopify

Модуль Яндекс&#160;Доставки легко настраивается и позволяет вызвать курьера прямо со страницы заказов в Shopify.

## Чем хорош Модуль

* Позволяет показывать стоимость Экспресс Доставки Яндекс&#160;Доставки в корзине.
* Перед отправкой заказа вы можете рассчитать стоимость услуги.
* Вы можете отправить в доставку один заказ или сразу несколько.
* Вы можете изменить настройки складов с помощью кнопки **Locations**.
* Если у вас еще нет кабинета партнера Яндекс&#160;Доставки, вы можете отправить заявку прямо из Модуля.
