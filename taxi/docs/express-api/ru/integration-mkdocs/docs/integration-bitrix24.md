# Интеграция с Битрикс24

Найти и настроить Яндекс&#160;Доставку можно в CRM Битрикс24.

## Чем хорош Модуль

* Доставку легко настроить, а менеджер может вызвать курьера прямо из карточки сделки.
* Перед отправкой заказа можно рассчитать стоимость услуги.
* После создания заявки на доставку удобно отслеживать статусы доставки, а также посмотреть номер телефона курьера.
* Интеграция разработана 1С-Битрикс. Если у Вас возникнут вопросы по работе модуля, обратитесь в [чат поддержки Битрикс24](https://helpdesk.bitrix24.ru/ticket.php?_ga=2.42163429.132530193.1607253101-1474700865.1596884140).

## Что нужно для интеграции с Битрикс24

* Для работы с Яндекс&#160;Доставкой в Битрикс24 необходимо иметь Кабинет Корпоративного Клиента Яндекс&#160;Доставки. Если у вас его нет, вы можете отправить заявку со страницы **Настройки доставки**.
* Прочитать инструкцию по настройке [можно здесь](https://helpdesk.bitrix24.ru/open/11604358/?sphrase_id=72298158).
