# Интеграция с Tillypad

Tillypad — это система автоматизации ресторанов с широким спектром возможностей. Она способна удовлетворить потребности и больших сетевых клиентов, и заведений с небольшим бюджетом. Интеграция Яндекс&#160;Доставки с Tillypad позволяет отправлять заказы и отслеживать с POS-терминала их доставку курьерами службы.

Интеграция доступна начиная с версии 10.1.12.

## Чем хороша интеграция

* Заявки на доставку создаются автоматически при назначении специального курьера в системе Tillypad. Отмена заявки также производится автоматически.
* В настройках устройства стыковки можно указать подходящий тариф и требования к оформлению заявки на доставку: минимальную сумму заказа, максимальную стоимость оценки доставки для отправки заказа в службу, необходимость кода подтверждения передачи товара.
* Время прибытия курьера указывается в заявке к моменту окончания приготовления блюд заказа.
* С POS-терминала можно узнать имя и телефон курьера, принявшего заявку на доставку, а также отслеживать время прибытия курьера в точки маршрута.
* Есть возможность отправлять в доставку заказы, которые оплачиваются картой курьеру.
* Интеграция разработана и поддерживается Tillypad.

## Что нужно для интеграции с Tillypad

Для работы с Яндекс&#160;Доставкой в Tillypad необходимо иметь Кабинет Корпоративного Клиента Яндекс&#160;Доставки. Если у вас его нет, отправьте заявку по ссылке, указанной в параметрах устройства стыковки.
Инструкция по работе интеграции находится [здесь](https://my.tillypad.ru/help101/russian/tpad.htm?htd-yandex-dostavka.html).
