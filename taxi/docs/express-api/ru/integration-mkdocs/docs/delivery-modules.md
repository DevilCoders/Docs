# Модули Яндекс Доставки

Установите Модуль Яндекс&#160;Доставки на платформу вашего интернет-магазина и управляйте заказами без сложных интеграций и сторонних сервисов.

## Простая настройка и быстрая доставка

* Легко подключить и не нужны ресурсы разработки. 
* Можно отправить заявку на доставку прямо из списка заказов. 
* Автоматически подтягиваются все данные о товаре, контактах и адресах. 
* Проще следить за статусом доставки.
