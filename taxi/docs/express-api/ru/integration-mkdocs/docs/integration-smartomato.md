# Интеграция со Смартомато

Смартомато — это онлайн-платформа для организации доставки ресторана. Интеграция Яндекс&#160;Доставки со Смартомато позволяет получить комплексное решение. Интеграция выполнена на всех участках приема заказа: сайт, виджет, мобильное приложение клиентов и мобильное приложение управления доставкой — везде будут доступны актуальная цена из Яндекс&#160;Доставки (еще до отправки заявки), актуальный статус активной доставки и полное управление аутсорсом.

## Чем хороша интеграция

* Можно выбрать подходящий тариф и спецтребования к доставке.
* Яндекс&#160;Доставке можно назначить любые зоны доставки, созданные в Смартомато.
* Расчет стоимости Яндекс&#160;Доставки для клиента ресторана происходит автоматически и начинается еще в процессе заказа. Актуальная цена с учетом настроенной политики субсидирования Смартомато отображается клиенту на всех этапах заказа на сайте или в приложении ресторана.
* Смартомато самостоятельно формирует заявку на доставку: адреса и контакты ресторана и клиента, а также другая информация по заказу подтягиваются автоматически.
* Заявка может передаваться в Яндекс&#160;Доставку:
  * автоматически после создания заказа;
  * автоматически после подтверждения заказа;
  * вручную через карточку заказа в Смартомато.
* После назначения курьера Яндекс&#160;Доставки его имя, телефон, данные авто отображаются в интерфейсе Смартомато.
* Курьер Яндекс&#160;Доставки прибывает в ресторан ко времени забора заказа, которое рассчитывается Смартомато на основании настроек ресторана.
* Все этапы Яндекс&#160;Доставки можно отслеживать в Смартомато.
* При отмене заказа в Смартомато заявка автоматически будет отменена в Яндекс&#160;Доставке.
* Информация о выбранном способе доставки и стоимости доступна в стандартных отчетах Смартомато.

Интеграция разработана Смартомато. Подробнее об интеграции узнайте в [Смартомато Help Center](https://help.smartomato.ru/ru/articles/5170378-интеграция-с-яндекс-go).

Задать вопросы по работе модуля вы можете техподдержке Смартомато [support@smartomato.ru](mailto:support@smartomato.ru).
