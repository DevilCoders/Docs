# Интеграция с RetailCRM

Интеграция Яндекс&#160;Доставки с RetailCRM позволяет заказывать доставку прямо из карточки сделки.
Подробная инструкция поможет выполнить настройки, а служба поддержки RetailCRM проконсультирует по частным вопросам.

## Чем хорош модуль

* Вы можете оформлять и отменять заявку на доставку из RetailCRM.
* Вы можете рассчитывать стоимость доставки в RetailCRM.
* В RetailCRM отображаются актуальные статусы доставки заказа.
* Можно получать трек-номер по заказу из Яндекс&#160;Доставки в RetailCRM.
* Можно печатать акт приема-передачи из RetailCRM.
* Интеграция разработана AnalitikaOnline. Задать вопросы по работе модуля можно в службу поддержки RetailCRM [support@retailcrm.services](mailto:support@retailcrm.services).

## Что нужно для интеграции с RetailCRM

* Для настройки интеграции воспользуйтесь [инструкцией](https://help.retailcrm.services/knowledge_base/item/200785?sid=46397).
