# Релизы

Ниже приведен список изменений в API от версии к версии.

### Версия 1.0.13 {#version-1-0-13}

**06.07.20**

- Добавлена новая операция [Получение кода подтверждения](../ref/v2/claims/IntegrationV2ClaimsConfirmationCode-docpage).

### Версия 1.0.12 {#version-1-0-12}

**03.07.20**

- Исправлена информация об окружениях.

### Версия 1.0.11 {#version-1-0-11}

**24.06.20**

- Добавлено поле с признаком источника заявки `referral_source`.

### Версия 1.0.10 {#version-1-0-10}

**19.06.20**

- Добавлены новые операции: [Получение доступных тарифов](../ref/v1/estimate/IntegrationV1Tariffs-docpage), [Оценка заявки и параметров автомобиля без создания оффера](../ref/v1/estimate/IntegrationV1CheckPrice-docpage).

### Версия 1.0.9 {#version-1-0-9}

**10.06.20**

- Добавлено поле с номером заказа клиента на точке `external_order_id` при создании заявки.

### Версия 1.0.8 {#version-1-0-8}

**21.05.20**

- Добавлен статус `failed`.

- Добавлен признак `resolution` в журнал для терминальных статусов.


### Версия 1.0.7 {#version-1-0-7}

**12.05.20**

- Добавлена информация о позиции исполнителя заказа.

### Версия 1.0.6 {#version-1-0-6}

**27.04.20**

- Добавлен номер телефона исполнителя заказа.

- Добавлен список ошибок при создании заявки.

- Добавлен список предупреждений при создании заявки.

### Версия 1.0.5 {#version-1-0-5}

**23.04.20**

- Механизм Уведомлений о смене статусов помечен устаревшим, вместо него следует использовать Журнал.

### Версия 1.0.4 {#version-1-0-4}

**21.04.20**

- Добавлена информация об оплате при получении.

- Добавлено поле для общего комментария к заказу (comment в корне заявки).

- Добавлена возможность отказаться от доставки до двери (см. `skip_door_to_door`).

### Версия 1.0.3 {#version-1-0-3}

**21.04.20**

- Добавлен API журнала изменений заказа.

### Версия 1.0.2 {#version-1-0-2}

**11.04.20**

- Добавлен API заказа с мультиточками.

- Исправлено описание времени жизни токена — теперь оно неограниченно долго.

- Добавлена операция получения информации по нескольким заявкам `v2/bulk_info`.

## Версия 1.0.1 {#version-1-0-1}

**07.04.20**

- Добавлено описание статусов заявки.

- Добавлен раздел уведомлений о смене статусов заявки.

- Добавлена возможность не отправлять автоматические нотификации отправителю/получателю (см. `skip_client_notify`).

- Добавлена возможность создавать заказ без необходимости возврата товаров в случае отмены заказа (курьер оставляет товар себе) (см. `optional_return`).

- Добавлена возможность создать заказ к определенному времени (не позднее 36ч) (см. `due`).

- Добавлена валюта к цене предложения такси (см. `pricing`).

- Добавлена итоговая цена выполнения заказа (с учетом отмен, платных ожиданий и т.д.) (см. `pricing`).

- Добавлена возможность выбрать размеры авто клиентом (без гарантий, что получится увезти).
