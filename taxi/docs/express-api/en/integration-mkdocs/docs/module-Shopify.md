# Module for Shopify

The Yandex Delivery module is easy to set up and enables you to call a courier directly from the Shopify order page.

## Module benefits

* Displays the cost of Yandex Express Delivery in the shopping cart.
* Calculate the service cost before sending an order.
* Send one or multiple orders for delivery at one time.
* You can change the depot settings using the **Locations** button.
* If you don't have a Yandex Delivery partner account yet, you can send a request directly from the module.
