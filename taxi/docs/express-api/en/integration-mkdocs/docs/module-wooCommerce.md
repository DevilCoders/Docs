# Module for WooCommerce

The Yandex Delivery module is easy to set up and enables you to call a courier directly from the WooCommerce order page.

## Module benefits

* Displays the cost of Yandex Express Delivery in the shopping cart.
* Calculate the service cost before sending an order.
* Send one or multiple orders for delivery at one time.
* "Pay courier by card" option is available.
* After creating a delivery claim, track the delivery status and view the courier's phone number, if necessary.
* If you don't have a Yandex Delivery partner account yet, you can send a request directly from the module.
* Integration developed and supported by Creative Motion.
