# Module for Opencart

The Yandex Delivery module is easy to set up and enables you to call a courier directly from the Opencart order page. The module supports all current Yandex Delivery plans.

## Module benefits

* Displays the cost of Yandex Express Delivery in the shopping cart.
* Delivery claim fields are automatically filled in with information from the order card.
* Create claims for immediate delivery or for a specific time.
* Calculate the service cost before sending an order.
* Choose the appropriate delivery plan or calculate the default cost.
* Send one or multiple orders in one delivery.
* "Pay courier by card" option is available.
* After creating a delivery claim, track the delivery status and view the courier's phone number.

## Module requirements

To use the module:

* You must have a Yandex Delivery Corporate Client Account. If you don't have one, you can send a request directly from the module.
* You will also need a Geoservices API [key](https://yandex.ru/dev/maps/geocoder/). The module contains detailed instructions on how to do this. It takes from several hours to several days to activate the key.
