# Module for Joomla

The Yandex Delivery module is easy to set up and enables you to call a courier directly from Joomla. The module is compatible with both JoomShopping and VirtueMart.

## Module benefits

* Calculate the service cost before sending an order.
* Send one or multiple orders for delivery at one time.
* After creating a delivery claim, track the delivery status and view the courier's phone number, if necessary.
* If you don't have a Yandex Delivery partner account yet, you can send a request directly from the module.

{% note warning %}

The cost calculation in the shopping cart works only with VirtueMart. Install an additional plugin.

{% endnote %}
