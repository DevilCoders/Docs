# eats-restapp-marketing

Сервис для управления рекламой ресторанов в Яндекс Еде.

## Как собирать доку

Нужно иметь установленным пакет `plantuml`.  
На мак можно установить через [brew](https://formulae.brew.sh/formula/plantuml).

```sh
brew install plantuml
```

Чтобы собрать документацию (по факту - срендерить картинки), нужно вызвать
`make` или `make docs`.
