Это папка для хранения кода аналитиков команды международки

Структура папки
В текущей папке видим две подпапки
1. prod - для продовых/регулярных процессов, таких как: 
- сбор данных для дашбордов
- регулярные выгрузки
- и т.п.
2. junk - тут храним код, который пишем во время выполнения задач. Выкладывается сюда с целью:
- не потерять 
- скинуть пулл реквест коллеге из команды для проверки

В каждой подпапке есть свои подпапки для проектов команды
- rida
- scooters
- yango 

Внутри этих подпапок для каждой задачи заводим свою папку, в которую кладем код
