# Начало работы

Чтобы начать работать с Яндекс Такси:

1. Получите токен для отправки запросов к API.
    1. Войдите в [Кабинет Корпоративного Клиента Яндекс Такси](https://business.taxi.yandex.ru/) с помощью выданного вам логина и пароля.
    1. Нажмите на ваш логин в правом верхнем углу Личного кабинета. Будет открыт блок с информацией о вашем аккаунте.
    1. В разделе **Токен для API** нажмите кнопку **Получить**. Вы будете перенаправлены на страницу oauth.yandex.ru.
    1. На странице oauth.yandex.ru нажмите кнопку **Разрешить**. Вы будете перенаправлены в Кабинет Корпоративного Клиента Яндекс Такси.
    1. В блоке информации о вашем аккаунте в разделе **Токен для API** будет отображен ваш OAuth-токен.
    
1. Токены для доступа к API действуют ограниченное время, после которого становятся недействительными. Используйте запрос [Обновить OAuth-токен](oauth-refresh.md) чтобы получить время жизни вашего токена.
    
    - Если время жизни токена подходит к концу — повторно выполните запрос [Обновить OAuth-токен](oauth-refresh.md). Дата устаревания вашего токена будет обновлена и вы сможете продолжать пользоваться текущим токеном.
    - Если время жизни токена истекло — получите новый токен в [Кабинете Корпоративного Клиента Яндекс Такси](https://business.taxi.yandex.ru/).
    
    {% note info %}
    
    Если токен утерян, для перевыпуска обратитесь в [Яндекс.Паспорт](https://passport.yandex.ru).
    
    {% endnote %}
    
1. Используйте OAuth-токен во всех запросах к API. Передавайте токен в заголовке:
    #### в версии API 1.0
    `Authorization: <значение вашего токена>`
    #### в версии API 2.0
    `Authorization: Bearer <значение вашего токена>`
    
    {% note info %}
    
    Для работы с API 1.0 используйте хост `https://business.taxi.yandex.ru`, а для работы с API 2.0 — хост `http://b2b-api.go.yandex.ru`.
    
    {% endnote %}
    
1. Для большинства запросов необходимо указывать идентификатор клиента (`client_id`). Получить значение можно при помощи запроса [Информация о клиенте](auth.md).
1. [Укажите информацию о кост-центрах](cost-center-create.md).
1. Создайте [роли пользователей](role-create.md).
1. Заведите учетные записи пользователей:
    #### в версии API 1.0
    [создание пользователя](user-create.md)
    #### в версии API 2.0
    [создание пользователя](api20/user-create.md)
    

## Заказ поездки {#section_gyz_b4t_5hb}

После того, как вы получили токен и указали информацию о ваших пользователях, вы можете использовать API Яндекс Такси для заказа поездок.

Процедура заказа выглядит следующим образом:

1. [Узнайте цену и условия предстоящей поездки и получите оффер](route-stats.md).
1. Создайте [черновик заказа](order-create.md).
1. При необходимости [измените данные заказа](order-change.md).
1. Передайте заказ [на обработку](processing.md).
1. При необходимости [отмените заказ](order-cancel.md).
