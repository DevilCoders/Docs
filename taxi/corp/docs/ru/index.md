# Введение

Яндекс Такси предоставляет корпоративным клиентам возможность взаимодействия с сервисом при помощи API.

- Начните работу с Яндекс Такси согласно [инструкции](concepts/quickstart.md).
- Ознакомьтесь с описаниями [запросов и ответов API](concepts/request-central.md).

В случае возникновения каких-либо проблем свяжитесь с вашим менеджером Яндекс Такси.
