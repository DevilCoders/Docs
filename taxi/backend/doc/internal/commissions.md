#### Комиссии


##### Ручка
Запрос в кабинет `https://taxi-cabinet.mobile.yandex.ru/api/inner/commissions_csv/?from=2016-03-01T03:00:40&to=2016-03-03T03:00:40` отдаст все транзакции со датой
создания >= 2016-03-01T03:00:40 и < 2016-03-03T03:00:40. Все времена в UTC.

Нужно передать HTTP-заголовок YaTaxi-Api-Key с очень секретным значением. Значение можно
получить у @aershov182


##### Формат ответа

Формат csv. Разделитель запятая.
Поля:

| Название | Описание | Пример значения |
| -------- | -------- | --------------- |
| id | Идентификатор транзакции в транзакционном логе | 56d5f014a141bbed7928b1a3 |
| order_id | Идентификатор заказа | 38e9d7894bba495a9deb472a8bb3adb4 |
| clid | clid | 1956789073 |
| real_clid | real_clid, может быть пустым | AF002B7FA9404CA398F15B11DAD327C0 |
| commission_delta | Изменение значения комиссии по сравнению с прошлым значением | 25.8302 |
| d_time | Время создания транзакции. | 2016-03-01T19:40:04+0000 |


Выдача будет состоять из заголовка:
```
id,order_id,clid,real_clid,commission_delta,d_time
```

И данных:
```
56d5f014a141bbed7928b1a3,38e9d7894bba495a9deb472a8bb3adb4,1956789073,AF002B7FA9404CA398F15B11DAD327C0,25.8302,2016-03-01T19:40:04+0000
```
