Принудительный сурж
=============

[TAXIBACKEND-3706](https://st.yandex-team.ru/TAXIBACKEND-3706)

Протокол
=========

Новые клиенты должны всегда передавать в `routestats` параметр:
```(javascript)
"supports_forced_surge": true
```

В случае наличия суржа в ответе `routestats` будет параметр:
```(javascript)
"service_levels": [
    {
        "forced_surge": {
            "value": 2.5,
            "title": "×2,5",
            "title_card":  "× 2,5",
            "description": "497 руб. первые 5 минут и 37 руб./мин далее",
            "reason": "Это вынужденная мера, так как спрос на такси вырос из-за дождя",
            "comment": "Вы сможете повторить поиск такси, когда спрос спадёт",
            "order_comment": "Действует коэффициент ×2,5"
        },
        ...
    },
    ...
]
```

В ручку `orders` нужно передавать параметр `forced_surge` только в случае
наличия принудительного суржа. Если его нет, то и параметр передавать не надо.

Пример для заказа, где пользователь подтвердил принятие суржа:
```(javascript)
"forced_surge": {
    "value": 3.14
},
```
