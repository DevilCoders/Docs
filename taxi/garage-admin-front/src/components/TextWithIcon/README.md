# TextWithIcon

Renders text with icon on the left.

## Props

| Name      | Type                                         | Default | Description    |
|-----------|----------------------------------------------|---------|----------------|
| label     | React.ReactNode                              | -       | text to render |
| icon      | React.ReactNode                              | -       | icon to render |
| type      | 'secondary' &#124; 'danger' &#124; 'warning' | -       | text styling   |
| className | string                                       | -       | CSS class name |
