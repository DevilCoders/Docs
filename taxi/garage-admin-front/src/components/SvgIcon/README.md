# SvgIcon

Renders an application-specific SVG icon from a predefined icon set.

## Available icons

* cab
* caliper
* fuel-benzine
* fuel-gas
* gearbox-auto
* gearbox-manual
* insurance
* phone
* repair
* star
* tire

## Props

| Name      | Type                                            | Default | Description        |
|-----------|-------------------------------------------------|---------|--------------------|
| icon      | [see available icons section](#available-icons) | -       | icon to render     |
| className | string                                          | -       | icon class name    |
| style     | React.CSSProperties                             | -       | icon inline styles |
