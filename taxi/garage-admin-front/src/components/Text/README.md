# Text

Basic text rendering component.
Wrapper around [Typography.Text](https://ant.design/components/typography/) component from 
[Ant Design](https://ant.design/docs/react/introduce) library.

## Props

| Name      | Type                                         | Default  | Description                |
|-----------|----------------------------------------------|----------|----------------------------|
| children  | React.ReactNode                              | -        | child nodes                |
| className | string                                       | -        | CSS class name             |
| code      | boolean                                      | -        | applies code style to text |
| strong    | boolean                                      | -        | displays text in bold      |
| type      | 'secondary' &#124; 'warning' &#124; 'danger' | -        | text styling               |
