# PhoneNumber

Displays the provided phone number, optionally prettified.

## Props

| Name          | Type                 | Default | Description                          |
|---------------|----------------------|---------|--------------------------------------|
| phone         | string &#124; number | -       | phone number                         |
| prefix        | string               | -       | text to show before phone number     |
| prettify      | boolean              | -       | prettify phone number if possible    |
| className     | string               | -       | CSS class name                       |
| iconClassName | string               | -       | CSS class name to be applied to icon |
