# Задача
Надо поддержать выделение блоков магазинов как на дизайне:
https://www.figma.com/file/yCwyDG2KykxXlpjJohbwbI/%F0%9F%8F%AA-%D0%95%D0%B4%D0%B0%D1%80%D0%BA%D0%B5%D1%82?node-id=1566%3A16391

# Зачем
Поддержать запуск аптек на Маркете

# Как делаем
- добавляем в настраиваемый `payload` (там, где сейчас приходит `title`, полный
  путь: `layout.[].payload`) виджета `places_multiline` поле `title_size` с
  вариантами:
    - `large`   // Текущий большой текст
    - `medium`  // Новый небольшой текст, как на дизайне для Аптек/Зоотоваров
- для десктопа нужно поддержать фон виджетов, поэтому добавляем в виджет `places_multiline` (объект `data.places_multiline.[].payload`) новое поле `background_color` в виде объекта:

```json
{
  // ...
  "payload": {
    // "places": ...,
    // "total_columns_hint": 6,
    "background_color": {
      "light": "#00BFDF",
      "dark": "#00BFDF"
    }
  }
}
```

- добавить виджет заголовка с картинкой и буковой “i”:
    - добавляем виджет `header_with_image`
    - поля виджета (в объекте `payload`):
        - `right_button_image` - опционально, картинка правой кнопки, где будет
          передаваться картинка с буквой "i"
        - `right_button_deeplink` - опционально, диплинк правой кнопки
        - `image_url` - url основной картинки. Может содержать поля `{w}` и
          `{h}` для подстановки ширины и высоты. Может быть типа SVG (для
          корректного отображения векторной картинки)
        - для десктопа: опциональное поле `background_color` в виде объекта:
          ```json
              "background_color": {
                "light": "#00BFDF",
                "dark": "#00BFDF"
              }
          ```
    - на нативках поле "background_color" поддерживать не нужно
    - если не пришло `right_button_image` ИЛИ `right_button_deeplink` - не
      показываем правую кнопку вообще

## Как будет работать
- добавляем следующую конфигурацию виджетов:
    - виджет `separator`
    - виджет `header_with_image`
        - картинка: лого еды + "Еда х Маркет" + экспресс, и диплинком на фулскрин
        - зависит: от любых виджетов `places_multiline` из этого блока
    - виджет `places_multiline`:
        - размер заголовка: `small`
    - виджет `places_multiline`:
        - размер заголовка: `small`
    - виджет `separator`
