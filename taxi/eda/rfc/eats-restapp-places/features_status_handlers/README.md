# Добавляем ручки со статусами фичей для плейсов

Тикет: [https://st.yandex-team.ru/EDAPARTNERS-2075](https://st.yandex-team.ru/EDAPARTNERS-2075)

Добавляются две новаых ручки
- /features-statuses-by-places - по заданному набору плейсов (или по всем плейсам парнера) вернет список статусов фич, группировка по плейсам.
- /features-statuses-by-features - по заданному набору плейсов (или по всем плейсам парнера) вернет список статусов фич, группировка по фичам.

Спека ручек в файле api.yaml
