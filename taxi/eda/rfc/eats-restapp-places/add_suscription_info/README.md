# Добавляем информацию о подписках в ручку search

Тикет: [https://st.yandex-team.ru/EDAPARTNERS-1821](https://st.yandex-team.ru/EDAPARTNERS-1821)

Добавилось поле subscription, в котором есть tariff_type, is_trial, need_alerting_about_finishing_trial, valid_until_iso
