## Задача
Быстро проверить, что добавление кастомных каруселей в карусели повысит заказы из них.

Тикет: https://st.yandex-team.ru/RETAILDEV-1564

## Предлагаемый вариант (доработки только на фронтах)
- добавить в эксперимент `eats_shops_home_carousels` поле:
    ```json
    {
        // ...
        "categoriesCarouselIncludesBanners": true
    }
    ```
- если это поле проставлено в `true`, то клиентские приложения в запрос в `get-categories` добавляют еще кастомные категории из ответа `menu/goods` (определять по признаку `"show_in" = ["banner_carousel"]`), в следующем порядке:
    1. кастомные категории
    2. все остальные категории (в порядке как сейчас)
