# Подзаголовок для баннера категории

## Разработка
Текст и цвет подзаголовка должен приходить с бэкенда и должны быть конфигурируемыми. Текст и цвет подзаголовка будет настраиваться в конфиге категорий EATS_PRODUCTS_CATEGORIES_SETTINGS.

### Бэкенд

#### Добавление в конфиг параметров subtitle и subtitle_color

Добавить в конфиг EATS_PRODUCTS_CATEGORIES_SETTINGS опциональный параметр subtitle строкового типа и опциональный subtitle_color. subtitle_color является массивом элементов типа ColorTheme.

ColorTheme:
|name|type|
|----|----|
|color|string|
|theme|enum(light, dark)|

#### Возвращать подзаголовок из конфига в ответе сервера
Добавить опциональное поле subtitle строкового типа и опциональное поле subtitle_color. subtitle_color является массивом объектов типа ThemedColor. Если поле не задано в конфиге - оно не возвращается.

# Цвет текста для категории

## Разработка
Цвет текста для каждой темы должен приходить с бэкенда и должен быть конфигурируемым. Цвет настраивается в конфиге категорий EATS_PRODUCTS_CATEGORIES_SETTINGS. В случае если для subtitle цвет не задан, то будет использоваться text_color.

### Бэкенд

#### Добавление в конфиг параметра text_color

Добавить в конфиг EATS_PRODUCTS_CATEGORIES_SETTINGS опциональный параметр text_color. Который будет являться массивом элементов типа ColorTheme.

#### Возвращать цвета из конфига в ответе сервера
Добавить опциональное поле text_color, которое является массивом объектов типа ThemedColor. Поле не будет возвращаться в случае, если для категории в конфиге не задан цвет текста.
