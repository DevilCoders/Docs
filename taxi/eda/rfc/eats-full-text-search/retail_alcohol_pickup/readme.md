### Описание
Хотим отображать алкоголь в поиске на главной, но при этом картинку замыливать.

Когда пользователь кликает алкогольный напиток c замыленной картинкой в поиске, его перекидывает в магазин - там пользователю покажется предупреждение про 18+, а затем, если пользователю есть 18 лет - то и все алкогольные товары.

### Реализация
В ответе ручки /eats/v1/full-text-search/v1/search в объекте описывающем [найденный товар в глобальном поиске](https://a.yandex-team.ru/arc/trunk/arcadia/taxi/uservices/services/eats-full-text-search/docs/yaml/api/api.yaml?rev=r9301850#L361) добавить свойство adult: 
```
  adult:
      type: boolean
      description: Товар с возрастным ограничением
``` 

В этом свойстве пробрасывать признак алкогольности.

На бэкенде это поле требуется брать из ответа ручки сервиса eats-nomenclature `/v1/products/info` (в ответе этой ручки уже есть такое поле).

На фронтах, если свойство adult пришло со значением true, то требуется замылить картинку.
