### Текущее состояние
- При поиске по магазину бейджик скидки не отображаются. При поиске с главной страницы бейджик скидки отображаются.
- В ответе ручки ``/eats/v1/full-text-search/v1/search`` - SearchResponse.blocks.payload(PlacesBlock).items есть поле ItemPreview, содержащее необязательное price_discount_value.
- Поле price_discount_value используется фронтами для отображения бейджа скидки при поиске с главной странице поиска.

### Что требуется
Что делаем сейчас:
- Со стороны бека нужно добавить опциональное поле price_discount_value в SearchResponse.blocks.payload(ItemsBlock).items.
- Новое поле price_discount_value должно быть полностью идентично уже существующему полю price_discount_value.
- Со стороны фронтов потребуется использовать новое поле price_discount_value для отображения бейджа скидки при поиске в магазине (по аналогии с поиском с главной страницы).
