### Описание
В рамках задачи по продаже алкоголя на самовывоз в алкомаркетах требуется показывать модальный диалог с просьбой подтвердить возраст, указанием условий покупки товаров и ссылкой на страничку с лицензиями.

### Ожидаемые изменения
Для таких магазинов с бэкенда в ответе ручки slug, в поле foundPlace будет приходить дополнительное поле. Объект с такой структурой:
```
  alcohol_dialog:
      type: object
      description: Источник данных для диалога подтверждения возраста
      additionalProperties: false
      required:
        - title
        - text
        - description
      properties:
          title:
              description: Заголовок окна
              type: string
          text:
              descriprion: Просьба подтвердить возраст
              type: string
          description:
              description: Правила покупки товаров в магазине. Может содержать текст с html тегами для экранирования ссылок.
              type: string
```

Данные для заполнения этого поля должны браться из Танкера по ключам из конфига. Конфиг нужно создать в рамках этой задачи. Схема конфига прилагается ниже.

Отправляемые с бэкенда поля title и text должны содержать обычный plain text для заполнения полей. 
Данные в поле description должны содержать текст с html тегами для корректного отображения ссылки (ссылок) на страницу с лицензиями.
На фронтах нужно корректно отображать значение description - обрабатывать html теги.

#### Конфиг

Предлагаю название EATS_CATALOG_ALCOHOL_DIALOG.
Схема:
```
description: Конфиг содержащий побрендовую информацию магазинов для взрослых которую требуется дополнительно отображать пользователям.
tags: [notfallback]
schema:
    type: object
    description: Ключ - числовой id бренда (завернутый в строку), значение - данные для заполнения диалогового окна.
    additionalProperties:
        $ref: '#/definitions/Dialog'
    required:
      - __default__
    properties:
        __default__:
            $ref: '#/definitions/Dialog'
        Dialog:
            type: object
            description: Источник данных для диалога подтверждения возраста
            additionalProperties: false
            required:
              - description
            properties:
                title:
                    description: Ключ танкера, по которому хранится заголовок окна
                    type: string
                    default: <todo: тут указать дефолтное значение ключа танкера для заголовка>
                text:
                    descriprion: Ключ танкера, по которому хранится текст с просьбой подтвердить возраст
                    type: string
                    default: <todo: тут указать дефолтное значение ключа танкера для основного текста диалога>
                description:
                    description: Ключ танкера, по которому хранится текст с правилами покупки товаров в магазине. Может содержать текст с html тегами для экранирования ссылок.
                    type: string
```
