# global lock клиент
клиент для глобальных блокировок через YT

### Параметры клиента:

**_token_** - oAuth токен пользователя для API YT (по умолчанию берем из переменной окружения YT_TOKEN)

**_yt.user_** - имя пользователя

**_yt.client.proxy_** - адрес YT

**_yt.timeout.global_** - по умолчанию 500

**_yt.timeout.failover_** - по умолчанию 1000
