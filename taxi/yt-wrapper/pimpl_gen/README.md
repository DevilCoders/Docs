## Генератор pimpl-заглушек для аркадийных типов

В `generate.h` в макросе `GENERATE` перечисляются типы, для которых генерируется
header, содержащий в одном неймспейсе
- forward declarations для перечисленных типов (именно для этого макрос принимает
  два аргумента: `STRUCT` и `CLASS`
- strong typdef-ы с указанием актуальных размера и выравнивания для генерации
  FastPimpl-заглушек

Использование:
```
pimpl_gen output-file.h
```

В случае, если текущее содержимое output-file.h совпадает с результатом работы генератора,
выходной файл не будет перезаписан.

Также см документацию в `generate.h`
