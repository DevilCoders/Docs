Разные layout создаются с целью не конфликтовать с проверенными testcases из тест пальмы
Т.е. необходимо для более удобно разграничения смысла тестов

Сейчас есть такие:
deltaLayout - используется для набора: "Сетка товаров" (Тест пальма)
routerLayout - используется для тестов связанных с роутером (без кейсов пальмы)
