# Startrek

Использование:
 ```sh
startrek <action> <release_ticket_key> <version_id> 
```

Примеры:
```sh
# Привязать тикеты к версии
./startrek attach TAXIREL-51920 268216

# Закрыть тикеты в релизе и проставить «Исправлено в версии»
./startrek close TAXIREL-51920 268216
 ```

Релизные тикеты можно найти здесь:
- https://st.yandex-team.ru/issues/412406

Версии заводятся здесь:
- https://st.yandex-team.ru/LAVKAWEB/versions
