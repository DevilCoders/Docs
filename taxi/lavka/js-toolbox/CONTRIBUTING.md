# CONTRIBUTING

-   По умолчанию пакеты собираются в папку `dist/`, и указан `main` равный `dist/src/index.js`.

-   Пакеты компилируются с помощью `tsc`.

-   Для тестов используется `jest`, который тестирует **скомпилированный js** код.

-   Для сторонних зависимостей лучше использовать версию "старше" (**^**):
    `"got": "^11.8.2"` (а не `"got": "11.8.2"`)
