Для проверки типов в шаблонах можно воспользоваться библиотекой **vue-type-check**
https://github.com/Yuyz0112/vue-type-check

# Установка:
```
npm i -g vue-type-check
```

# Запуск

```
vtc --workspace ./ --srcDir ./src
```

# Возможные проблемы

1) Утилита корректно работает только с однофайловыми vue компонентами в которых

есть блок \<script lang="ts">

2) Есть проблемы при работе в функциональными компонентами
