## Расширения middleware

### `x-stall-tvm_or_token`

Данное расширение позволяет включить авторизацию для ручки в виде TVM или
по токену.

Формат тега в OpenAPI:

```yaml
x-stall-tvm_or_token:
    tokens: foo.bar.baz
    tvm: abc.cde.def
```

Значения полей:

- `tokens` - указывает на секцию в конфиге откуда взять токены для
  проверки авторизации.
  Если указано, то берётся заголовок `Authorization: (token|bearer) <значение>`
  и значение токена сверяется со значением (значениями) в конфиге.
 
 В конфиге может лежать просто строка, массив строк.
 Если строка содержит запятые - это значит она содержит несколько токенов.

- `tvm` - указывает на секцию в конфиге откуда взять настройки TVM.

#### Особенности

- если пришёл токен и он прошёл проверку - TVM не проверяется
- при проблемах доступности TVM может возвращать 504
