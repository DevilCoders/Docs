# SignalQ Admin

## [Wiki](https://wiki.yandex-team.ru/taxi/partner_products/frontend/projects/signalq-admin/)

## Development

Add `signalq-admin.localhost.yandex-team.ru` to your [hosts](<https://en.wikipedia.org/wiki/Hosts_(file)>).

```bash
git clone git@github.yandex-team.ru:taxi/superweb.git
cd superweb/@apps/signalq-admin
yarn
yarn mkcert
yarn start
```
