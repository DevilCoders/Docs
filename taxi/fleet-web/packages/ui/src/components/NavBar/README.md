# NavBar

A group of links which allow navigation between internal (via `NavBar.Link`) and
external (via `NavBar.ExternalLink`) routes.

## NavBar.Link

Component which allows navigation between inner routes, handled by `react-router`.

Based on `NavLink` component from `react-router`.

## NavBar.ExternalLink

Component which allows navigation to external pages. Based on HTML `<a>` tag with
appropriate styles applied.
