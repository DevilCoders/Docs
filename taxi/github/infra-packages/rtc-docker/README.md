Набор докерфайлов для костыльной сборки имеющихся инфраструктурных сервисов в докер-образы.

strongbox_client_poc.py - proof of concept клиента к стронгбоксу, который, впрочем, за неготовностью стронгбокса ходит в секретницу напрямую. Ожидает, что будут проброшены переменные окружения YAV_TOKEN (oauth-токен для авторизации в секретнице) и YAV_SEC_ID (айдишник секрета, у которого в поле value лежит полный текст taxi.json). Как минимум это позволяет запустить сервисы.
