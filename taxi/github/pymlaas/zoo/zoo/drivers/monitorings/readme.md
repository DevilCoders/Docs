- Может существовать только 1 конфиг в 1 дату (потому что дата - это id эксперимента).

- Если два эксперимента запустили в один день, придется положить их в 1 конфиг (с соответствующими тестовыми и контроьными группами)

- В конце названия эксперимента обязательно должна быть дата, где '-' заменены на '_'

- Если контроль фигурирет дважды (1 и тот же путь для двух тестовых групп), статистики для него посчитаются дважды
