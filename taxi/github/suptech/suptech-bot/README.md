# suptechcatbot
Селфсервис саптеха

##### Команды:
/start - инициация бота
/help - даёт справку о боте
/forgot_password_b2b - сброс пароля в трекере
/get_checks - ищет чеки по номеру телефона
/createtask - создаёт таск на саптех
/checktask - проверяет созданные таски в саптехе
/feature - позволяет оставить пожелание на автоматизацию и не создавать при этом таск
/yql_select - позволяет выгружать дополнительные данные о водителе
/bugtracker - позволяет обновить и выгрузить данные по багам для багхантеров
/tagging - добавление водителей/клиентов в таблицы для тегирования бекендом
/user_rating - проверка рейтинга и отзывов на клиента
/get_chat_id - отображение id чата


#### Организация проекта
Проект разделён на несколько папок по теме.

**configurations** содержит конфиги для ботов: 
* config_tree.py - класс с инициацией модулей ручек Трекера, стаффа. Импорт конфига с кнопками и текстами
* keys.py - методы для вызова ключей из секретов кулауда.
* logger.py - один логгер, чтоб править всеми.

**bot_handlers** содержит хендлеры для ботов, каждый файл представляет собой одну команду.
При добавлении новой команды в бот, помимо файла в bot_handlers, нужно добавить imprort первичной функции команды в __init__

**functions** содержит все функции, используемые в проекте. Обработка файлов. Запуск кверей. Создание задач и тому подобное
