#Аптеки: 
## Шаг 0
аптеки реализуются новым рестораном в еде, как следствие шагом 0 делаем:
1) **Точка входа**
шорткат с диплинком, который ведет в ресторан аптеки:
будет реализован на основе медиашорткатов
2) **Доработки**
В ручке GET /internal-api/v1/orders?external_ref=...&service=... коллеги из еды добавляют признак аптеки
```
business:
    type: string
    enum: [restaurant, store, pharmacy]
    example: pharmacy
```
на оснвое этого признака:
* необходимо вести платежи через другие терминалы в трасте (по юридическим причинам):
 * https://st.yandex-team.ru/TAXIBILLING-2985 - подключение терминалов в transactions
 * https://st.yandex-team.ru/TAXIBACKEND-28090 - со стороны payments-eda выбор нужного сервиса
* необходимо ограничивать способы оплаты
 * https://st.yandex-team.ru/TAXIBACKEND-28004 - ограничение для оплаты бейджами
