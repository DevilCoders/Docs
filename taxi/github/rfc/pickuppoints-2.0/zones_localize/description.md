* Ручка принимает набор uri для получения имен точки.

* Для каждого uri формируется при возможности три имени: zone_name, point_name, choice_name.

* Choice_name выдается при наличии у зоны более одного choice.

* При наличии у зоны choice'ов проверяется наличие у точки choice_id. При отсутствии репортим в LOG_ERROR().
