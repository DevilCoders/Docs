В totw необходимо передавать интервал подачи.

В request уже передается due с временем "начала" подачи, добавим к нему поле till_due с временем окончания
```
{
    "request":  {
        "due": "2013-04-11T19:58:24+0400",
        "till_due": "2013-04-11T20:08:24+0400",
        ...
    },
    ...
}
``` 

В заголовки ответа добавим X-Yataxi-Polling-Interval чтобы контролировать частоту полинга и задавать ее, например,
в зависимости от статуса заказа. Так сможем ограничить нагрузку от предзаказов ожидающих назначения.

Изменения для нового дизайно описаны в design_changes.md
