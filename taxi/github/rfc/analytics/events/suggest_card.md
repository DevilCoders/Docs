**Здесь описываем события относящиеся к шорткатам**

`Main.SuggestCard.Closed` - Свернули большую карточку с саджестом (бывшие баблы)
Параметры:
* CommonParams
* GridID - айдишник конфигурации сетки в выдаче саджеста
* height = center/fullscreen - высота, с которой решили свернуть карточку (стандартное положение/развернута на максимум)
