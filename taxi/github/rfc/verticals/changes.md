## изменения

###конфиг мод

* добавлено переопределение элементов интерфейса
* добавлено описание зоны для мод по зонам
* добавлено описание плашки для мод с тарифами
* добавлено поле с экспериментом, допускающим моду

###zoneinfo

* в запрос добавлен флаг modes в запрос
* в ответ модам добавлено переопределение элементов интерфейса
* в ответ модам добавлен список тарифов
* в списке тарифов сбрасываются моды на дефолт, если нет флага modes

###personalstate

* добавлено selected_mode
* добавлено описание по модам: выбранный класс и параметры мультикласса

###routestats

* в запрос добавлено selected_mode
* в запрос добавлено описание по модам: выбранные классы и параметры мультикласс
* в ответ добавлены плашки с описанием моды или класса
