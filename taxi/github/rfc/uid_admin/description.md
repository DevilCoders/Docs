Админка с информацией по UID'ам.
==========================

Нужно для того, чтобы дать информацию саппортам и разработчикам об связях между UID'ами.

Будет доступно три функционала (три ручки):
1) информация об UID (тип, phone_id, связанные фониши, привязанный портал, etc)
2) информация о связи двух UID (текстовое описание)
3) ручка развязывания фониша от портала через https://wiki.yandex-team.ru/users/andrey1931/OtvjazyvanieFonishnogoAkkauntaPoUidam/
