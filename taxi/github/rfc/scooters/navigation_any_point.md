## Задача:
https://st.yandex-team.ru/SCOOTERSMANAGER-15

## Цели:
Хотим дать возможность пользователям строить маршрут навигации до произволльной точки через клиентское приложение, а те только выбирать парковку для построение навигации.
Для этого на клиентах будет добавлен новый экран с картой, на котором пользователем с помощью передвижения пина сможет указать любую точку, в том числе парковку.
Для того:
- при приближании пина близко к парковке, пин должен притягиваться. 
- выбранные паркоки должны отображаться так, чтоб сочетались с наложением пина на них.

## API
### Предложение:

В тело запрос `layers/v2/objects` при `"state.screen": "choose_b"` в структуру `state.scooters` добавляем булевое поле `"pin_destination_selection": true`

**`Request layers/v2/objects`**
```jsonc
{
    "state": {
        ...
        "screen": "choose_b",
        "scooters": {
            "pin_destination_selection": true //флаг определяет особый режим работы для выбора произвольной точки навигации.
        }
        ...
    }
}
```

В ответе `layers/v2/object` должен быть список парковок самокатов, где в выбранном состоянии парковки должно быть изображение стилизованное под пикап-поин, чтоб на нем мог стоять пин. 
Так же в каждой парковки должны присутствовать следующие поля и объекты

**`Response layers/v2/objects`**
```jsonc
{
    "style": {
        ...
        "image": //такая же тапабельная иконка парковки как и в обычном "choose_b",
        "selected_image": //специальная иконка похожая на пикап-поинт
        ...
    }
    ...
    "properties": {
        ...
        "behavior": {
            "stick_pin": true, //включает притягивание для этой точки, при попадании пина в определённый радиус
            "stick_pin_scale": 1.2 //скейл для анимации при срабатывании притягивания.
        },
        "options": [
            {
                "on": "select",
                "action": {
                    "type": "select_as_navigation_destination" //выбор парковки для конечного пункта при построении маршрута
                }
            }
        ]
        ...
    }
    ...
}
```

### Примечание

При это предлагаяю предсмотреть возможность использовать все остальное поведение `"state.screen": "choose_b"`, а конкретно возможность предвыбора парковки сервером при указании поля `state.scooters.destination`
