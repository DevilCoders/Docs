# pickup-points-manager

## Задачи

* Загружать в базу точки из geojson по запросу из админки
* Выгружать из базы точки по BBox по запросу из админки
* Удалять по одной точке из базы по запросу из админки
* Быстро искать точки в заданном радиусе по запросу из ml (нагруженная ручка)

## Как решим?

* Ручки, работающие с админкой являются не нагруженными и ходят напрямую в базу
* Нагруженная ручка для ml работает с кэшом. База точек имеет малый размер и поместится в кэш.

## Формат данных в базе

* id точки - сгенеренный
* lon, lat - проверка на корректность
* iconCaption - имя или комментарий к точке не проверяется и может быть пустым
