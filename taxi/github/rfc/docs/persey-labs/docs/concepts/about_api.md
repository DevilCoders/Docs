# Описание API

API системы тестирования на коронавирус — это набор HTTP-методов, с помощью которых лаборатория будет взаимодействовать с системой Яндекса:

 * [получать заявки](../ref/get-buckets.md) на выполнение тестов;
 * [получать список отделений](../ref/get-labs-list.md), которые участвуют в программе;
 * [обновлять статус заявок](../ref/set-lab-status.md), например сообщать о завершении теста;
 * [сохранять штрихкоды](../ref/put-barcode.md) заявок в Яндексе.

Взаимодействие с API осуществляется по протоколу HTTPS. Сервис возвращает данные в формате JSON.

Для работы с API необходима [авторизация](authorization.md).
