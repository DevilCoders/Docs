# О проекте

«Система тестирования на коронавирус» — это одно из направлений проекта [Помощь рядом](https://help.yandex.ru/). Эта программа призвана помочь лабораториям проводить тесты
на коронавирус у пациентов на дому. В рамках этой программы:

- Лаборатория будет ежедневно получать от Яндекса список заявок на проведение теста. Для каждой заявки будет указана информация о пациенте и его адрес.

- Заявки, которые получит лаборатория, будут распределены по бригадам (сумкам) — на основе запланированных маршрутов. Маршруты по выездным точкам строятся таким образом, 
 чтобы за смену бригада могла посетить как можно больше пациентов.
 
- Яндекс поможет организовать выезды к пациентам. В назначенные дни в лабораторию будет приезжать водитель, который будет доставлять медработника к пациентам. 

## Интеграция лаборатории с системой Яндекса

В рамках проекта Яндекс предоставляет инфраструктуру: 

* [HTTP API](concepts/about_api.md) — набор [методов](ref/about.md), с помощью которых лаборатория сможет получать заявки, сообщать о готовности к тесту или обновлять статус заявок.
* Мобильное приложение «Помощь рядом» — помогает медработникам во время выезда к пациентам. 

## Как это работает

Пациенты, которые хотят выполнить тест на дому, отправляют заявку в Яндекс. Заявки группируются по территориальному признаку и распределяются по лабораториям. 

Ежедневно лаборатория запрашивает у Яндекса список заявок на текущий день. Полученные заявки лаборатория регистрирует в лабораторной информационной системе (ЛИС) и затем 
 подготавливает необходимый комплект документов. Когда документы будут готовы, лаборатория сообщает Яндексу о готовности к проведению теста. В назначенный день Яндекс присылает водителя.

Во время объезда медработник пользуется приложением «Помощь рядом» — оно показывает адреса пациентов, а также позволяет фиксировать ход выполнения заявок
 (прибыл на адрес, взял биоматериал).
 
По окончании смены водитель доставляет медработника обратно в лабораторию. Лаборатория проводит тест и сообщает результат пациенту. В Яндекс
 лаборатория сообщает только о факте завершения теста — сами результаты не передаются.

Подробнее см. в разделе [Процесс взаимодействия лаборатории с Яндексом](concepts/workflow.md).

## Как присоединиться к программе

Если ваша лаборатория выполняет тесты на коронавирус, вы можете присоединиться к программе тестирования. Чтобы принять участие:

1. Оставьте заявку на почту [covid@yandex-team.ru](mailto:covid@yandex-team.ru). С вами свяжутся наши специалисты для обсуждения деталей.
2. Подпишите договор с Яндексом.
3. После того как договор будет подписан, вы получите [доступ к API](concepts/authorization.md). Вам будут выданы *логин* и *API-ключ* — их необходимо будет
указывать во всех запросах к API. Кроме того, для лаборатории будет выдан *строковый идентификатор* — по нему необходимо будет [получать список заявок](ref/get-buckets.md).
4. Используя [API](concepts/about_api.md), настройте взаимодействие вашей ЛИС и системы Яндекса.

{% note "Примечание."%}
Способ получения авторизационных данных (логина, API-ключа, идентификатора) обсуждается индивидуально с каждой лабораторией. По умолчанию данные высылаются по электронной почте.
{% endnote %}
