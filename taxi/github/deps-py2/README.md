# Common dependencies of taxi projects on Python 2

## Installation

### Ubuntu

You can directly install `taxi-deps-py2-common` with `apt` utility:

```
sudo apt install taxi-deps-py2-common
```

### From scratch

You may want to install deps-py2 into your local virtaulenv:

```
# From local copy
pip install -i https://pypi.yandex-team.ru/simple/ .
```
