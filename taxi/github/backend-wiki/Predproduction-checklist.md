### Условия, которые службе такси необходимо выполнить для выхода в продакшен:

* заполнен [профиль][profile]. Обязательно должна быть заполнена часть профиля, которая называется Подключение. Остаться пустым может только  URL черного списка.
* отсутствуют ошибки при импорте тарифов, тарифов 1.0, профилей водителей. Импорт тарифов считается успешным, если Яндекс.Такси удалось импортировать **все тарифы** без ошибок. Импорт водителей считается успешным, если Яндекс.Такси удалось импортировать **хотя бы одного водителя** без ошибок. Статусы импортов можно смотреть на [странице импортов][imports].
* каждому импортированному тарифу присвоена категория. Для этого обратитесь к [taxitechsupport@yandex-team.ru][mailto]. Как только категории будут присвоены, они отобразятся на [странице импортов][imports] в разделах импорт тарифов.
* корректно настроена передача треков водителей. Для настройки треков необходимо внести адреса хостов, с которых будут передаваться треки, в [профиль][profile] и корректно передавать их ([документация по трекам][track_doc]). Трек передан успешно, если запрос на передачу получает ответ 200 OK, и водитель отобразился [на карте][map].
* успешно выполнены в тестовой среде все типы поддерживаемых таксопарком заказов (срочные и отложенные). Если таксопарк поддерживает пожелания клиента, он должен продемонстрировать, что умеет выполнять успешно заказы с пожеланиями.
