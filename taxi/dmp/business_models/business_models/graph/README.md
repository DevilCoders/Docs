## Описание главного конфига

  * __logs_path__: (_str_) Подпапка, где будут сохранится логи
  * __scripts_path__: (_str_) Подпапка со скриптами
  * __configs_path__: (_str_) Путь, где хранятся конфиги для моделей
  * __saved_models_path__: (_str_) Подпапка, где будут сохранится модели
  * __default_configurations__: (_list of dicts_) Список конфигураций по умолчанию — если внутри вершины
  графа не указан свой список конфигураций. Конфигурация задает общие параметры одного предсказания для вершины графа, в каком скейле,
  на сколько точек предсказывать и тип предсказания. Для каждого элемента списка формируется свой результат
  
    [
    * __scale__: (_str_)
    * __ndots_predict__: (_int_)
    * __type__: (_str_)
    
    ]
    
  * __graph__: (_dict_) Граф выполнения операций: задает вершины и их последовательность выполнения
    * __node1__: (_int_) приоритет вершины
    * _etc._

  * __node1__: (_dict_)
    * __source_tables__: (_dict_) Словарь с источниками данных, которые необходимо загрузить в менеджер данных для выполнения
        * __info__: (_dict_) Название таблицы
            * __method__: (_str_) Метод _reader_'a, который будет использоваться для скачивания
            * __name__: (_str_) Параметры метода
            * __scale__: (_str_) Параметры метода
            * __etc__:
        * _etc._
        
    * __configurations__: (_list of dicts_) Конфигурация задает общие параметры одного предсказания для вершины графа, в каком скейле,
  на сколько точек предсказывать и тип предсказания и величины, которые мы хотим предсказать. Для каждого элемента списка формируется свой результат. 
  Если __configurations__ не задан, то используется __default_configurations__, но необходимо указать __predicted_values__ на уровне ключа __node__
        
        [
        * __scale__: (_str_)
        * __ndots_predict__: (_int_)
        * __type__: (_str_)
        * __predicted_values__: (_dict_)
          * __supply_hours__: (_str_)
          * _etc._
          
        ]
          
    * __model__: (_str_) Название модели-наследника __Model__
    * __model_parameters__: (_str_) Путь к конфигу модели, файл с параметрами инициализации, фита и форкаста
    * __output_tables__: (_dict_) Указываются таблицы, куда будет выгружаться предсказания
      * __forecast__: (_dict_) Параметры для метода _runner.upload_forecast_, если пустой словарь, то не выгружается
      * __fact__: (_dict_) Параметры для метода _runner.upload_forecast_, если пустой словарь, то не выгружается
      * __table__: (_dict_) Параметры для метода _runner.upload_raw_table_, если пустой словарь, то не выгружается
  
  * __node2__
  * __etc.__
  
  ## Пример конфига модели (MultipleConfigs)
  
  
* __init__: Параметры для инициализации модели
    * __prediction_model__: _CohortModel_
    * __newbie__: _@new_users_ Имя таблицы в менеджере данных
    * __prediction_model_configs__: _forecast_configs/CohortModel/demand_
    * __newbie_name": _users_
* __fit__: Параметры для метода _fit_
    * __data__: _@info_ Имя таблицы в менеджере данных
    * __key_name__: _visited_users_
    * __value_name__: _sessions_wo_geo_break_
* __forecast__: Параметры для метода _forecast_
    * __data__: _@info_ Имя таблицы в менеджере данных
    * __key_name__: _visited_users_
