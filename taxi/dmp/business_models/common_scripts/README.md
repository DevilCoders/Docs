# Описание данных
Папка для обмена полезными скриптами, проверяющими гипотезы или использующими библиотеку, но не являющимися ее частью.

## Как пользоваться?
Сделал что-то полезное -> добавил скрипт (с адекватным названием) -> добавил небольшое описание сюда -> профит!

## Что есть?

**module_test.ipynb** - небольшие примеры использование первой версии библиотеки (когда она еще называлась predictions)

**demand_vs_supply_regression_(not_final).ipynb** - первые шаги в поиске производственной функции (поездки через demand и supply)
