# mention-bot
Telegram бот для вызова пользователей в чат по хэштегу.

### **Функционал** 
Бот - @UserGroupBot

Используйте #Название_группы что бы призвать всех пользователей группы.
Добавив тэг #randomN можно призвать N случайных пользователей из группы.

**Команды:**

/members <i>название_группы</i> - отображает состав группы
/chat_id - отображает id чата
/groups - отображает список всех групп в чате

**Команды администратора чата:**
/add_user_to_group <i>пользователь</i> <i>название_группы</i> - добавляет пользователя в группу. Можно добавить несколько пользователей, прописав их через "," или ";"
/remove_user_from_group <i>пользователь</i> <i>название_группы</i> - удаляет пользователя из группы. Можно удалить несколько пользователей, прописав их через "," или ";".'
