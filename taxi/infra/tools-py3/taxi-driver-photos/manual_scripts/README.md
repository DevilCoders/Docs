# Ручная загрузка фото в MDS S3 (временное решение)

Скрипт берет фото из папки загрузки, обрезает их и помещает в папку выгрузки, и потом загружает в S3 MDS

Работает в 8 потоков

## Настройка s3cmd
1. Получаем доступ к s3, namespace driver-photos (спросить у Жени Кацевмана, он может дать доступ, либо обратиться к админам MDS)
2. Далее поставить и настроить https://s3tools.org/s3cmd (или через brew install)
3. `s3cmd` клиент должен быть настроен глобально (раньше было: лежать в `../s3cmd-2.0.2/s3cmd` по отношению к скрипту)


## Запуск скрипта
`python3 process_and_upload.py ПАПКА_С_ФОТО OUTPUT_ПАПКА СОЛЬ ФАЙЛ_СО_СПИСКОМ_ВУ`
