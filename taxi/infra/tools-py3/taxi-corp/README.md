# Corp Dev Tools

Папка indexes содержит автоматически сгенерированные скрипты миграции индексов.
Это сделано для упрощения ревью, чтоб видеть какие индексы добавили/удалили.

Папка pd_migrations содержит одноразовые скрипты для миграции на сервис персональных данны.

Папка st содержит одноразовые скрипты с тикетом в названии.
Префикс **t** - очередь TAXIBACKEND, префикс **c** - очередь CORPDEV.

В корне находятся регулярные скрипты:
* delete_by_ids - удаляет записи по id из указанной коллекции
* delete_clients - удаляет клиентов и связанные с ними сущности
* drive_groups_change_client_id - перепривязывает дескрипшены с одного клиента на другого
* get_balance - выводит баланс по списку сервисов и контрактов
* set_client_features - изменяет фичи у списка клиентов
* eats_promocodes_unset - отвязывает промокоды еды
* set_is_auto_activate_to_clients - изменяет services.taxi.is_auto_activate у списка клиентов
* reschedule_stq_tasks – перезапланировать stq-таски
