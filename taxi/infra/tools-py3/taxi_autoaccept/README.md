# autoaccept tools
## Скрипт для редактирования списка тумблеров

#### Добавление нового тумблера

Добавляет тумблер с именем NAME:
```
driver_options.py create-switch --name NAME
```
При успешном выполнении, скрипт выведет идентификатор созданного тумблера.

По умолчанию, созданный тумблер видим для водителей. Если требуется создать невидимый тумблер, нужно указать опцию ```-vF```:
```
driver_options.py create-switch --name NAME -vF
```
Создать отмеченный по умолчанию тумблер можно добавив опцию ```-с```:
```
driver_options.py create-switch --name NAME -c
```
Создать недоступный для изменения водителями тумблер можно добавив опцию ```-eF```:
```
driver_options.py create-switch --name NAME -eF
```
Все опции создания тумблера можно комбинировать в одной команде в любом порядке.
#### Просмотр существующих тумблеров
```
driver_options.py list
```
#### Изменение свойств тумблера
Чтобы изменить свойства тумблера, понадобится его идентификатор. Идентификатор выводится при создании тумблера командой ```create-switch``` и при просмотре существующих тумблеров командой ```list```.
Чтобы изменить имя тумблера:
```
driver_options.py update-switch --id ID --name NEW-NAME
```
Сделать тумблер видимым для водителя:
```
driver_options.py update-switch --id ID -v
```
Сделать тумблер невидимым для водителя:
```
driver_options.py update-switch --id ID -vF
```
Сделать тумблер доступным для изменения в интерфейсе Про:
```
driver_options.py update-switch --id ID -e
```
Сделать тумблер недоступным для изменения в интерфейсе Про:
```
driver_options.py update-switch --id ID -eF
```
Сделать тумблер отмеченным по умолчанию:
```
driver_options.py update-switch --id ID -с
```
Снять отметку по умолчанию для тумблера:
```
driver_options.py update-switch --id ID -сF
```
#### Изменение порядка отображения тумблеров
Про выводит тумблеры упорядоченно. Чтобы переместить тумблер с идентификатором ID в позицию перед тумблером с идентификатором NEXT-ID:
```
 driver_options.py update-switch --id ID --next-id NEXT-ID
```
Для перемещение тумблера с идентификатором ID в конец списка:
```
driver_options.py update-switch --id ID -x
```
Все опции редактирования тумблера можно комбинировать в одной команде в любом порядке.
#### Удаление тумблера
```
driver_options.py delete --id ID
```
**Удаление - необратимая операция. При удалении, все манипуляции водителей с тумблером также удаляются.**
