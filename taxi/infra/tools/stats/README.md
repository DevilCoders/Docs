stats
=====

* [calc_vs_real_cost.py](calc_vs_real_cost.py) - generate statistics on average calculator cost error.

* [coupon_orders.py](coupon_orders.py) - aggregates coupon order by park for given period (initially, [TAXIBACKEND-1812](https://st.yandex-team.ru/TAXIBACKEND-1812)).

* [get_chargeback_orders.py](get_chargeback_orders.py) - script for getting chargebacks.

* [parks_orders.py](parks_orders.py) - script for get statistics about how many orders was and how many coupon orders (compensations in regions).

* [platform_dist.py](platform_dist.py) - build distribution for platforms (Android, iPhone etc) by date for orders completed in last 45 days.

* [ride_report_failure.py](ride_report_failure.py) - calculate ride reports failure rates for [TAXIBACKEND-2969](https://st.yandex-team.ru/TAXIBACKEND-2969).

* [yt_ratings_distribution.py](yt_ratings_distribution.py) - various distributions related to new ratings ([TAXIBACKEND-2989](https://st.yandex-team.ru/TAXIBACKEND-2989)).

* [print_cpp_timings.py](print_cpp_timings.py) - print timings of taxi-protocol

* [yt_service_interaction.py](yt_service_interaction.py) - calculates a graph of HTTP interaction between production hosts based on YT taxi-access-log data ([TAXICOMMON-1329](https://st.yandex-team.ru/TAXICOMMON-1329))
