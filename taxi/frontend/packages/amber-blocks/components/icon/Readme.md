Иконки

Иконки рядом с текстом могут быть выравнены через параметр `valign`, в остальных случаях рекомендуется способ 

```jsx
const Exit = require('../icon/Call').default;
<React.Fragment>
    <div style={{fontSize: '12px', lineHeight: '48px', display: 'flex', alignItems: 'center'}}>
        <Exit/><span>Это маленький текст</span>
    </div>
    <div style={{fontSize: '30px', lineHeight: '48px', display: 'flex', alignItems: 'center'}}>
        <Exit/><span>Это большой текст</span>
    </div>
</React.Fragment>
```
