# Package @yandex-taxi/fintech-common-server

Пакет общих компонентов используемых при интеграции с, или развертывании в финтехе.

## Использование

```bash
~$ npm install --save @yandex-taxi/fintech-common-server
```

## Паблишинг
1. Меняем код;
2. npm run test
2. Инкрементим версию в package.json;
3. npm publish;
