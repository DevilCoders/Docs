# Автоматически маунтить нужные папки по включению систему

**Проблема:** при перезагрузке нужно руками маунтить арк. А если папок намаунчено много было, то руками запускать скрипты неудобно. Поэтому можно сделать скрипт, который маунтит что нужно и куда нужно, а потом настроить автоматический запуск скрипта на login через Automator (для MacOS) или через profile.d (для unix-жителей).
