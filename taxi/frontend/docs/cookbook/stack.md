# Общие правила фронтенд сервисов такси

Обязательно: `typescript`
Рекомендовано:` react, redux, nodejs, exressjs, nginx, npm, webpack`
Допустимо: `effector, nestjs`
Библиотека компонент: [go-web-components](https://a.yandex-team.ru/arc/trunk/arcadia/taxi/frontend/packages/go-web-components)

Проект должен удовлетворять [требованиям](https://oko.yandex-team.ru/school/common) базовой [школы разработки яндекса](https://clubs.at.yandex-team.ru/verstka/6370)
