## Назначение

Здесь лежит генерилка типов, которая из бэкендных .yml файлов генерирует тайпинги для typescript.
Вероятно, стоило бы перевести весь проект на что-то подобное. Единственный вопрос - это что лучше -
класть все тайпинги в одно место или раскидывать по проектам, в которых они используются.
