# `@logistics-frontend/client-core`


## Назначение

Здесь лежит код, который зависит от сущности "корпоративный клиент" и авторизации в корп кабинете ради переиспользования в
более прикладных вещах, которым надо работать с этими сущностями. Не надо добавлять зависимость от этого пакета туда, 
где понятия "текущего клиента" не существует, например, в @yandex-taxi/public или @logistics-frontend/core.
