# Вебвью: служба поддержки для yaproxy
Адрес: /webview/yaproxy/help

**Тестовые параметры**
- Логинимся в тестовом пасморте
- Выставляем заголовок x-yataxi-userid (значение берём из поля sessionid в приложении)
- Идём в проксю https://ya-authproxy.taxi.tst.yandex.ru (проксю надо настроить на свой стенд, ссылка ниже)

Настроить урлы прокси можно [тут](https://tariff-editor.taxi.tst.yandex-team.ru/dev/configs/edit/YA_AUTHPROXY_ROUTER_RULES_2?name=ya_aut)
