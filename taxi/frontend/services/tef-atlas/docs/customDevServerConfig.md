# Собственная конфигурация дев-сервера

При запуске дев-сервера вебпака иногда возникает необходимость изменить 
конфигурацию, используя переменные, специфичные для конкретного окружения 

Для того чтобы это можно было сделать не меняя файлы, находящиеся 
в системе контроля версий, нужно создать или изменить уже существующий файл 
`webpack/env.config.js`

Он экспортирует две конфигурации сервера:
 
- localServerConfig – конфиг сервера, который обычно запускается на локальной машине
- virtualServerConfig – конфиг сервера, который обычно запускается на виртуальной машине

Если файла с конфигом нет, то будет использована дефолтная настройка из конфига вебпака

### Пример файла `webpack/env.config.js`:

```javascript
// Изменение настроек локального дев-сервера
exports.localServerConfig = {
    port: 9999, // Переопределение порта
    server: {
        options: {
            cert: 'another-cert.pem' // Переопределение глубоко вложенных свойств
        }
    },
    additionalOption: true // При необходимости можно добавить другие опции
};

// Изменение настроек виртуального дев-сервера
exports.virtualServerConfig = {
    ipc: `path/to/custom-socket.sock`
};
```
