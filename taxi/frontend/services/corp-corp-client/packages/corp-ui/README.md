## Назначение

Здесь лежат все визуальные компоненты КК, которые ничего не знают о самой логике КК и которые
можно повторно использовать. Всё, что узнаёт о специфике КК (клиенте, юзере, заказе и прочем) -
стоит класть в отдельные проекты вроде @yandex-taxi/corp-client-core и т.д.
