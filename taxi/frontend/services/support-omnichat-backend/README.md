## Установка и разработка

### Предварительная настройка

скачать [файл настроек локального TVM демона](https://yav.yandex-team.ru/secret/sec-01g1g29bes5smmtz6zpg7e35a3/explore/versions) и положить в корень домашней папки пользователя с названием .tvm-support-omnichat-backend (`~/.tvm-support-omnichat-backend.json`)

### Запуск

`cd services/support-omnichat-backend/`

`npm run install`

`npm start`

## Сборка

`npm run build`
