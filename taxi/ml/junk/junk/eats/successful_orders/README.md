## Предсказание вывоза заказов  (или отмены из-за нехватки курьеров)

Задать пусть всех файлов нельзя, все собирается сюда
`//home/taxi_ml/dev/eats/successfully_orders/mvp/`

В `constants.py` указываются нужные даты, после этого можно запустить 
`feature_generation.py`, который подготовит все данных для обучения
и поделит выборку на train, val, test. Здесь же можно посмотреть на то,
какие фичи и как генерируются. 

В `train.py` задаются параметры моделей, которые нужно обучить. Можно задать
сразу несколько вариантов. `.cbm` не сохраняются, но предсказания моделей на тестовой выборке
сохраняются в основной папке.

В `Cancellations Metrrics.ipynb` есть минимальный код для загрузки предсказаний.\
Чтобы не грузить полностью датасет с фичами, руками (через YQL) был собран небольшой датасет
с фичами типа время заказа в `//home/taxi_ml/dev/eats/successfully_orders/mvp/training_short`
