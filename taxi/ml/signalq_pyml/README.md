### SignalQ PyML

Это модуль из которого собирается пакет для камеры `SignalQ`, он является сабмодулем в репозитории https://bb.yandex-team.ru/projects/TAXI/repos/signalq2/browse сборка этого модуля осуществляется в среде того проекта.

`requirements.txt` тоже свои, так как тестовая и прод среда - arm девайсы и катить все из общих `requirements.txt` не получится,
`requirements.txt` из этого пакета доустанавливаются в окружении основной среды

`projects/projects/nirvana` — папка с кодом регулярных процессов, например для загрузки пулов и выгрузки разметки в Толоке.

`projects/projects/offline` — папка с кодом для запуска на зомбиках с и (возможно) нирване с GPU,
для работы могут понадобиться сторонние библиотеки с лицевымии детекторами и другими CV алгоритмами, например `FaceEngine`

`signalq_pyml` — папка с кодом, который запускается и офлайн и на камере
