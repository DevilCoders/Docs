Словарь аргументов подклассов `Processor`:
* размер временной истории – `window_length`
* размер истории, ограниченной количеством элементов – `history_size`

Рекомендации по разработке:
* порядок `head_pose` – `roll`, `pitch`, `yaw`
* порядок `gaze` – `pitch`, `yaw`

Нейминг модулей:
* стараемся убирать redundancy из названий (не было дублирования на разных уровнях)

Нейминг функций:
* `(verb)_(instance_to_return)_(Optional[instance_to_act_on])`

Структура классов:
* Публичные методы вниз
* Объявления приваатных рядом с их использованием
