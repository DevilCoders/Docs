# Авторизация

## По TVM
Мы начнем с простой авторизации - по TVM. Где-то внутри нашей системы будет жить конфиг, который описывает какие TVM имеют доступ к каким пайплайнам.

```yaml
tvm-auth:
    taxi:
      - 12345
      - 56789
    scooters:
      - 86501
admin:
  - 50876
  - 459002
```

Записи в tvm-auth - это указания для каждого пайплайна кто имеет к нему доступ.
Записи в admin - это TVM которые имеют доступ ко всему. Это для админки где мы пока будем редактировать конфиги.
