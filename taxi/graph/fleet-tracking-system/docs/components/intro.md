## Основные компоненты системы

Внутри, у нас есть 4 основных компонента:
1. Yagr - входная точка для пользовательских каналов
2. Yaga - притягиватель и предсказатель
3. Internal-trackstory - сервис для хранения шорттреков
4. Yaga-metrolog - вспомогательный сервис для логирования притянутых позиций в
   Yt и для сбора сложной статистики.
