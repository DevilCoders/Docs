Утилита check_backup умеет делать следующее:

- сравнивать два дампа (dump_fullarc_to_yt) и определять по ним, корректен ли бекап;
- находить по сервису (service+ctype) детали конфигурации: как называется ферриман, путь к последнему снапшоту (бекапу) и проч.;
- выполнять большой "батч" с отщеплением последнего стейта в YT и сравнением его с последним бекапом во многих сервисах сразу

Пожалуйста, прочтите документацию на вики: https://wiki.yandex-team.ru/users/yrum/checkbackup/
