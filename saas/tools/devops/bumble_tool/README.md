### Bumble creation tool
You can provide all tokens separately or get ALL_IN_ONE token [here](https://oauth.yandex-team.ru/authorize?response_type=token&client_id=7999105bc18c4a52b47afddb9ce12520)
and provide it via `BUMBLE_TOKEN` environment variable or `~/.bumble/token` file.

To add new scopes to token edit passport [application](https://oauth.yandex-team.ru/client/7999105bc18c4a52b47afddb9ce12520)


After new CTYPE creation, deploy zerodiff on [DM](https://nanny.yandex-team.ru/ui/#/services/catalog/saas_production_deploy_manager/)
