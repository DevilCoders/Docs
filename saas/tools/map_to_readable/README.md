## `map_to_readable`

### Что это?
Инструмент для того, чтобы превратить таблицу ферримана или таблицу с документами (например, такая появляется на промежуточном этапе работы `standalone_indexer`) в человекочитаемую таблицу, по которой также удобно запускать разного рода запросы.

### Как запустить?
* `./map_to_readable $PROXY $TABLE_TO_READ $DST_PATH [$ZONE_VIEW_PATH]`
  В качестве `$TABLE_TO_READ` может выступать таблица из стейта ферримана (`.../ytpull/123456789.full`), может выступать временная таблица `standalone_indexer` (`.../processed.docs`). Также годится любая таблица, в которой в колонке document записан протобуф `NRTYServer.TMessage.TDocument` или в колонке value записан протобуф `NRTYServer.TMessage`. 
  Если нужно посмотреть глазами на зоны, можно их вывести в отдельную таблицу, дописав последний аргумент (будет url и текстовое представление зоны).
