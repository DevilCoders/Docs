## Базовая школа фронтенда

Описывает технологии, обязательные для использования во всех сервисах Яндекса (в контексте кода, выполняемого в браузере).

### Фронтенд технологии

|||
| -------- | ------ |
| Язык | TypeScript / JavaScript |
| Шаблоны | React |
| Пакетный менеджер | NPM ||
