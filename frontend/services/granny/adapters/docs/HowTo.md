## Нужно реализовать новый сниппет/колдунщик на бабуле. Что делать ?

Общие принципы идентичны web4. Детали можно посмотреть [в доке для web4.](http://docs.serp.yandex.ru/docs/HowTo)(За исключением пункта "Если данные сниппетные", т.к. в бабуле нет этих механизмов, можно писать на Конструкторе только для данных из ``searchdata.docs.*.construct``).

## Важные отличия от web4

- В бабуле нет legacy адаптера(Дока про легаси лежит [здесь](http://docs.serp.yandex.ru/docs/Adapters/#-legacy). Пункт "Адаптер Legacy")
- Конструктор работает только на данных в `doc.construct`.
- Список поддерживаемых блоков конструктора в бабуле можно посмотреть пока только здесь **`https://github.yandex-team.ru/serp/granny/tree/dev/construct/blocks-common`**
