## Основные параметры

    {
        "content_type": "accordion",
        "content": [
            {
                "title": "Основные возможности",
                "content": <Markup>
            },
            {
                "title": "Особенности",
                "content": <Markup>
            }
        ]
    }

## Дополнительные параметры

### Expanded

Открывает или закрывает раздел. По-умолчанию `false`

    {
        "content_type": "accordion",
        "content": [
            {
                ...,
                "expanded": boolean
            }
        ]
    }
