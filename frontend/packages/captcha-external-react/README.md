# captcha-external-react

Пакет с React-обвязкой над SmartCaptcha.

## Разработка

1. Установка зависимостей:

```bash
npm ci
```

2. Запуск:

```bash
npm start
```

## Публикация

1. Сборка проекта

```bash
npm run build
```

2. Подготовка сборки к публичной выкладке

```bash
npm run prepare:public
```

3. Выкладка

```bash
cd build && npm publish
```
