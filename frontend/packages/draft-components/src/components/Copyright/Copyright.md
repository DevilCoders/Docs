# Copyright

<!-- description:start -->
Компонент копирайта.
<!-- description:end -->
## Способы подключения
```ts
// src/App.ts
import React from 'react'
import { Copyright } from '@yandex-lego/draft-components/Copyright'

const App = () => {
  return (
    <Copyright start={1997} />
  )
}
```
## Примеры использования

### Даты

За год начала отвечает свойство `start`.

{{%story::desktop:lego-draft-components-copyright-desktop--dates%}}

### Сервис

Если данные о вашем сервисе есть в пакете [@yandex-lego/services](https://github.yandex-team.ru/search-interfaces/frontend/tree/master/projects/lego/packages/services), то вы можете добавить информацию о нем в компонент.

```ts
import { serviceName, serviceUrl } from '@yandex-lego/services/lib/cjs/index';

const name = serviceName(service, contentRegion);
const serviceHref = serviceUrl(service, contentRegion);
...
<>
  <Copyright url={serviceHref} linkText={name} {...props} />
</>

```
