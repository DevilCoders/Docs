### Доступные спецификаторы
`var date = new Date(2014, 2, 4, 12, 12, 12); // 04.03.2014 12:12:12`

## strftime

`%a` - Сокращенное название дня недели
```javascript
    strftime('%a', date); // вт
```

`%A` - Полное название дня недели
```javascript
    strftime('%A', date); // вторник
```

`%b` - Аббревиатура названия месяца в РП
```javascript
    strftime('%b', date); // мар, но мая
    strftime('%~b', date); // мар, но май в ИП
```

`%B` - Полное название месяца в РП
```javascript
    strftime('%B', date); // марта
    strftime('%~B', date); // март, модификатор переводит в ИП
```

`%c` - Предпочитаемое отображение даты и времени, в зависимости от текущей локали, %Y-%m-%d %H:%M:%S
```javascript
    strftime('%c', date); // 2014-03-04 12:12:12
```

`%C` - Двухзначный порядковый номер столетия (год, деленный на 100, усеченный до целого), 19 для 20-го века
```javascript
    strftime('%C', date); // 20
```

`%d` - Двухзначное представление дня месяца (с ведущими нулями), От 01 до 31
```javascript
    strftime('%d', date); // 04
```

`%D` - Дата в виде месяц/день/год
```javascript
    strftime('%D', date); // 03/04/14
```

`%e` - День месяца в виде десятичного числа (1-31) в двух-символьном поле, с ведущим пробелом, если он состоит из одной цифры
```javascript
    strftime('%e', date); // '_4'
```

`%F` - Дата в виде "год-месяц-день" (ISO 8601: %Y-%m-%d)
```javascript
    strftime('%F', date); // 2014-03-04
```

`%g` - Двухзначный номер года в соответствии со стандартом ISO-8601:1988
```javascript
    strftime('%g', date); // 14
```

`%G` - Полная четырехзначная версия %g (ISO 8601:1988 week-based year)
```javascript
    strftime('%G', date); // 2014
```

`%h` - Сокращенное название месяца, псевдоним %b
```javascript
    strftime('%h', date); // мар
```

`%H` - Двухзначный номер часа в 24-часовом формате, с ведущим нулем если он состоит из одной цифры. От 00 до 23
```javascript
    strftime('%I', new Date(2014, 2, 4, 5)); // 05
```

`%I` - Двухзначный номер часа в 12-часовом формате,  ведущим нулем если он состоит из одной цифры. От 01 до 12
```javascript
    strftime('%I', new Date(2014, 2, 4, 5)); // 05
```

`%j` - Порядковый номер дня в году, 3 цифры с ведущими нулями, От 001 до 366
```javascript
    strftime('%j', date); // 063
```

`%k` - Двухзначное представление часа в 24-часовом формате, с пробелом перед одиночной цифрой. От '_0' до 23
```javascript
    strftime('%k', new Date(2014, 2, 4, 5)); // '_5'
```

`%l` - Час в 12-часовом формате, с пробелом перед одиночной цифрой,  От ' 1' до 12
```javascript
    strftime('%l', new Date(2014, 2, 4, 5)); // '_5'
```

`%m` - Двухзначный порядковый номер месяца, От 01 (январь) до 12 (декабрь)
```javascript
    strftime('%m', date); // 03
```

`%M` - Двухзначный номер минуты с ведущим нулем. От 00 до 59
```javascript
    strftime('%M', new Date(2014, 2, 4, 5, 6)); // 06
```

`%n` - Символ перевода строки ("\n")

`%p` - 'AM' или 'PM' в верхнем регистре, в зависимости от указанного времени, AM для 00:31, PM для 22:23

`%P` - 'am' или 'pm' в зависимости от указанного времени, am для 00:31, pm для 22:23

`%r` - Время в 12 часовом формате. Тоже что и "%I:%M:%S %p"
```javascript
    strftime('%r', new Date(2014, 2, 4, 12, 6, 6)); // 12:06:06 PM
```

`%R` - Время в 24 часовом формате HH:MM. Тоже что и "%H:%M"
```javascript
    strftime('%R', new Date(2014, 2, 4, 20, 6, 6)); // 20:06
```

`%s` - Метка времени Эпохи Unix

`%S` - Двухзначный номер секунды, От 00 до 59
```javascript
    strftime('%S', new Date(2014, 2, 4, 20, 6, 6)); // 06
```

`%t` - Символ табуляции ("\t")

`%T` - ISO 8601 формат времени HH:MM:SS. Тоже что и "%H:%M:%S"
```javascript
    strftime('%T', new Date(2014, 2, 4, 20, 6, 6)); // 20:06:06
```

`%u` - Порядковый номер дня недели согласно стандарту ISO-8601
```javascript
    strftime('%u', date); // 2
```

`%U` - Порядковый номер недели в указанном году, начиная с первого воскресенья в качестве первой недели
```javascript
    strftime('%U', date); // 9
```

`%V` - Порядковый номер недели в указанном году в соответствии со стандартом ISO-8601:1988, счет начинается с той недели, которая содержит минимум 4 дня, неделя начинается с понедельника
```javascript
    strftime('%V', date); // 10
```

`%w` - Порядковый номер дня недели, От 0 (воскресенье) до 6 (суббота)
```javascript
    strftime('%w', date); // 2
```

`%W` - Порядковый номер недели в указанном году, начиная с первого понедельника в качестве первой недели
```javascript
    strftime('%W', date); // 9
```

`%x` - Предпочитаемое отображение даты, без времени. Тоже что и "%m/%d/%y"
```javascript
    strftime('%x', date); // 03/04/14
```

`%X` - Предпочитаемое отображение времени в зависимости от локали, без даты. Тоже что и "%H:%M:%S"
```javascript
    strftime('%X', date); // 12:12:12
```

`%y` - Двухзначный порядковый номер года с ведущим нулем, 09 для 2009
```javascript
    strftime('%y', new Date(2001, 2, 4)); // 01
```

`%Y` - Четырехзначный номер года, 2014
```javascript
    strftime('%Y', date); // 2014
```

`%z` - Смещение временной зоны относительно UTC (пример -0500)
```javascript
    strftime('%z', new Date('Tue Mar 04 2014 12:12:12 GMT+0400 (MSK)')); //+0400
```

`%Z` - Аббревиатура временной зоны.
```javascript
    strftime('%Z', new Date('Tue Mar 04 2014 12:12:12 GMT+0400 (MSK)')); // MSK
```

`%%` - Символ процента ("%")

### Доступные модификаторы

`-` -  нет ведущего пробела или нуля

`0` - ведущий ноль

`_` - ведущий пробел

`~` - в ИП (только для месяца)

## mtime

`mtime(date, [options]);` - Возвращает разницу между date и настоящим моментом в различных строковых форматах.

`options = undefined || {}` - Дефолтный формат
```javascript
    mtime(date); // 1 секунда назад || 1 минута назад || 1 час назад || вчера || позавчера
```

`options.short = true` - Cокращенный формат
```javascript
    mtime(date, { short: true }); // 1 с. назад || 1 мин. назад || 1 ч. назад || вчера || позавчера 
```

`options.brief = true` - Сокращенный формат без слова "назад"
```javascript
    mtime(date, { brief: true }); // 1 сек. || 1 мин. || 1 час || вчера 
```
`options.now = new Date('2017-01-06T12:00:00+0000')` - Время, разницу между которым нужно находить
