# pt-table

Паттерн для создания простых плоских таблиц. Набор модификаторов покрывает максимальное количество как стилистических так и структурных потребностей. Также есть возможность маркеровать строки по статусу.

### Модификаторы блока

| Название | Значения | Описание |
| -------- | -------- | -------- |
| `view` | `default` `ghost` | Тип отображения |
| `border` | `all` | Добавление рамок |
| `stripe`  | `even` `odd` | Полосатость |
| `space-v` | `xs` `s` `m` `l` | Внутренние отступы по вертикали  |
| `space-h` | `xs` `s` `m` `l` | Внутренние отступы по горизонтали |
| `space-a` | `xs` `s` `m` `l` | Внутренние отступы по всем сторонам |

### Элементы блока
 
| Элемент | Описание |
| --------| -------- |
| [col](#elems-col) | Ячейка |
| [row](#elems-row) | Строка |

```js
{
	block: 'pt-table',
	elemMods: { view: 'success', border: 'all' },
	content: [
		{
			elem: 'row',
			elemMods: { view: 'head' },
			content: 'Heade row'
		},
		{
			elem: 'row',
			content: 'First row'
		},
		{
			elem: 'row',
			content: 'Second row'
		},
		{
			elem: 'row',
			content: 'Third row'
		}
	]
}
```

### Элементы блока
 
<a name="elems-col"></a>
#### Элемент `col`

Элемент, отвечающий за ячейку.

##### Модификаторы элемента блока

| Модификатор элемента | Допустимые значения | Описание |
| -------------------- | ------------------- | -------- |
| `width` | `5` `10` ... `100` | Ширина в процентах |
| `align` | `center` `right` `left` | Выравнивание |
| `grow` | `evenly` | Приоритет на сохранение пропорции |

<a name="elems-row"></a>
#### Элемент `row`

Строки могут отделяться друг от друга полосой или отступом, если не задан модификатор border.

##### Модификаторы элемента блока

| Модификатор элемента | Допустимые значения | Описание |
| -------------------- | ------------------- | -------- |
| `view` | `head` | Тип отображения как заголовка |
| `border` | `bottom` `top` | Рамки |
| `status` | `default` `error` `success` `warnings` | Статус |
