# counter

Блок счетчиков для отображения количества чего-либо в смысловом блоке (оповещения, события, итд).

### Модификаторы блока

| Модификатор | Значение | Описание |
|------------ | -------- | -------- |
| `size` | `xs` `s` | Размер |
| `view` | `alert` | Фон |

```js
{
	block: 'counter',
	mods: { view: 'alert', size: 's' },
	content: '2'
}
```
