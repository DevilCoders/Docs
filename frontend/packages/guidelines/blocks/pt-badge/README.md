# pt-badge

Блок для маркировки статуса сущности. Например, статус платежа.

### Модификаторы блока

| Название | Значения | Описание |
| -------- | -------- | -------- |
| `view` | `success` `warning` `alert` `default` | Тип отображения |

```js
{
	block: 'pt-badge',
	mods: { view: 'success' }
}
```
