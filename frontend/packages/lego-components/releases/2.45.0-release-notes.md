### [themekit: Починить раскрытие переменных с учетом мапинга][ISL-8767]

При создании css файлов темы для алиасов теперь создаются ссылки на используемые алиасы.

Пример:

```diff
:root {
  --color-a: #fff;
-  --color-b: #fff;
+  --color-b: var(--color-a);
}
```
