### [Tumbler.ts: Ошибка в токенах][ISL-9103]

Токены про background `tumbler-view-default-button-*`переехали из скоупа button

Токен `tumbler-view-default-fillColor-base` переехал под скоуп button

Формат для yml:

```yml
fillColor:
      base:
        value:
      disabled:
        value:
      hovered:
        value:
      checked:
        base:
          value:
        disabled:
          value:
        hovered:
          value:

    button:
      fillColor:
        base:
          value:
```
