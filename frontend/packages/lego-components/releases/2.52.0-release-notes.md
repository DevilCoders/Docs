### render-override

Была реализована библиотека предоставляющая возможность переопределять внутренние элементы составных компонентов, более подробнее можно прочитать в [документации](https://github.yandex-team.ru/search-interfaces/frontend/blob/master/packages/lego-components/src/lib/render-override/README.md) к самой библиотеки.

### Select

В компонент `Select` была добавлена возможность переопределения внутренних элементов.

Список доступных переопределений:

* `renderTrigger` — Переопределение компонента `Trigger`
* `renderTriggerIcon` — Переопределение компонента `Icon` внутри компонента `Trigger`
* `renderMenu` — Переопределение компонента `Menu`

**Пример**

```tsx
<Select
  renderMenu={(props, Menu) => (
    <div>
      <input placeholder="Search..." />
      <Menu {...props} />
    </div>
  )}
/>
```
