# Разработка

**Установка зависимостей:**

Необходимо установить зависимости в корне Капчи, затем установить зависимости для Внешней Капчи.

```sh
captcha -> npm ci
captcha -> cd apps/external
external -> npm ci
```

**Запуск проекта:**

```sh
npm start
```

**Сборка проекта:**

```sh
npm run build
```
