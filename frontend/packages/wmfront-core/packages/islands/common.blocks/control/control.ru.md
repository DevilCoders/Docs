# control

Блок `control` является служебным блоком. Содержит базовую функциональность, которая используется для работы некоторых блоков-контролов библиотеки `islands`: 

* [attach](../../attach.ru.md)
* [button](../../button.ru.md)
* [button2](../../button2.ru.md)
* [checkbox](../../checkbox.ru.md)
* [check-button](../../check-button.ru.md)
* [input](../../input.ru.md)
* [radio-button](../../radio-button.ru.md)
* [radiobox](../../radiobox.ru.md)
* [select](../../select.ru.md)
