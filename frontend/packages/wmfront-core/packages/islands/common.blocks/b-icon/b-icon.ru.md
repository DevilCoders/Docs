## Описание

**Статус блока**: **deprecated**

Блок используется только в старых блоках (`b-form-button`, `b-form-attach`, `b-menu` и т.д).
Как только для них будет создана замена, `b-icon` будет удален.
Вместо `b-icon` используйте [icon](../icon/icon.md) для иконок и (image)[../image/image.md] для картинок.

Блок `b-icon` создает контейнер для изображения. Предоставляет возможность добавлять и редактировать изображение без необходимости правки родительского блока.

По умолчанию тег `<img>` выводит однопиксельную прозрачную картинку (.gif) с пустым атрибутом `alt`.

```js
{
	block: 'b-icon'
}
```

Представление блока в HTML, после BEMHTML-преобразования:

```html
<img class="b-icon" src="//yandex.st/lego/_/La6qi18Z8LwgnZdsAr1qy1GwCwo.gif" alt=""/>
```


```js
{
	block: 'b-icon',
	url: '/url'
}
```

Представление блока в HTML, после BEMHTML-преобразования:

```html
<img class="b-icon" src="/url"/></div>
```

## BEMJSON-поля блока
Все ниже перечисленные поля необязательные.

* `alt` – устанавливает альтернативный текст для изображений. Такой текст позволяет получить текстовую информацию о изображении при отключенной в браузере загрузке изображений.
* `width` – устанавливает ширину изображения. Допускается использовать значения в пикселах или процентах. Если установлена процентная запись, то размеры изображения вычисляются относительно родительского элемента — контейнера, в котором расположен тег  `<img>`.
* `height` – устанавливает высоту изображения. Допускается использовать значения в пикселах или процентах. Если установлена процентная запись, то размеры изображения вычисляются относительно родительского элемента — контейнера, в котором расположен тег `<img>`.
* `url` – задавая `url`, можно установить `b-icon` новое изображение, со своим `alt`, шириной и высотой.


## Способы использования блока
Использовать блок можно двумя способами:

* через `url`;
* через модификатор.

### Использование блока через url

Задаем в поле `url` свой путь к изображению, BEMJSON:

```js
{
	block: 'b-icon',
	url: 'https://yastatic.net/icon.png',
	alt: 'My icon'
}
```

Этот способ рекомендуется для добавления одиночных картинок.

### Использование блока через добавление модификатора

Добавляем блоку модификатор, BEMJSON:

```js
{
	block: 'b-icon',
	mods: {type: 'simple'},
	alt: 'My icon'
}
```
В CSS задаем размеры изображения и путь к нему:

```css
.b-icon_type_simple
{
    width: 16px;
    height: 16px;
    background: url(icons-sprite.png) no-repeat -50px 0;
}
```
Этот способ рекомендуется при использовании спрайтов.


_При помощи CSS-свойства `background: url(...);` в блоке можно использовать как спрайты, так и одиночную иконку._

## Модификаторы

### size-*

Позволяют отображать специальные иконки различных размеров.
