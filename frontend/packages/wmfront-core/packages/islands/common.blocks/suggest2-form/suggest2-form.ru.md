# suggest2-form

Блок `suggest2-form` — это форма-контейнер(`<form>`), которая содержит все контролы саджеста (`button`, `input`, `popup`) и блоки [suggest2-*](../suggest2/suggest2.ru.md#%D0%92%D1%81%D0%BF%D0%BE%D0%BC%D0%BE%D0%B3%D0%B0%D1%82%D0%B5%D0%BB%D1%8C%D0%BD%D1%8B%D0%B5-%D0%B1%D0%BB%D0%BE%D0%BA%D0%B8). Блок используется для:

* первоначальной инициализации блока `suggest2` в DOM-дереве, при загрузке страницы с саджестом.
* доступа ко всем вышеперечисленным вложенным блокам формы и их инициализации.

Чтобы саджест взаимодействовал с формой на странице, блок `suggest2-form` миксуется к форме с указанием JS-инициализации (`js: true`).

Если необходимы [счетчики](../suggest2-counter/suggest2-counter.ru.md), то дополнительно к `suggest2-form` нужно примиксовать и блок `suggest2-counter`.

> **NB** *Обратите внимание, что форма обязательно должна содержать блок [suggest2](../suggest2.ru.md).*

## Подробнее
### Модификаторы блока

<a name="mods-disabled"></a>
#### Модификатор `disabled`

Отвечает за установку неактивного состояния («режим ожидания») для вложенных блоков формы: [suggest2](../suggest2/suggest2.ru.md), [suggest2-counter](../suggest2-counter/suggest2-counter.ru.md), [suggest2-input](../suggest2-input/suggest2-input.ru.md), [suggest2-popup](../suggest2-popup/suggest2-popup.ru.md).

Способы установки: JS.<br>
Допустимое значение: `yes`.

> **NB** Чтобы использовать модификатор, подключите его в зависимости на проектом уровне._


###  Элементы блока

<a name="elems-node"></a>
#### Элемент `node`

Определяет вспомогательный контейнер внутри формы («узел»), для подключения каждой пары поля ввода с саджестом на странице. Используется для доступа и управления вложенными блоками формы. Необходимо выполнение следующих условий:

* Элемент `node` обязательно содержит единичные экземпляры блоков `input`, `popup` и/или `button`.
* К блокам `input` и `button` примиксованы соответствующие одноименные элементы блока `suggest2-form`.
* Блок `suggest2` примиксован к блоку `popup` с указанием JS-параметров (параметр `url` — обязательный).

```js
{
    block: 'form',
    tag: 'form',
    js: true,
    mix: {block: 'suggest2-form', js: true}, // Необходима возможность взаимодействия с формой.
    content: {
        block: 'suggest2-form',
        elem: 'node', // Добавляем узлы для каждого саджеста
        content: [
            {
                block: 'input', // Знание об inpute
                mods: {size: 'm', theme: 'normal'},
                mix: {block: 'suggest2-form', elem: 'input'},
                content: {elem: 'control'}
            },
            {
                block: 'popup', // Саджест примиксовывается к любому popup
                mods: {autoclosable: 'yes', adaptive: 'no', animate: 'no'},
                mix: [{
                    block: 'suggest2',
                    mods: {theme: 'normal', size: 'm'},
                    js: {url: '//suggest-server.lego-dev.dev.yandex-team.ru'}
                }, {
                    block: 'suggest2-detect', js: true // Микс с suggest2
                }],
                content: {elem: 'content'}
            }
        ]
    }
}        
```
