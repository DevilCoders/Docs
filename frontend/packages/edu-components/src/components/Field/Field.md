# Field

<!-- description:start -->
Компонент для отображения поля ввода текста или числа.
<!-- description:end -->

## Пример использования

```typescript jsx
import React, { FC } from 'react';
import { compose } from '@bem-react/core';
import { withRegistry, Registry } from '@bem-react/di';
import {
  cnField,
  Field as FieldBase,
  withTypeNumber,
  withWidthComputed,
  IFieldProps as IFieldDesktopProps,
} from '@yandex-int/edu-components/Field/desktop';

// Соберите Textinput с нужной конфигурацией
import { Textinput } from './Textinput';

const fieldRegistry = new Registry({ id: cnField() });

fieldRegistry.set('Textinput', Textinput);

export interface IFieldProps extends IFieldDesktopProps {
  width?: 'computed';
  type?: 'text' | 'number';
}

export const Field = compose(
  withRegistry(fieldRegistry),
  withWidthComputed,
  withTypeNumber,
)(FieldBase) as FC<IFieldProps>;

const App = () => {
  return <Field view="platform" size="m" type="number" value="12345" width="computed" />;
};
```

## Модификаторы

<!-- modifiers:start -->
### type

Задает тип содержимого. Форматирует числа.

**Допустимые значения:** `"number"`.

### width

Задает автоматический подсчет ширины компонента.

| Свойство      | Тип          | По умолчанию | Описание                                  |
| ------------- | ------------ | ------------ | ----------------------------------------- |
| width?        | `"computed"` | —            | Автоматический подсчёт ширины компонента. |
| letterWidth?  | `number`     | `0.8`        | Ширина одного символа.                    |
| numberWidth?  | `number`     | `0.66`       | Ширина одной цифры.                       |
| dividerWidth? | `number`     | `0.2`        | Ширина одного разделителя.                |
<!-- modifiers:end -->
