### readOnly

Задает изменяемость селекта и добавляет ему статус.

<!-- props:start -->
| Свойство  | Тип                                                   | Описание                        |
| --------- | ----------------------------------------------------- | ------------------------------- |
| readOnly? | `false \| true`                                       | Сделать ли селект неизменяемым. |
| status?   | `"initial" \| "correct" \| "incorrect" \| "accepted"` | Статус селекта.                 |
<!-- props:end -->
