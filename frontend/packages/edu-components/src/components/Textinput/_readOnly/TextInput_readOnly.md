### readOnly

Задает изменяемость текстового поля и добавляет ему статус.

<!-- props:start -->
| Свойство  | Тип                                                   | Описание                       |
| --------- | ----------------------------------------------------- | ------------------------------ |
| readOnly? | `false \| true`                                       | Состояние «только для чтения». |
| status?   | `"initial" \| "correct" \| "incorrect" \| "accepted"` | Статус текстового поля.        |
<!-- props:end -->
