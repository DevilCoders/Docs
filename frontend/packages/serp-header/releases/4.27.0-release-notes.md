### [Поддержать в шапке и в services кейс с разными доменами для главной и поиска][ISL-9306]

В serp-header поддержана возможность указать в качестве поиска другой сервис в табах.
Например, основной сервис живет на одном домене (yandex.ru/news), а поиск на другом (newssearch.yandex.ru).
Для того, чтобы это сделать, надо в пакете services для нужного сервиса, указать в секции search id нужного сервиса.
```json
"search": {
    "id": "newsearch"
}
```
При построение ссылок в табах, шапка сформулирует ссылку относительного другого сервиса.
