### [Кастомизация пунктов меню в User2][ISL-8949]

В блоке `user` появился новый проектный уровень `custom-items`.
Он нужен для того, чтобы полностью формировать список элементов меню.

В нем будут отображаться только те пункты, которые переданы в `customMenuItems`.

В `User2Vanilla` можно взять с dist нужный блок и передать `getContent` в компонент.

```tsx
import User2HTML from '@yandex-lego/serp-header/dist/custom-items/user2.desktop';

<>
    <User2Vanilla
        {...userProps}
        getContent={User2HTML.getContent}
    />
</>
```
