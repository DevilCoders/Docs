## Пояснения

<a name="ISL-9067"></a>
### [Доделки по EU в Видео][ISL-9067]

В блоке user появилась удобная опция для фильтрации списка меню в блоке
пользователя - allowedActions. Опция, если она задана, работает по принципу
whitelist и позволяет отфильтровать пункты меню по модификатору action.
Фильтрация не применится для пунктов из customMenuItems.

```javascript
    {
        tld: 'ru',
        ctx: {
            secretKey: secretKey(306379094),
            uid: 306379094,
            yu: 7390764501529155693,
            avatarId: '20706/226291201-19837531',
            name: 'Denis Smirnov',
            retpath: "//yandex.ru?text=fass'asa",
            passportLinkParams: 'test=1',
            hasPlus: true,
            allowedActions: [ 'mail', 'passport', 'plus'],
            accounts: [
                {
                    uid: 123456789,
                    name: 'Александр Баумгертнер',
                    pic: {
                        avatarId: '21377/28004787-1253930',
                    },
                    subname: { provider: 'fb' },
                    ticker: { count: 10 }
                },
            ],
        },
    }
```
