### [Ванильные компоненты: бандлы под языки][ISL-8704]

В сборке serp-header появилась возможность подключать точечно dist.

Теперь у каждого блока в dist появились файлы под key и lang.
Если вы подключаете блок на клиенте, это поможет вам сократить бандл.

Если раньше вы подключали:

```
// 300кб без gzip
import * as UserHTML from '@yandex-lego/serp-header/dist/base/user2.desktop';
```

Теперь можно использовать:

```
// 40кб без gzip
import * as UserHTML from '@yandex-lego/serp-header/dist/base/user2.desktop.{key}.{lang}';

// вероятнее всего вам нужен
import * as UserHTML from '@yandex-lego/serp-header/dist/base/user2.desktop.base.ru';
```

Если вы использовали `User2Vanilla`, можно сделать импорт UserHTML с нужного файла и передать в пропс `getContent`.
