### BREAKING CHANGES

* **testpalm-cli:** Аргумент "suppress-exist-project-error" удалён.

Сначала пытаемся получить проект, чтобы не пытаться создать уже существующий проект.

Ранее ручки получения проекта в новом TestPalm API не было.
* **testpalm-cli:** Теперь утилита требует OAuth-токен из переменной окружения `TESTPALM_OAUTH_TOKEN`.
* **testpalm-cli:** Новый пакет
