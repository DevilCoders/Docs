### BREAKING CHANGES

* **abc-client:** В новой версии API (v4) по умолчанию отдаётся минимальный набор данных для каждой ручки. Разработчики призывают всегда использовать Query-параметр `fields`, в котором можно перечислить все необходимые программе поля, которые должно вернуть API.
* **abc-client:** Добавлена пейджинация, сейчас ручки с поддержкой пейджинации возвращают объект c единственным полем `result`.
