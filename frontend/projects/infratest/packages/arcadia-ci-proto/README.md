# arcadia-ci-proto

Скомпилированные proto файлы для чтения сообщений [Arcadia CI] в Logbroker.

## Установка

```console
npm i "@yandex-int/si.ci.arcadia-ci-proto --registry=https://npm.yandex-team.ru
```

## Компиляция

```console
npm run proto
```

[Arcadia CI]: https://docs.yandex-team.ru/ci/event/event
