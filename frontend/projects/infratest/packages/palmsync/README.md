# palmsync

Инструмент для валидации тестовых сценариев проекта и их синхронизации в проект сервиса [TestPalm](https://testpalm.yandex-team.ru).

```bash
npm install @yandex-int/palmsync --registry https://npm.yandex-team.ru
```
