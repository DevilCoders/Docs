# Skipper
Инструмент для автоматизации работы с заскипамми. Основной функционал:
- Запуск погонятора для заскипов, для выбранной очереди или задач
- Генерация(обновление) родительской задачи(отчет) с массовыми прогонами
- Фильтрация запуска по:
  - платформе(desktop/touch-phone/touch-pad)

### Перед началом работы
Для работы инструмента необходимо указать токен пользователя, от имени которого будет осуществляться работа с [Трекером](https://st.yandex-team.ru/).
1. Получите токен по [ссылке](https://oauth.yandex-team.ru/authorize?response_type=token&client_id=5f671d781aca402ab7460fde4050267b)
2. Пропишите в консоль

   ``export TRACKER_API_TOKEN=<ваш токен>``
3. Лучше сразу добавить в ``.bashrc``/``.zshrc`` чтобы не вводить переменную каждый раз, можно сделать командой

### Запуск

Выполните команду

``npm run skipper -- <[TASKS]|QUEUE>``

Где первым параметром указывается очередь, либо задачу(-и).
- Будет запущен погонятор на каждую задачу(в среднем 2 минуты на каждую задачу).
- В каждой задаче будет добавлен комментарий с запущенной задачей.
- После завершения прогона, необходимо разобрать отчеты в запущенных задачах

### Дополнительные параметры
- ``-p create|<QUEUE>|<TASK>`` - указать куда сгенерировать отчет
  - ``create`` - создает новую задачу в очереди, выбранной для прогона(либо первой задаче)
  - ``<QUEUE>`` - создает задачу в указанной очереди
  - ``<TASK>`` - обновляет описание в указанной задаче, заменяет его на отчет(ВНИМАНИЕ! Описание полностью перезаписывается)
- ``-d <desktop|touch-phone|touch-pad>`` - ограничение по прлатформе
