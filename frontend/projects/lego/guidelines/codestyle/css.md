CSS Styleguide
==============

Разработка документа ведется в репозитории https://github.yandex-team.ru/lego/islands-codestyle.


Последовательность свойств
--------------------------

Типы свойств:
- Позиционирование
- Отображение
- Размеры
- Таблицы/Списки
- Текст
- Шрифт
- Цвета/Эффекты
- Прочее

Свойства группируются по типам.
Группы следуют в перечисленном выше порядке и разделяются пустой строкой.
Для сортировки свойств используется утилита [csscomb].
Конкретную последовательность свойств можно посмотреть в [конфиге].

[csscomb]: https://github.com/csscomb/csscomb.js
[конфиге]: https://github.yandex-team.ru/lego/islands-codestyle/blob/dev/configs/islands/.csscomb.json

Пробелы, кавычки, переводы строк, точка с запятой
-------------------------------------------------

- Для индентации используется 4 пробела
- Знаки табуляции не применяются
- Строки обрамляются одинарными кавычками
- Кавычки используются только если это необходимо
- Правила отделяются друг от друга одной пустой строкой
- Все [комбинаторы] кроме [descendant] отбиваются пробелом с обеих сторон
- Открывающая фигурная скобка не переносится на новую строку
- Двоеточие отбивается пробелом только с правой стороны
- Точка с запятой ставится у каждого свойства
- Группа свойств с вендорными префиксами выравнивается лесенкой (по убыванию длины)
- В конце файла всегда делается перевод строки

[комбинаторы]: http://www.w3.org/TR/css3-selectors/#combinators
[descendant]: http://www.w3.org/TR/css3-selectors/#descendant-combinators

```css
.link {
    color: blue;
}

.paragraph {
    padding: 10px;
}

.paragraph > .link {
    text-decoration: none;
}

.link {
    font-family: 'Goudy Bookletter 1911', sans-serif;

    -webkit-border-radius: 3px;
       -moz-border-radius: 3px;
            border-radius: 3px;
    background: -webkit-linear-gradient(top, #fff 0, #eee 100%);
    background:    -moz-linear-gradient(top, #fff 0, #eee 100%);
    background:         linear-gradient(to bottom, #fff 0, #eee 100%);
}
```

Прочее
------

### Цвет

Если используется шестнадцатеричный формат, буквы указываются в нижнем регистре.

```css
.link {
    color: #fff;
}
```

Если это возможно, используем три порядка вместо шести (`#сссccc` → `#ccc`).

```css
.link {
    color: #ccc;
}
```

### background-image

Функция `url` всегда указывается в самом конце, т.к. может содержать длинные адреса или контент, закодированный в `base64`.

### Регистр

Названия элементов в селекторах пишутся в нижнем регистре.

```css
li > a {
    color: red;
}
```

### Дробные величины

В дробных значениях меньше одного не пишется ноль.

```css
.paragraph {
    padding: .5em;
}
```

### Нулевое значение

Нулевое значение записывается без единиц измерения.

```css
.paragraph {
    margin: 0;
}
```
