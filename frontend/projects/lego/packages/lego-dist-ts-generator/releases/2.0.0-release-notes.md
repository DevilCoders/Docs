## Пояснения

* Добавлена возможность указывать конфиг через аргументы `--config path/to/config.js`, по умолчанию это `.dist.config.js`
* Запуск скрипта теперь не требует явной передачи названия пакета `CONTRIB_NAME`, в связи с этим изменился и формат конфига, теперь конфигурация может содержать настройки только для одного пакета:
    ```js
    module.exports = {
        srcPath: './src',
        ignore: ['...'],
        tsconfig: 'tsconfig.dist.json',
        distPath: './',
    };
    ```
* Свойство `ignore` в конфиге имеет теперь тип массива `ignore: ['a.ts', 'b.ts']`
