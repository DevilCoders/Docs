### [iver: версионирование приватных пакетов][ISL-8501]

`Iver` теперь вычисляет версии и приватных пакетов тоже.
Чтобы включить приватый пакет для версионирования, надо проставлять `autoreleases: true`.

Это необходимо для того, чтобы проставлять версии в сервисах после влития и при отведении релизов не останавливать очередь, чтобы запушить измененную версию.

При отведении релизов на нужные версии будет ставиться тег.
