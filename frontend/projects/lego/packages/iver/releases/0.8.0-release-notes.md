### [iver: "Пояснения" не подтягиваются в автосгенеренный файл с release notes][ISL-8766]

В `iver` была сломана функциональность про формирование notes.
Эта функциональность вернулась и теперь можно писать примечания к пакету, что для минорных и мажорных версий проставлялись записи.
