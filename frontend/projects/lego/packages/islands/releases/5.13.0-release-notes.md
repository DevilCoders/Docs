# islands 5.13.0

Изменения, вошедшие в релиз islands 5.13.0, разделены на группы:

* [минорные изменения](#Минорные-изменения) — задачи, в рамках которых произошло расширение API блоков;
* [исправлено и улучшено](#Исправлено-и-улучшено) — различные исправления, не затрагивающие API блоков.

Все важные изменения подробно описаны [ниже](#Пояснения).

## Пояснения

<a name="ISL-4100"></a>
### [dropdown2: Разработать в новом дизайне][ISL-4100]

Реализован новый дизайн для блока `dropdown2`.

Новый дизайн находится в состоянии unstable.

<a name="ISL-4231"></a>
### [m-editor: Добавить поддержку size и placeholder][ISL-4231]

В блоке `m-editor` появилась возможность передавать произвольное значение модификатора `size` в блок `input`.
Ранее значение модификатора всегда было `s`.

Также, появилась возможность передать в `input` поле `placeholder`.

<a name="ISL-4283"></a>
### [popup2, menu, select2, tooltip: Разработать в новом дизайне][ISL-4283]

Для платформы `desktop` реализован новый дизайн для блоков `popup2`, `menu`, `select2`, `tooltip`.
Новый дизайн находится в статусе `experimental`.

<a name="ISL-4085"></a>
### [radio-button: Разработать в новом дизайне][ISL-4085]

В блоке `radio-button` появился новый дизайн.

```js
({
    block: 'radio-button',
    mods: {theme: 'normal', size: 'm', view: 'default', tone: 'default'}
    content: ...
})
```

<a name="ISL-4150"></a>
### [slider: Разработать в новом дизайне][ISL-4150]

В блоке `slider` появился новый дизайн.

```js
({
    block: 'slider',
    mods: {view: 'default', tone: 'default', theme: 'normal', size: 'n'},
    ...
})
```

<a name="ISL-4369"></a>
### [tabs-menu: добавить модификатор для внешних ссылок][ISL-4369]

Для `tabs-menu` добавлена проверка при установке модификатора активного элемента меню `_active_yes`. Модификатор не будет
выставляться, если у элемента есть модификатор `_outer_yes`. Его можно использовать для внешних ссылок в меню,
которые не должны переходить в активное состояние.

<a name="ISL-4282"></a>
### [textinput: Разработать в новом дизайне][ISL-4282]

Для платформы `desktop` реализован новый дизайн для блока `textinput`.
Новый дизайн находится в статусе `unstable`.

<a name="ISL-4404"></a>
### [i-font: Правки в шрифте YS][ISL-4404]

Несколько небольших изменений в YS
  - добавлены новые и старые знаки валют
  - добавлены альтернативный знак номера (для ретроградов)
  - уточнен размер знака градуса.
