## Пояснения

<a name="ISL-2027"></a>
### [link_pseudo_yes: Не предотвращать переход по ссылке, если зажаты CMD/CTRL][ISL-2027]

Для блоков `button2` и `link` c модификатором `pseudo_yes`
добавлена возможность передавать `domEvent` и `isModKey` в BEM-событие `click`.

Булев модификатор `isModKey` выставляется,
если зажата одна из клавиш-модификаторов (`shift`, `meta` (`cmd`, `win`), `ctrl`, `alt`)
при нажатии на кнопку или псевдоссылку.

Модификатор используется для отключения бизнес-логики сервиса, если он установлен.

<a name="ISL-2677"></a>
### [slider: Слайдер не должен вылезать за сетку][ISL-2677]

Для блока `slider` добавлен модификатор `contained`, предотвращающий выход ползунка за границы блока путём добавления блоку отступов.

<a name="ISL-2515"></a>
### [tabs: Научить блок сбрасывать выделенную вкладку][ISL-2515]

В API блока `tabs` добавлен метод `deactivate`, который снимает выделение с выбранной вкладки.
В `tabs-menu` добавлен обработчик события удаления модификатора `active`, в котором при удалении модификатора программно создаётся событие `change`.

<a name="ISL-2292"></a>
### [Поправить font-face у компонентов][ISL-2292]

Поправлен font-face у компонентов — в свойство `font-family` было проставлено 'Helvetica Neue', Helvetica, Arial, sans-serif.
