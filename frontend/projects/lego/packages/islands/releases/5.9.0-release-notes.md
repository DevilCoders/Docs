<a name="ISL-4032"></a>
### [radio-button.react: Сделать возможным указывать произвольные атрибуты][ISL-4032]

**Атрибут `controlAttrs` в [control](../../common.blocks/control/control.react.js)**

Добавлена поддержка атрибута `controlAttrs` для базового react-блока `Control`. Все свойства указанные в `controlAttrs` по умолчанию станут атрибутами dom-элемента блока, если это не переопределено в потомках.

Пример:
```jsx
<RadioButton
    theme='normal'
    size='s'
    value={1}>
    <RadioButton.Radio
        controlAttrs={{title: 'номер один'}}
        value={1}>
        номер один
    </RadioButton.Radio>
    <RadioButton.Radio
        controlAttrs={{title: 'номер два'}}
        value={2}>
        номер два
    </RadioButton.Radio>
</RadioButton>

```
