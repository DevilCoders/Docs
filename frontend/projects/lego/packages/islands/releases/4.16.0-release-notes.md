### [share-dropdown: Открывать страницы в заданных окнах и фреймах][ISL-2395]

В блоке `share-dropdown` добавлена возможность управлять атрибутом `target` элементов `share-dropdown__share-item`.
```javascript
({
    block: 'share-dropdown',
    content: [
        {
            elem: 'share-item',
            target: '_self',
            service: 'twitter'
        }
   ]
});

```
Если `target` не передан, то, как и раньше, будет выставляться значение `_blank`.
