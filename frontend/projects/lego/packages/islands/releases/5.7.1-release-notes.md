## Пояснения

<a name="ISL-2030"></a>
### [social-brocker: После авторизации в FB окно авторизации не закрывается][ISL-2030]

Блок `social-broker` был доработан для работы с новым поведением Facebook.

При обновлении не забудьте обновить у себя закрывающую страничку `social-broker__closer.html`, которая, скорее всего,
у вас обрабатывается особым образом, поскольку её адрес должен быть внутри домена вашего сервиса.
