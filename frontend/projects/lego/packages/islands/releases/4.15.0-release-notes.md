## Пояснения


<a name="ISL-2972"></a>
### [i-services: Добавить сервис webmaster-api][ISL-2972]

В блок `i-services` добавлен сервис «[API Вебмастера](https://tech.yandex.ru/webmaster/)». Вы можете использовать его через идентификатор `webmaster-api`.

<a name="ISL-2976"></a>
### [Отправлять счетчики через navigator.sendBeacon][ISL-2976]

Поддержана отправка счетчиков через метод `navigator.sendBeacon`.

Он позволяет решить проблему неуспевших сработать счетчиков из-за закрытия вкладки, окна и браузера.
