## Пояснения

<a name="ISL-6419"></a>
### [textarea: добавить модификатор view=default][ISL-6419]

В блоке `textarea` появилась возможность использовать `view_default`.
Доступно использование `view` совместно с `tone` [default, grey, red, dark]
По умолчанию блоку выставляется `view_classic` – вид textarea, как было раньше.

<a name="ISL-5844"></a>
### [Улучшение UserPic][ISL-5844]

Улучшен компонент `user-pic` - теперь используется тег  `img` с атрибутами `src` и `srcset` (вместо 2х `div`)
