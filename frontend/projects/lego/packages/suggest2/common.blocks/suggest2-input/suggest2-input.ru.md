# suggest2-input

Блок `suggest2-input` – адаптер, для подключения к саджесту блоков `input` из разных библиотек (не только `islands-*`, но и библиотек сервисов) с помощью унифицированного API.

Блок [suggest2](../suggest2/suggest2.ru.md) отслеживает изменения в поле ввода поискового запроса (блок `input`) и сам перестраивает список подсказок.

> **NB** Текущая версия блока `suggest2-input` поддерживает только адаптер для блока [input](../input/input.ru.md).

## Подробнее

<a name="mods-disabled"></a>
### Модификатор `disabled`

Отвечает за неактивное состояние блока.

Выставляется автоматически при выставлении неактивного состояния блоку [suggest2-form](../suggest2-form/suggest2-form.ru.md#%D0%9C%D0%BE%D0%B4%D0%B8%D1%84%D0%B8%D0%BA%D0%B0%D1%82%D0%BE%D1%80-disabled).

Способы установки: JS.<br>
Допустимое значение: `yes`.

<a name="mods-disabled"></a>
### Модификатор `complete`

Добавляет методы для работы с автодополнением.

Установить модификатор можно либо доопределив код инициализации `suggest2-form`, либо на стадии инициализации или runtime в методе [setMod блока suggest2](https://github.yandex-team.ru/lego/islands/tree/dev/common.blocks/suggest2-form/suggest2-form.js#L98).

Чтобы автодополнение саджеста работало корректно, в подключаемом блоке `input` должны быть реализованы следующие методы:

* `showAhead` — устанавливает текст дополнительной подсказки.<br>
    Аргументы:  
    * `filler {String}` – невидимая часть дополнительной подсказки.   
    * `hint {String}` – видимая часть дополнительной подсказки.

    Возвращаемое значение: `{String}`. Полный текст дополнительной подсказки.  
* `clearAhead` — очищает текст дополнительной подсказки.  
      Возвращаемое значение: {Boolean} `false`.


Способы установки: JS.<br>
Допустимое значение: `auto`.

> Чтобы использовать модификатор, подключите его в зависимости блока со значением `auto` на проектном уровне._
