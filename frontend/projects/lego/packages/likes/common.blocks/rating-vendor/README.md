# rating-vendor
Отображение рейтинга с цветным фоном. Цвет фона подбирается в зависимости от значения рейтинга.

## Обзор

### BEMJSON-поля блока

| Поле | Тип | Описание |
| ---- | --- | -------- |
| [value](#field-value) | `Number` | Значение рейтинга (от 0 до 10). |
| [base](#field-base) | `Number` | Максимальное значение рейтинга (по умолчанию 10). |
| [decimalSeparator](#field-decimalSeparator) | `String` | Десятичный разделитель (по умолчанию ","). |

### Модификаторы блока

| Модификатор | Допустимые значения | Способы установки | Описание |
| ----------- | ------------------- | ----------------- | -------- |
| [type](#mods-size) | `s`, `m` | `BEMJSON` | Стилевое оформление. |

## Подробнее

### BEMJSON-поля блока

<a name="field-value"></a>
#### Поле `value`

Значение рейтинга (от 0 до 10). Влияет на цвет фона.

Тип: `Number`.

### Пример
```
{
    block: 'rating-vendor',
    value: 8.5
}
```

<a name="field-base"></a>
#### Поле `base`

Максимальное значение рейтинга (по умолчанию 10). Влияет на цвет фона.
Допустимые значения для base: 5 и 10

Тип: `Number`.

### Пример
```
{
    block: 'rating-vendor',
    value: 4.5,
    base: 5
}
```

<a name="field-decimalSeparator"></a>
#### Поле `decimalSeparator`

Десятичный разделитель (по умолчанию ","). Для реализации поддержки языков, где в качестве разделителя используется не запятая (как в русском), а, например, точка (как в английском).

Тип: `String`.

### Пример
```
{
    block: 'rating-vendor',
    value: 8.5,
    decimalSeparator: '.'
}
```

### Модификаторы блока

<a name="mods-size"></a>
#### Модификатор `size`

Размер паддиинга вокруг значения рейтинга.

Способы установки: `BEMJSON`.

| Допустимые значения | Описание |
| ------------------- | -------- |
| `s` | Малый размер. |
| `m` | Средний размер (по умолчанию). |
