# rating-dynamic
Блок выставления оценки в виде звёздочек.

## Обзор

### Модификаторы блока

| Модификатор | Допустимые значения | Способы установки | Описание |
| ----------- | ------------------- | ----------------- | -------- |
| [theme](#mods-theme) (req) | `border`, `solid` | `BEMJSON` | Стилевое оформление. |
| [size](#mods-size) (req) | `auto`, `m` | `BEMJSON` | Размер звёзд. |
| [ugc](#mods-ugc) | `simple` | `BEMJSON` | Отправка оценки в ugcdb. |
| [edge-bug](#mods-edge-bug) | `true` | `BEMJSON` | Фикс бага для браузера Edge. |

req — обязательный модификатор.

### BEMJSON-поля блока

| Поле | Тип | Описание |
| ---- | --- | -------- |
| [value](#field-value) | `Number` | Значение рейтинга (от 1 до 5). |
| [ugcToken](#field-ugcToken) | `String` | Специальный токен для сохранения оценки в ugcdb. |

## Подробнее

### Модификаторы блока

<a name="mods-theme"></a>
#### Модификатор `theme`

Определяет внешний вид звезд.

Способы установки: `BEMJSON`.

| Допустимые значения | Описание |
| ------------------- | -------- |
| `border` | Светлые звезды с обводкой – стандартный вид для отображения на белом фоне страницы. |
| `solid` | Звезды равномерно залитые одним цветом – подходит для отображения на темном фоне или поверх картинок (на постерах фильмов). |

<a name="mods-size"></a>
#### Модификатор `size`

Размер звёзд.

Способы установки: `BEMJSON`.

| Допустимые значения | Описание |
| ------------------- | -------- |
| `auto` | Подстраивается по ширине контейнера. |
| `m` | Средний размер. |

<a name="mods-ugc"></a>
#### Модификатор `ugc`

Отправляет оценку в ugcdb. Для отправки ajax-запроса использует `i-request_type_ajax`.

Способы установки: `BEMJSON`.

| Допустимые значения | Описание |
| ------------------- | -------- |
| `simple` | Простая отправка оценок в ручку ugc (в будущем может потребоваться более сложная отправка с сохранением в LocalStorage при отсутствии интернета и отправки оценок в следующих сессиях). |

<a name="mods-edge-bug"></a>
#### Модификатор `edge-bug`

Фикс для бага с выделением звезд в Edge, проявляется для `_theme_solid`. См. [видео](https://jing.yandex-team.ru/files/sladex/rating-dynamic-solid-edge-17-bug.mp4). В отдельном модификаторе, чтобы код фикса не попадал остальным браузерам. Выставление модификатора предполагается на стороне сервера.

Способы установки: `BEMJSON`.

| Допустимые значения | Описание |
| ------------------- | -------- |
| `true` | Boolean значение. |

### BEMJSON-поля блока

<a name="field-value"></a>
#### Поле `value`

Значение рейтинга (от 1 до 5).

Тип: `Number`.

<a name="field-ugcToken"></a>
#### Поле `ugcToken`

Специальный токен для сохранения оценки в ugcdb. Тот же самый токен может использоваться и для редактирования, и для удаления оценки.

Тип: `String`.

### Пример
```
{
    block: 'rating-dynamic',
    theme: 'border',
    size: 'm',
    value: 5
}
```
