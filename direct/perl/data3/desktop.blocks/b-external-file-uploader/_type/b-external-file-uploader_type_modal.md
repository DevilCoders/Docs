#b-external-file-uploader_type_modal#

##Описание##
Модификатор для блока b-external-file-uploader из bricks  для показа блока в модальном окне

Для данного модификатора пояляются новые публичные методы.

* `show` - открывает попап
* 'hide' - закрывает попап
