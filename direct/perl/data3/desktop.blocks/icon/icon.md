#icon#

Блок с иконками
Конкретная иконка определяется модификаторами вида size-13_info, size-16_delete

Пример:
{
    block: 'icon',
    mods: { 'size-16': 'plus' },
    mix: [{
        block: 'link',
        elem: 'icon'
    }]
}

Важно! Для удобства все иконки перенесены в bricks, в блок на уровне Директа иконки не добавляем!
