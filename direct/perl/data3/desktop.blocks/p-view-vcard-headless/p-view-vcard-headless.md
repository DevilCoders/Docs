#p-view-vcard-headless#

##Описание##
Страница с контактной информацией о визитке без шапки и футера, для использования в iframe (страница мастера визиток, страница кампании и т.д.)

###Автор### 
[coffeeich](https://staff.yandex-team.ru/coffeeich)
[heliarian](https://staff.yandex-team.ru/heliarian)

###Где используется###

Используется на страница мастера визиток `p-manage-vcards` в блоке `b-vcards` для просмотра визитки.
