#p-feeds-history#

##Описание##
Страница детализации ошибок для фида.

Использует технологию [History API](https://github.com/bem/bem-history) для обновления интерфейса без перезагрузки страницы.
Состояние интерфейса транслируется блоку `b-feeds-history` для обновления его содержимого.  

###Автор### 
[coffeeich](https://staff.yandex-team.ru/coffeeich)
[eemelin](https://staff.yandex-team.ru/eemelin)

###Взаимодействие с другими блоками###
Содержит в себе блок `b-feeds-history`, который умеет обновлять своё содержимое при изменении его view-модели.
