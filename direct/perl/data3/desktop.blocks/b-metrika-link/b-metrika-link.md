#b-metrika-link

Блок для отрисовки кнопки "Перейти в метрику" (сейчас используется только в статистике)

##Параметры контекста
 * metrikaCounters - счетчик метрики,
 * cid - cid кампании,
 * noFilter - не нужна сегментация
 * dateFrom - начало
 * dateTo - конец
 * goalId - цель метрики
 * noGoal - не учитывать цель,
 * noPeriod - не учитывать даты
