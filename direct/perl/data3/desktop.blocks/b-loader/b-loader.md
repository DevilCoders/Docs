####Это универсальный блок, который отображает внутри себя спиннер
Он растягивается на всю ширину родителя, затемняет его контент и показывает по середине себя спиннер.
Его можно всунуть куда угодно, главное чтобы родитель был с `position: relative`

##Пример
```js
BEM.blocks['b-loader'].getInstance($('.b-page').bem('b-page')).show()
```
