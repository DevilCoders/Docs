#Текстовая метка с иконкой
###maintainer
@johnson

---

###Use
{
    block: 'icon-text',
    mods: { 
        size: 'xs',
        theme: 'info'
    },  
    text: {
        ...все, что угодно
    }
}



###Mods
size: xs, s, m
theme: info, notice



###Deps
{ block: 'icon-text', mods: { size: 'xs', theme: 'info' } }



###Icon
иконка - блок 'icon' с модификатором, например: { 'size-12': 'info' }
