# Залогиниться в корпоративный Office 365

## Теория { #theory }

Office 365 &mdash; это как гуглодоки, только от Microsoft.
Там есть Word, Excel и много других штук.

У Яндекса есть корпоративная интеграция. Лицензии выданы всем сотрудникам.  

Мы используем Microsoft.Формы для проверочных опросов к некоторым статьям документации. 


## Как залогиниться { #guide }

1. Зайди в проверочный опросник: [ссылка](https://forms.office.com/Pages/ResponsePage.aspx?id=91gB6oFweEaw7X55tD91WioDI29cj_FKvM5f4YSFsflUQ1JRSTc5S000MzU5NTdLN0RCRVQ0N1pMMS4u)  
Если видишь вопрос про 2 x 2, то все уже хорошо, дальше ничего делать не надо.
2. Зайди на <https://office.com/>
3. Введи свою корпоративную почту в формате `login@yandex-team.ru` и нажми `Далее`.
4. Если проредиректило на Паспорт, введи свой доменный пароль.  
Если не проредиректило, должно быть уже все хорошо. 
5. Success!  
Для проверки снова зайди в проверочный опросник: [ссылка](https://forms.office.com/Pages/ResponsePage.aspx?id=91gB6oFweEaw7X55tD91WioDI29cj_FKvM5f4YSFsflUQ1JRSTc5S000MzU5NTdLN0RCRVQ0N1pMMS4u)

Если в п.5 опросник все равно не открывается: пиши на help@ "не логинит в Microsoft.Формы", прикладывай скриншоты браузера.
