# Базы данных, используемые в Директе

## Базы данных продакшена
В Директе используется нескольно различных баз данных: <br> <br>
**MySQL** - исторически самая старая база в сервисе. Содержит практически всю информацию о клиентах, кампаниях и других сущностях в Директе. <br>
**YT** - Yandex Tables. Внутренняя база данных Яндекса. Долго выполняются пишущие запросы, при этом быстро выполняются читающие запросы. Активно используется для загрузки больших объемов данных: много объектов, или статистика. <br>
**YDB** - Yandex DataBase. Еще одна внутренняя база данных Яндекса. Взаимодействует в основном с Java-компонентами Директа. В ней хранятся настройки пользователя (выбранные в интерфейсе столбцы/фильтры, например), сокращенные ссылки для гридов, "заявки" на создание кампаний в Мастере кампаний <br>
**GrUT** - Grand Unified Technology. Новая внедряемая единая база данных Рекламы. Подробнее можно почитать [тут](https://sankear.at.yandex-team.ru/31).

## Базы данных разработческие среды
Существует несколько баз: <br><br>
**prod** - база продакшена. Доступна для читающих запросов. Модифицировать руками нельзя <br>
**test(ts)** - тестовая база для релизных стендов. <br>
**dt(devtest)** - база тестирования на бетах/в бранчах
**dev7** - база тестирования на бетах/в бранчах. Как правило, используется для первичного тестирования миграций.
