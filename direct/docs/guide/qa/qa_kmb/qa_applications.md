# Текстовое краткое описание
**BEM + perl** - исторические самое "старое" приложение, на котором долгое время существовал Директ. Код фронтовой и бэкендной частей хранится совместно. Фронтовые страницы написаны на *BEM* и *tt2*. Бэкенд на *perl*. Количество кода в данной компоненте уменьшается, так как весь новый функционал реализуется в новых компонентах, а старый функционал и страницы переписываются.
<br>

**dna** - компонента, отвечающая за фронтенд профессионального кабинета Директа. Написана на *React + TypeScript*
<br>

**uac** - компонента, отвечающая за фронтенд кабинета для новичков. Написана на *React + TypeScript*. В репозитории Директа находится в разделе *frontend*
<br>

**java-web + kotlin** - бэкенд компонента, отвечающая за взаимодействие с фронтовым кодом как dna, так и uac. Часть, взаимодействующая с dna, написана на *Java*, часть, взаимодействующая с uac - *Kotlin*
<br>

**canvas** - приложение, отвечающие конструкторы креативов в Директе: графические креативы и видеокреативы. Также разбито на 2 компоненты: фронт и бэк
<br>

**java-jobs** - бэкенд компонента, отвечающая за различные автоматически выполняющиеся скрипты. Написана на Java
<br>

**java-api5** - бэкенд компонента, отвечающая за код внешнего API, предоставляемого пользователям. Написана на Java
<br>

**java-intapi** - бэкенд компонента, отвечающая за IntAPI. Написана на Java
<br>

**другие java-приложения** - различные бэкенд компоненты, применимые в различных областях: для просмотра логов, выполнения oneshot-миграций, тестовых степов для тестов и т.д. Написаны на Java
<br>
