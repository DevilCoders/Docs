# Как авторизоваться на стенде crowdtest

Стенд: <https://release.crowdtest.direct.yandex.ru>
  
Стенд открыт наружу, поэтому для доступа к нему используется 2-факторная авторизация. Настроить доступ для себя можно [по инструкции](https://wiki.yandex-team.ru/passport/yateamtotp/#dljaoutstaff-sotrudnikov/vneshnixkonsultantov)  
  
  
_Далее скопированный текст из инструкции_  
  

### Как включить?

Рассматривается на примере приложения Я.Ключ: [Android](https://play.google.com/store/apps/details?id=ru.yandex.key&hl=ru), [iOS](https://itunes.apple.com/ru/app/%D1%8F%D0%BD%D0%B4%D0%B5%D0%BA%D1%81-%D0%BA%D0%BB%D1%8E%D1%87-%D0%BE%D0%B4%D0%BD%D0%BE%D1%80%D0%B0%D0%B7%D0%BE%D0%B2%D1%8B%D0%B5-%D0%BF%D0%B0%D1%80%D0%BE%D0%BB%D0%B8/id957324816?mt=8)

**Можно включить через IDM:**  
* Заходим на <https://idm.yandex-team.ru>
* Выбираем: Система - Мобильные пароли, роль - [Одноразовый пароль (TOTP)](https://nda.ya.ru/3UVJH2), логин - свой логин (или логин подчинённого, для кого запрашиваем), жмём "Запросить"
* На телефон придёт смс, содержащее секрет
* Открываем Я.Ключ, нажимаем на кнопку с плюсом, жмём "Добавить вручную", "Выбрать другой сервис". Выбираем: сервис - "Другой сервис", название сервиса - любое, логин - ваш логин (без собаки и домена), секретный ключ - секрет из СМС, нажимаем кнопку ввода на клавиатуре смартфона.

**Если есть возможность войти из-под VPN, то создать секрет можно через Паспорт:**
* Логинимся в ятиме
* Заходим на <https://passport.yandex-team.ru/profile>
* Жмём "Создать новый секрет для TOTP"
* Открываем Я.Ключ, нажимаем на кнопку с плюсом, сканируем полученный QR-код

### Как пользоваться?
* Заходим на <https://passport.yandex-team.ru>
* Вводим логин и доменный пароль, жмём Войти
* На следующем шаге вводим одноразовый пароль (6 цифр) из приложения. Внимание: пароль живёт всего 30 секунд, так что вводить надо быстро.
  
**Надо помнить, что при авторизации из внешней сети авторизация привязывется к IP. Так что при подключении к другой сети, включении VPN и т.п. придётся переавторизоваться.**  
  

### Как выключить?

Только через IDM, отозвав роль.  

**Если 2фа включалась через Паспорт, то в IDM отключить её можно будет не ранее следующего дня.**  
  

### Что делать, если одноразовый пароль из Ключа не подходит?

Убедитесь, что:
1. на предыдущем шаге вы уже ввели логин и доменный пароль (в вебовском домике)
2. на девайсе, где установлен Ключ, верно выставлено время
3. в Ключ TOTP-секрет введён без опечаток и лишних пробелов (можно попробовать удалить аккаунт из Ключа и добавить заново)
4. при добавлении аккаунта в Ключ использовался пункт "другой сервис" (а не "Яндекс")
5. одноразовый пароль вводится без опечаток
  
Можно также попробовать удалить аккаунт из Ключа, перезапросить роль (отозвать и запросить заново) и снова добавить аккаунт (с новым секретом, который придёт в смс) в Ключ.
