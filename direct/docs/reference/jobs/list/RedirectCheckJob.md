## RedirectCheckJob

### Продукт/подсистема

Кликовые ссылки баннеров независимо от типа.


### Роль в работе продукта/подсистемы

В качестве кликовой ссылки баннера пользователи нередко указывают ссылку на сервис-счетчик, который уже редиректит на лендинг. Данная джоба вычисляет конечные домены по цепочке редиректов и обновляет домены баннеров.


### Датафлоу

- Достает id баннеров из очереди [ppc.redirect_check_queue](https://direct-dev.yandex-team.ru/db/ppc/tables/redirect_check_queue.html)
- Для ссылок из этих баннеров вычисляет конечные ссылки по цепочке редиректов (с помощью сервиса Zora).
- На основе полученных конечных ссылкок обновляет домены в баннерах, а так же сбрасывает некоторые статусы (при необходимости переотправляет в модерацию и БК/Caesar).
