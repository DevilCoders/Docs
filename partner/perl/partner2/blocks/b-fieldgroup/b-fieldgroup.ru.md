Состоит из поля label и поля field. В них отображается то, что передано (текст или блоки).

###Параметры bemjson
- {boolean|object} js По умолчанию не инициализируется
- {string|bemjson} content Содержание поля field
- {string|bemjson} label Содержание поля label
- {string} labelName Атрибут name поля label
- {string} labelValue Значение поля label
- {string} name Атрибут name поля field
- {string} error Текст ошибки

###Флаги
- checked yes Если установлен nullable yes - checkbox будет checked.
