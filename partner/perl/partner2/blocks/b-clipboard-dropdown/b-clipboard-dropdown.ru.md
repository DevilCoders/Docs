Переданный текст будет отображен в блоке b-form-textarea в выпадашке. Для копирования в буфер чуть ниже будет кнопка. Выпадашка отображается по клику на блок-триггер (ссылка по умолчанию).

По сути - это блок b-clipboard в выпадашке.

###Параметры bemjson
- {string} text Текст для копирования
- {string} derection Направление выпадания попапа
- {string} switcher Текст блока-триггера
