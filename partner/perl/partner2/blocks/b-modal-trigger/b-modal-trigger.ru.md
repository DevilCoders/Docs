Содержит кнопку-триггер и модальную форму. По клику на триггер показывает форму.

###Параметры bemjson
- {string} content Контент триггера
- {string} url Сыылка для загрузки формы
- {object} formMods Моды для формы
- {object} triggerMods Моды для триггера
- {string} js.refresh Название параметра, из которого нужно взять текст для обновления контента триггера, при событии action на форме
