Использует b-fieldgroup и b-form-colorpicker.
Отображает надпись рядом с контролом выбора цвета.

###Параметры bemjson
- {string} name Название поля для формы, будет передан в b-form-colorpicker
- {string} content Выбранный цвет, будет передан в b-form-colorpicker
- {bemjson|string} label Текст надписи
- {bemjson|string} error Текст ошибки
