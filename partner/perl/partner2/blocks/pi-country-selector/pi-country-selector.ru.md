Используется блоком `pi-contract-form`. В js-параметры ожидает массив `countries`. Массив должен состоять из объектов, в которых должны присутствовать обязательные поля `name` и `id`. Например: `[{"name":"Киргизия","id":"207"},...,{"name":"США","id":"84"}]`

События
=======

- `change` Срабатывает, когда меняется текущая страна. В обработчики передается объект страны из массива `countries`. Например: `{country: {"name":..., "id":...}}`. Так же событие срабатывает в случаях, когда выделение снимается с какой-либо страны, в таком случае в обработчик передается объект с пустым полем `country`, вот так: `{country: null}`.

Методы
======

- `getCountryById(id)` Возвращает первый объект данных страны из массива `countries` с указанным `id`.
- `val()` Возвращает объект данных текущей выбранной страны.
