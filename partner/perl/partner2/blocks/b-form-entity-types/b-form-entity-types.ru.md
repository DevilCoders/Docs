В данный блок зашит блок *b-form-select* и блок *b-form-input*.
Рисует контрол с выпадающим списком и при выборе определенного пункта меню показывает текствое поля для ввода. Если форма не проходит валидацию из-за этого элемента на сервере, то отображает ошибку.

Возможные свойства:

* itemCustomValue: значение, с которым нужно сверять выбранный пункт списка (для отображения текстового поля)
* errorText: описание ошибки,
* lastInput: последний ввод пользователя, собственно что и привело к ошибке (нужно, чтобы показать самому пользователю)
* lastOption: последний выбранный пункт пользователя
* fieldName: ключ, который добавляется в атрибут name у тегов select и input

В моду content передаются пункты меню для списка (элемент *option*, описание которого можно посмотреть в документации блока b-form-select).
