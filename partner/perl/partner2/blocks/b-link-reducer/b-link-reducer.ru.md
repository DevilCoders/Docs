Блок представляет из себя псевдоссылку или кнопку, по нажанию на которую выпадает попап, при этом происходит ajax запрос на сервер с url текущей страницы и параметрами, если длина параметров превышает заданное количество символов (по умолчанию 256).

Т.к. блок используется, в том числе, из tt2, то решила, что параметры должны в него попадать уже заэнкодиные.

=== Параметры bemjson
- {string} [direction] Направление раскрытия попапа
- {string} [url] Ссылка, на которую будет уходить ajax запрос. Дефолт '/short_link/add'
- {string} [pageUrl] Ссылка на страницу, будет отправлена в сокращатель. Дефолт document.location.pathname
- {number} [minParamsLength] Минимальное число символов в параметрах ссылки, при котором произойдет отправка ссылки в сокращатель. Дефолт 256
- {string} [urlParams] Параметры ссылки. Должны быть уже заэнкодины. Дефолт - пустая строка
