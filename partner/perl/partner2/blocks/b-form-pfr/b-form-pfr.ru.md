Блок с 4 текстовыми полями для ввода номера пфр.
При вводе фокус переходит на следуюее поле, если длина значения текущего поля равна максимальной длине этого поля. Также фокус переходит на предыдущее поле, при удалении текста через backspace.

###Параметры bemjson
- {string} error Текст ошибки
- {string} label
- {string} namePrefix Префикс для атрибута name контролов
- {object} value
- {string|number} value.pfr1
- {string|number} value.pfr2
- {string|number} value.pfr3
- {string|number} value.pfr4
