По нажатию на ссылку показывается форма: поле с автокомплитом для выбора пользователя, кнопка Вход.

Позволяет зайти под логином другого пользователя. Полезно для отлавливания багов, тестирования.

Для работы нужны 2 параметра:
dataProvider - ссылка, будет передана в поле с автокомплитом, нужна для добывания данных;
formAction - ссылка, нужна для авторизации пользователя, будет использована как action для form.
