Данный блок содержит несколько полей формы для редактирования мультиформатов блоков RTB.

### Опции `bemjson`

* *string* `prefix` - Префикс, добавляемый в `html` атрибут `name`.
* *array* `directBlocks` - Массив опций для блока **"Формат Директа"**, `b-form-select`. Формат элементов массива следующий:
    * `{label: 'название', value: 'значение'}`.
* *array* `mediaBlocks` - Массив опций для блока **"Формат медийной рекламы основной"**, `b-form-select`. Формат элементов массива следующий:
    * `{label: 'название', value: 'значение'}`.
* *array* `dspBlocks` - Массив опций, используемые для отображения содержимого блока **"Мультиформаты для DSP"** (преобразуются в список блоков `b-form-checkbox`). Формат элементов массива следующий:
    * `{label: 'название', value: 'значение', selected: 'selected'}`.
* *object* `value` - Предустановленные значения блоков `b-form-select` и зависимого блока "Количество объявлений" `limit`. Необязательная. Может содержать следующие поля:
    * *string* `directBlock` - Значение заранее выбранной опции блока **"Формат Директа"**, которая должна быть в массиве `directBlocks`. В противном случае будет выбрана первая опция в списке.
    * *string* `mediaBlock` - Значение заранее выбранной опции блока **"Формат медийной рекламы основной"**, которая должна быть в массиве `directBlocks`. В противном случае будет выбрана первая опция в списке.
    * *string|number* `limit` - Выбранное **"Количество объявлений"**. Данный блок будет отображен, если `directBlock` = `horizontal` || `vertical`.
