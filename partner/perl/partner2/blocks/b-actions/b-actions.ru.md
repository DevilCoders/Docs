По клику на кнопке выпадает список доступных действий (фактически – список ссылок).

Свойства блока
==============

* `caption` – текст кнопки
* `actions` – список действий, массив объектов вида:

    {
        url: 'URL действия',
        label: 'Название действия (текст ссылки)',
        confirm: 'Вопрос подтверждения действия ("Вы уверены?")'
    }
