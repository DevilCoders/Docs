Блок, который отражает статус перезабора статистики. Работает в паре с `b-web-stat-updater`. Может показать три состояния:

1. Прогресс бар
2. Описание ошибки + кнопка
3. Просто сообщение

#### Методы

+ `id`: Айди блока (нужно для AJAX)
+ `isRunning`: Флаг, показывающий обрабатывается сейчас процесс или нет (в виде прогресс бара)
+ `showBar`: Показывает / Обновляет прогресс бар. В качестве аргументов предаются проценты и время выполнения
+ `showError`: Показывает сообщение об ошибке (переданной в аргументе), а еще кнопку *Продолжить*
+ `showText`: Показывает сообщение (переданной в аргументе)

#### Параметры (св-ва входного *bemjson*-a)

+ `update_button`: объект с параметрами кнопки
    + `label`: Текст кнопки
    + `url`: Урл для AJAX запросов
+ `id`: айди блока (нужен для AJAX запросов)
+ `multistate`: состояние процесса
+ `progressbar`: статус выполнения процесса
    + `progress`: проценты
    + `estimated`: время
+ `error`: сообщение об ошибке
