Попап с ромашкой и текстом. Располагается по центру экрана.

###Параметры bemjson
- {string} text Текст, который будет отображен рядом с ромашкой

###Статические методы
####getInstance
Вернет экземпляр блока. Полезно, когда хочется расположить только один блок на странице и использовать его несколько раз.

###Публичные методы
####show
Отобразит попап с ромашкой.

- {string|bemjson} content Текст, который будет отображаться рядом с ромашкой

####hide
Спрячет попап.
