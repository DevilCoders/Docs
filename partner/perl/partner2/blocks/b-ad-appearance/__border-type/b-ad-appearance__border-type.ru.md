Если тип рамки утсновить в none, то цвет рамки будет задисейблен.

###Параметры bemjson
- {string} type Установленный тип рамки
- {string} color Установленный цвет рамки
- {string} prefix Префикс для аттрибута name
- {string} everyAd Можно ли установить тип рамки в значение everyAd. Принимаемые значения 'yes'|'no'. По умолчанию 'yes'.
