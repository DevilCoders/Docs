###Параметры bemjson
- {string} [error] Общая ошибка блока
- {object} [errors] Ошибки по каждому контролу
- {string} [label] Общее название-заголовок блока
- {string} namePrefix Строка, которая будет добавлена к атрибуту name каждого контрола
- {object} [value] Значения контролов блока
- {string} bankbicURL Ссылка для получения сведений о банке
