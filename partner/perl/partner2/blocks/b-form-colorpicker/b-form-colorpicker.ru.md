Использует `b-form-input` с модификатором `pattern`. Применяется, например, в конструкторе внешнего вида блока RTB (`templates/common/form/rtb_block_appearance.bem.tt2`).

###Параметры bemjson
- {string} id Аттрибут id dom-элемента блока
- {string} content Выбранный цвет
- {string} name Аттрибут name для b-form-input
- {string} mods.disabled Флаг доступности поля, передается в b-form-input
- {bemjson|string} error Текст ошибки
