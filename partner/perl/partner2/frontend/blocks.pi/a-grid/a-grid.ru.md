### Описание

Содержит набор стилей для реализации сеточной верстки.

### Использование

Предполагается подмешивать данный блок и его элементы к другим узлам для добавления требуемых стилей. Данный блок является оберткой для его содержимого. Для реализации колонок стоит подмешивать элемент `col` с разными модификаторами.
Также можно использовать явно в `bemjson`. Есть небольшой набор шоркатов.

### Объявление блока на странице

```
{
    block: 'a-grid',
    content: [
        {
            elem: 'col',
            elemMods: {fixed: 220},
            content: 'Заголовок'
        },
        {
            elem: 'col',
            elemMods: {'fixed-offset': 220},
            content: 'Какой-нибудь элемент'
        }
    ]
}
```
Короткая запись.
```
{
    block: 'a-grid',
    cols: [
        {fixed: 220, content: 'Заголовок 2'},
        {'fixed-offset': 220, content: 'Какой-нибудь элемент 2'}
    ]
}
```
