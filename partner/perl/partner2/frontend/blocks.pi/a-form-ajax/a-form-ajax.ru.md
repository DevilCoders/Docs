Блок для загрузки формы с другого URL. Используется AJAX, а сама форма передается в формате JSON. Форма загружается GET запросом, а данные отправляются POST запросом (по умолчанию добавляется параметр ajax=1 к этим запросам).

Во входном `bemjson` есть возможность указать следующие параметры:
* `autoLoad`: Говорит блоку, нужно ли загружать форму сразу, как инициализировался блок
* `url`: Ссылка, по которой можно загрузить форму
* `sendUrl`: Ссылка, для отправки даннных формы через AJAX (по умолчанию используется атрибут `action` формы или параметр `url`)

### Пример входного bemjson

```
{
    block: 'a-form-ajax',
    {
        autoLoad: true,
        url: '/settings'
    }
}
```
