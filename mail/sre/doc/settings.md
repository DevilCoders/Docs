### Что делает сервис
Сервис предоставляет API для доступа к настройкам почты пользователей.

### Импакт на пользователей от отключения сервиса
Невозможность установки новых IMAP/POP3 сессиий  
Веб и тач переходят в ограниченый режим работоспособности (отображается дефолтная тема, не работают подписи)

### Импакт остановки тачки/ДЦ
Балансер должен перенаправить нагрузку на другие тачки/ДЦ, нагрузка на них возрастет пропорционально.

### Особенности деплоя
При деплое обычно наблюдается небольшой всплеск по таймингам, возможные 500-ки должны скомпенсироваться ретраями.

### Критичные метрики
#### ymsettings_5xx
Процент ответов 5хх с конкретной тачки, агрегируется по количеству загоревшихся тачек
#### ymsettings_4xx
Процент ответов 4хх с конкретной тачки, агрегируется по количеству загоревшихся тачек

### Куда ходит
#### mdb (macs)
Чтение и запись настроек
#### sharpei (cluster_client)
Получение шарда mdb для пользователя
#### blackbox (cluster_client)
Получение параметров паспорта (from_name, default_email)
#### sanitizer (cluster_client)
Проверка хранящейся подписи к письму
