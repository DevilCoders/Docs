Библиотека micronaut-ydeploy
============================

Данная библиотека предоставляет доступ к метаданным окружения сервисов, запущенных в `Y.Deploy`, посредством бина `CloudMetadataProvider` из [utils](java_utils.md).

{% note warning %}

Сервис должен быть запущен  в окружении `ydeploy` (имеется в виду `micronaut`-окружение), чтобы иметь возможность получить доступ к метаданным облака.

{% endnote %}

Метаданные описываются типом `YDeployCloudMetadata`.
