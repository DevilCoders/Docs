## Утилита для фильтрации отчета о регистрации кассы из налоговой

Используется в [процессе R5](https://hitman.yandex-team.ru/projects/trust_cashes/trust_registration_5/?jobCreationTimeQuery=YEAR&jobPage=1&pageSize=20) для того, чтобы оставить только одну кассу в csv файле из ФНС.

С входным и выходным файлом работается в кодировке cp1251. В качестве разделителя используется символ `;`.

Из csv файла остается только одна касса с заданным серийным номером и серийным номером ФН.

Если строчек, удовлетворяющих условию несколько или нет, то возникнет ошибка.


### Параметры

- `--serial-number` – Серийный номер кассы, которую надо оставить. Можно указывать без ведущих в начале 0.
- `--fiscal-storage-sn` – Номер фискального накопителя в записи, которую надо оставить
- `--input-filename` – Имя входного csv файла
- `--output-filename` – Имя выходного csv файла
