Скрипт добавляет метку firm_title ко всем метрикам в проекте trust_cashregisters в соломоне.

Скрипт скачивает все метрики c 2019-12-20 до текущего момента, по ИНН в метке firm в скачанных данных добавляет к ним метку firm_title и пушит данные обратно.

Для работы скрипта необходимо заполнить в коде переменную TOKEN значением токена для соломона и создать окружение с помощью

```
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```
