### Касса в докере
Сборка через docker build.

Запуск например вот так:

```
docker run -it ${image_tag} /bin/bash /kkt/setup_and_run_kkt.sh 01234012340123400002 9999000000000002
```
Где аргументы - серийник кассы и серийник фискального накопителя.

--------
Inspired by [qemu_fr](../../qemu_fr).
Пока дублируются с исходниками:
- templates. Совпадение почти полное, пока не уверен стоит ли их собирать в одно место.
- прошивка (data/FR). Тоже не уверен стоит ли держать в одном месте.
- start.sh претерпел небольшие изменения. Насколько понимаю основано всё вообще на файлике с настоящей кассы.
