# How-to: обновление сертификатов

Для "облегчения" KRL (Key revocation lists) и улучшения безопасности, Skotty всегда получает временные сертификаты. Это означает, что периодически их нужно обновлять.
Для обновления необходимо запустить `skotty renew` и пройти мастер. При использовании Yubikey процесс обновления должен пройти полностью автоматически:
![renew](_assets/skotty_renew.png)

Процесс обновления никак не должен затронуть `legacy` ключ, поэтому о нем можно не волноваться :)
