# Настройка SSH сервера: CAuth

### CAuth 1.0
Текущая версия CAuth, работая поверх OpenSSH, умеет работать с SSH-сертификатами by design. Достаточно загрузить ключи на Стафф и дальше все раскатится само. Поддержки passwordless sudo не планируется, поэтому пользователю Skotty, кроме загрузки ключей на Стафф, ничего делать не нужно.

### CAuth 2.0
Новая версия CAuth как умеет в SSH-сертификаты Skotty из коробки, так и сам настраивает PAM для работы с passwordless sudo. Пользователю Skotty все так же, ничего дополнительно предпринимать не нужно.
