# Gideon

Container aware сборщик проб для Linux. Для работы нужно ядро 4.17.0+, собирается (без приседаний) для 4.19.x (будет пофикшенно в будущем, см. [BPF CO-RE](https://facebookmicrosites.github.io/bpf/blog/2020/02/19/bpf-portability-and-co-re.html))

Вводная документация: https://wiki.yandex-team.ru/security/gideon/
Интеграция с Portoshell: https://wiki.yandex-team.ru/security/gideon/portoshell/
Интеграция с SSHD с помощью PAM: https://wiki.yandex-team.ru/security/gideon/pam/
