# burp-active-scanner

# Overview
Burp-active-scanner is active scanner and crawler extension for BurpSuitePro.
Plugins turns BurpSuite to standalone web application security scanner.

# Usage

* Build FatJar with  `ya make`
* Change and save burp_project_config.json and burp_user_config.json
* Put path to config in MOLLY_CONFIG Environment variable
