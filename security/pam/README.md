# PAM
This repo contains pam modules. They are wrapped into the debian packages for the ease of the installation.  
Pam stands fot Pluggable Authentication Modules. It is used for setting up a configurable authentification.
## Usefull links

about pam: man 8 pam  
loginuid: man 8 pam_loginuid

### Installing

Just add the repository to the ```/etc/apt/sources.list``` and install the package.
