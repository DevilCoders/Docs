Если вам нужен конечный тул, который можно запускать для поиска: [security/ant-secret/secret-search/bin](https://a.yandex-team.ru/arc/trunk/arcadia/security/ant-secret/secret-search/bin)

Если библиотечки то есть два публичных интерфейса:
