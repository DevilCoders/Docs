# C3PO Robot

![](https://center.yandex-team.ru/api/v1/user/robot-c3po/photo_17684/300.jpg)

## Project structure

* **bin** - contains C3PO binary and sources of main entry point
* **components** - Core components and plugins source code
* **docker** - crontab, supervisor and C3PO plugin configs brought together to build a C3PO package using Dockerfile for Deploy
