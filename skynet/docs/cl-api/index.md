# Структура документа

1. [Общие сведения о Skynet](introduction.md): назначение {{ serviceName }}, краткие сведения об установке на серверы, список сервисов, перечень доступных программных интерфейсов.
    
1. [Объекты управления](targets-classif.md): описание объектов, для управления которыми используется sky``. Сведения о доступных способах указания серверов.
    
1. [Группы команд](commands-groups.md): информация о группировке команд `sky`, используемой в документе.
    
1. [Зависимости серверов и групп](sky-list.md): описание команд, возвращающих сведения о взаимосвязи объектов (например, о группах, к которым относится хост).
    
1. [Технические данные](sky-top.md): описание команд, возвращающих сведения о нагрузке на серверы (например, [среднее количество процессов в очереди](sky-load.md)).
    
1. [Исполнение удаленных команд](sky-run.md): описание команд, предоставляющих возможность выполнять произвольные команды на удаленных серверах.
    
1. [Работа с данными](sky-share.md): описание команд, используемых для предоставления доступа к ресурсам (каталогам и файлам) указанного сервера и дальнейшей загрузки данных ресурсов на прочие серверы.
    
Информация о выпущенных версиях документа приведена в разделе [Релизы](releases.md).
