# Введение

**Документ содержит** инструкции по использованию одного из программных интерфейсов {{ serviceName }} — command-line API (`sky`).

{{ serviceName }} — это набор сервисов, предоставляющих возможность управлять серверами Яндекса, в том числе — удаленно запускать команды и эффективно копировать каталоги и файлы на группе серверов.

В документе приведена общая информация о {{ serviceName }}, рассмотрены основные сервисы, приведен список объектов управления, описаны команды `sky`.

**Документ актуален** для {{ serviceName }} версии 16.12.0 и старше.

**Целевая аудитория документа** — разработчики и администраторы, использующие `sky` для удаленного выполнения процедур на серверах.

**Документ не содержит** подробных сведений об установке {{ serviceName }}, инструкций по использованию Python API.
