#### Релиз 2.0. Декабрь 2018 г.

- Документ переработан.
- Добавлено описание команд:
    - [Список доступных конфигураций (sky listconf)](sky-listconf.md).
    - [Доступные инстансы (sky listinstances)](sky-listinstances.md).
    - [Время работы после загрузки (sky uptime)](sky-uptime.md).
    - [Выполнение команды в контейнерах porto (sky portorun)](sky-portorun.md).
    - [Удаленный доступ к контейнерам porto (sky portoshell)](sky-portoshell.md).
    - [Копирование и запуск бинарного файла с текущего хоста на удаленные (sky launch)](sky-launch.md).
    - [Копирование данных в (из) удаленного контейнера porto (sky portocopy)](sky-portocopy.md).
    
- Добавлен раздел [Глоссарий](glossary.md).
- Удалены разделы (команды больше не используются):

    - Список процессов (sky stat).
    - Статистика использования ресурсов (sky rusagestat).

#### Релиз 1.3. Апрель 2014 г.
Обновлен раздел [Хосты группы (sky list)](sky-list.md):
- Добавлено описание новых [форматов вывода](sky-list.md#response-format) ([JSON](sky-list.md#json) и [MessagePack](sky-list.md#messagepack)).
- Добавлено описание ключа `-f` (`--format`).
- Актуализировано описание ключа `--cut-domain`.
Раздел [Проверка доступности хоста по cqudp (sky ping)](sky-ping.md) дополнен описанием ключей `--extended` и `--check-writable`. Для всех команд уточнено описание ключа `-l` (`--long_names`).
