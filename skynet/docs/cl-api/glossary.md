# Терминология

#### Xост {#host}

_Синоним: host_.

Уникальный идентификатор сервера.

#### Группа хостов {#host-group}

Статическая группа хостов, объединенных по определенному признаку.

Каждый хост может входить в неограниченное количество групп.

#### Инстанс {#instance}

Экземпляр программы.

На одном может быть запущено неограниченное количество инстансов. Конкретный инстанс может быть запущен на одном хосте. Прочим экземплярам программы назначаются отличающиеся идентификаторы.

#### Группа инстансов {#instance-group}

Группа инстансов, объединенных по определенному признаку. В большинстве случаев объединяются инстансы, запущенные на разных хостах.

#### Шард {#shard}

Логическая единица, используемая для деления поисковой базы.

#### Контейнер {#container}

_Синоним:_ _Слот_, _Porto-контейнер_.

Изолированное окружение для запуска процессов. Является основной единицей системы контейнерной изоляции [Porto](https://wiki.yandex-team.ru/porto/).
