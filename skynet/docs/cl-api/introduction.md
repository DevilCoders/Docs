# Общие сведения о Skynet

Skynet — это набор сервисов и утилит, используемых для управления серверами Яндекса. Среди клиентов Skynet — Поиск, БК, Маркет, Карты и прочие сервисы.

Ключевые возможности Skynet:

* Одновременное выполнение команд на большом количестве удаленных серверов.
* Эффективное использование сетевых ресурсов.
* Высокая скорость копирования ресурсов между серверами.
* Использование отказоустойчивой модульной системы формирования и доставки отчетов.

[Ограниченный набор](commands-groups.md) функциональных возможностей {{ serviceName }} доступен из командной строки посредством программного интерфейса `sky`.
