# Параметры установки и функционирования

Параметры установки и функционирования {{ serviceName }} задаются в сервисе [Genisys](https://genisys.yandex-team.ru). Чтобы получить доступ к Genisys, напишите на рассылку  [skynet@yandex-team.ru](mailto:skynet@yandex-team.ru).

Для сборки и хранения {{ serviceName }} используется [Sandbox](https://sandbox.yandex-team.ru/). Версия {{ serviceName }}, которая должна быть установлена на группе серверов, определяется по идентификатору задачи Sandbox.

В {{ deploy-and-configService }} вы можете определить версию {{ serviceName }} через [псевдоимена](aliases.md) (_алиасы_). Например, назначить номер текущей актуальной версии алиасу <q>stable</q>, а группам, на которые ее необходимо установить, поставить в соответствие данное псевдоимя. В дальнейшем изменение текущей версии требуется вносить только в запись об алиасе.

Также в {{ deploy-and-configService }} вы можете создать специфичную конфигурацию одного или нескольких сервисов  {{ serviceName }}, если для сервера необходима конфигурация сервиса отличная от базовой.

Подробную информацию о настройке параметров установки и функционирования {{ serviceName }} вы можете найти в разделах:

- [Версия Skynet](skynet-version.md)
- [Алиасы](aliases.md)
- [Конфигурация сервисов](services-configuration.md)
