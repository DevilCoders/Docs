# Установка
Skynet устанавливается на серверах с операционными системами:
- linux и glibc 2.11+ (например, ubuntu lucid и старше);
- macOS.

Порядок установки {{ serviceName }} на сервер представлен на рисунке ниже.
![image](_images/installation.png)
1. Утилита `gosky` (пакет `yandex-gosky`) устанавливается на целевой сервер.
    
1. Утилита `gosky` запрашивает в [{{ deploy-and-configService }}](https://genisys.yandex-team.ru) сведения о версии {{ serviceName }}, которая должна быть установлена.
    
1. Указанная версия `skynet.bin` загружается на сервер и устанавливается.
    

После первичной установки {{ serviceName }} утилита `gosky` обеспечивает регулярное обновление {{ serviceName }} до актуальных версий.

На боевых серверах установлены совпадающие версии {{ serviceName }}. Различия
 возможны только при тестировании новых функциональных возможностей или проведении
 экспериментов с конфигурациями.

Поддерживается обратная совместимость версий. Если по каким-то причинам обратная совместимость невозможна, выполняется принудительная установка идентичных версий {{ serviceName }} на всех серверах.
