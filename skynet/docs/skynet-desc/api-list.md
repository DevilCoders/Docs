# Программные интерфейсы

Доступ к функциональным возможностям {{ serviceName }} обеспечивается программными интерфейсами: Python API и command-line API (далее — sky).

Python API используется, если вызов функциональных возможностей {{ serviceName }} выполняется из исходного кода, написанного на Python.

{% note alert %}

Python API необходимо использовать с версией Python, распространяемой с {{ serviceName }}.

{% endnote %}


sky используется администраторами. Предоставляет доступ к ограниченному набору функциональных возможностей {{ serviceName }} из командной строки.
