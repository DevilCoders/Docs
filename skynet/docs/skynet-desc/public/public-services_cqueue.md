# CQueue / CQudp

Используется для одновременного запуска команд на большом количестве серверов. Позволяет выполнять Python-код.

Преимущества по сравнению с `ssh`:

- Высокая скорость выполнения команд.
- Одновременное выполнение на большом количестве серверов.
- Агрегация результатов со всех серверов, на которых выполнена команда.

Реализации сервиса: `CQudp` и `CQueuе` (устаревшая).

[Программные интерфейсы](../api-list.md) {{ serviceName }} поддерживают обе версии сервиса, однако настоятельно
 рекомендуется использовать `CQudp`, к преимуществам которого относятся:
- Более высокая сходимость и стабильность.
- Корректная работа с IPv6-only-хостами.
