# procman

API для запуска, управления и мониторинга процессов {{ serviceName }}. Является аналогом демона инициализации демонов `systemd`, используемого в Linux.

Ключевые возможности:

- Тегирование запускаемых процессов.
- Автоматический перезапуск отказавших процессов.
- Запуск процессов от имени разных пользователей.
- Aгрегация stdout и stdin в единый лог procman.
