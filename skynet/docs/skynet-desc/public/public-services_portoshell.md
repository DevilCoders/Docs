# portoshell

CLI-сервис, предоставляющий авторизованный доступ к Porto-контейнерам без использования CAuth и
 SSH. Используется для отладки контейнеров. Обращения возможны от имени пользователя, создавшего
 контейнер, или суперпользователя (root).

Доступ к контейнерам по SSH может быть ограничен рядом причин, например:
- Сетевой доступ невозможен.
- Файловая система контейнера ограничена.
- LDAP не настроен.

portoshell обходит данные ограничения и обеспечивает прямой доступ к контейнерам.
