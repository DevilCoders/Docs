# copier

Используется для копирования файлов между серверами, на которых установлен {{ serviceName }}. Позволяет одновременно копировать файлы на неограниченное количество серверов. Для передачи данных используется модифицированный протокол BitTorrent.

Передача файлов состоит из следующих этапов:

- Предоставление доступа к файлам (<q>расшаривание</q>). В результате формируется
 идентификатор, по которому доступны соответствующие ресурсы.
- Скачивание файлов по полученному идентификатору.

Преимущества по сравнению с прочими способами копирования файлов:

- Отсутствие необходимости знать точный адрес, по которому доступны передаваемые файлы
 (скачивание выполняется по идентификатору). Расположение ресурсов в рамках файловой системы при
 этом может меняться.
- Возможность эффективно копировать файлы с одного сервера на множество других.
- Корректная балансировка нагрузки на сеть.

{% note info %}

Помимо модифицированного BitTorrent `copier` также поддерживает передачу данных посредством rsync, HTTP, Sandbox и пр.

{% endnote %}
