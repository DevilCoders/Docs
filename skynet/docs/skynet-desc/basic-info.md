# Общие сведения

{{ serviceName }} — это набор сервисов и утилит, используемых для управления серверами Яндекса. Среди клиентов {{ serviceName }} — Поиск, БК, Маркет, Карты и прочие сервисы.

Ключевые возможности {{ serviceName }}:

- Одновременное выполнение команд на большом количестве удаленных серверов.

- Использование отказоустойчивой модульной системы формирования и доставки отчетов.
