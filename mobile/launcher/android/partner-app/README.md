Есть две активные ветки: 

<b>master</b> (ex. Partners)

Ветка содержит конфигурации для партнеров. Для каждого партнера отведен свой flavour. 

Есть возможность сборки как конфигов, так и шаблонов проектов для отправки партнерам.

После сборки шаблона проекта на выходе получается .zip файл с проектом, который готов для отправки партнеру. 
При сборке требуется в параметре 'partner' указать партнера для которого собирается шаблон: 

    ./gradlew app:assembleTemplate -Ppartner=flylow

<b>template</b> (ex. base)

Ветка содержит шаблон проекта конфигурации, который  отправляется OEM для самостоятельной подготовки конфигурации. В шаблон добавляются список параметров с описанием, которые мы ходим позволить менять OEM. Обычно список параметров будет меньше, чем мы используем в master. После сборки проекта на выходе получается .zip файл с проектом, который готов для отправки партнеру. 

<b> В полученном проекте нет изменений, присутствующих в конфиге партнера</b>

При сборке требуется в параметре 'partner' указать партнера для которого собирается шаблон: 

    ./gradlew assembleTemplate -b distribution/distribution.gradle -Ppartner=flylow
