# Я.Продавец

Приложение маркета для продавцов на базе Flutter SDK

## Запуск

Flutter SDK рекумендуется устанавливать через [FVM](https://fvm.app), а настройку окружения выполняем по [стандартной схеме](https://docs.flutter.dev/get-started/install/macos)
Перед началом работы также потребуется настроить окружение для работы с внутренним менеджером пакетов unpub. Сделать это можно по [инструкции](https://wiki.yandex-team.ru/taxi/mobile-infra/unpub/#polzovateljamservisa)

После чего можно выполнить команду установки зависимостей
```
flutter pub get
```
Затем необходимо сгенерировать файл конфигурации в debug-режиме или prod
```
make config:dev
// или
make config
```

Далее необходимо установить зависимости для iOS
```
cd ios && pod install
```

Запускаем эмулятор
```
open -a Simulator
```
Стартуем сборку проекта с помощью команду run или через инструменты IDE
```
flutter run
// или FVM (если настроили)
fvm flutter run
```

После первой сборки вы можете получить ошибку об неиспользуемоый переменной сертификата. Это связано с ошибкой в одной из зависимости для iOS. Для решения проблемы необходимо открыть xCode, запустить сборку проекта (CMD+B), далее всплывет ошибка с указанием места. Необходимо закомментировать часть кода с неиспользуемой переменной и подтвердить Unlock. После чего можно снова запустить сборку проекта проект.
