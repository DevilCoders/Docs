# Клиентские события для Тестопитека

### Общая информация 
- Каждое клиентское событие описывается строковым именем и словарем: ``Map<string, any>``

### Процесс логирования на клиенте
Для настройки процесса логирования при помощи библиотеки клиентских собыйтий Тестопитека необходимо:
- Создать реализацию интерфейса `EventReporter` определив в нем метод `report(event: LoggingEvent)`. 
- `LoggingEvent` хранит в себе имя и, список атрибутов, которые необходимо отправить. 
- Зарегистрировать реализацию в `EventusRegistry.setEventReporter(reporter: EventReporter)`.
- Для отправки события необходимо вызвать `EventusRegistry.report()`
