### Баги

* Если remote HEAD не находится в первой десятке комитов, то мы не можем определить, что запушено.
* При рефреше сбивается размер подгруженных в истории коммитов.
* Когда находишься в Changes и пробуешь выделить несколько файлов с помощь Shift + стрелочки вниз/вверх, происходит что-то странное.
* При переключении на Arcane из другого приложения с помощью Cmd+Tab или по иконке в Dock происходит большая задержка.
* Не показываются новые файлы в новой папке.
* Нельзя сделать amend, если в stage нет файлов, чтобы просто поменять имя коммита.
* В описании комита если попереключать табы бывает наложение одного view на другое(похоже на баг SwiftUI).

### Сборка

Версия Xcode: 11.4

Собрать приложение можно через Xcode или через fastlane:

```bash
rvm use .
bundle install
bundle exec fastlane build_arcane
```

Также в проекте подключен линтер [SwiftLint](https://github.com/realm/SwiftLint) *(версия 0.39.2)*, настройки которого можно посмотреть в `.swiftlint.yml`. Данный линтер в данный момент опционален, т.е. если он у вас нее установлен никаких проверок производиться не будет.
