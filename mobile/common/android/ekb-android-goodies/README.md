# ekb-android-goodies

Ответственный: [Порошин Иван](https://staff.yandex-team.ru/poroshin-ivan).

**Android Goodies** -- это библиотека общих компонент для Андроид-приложений, разрабатываемая [Группой разработки Android приложений](https://staff.yandex-team.ru/departments/yandex_infra_tech_ekbinterface_android/). Библиотека решает проблему дублирования кода и предоставляет инфраструкурные решения, приводящие приложения к единообразию.

Документация на вики: https://wiki.yandex-team.ru/users/poroshin-ivan/agoodies/
