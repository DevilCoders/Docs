## About
uikit - yandex flavor for material design

## Contents
 - `colors` - basic colors (no dependencies)
 - `styles` - basic styles (depends on support design library)
 - `fonts` - yandex sans fonts (depends on support appcompat). Depends on all other fonts libraries.
 - `fonts-ya-light` - YS text light
 - `fonts-ya-regular` - YS text regular and YS text italic
 - `fonts-ya-medium` - YS text medium
 - `fonts-ya-bold` - YS text bold
 - `fonts-ya-display-light` - YS display light
 - `fonts-ya-display-regular` - YS display regular
 - `fonts-ya-display-medium` - YS display medium
 - `fonts-ya-display-bold` - YS display bold

## Usage
```
ext.uiKitVersion = '0.6.0'
implementation "com.yandex.android.uikit:colors:$uiKitVersion"
implementation "com.yandex.android.uikit:styles:$uiKitVersion"
implementation "com.yandex.android.uikit:fonts:$uiKitVersion"
implementation "com.yandex.android.uikit:fonts-ya-light:$uiKitVersion"
implementation "com.yandex.android.uikit:fonts-ya-regular:$uiKitVersion"
implementation "com.yandex.android.uikit:fonts-ya-medium:$uiKitVersion"
implementation "com.yandex.android.uikit:fonts-ya-bold:$uiKitVersion"
implementation "com.yandex.android.uikit:fonts-ya-display-light:$uiKitVersion"
implementation "com.yandex.android.uikit:fonts-ya-display-regular:$uiKitVersion"
implementation "com.yandex.android.uikit:fonts-ya-display-medium:$uiKitVersion"
implementation "com.yandex.android.uikit:fonts-ya-display-bold:$uiKitVersion"
```
