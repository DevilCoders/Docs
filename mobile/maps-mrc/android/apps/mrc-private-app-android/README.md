Это приватное приложение Зеркал, используемое для сбора данных по заданиям. В настоящий момент в нем поддерживается три типа заданий:
- фотографирование объездов дорог
- сбор радио-карт схем помещений
- пешеходные задания актуализации данных "последней мили".

### Как перейти на новую версию mapkit

1. Выбрать релизную версию mapkit из [этого списка](http://artifactory.yandex.net/artifactory/yandex_mobile_releases/com/yandex/maps/maps.mobile/), соответствующие релизные ветки расположены [тут](https://a.yandex-team.ru/arc/branches/maps-mobile-releases/)
2. Обновить зависимость [тут](https://a.yandex-team.ru/arc/trunk/arcadia/mobile/maps-mrc/android/apps/mrc-private-app-android/app/build.gradle?rev=r8352654#L54)
