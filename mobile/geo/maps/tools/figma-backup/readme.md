### Утилита предназначена для создания локальных бекапов .fig файлов

**Оригинальная версия:** <https://github.com/ritds/figma_backup>

Для работы необходимо:
1. python 3, pip 3
2. установленная node v16.13.0

Настройка:
1. в файле manage.sh необходимо в переменные figma_login и figma_password прописать логин / пароль учетной записи figma
2. в файле get_figma_files_list.yml необходмо указать
    1. **access_token** - figma token учетной записи 
    2. **teams** - список идентификаторов команд (можно подсмотреть в урле https://www.figma.com/files/<...>/team/<нужный ид команды>/...)
    3. **age_limit** - максимальное количество дней со времени последнего изменения файла (используется для фильтрации файлов при регулярном бекапе)

Запуск
1. При первом запуске необходимо установить npm зависимости, для этого нужно выполнить `npm i`
2. Выполнить `./manage.sh`

Файлы сохраняются в папку ./work/figma/<текущая дата>
