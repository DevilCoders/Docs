# Генерилка мультиплатформенных модулей

Запускается командой:

	cd tools && ./gradlew :multiplatform-module-creator:run --args='<new nultiplatofrm module name>'

`<new nultiplatofrm module name>` - название нового модуля

Полное описание генерилки: [https://wiki.yandex-team.ru/maps/mobile/dev/multiplatform/generilka-multiplatformennyx-modulejj/](https://wiki.yandex-team.ru/maps/mobile/dev/multiplatform/generilka-multiplatformennyx-modulejj/)
