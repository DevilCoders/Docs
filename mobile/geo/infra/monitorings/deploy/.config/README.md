

### ./cron
Описание, когда и какие cron таски будут запускаться. <br />
В начале файла перечислены переменные окружения, которые будут переданы в cron таски. <br />

### ./checks
Сигналы, из которых данные будут прорастать в Голован. <br />
Для описания каждого из сигналов есть по скрипту, которые лежат в папке [.config/checks](./.config/checks). <br />
Эти скрипты отдают содержимое файлов, в которые наши cron таски сложили значения. <br />
