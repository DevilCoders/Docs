# Progress
Используется для отображения прогресса выполнения чего-либо. [Подробнее](https://ru.bem.info/libs/bem-components/v2.3.0/desktop/progressbar/)

## Использование
```jsx
var Progress = require('bem-react/lib/Progress');

<Progress val={50} />
```

## Параметры

#### theme `String`
Тема: `islands`

#### val `Number`
Процент выполнения
