Small Flask application that allows to start shooting and find out its status.

# Build and run
## Arcadia way
```
$ cd firestarter/bin/
$ ya make
$ ./firestarter
```
## Python way
```
$ cd firestarter/
$ virtualenv venv_fs
$ source venv_fs/bin/activate
$ python -m pip install -r reqs.txt -i https://pypi.yandex-team.ru/simple
$ cd src/
$ python main.py
```
