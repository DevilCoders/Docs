## Краткое описание
Клиент использует SUP API для отправки пуш-уведомлений.

Подробный список параметров для отправки можно найти в [документации SUP](https://doc.yandex-team.ru/sup-api/concepts/push-create/index.html).

Чтобы начать пользоваться клиентом, нужно зарегистрировать проект в SUP. [Подробная инструкция как это сделать.](https://doc.yandex-team.ru/sup-api/common/registration.html)
