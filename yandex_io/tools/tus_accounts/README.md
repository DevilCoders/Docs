Простая тулза для создания tus аккаунтов

Для создания например 5 аккаунтов с плюсом нужно запустить:

# ./tus_accounts --first-login-suffix-number 0 --accounts-num 5 --with-yandex-plus --login-prefix "yndx-io-functests-plus-"

В выхлопе утилиты будет список созданных логинов, которые нужно передать медиа-сервисам, чтобы на них навесили Яндекс.Плюс.
По дефолту аккаунты имеют префикс **yndx-io-functests-**.
Для аккаунтов с плюсом нужно испльзовать префикс: **yndx-io-functests-plus-**
Аргумент **--first-login-suffix-number** нужен, чтобы указать первый свободный индекс аккаунта (сейчас не поддержан листинг аккаунтов. Первый свободный аккаунт нужно искать по тикетам :)).
