## Утилита для получения oauth code для Yandex.IO

Нужна для авторизации демонов в обход стандартной схемы (через sound_initd,
также известной как "пилик").

Для авторизации во внутренних сервисах получает токен по SSH-ключу.
Для создания кода авторизации, принимает куку `Session_id` от ****внешнего*
домена в аргументе `--sessionid`.
