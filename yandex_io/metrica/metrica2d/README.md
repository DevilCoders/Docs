# metrica2d

## Что это?
Будущая замена metricad, которая будет уметь отправлять метрики в несколько мест (AppMetrica, Clickdaemon).

## Зачем переписывать?
Текущая реализация metricad сильно завязана на аппметрике, поэтому ее сложно научить слать метрики в разные места.

## Почему metrica2d, а не рефакторинг appmetrica?
Чтобы как можно быстрее написать отправку метрики в кликдемон и не блочить ее проверкой рефакторинга аппметрики. После того, как metrica2d будет готов и проверен на кликдемоне, будет рефакторинг аппметрики и metrica2d полностью заменит metricad
