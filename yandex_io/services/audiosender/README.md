Серверная часть аудиосендера, запускаемая на устройстве, с которого хочется записать звук. Клиентское приложение запускается на Android. Связь осуществляется через веб-сокеты, для отладки без клиентского приложения можно воспользоваться утилитой `websocat`. Описание протокола можно найти здесь: https://wiki.yandex-team.ru/users/yashrk/AudioSenderProtocol/.

## Посылаемый звук
Т.к. сервер воспринимает только одноканальный 16-битовый звук, то желательно, чтобы звук писался в этом формате.

## Сборка

Т.к. audiosender вошёл в состав сборки yandex_io по умолчанию, никаких дополнительных действий и отличий от дефолтной сборки больше нет.

## Запуск
Пока для запуска есть одна возможность - добавить флаг `"audiosenderMode": true` в конфиг космодрома для устройства.
В течение минуты устройство подхватит флаг и перезапустит сервисы `yiod` и `audiod` в режиме `audiosender`. `AliceService` в таком режиме запущен не будет. Однако возможность настроить колонку (пропиликать) останется.

## Конфигурация

Настройки `audiosender`, как и остальных сервисов, находятся в соответствующей секции `quasar.cfg`. Конфигурация записи звука, с которой перезапускается `audiod` при получении флага `audiosenderMode` находится внутри секции `audiod::audiosender`.
