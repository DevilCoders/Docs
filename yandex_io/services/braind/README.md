### Braind

## Meaning

Braind is a service that hold most of the states  that Quasar Services have. 
This state will be provided to IO SDK User. The state can be used for different purposes (i.e. LED animations)

## Braind DeviceState 

* Mics State 
* Alice State
* Bluetooth State
* Configuration State
* Update State
* Volume State
* Alarms/Timers State
