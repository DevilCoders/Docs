# stories-utils

https://pages.github.yandex-team.ru/morda/stories-utils/

Всякие мелкие скрипты для подготовки сториз

## Фоны для погодных сториз

1. `node ./index.js <папка с деревом файлов бэкграундов>`
Сгенерируется html файл

2. Открыть это файл, потыкать галочки/цвета, В самом низу нажать кнопку
Сгенерируется json файл

3. `node ./optimize.js <папка с деревом файлов бэкграундов> <файл из п.2>`

Видео будут подрезаны и кадрированы.
Рядом в папке `<папка с деревом файлов бэкграундов>-preview` будет
создано дерево файлов с тизерными картинками и tsv файл экспорта.

4. Залить видео из `<папка с деревом файлов бэкграундов>` в `s3://home/stories/weather_gifs` 

5. Залить видео из `<папка с деревом файлов бэкграундов>-preview` в `s3://home/stories/weather_gifs_preview`

6. Импортировать в мадм экспорт из п.3

Для п.4-5 рекомендуется использовать awscli.
