# Регламент по добавлению новых проверок
* Все портальные проверки заводятся через [шаблоны алертов в Golovan](https://wiki.yandex-team.ru/golovan/userdocs/templatium/alerts/) и/или через [portal-checks-generator](https://github.yandex-team.ru/morda/admin/tree/master/portal-checks-generator#%D0%B3%D0%B5%D0%BD%D0%B5%D1%80%D0%B0%D1%82%D0%BE%D1%80-%D0%B0%D0%BA%D1%82%D0%B8%D0%B2%D0%BD%D1%8B%D1%85-%D0%B8-%D1%85%D0%BE%D1%81%D1%82%D0%BE%D0%B2%D1%8B%D1%85-%D0%BF%D1%80%D0%BE%D0%B2%D0%B5%D1%80%D0%BE%D0%BA-%D0%B2%D1%81%D0%B5%D1%85-%D1%81%D0%B5%D1%80%D0%B2%D0%B8%D1%81%D0%BE%D0%B2-%D0%B3%D0%BB%D0%B0%D0%B2%D0%BD%D0%BE%D0%B9-%D1%81%D1%82%D1%80%D0%B0%D0%BD%D0%B8%D1%86%D1%8B).
* Каждый портальный [агрегат](https://docs.yandex-team.ru/juggler/aggregates/basics) в Juggler соответствует одному сервису (пример для Геохелпера: [host=portal.geohelper](https://juggler.yandex-team.ru/aggregate_checks/?query=host%3Dportal.geohelper)).
* На каждый сервис должно быть не более одного отдельного шаблона в Golovan и/или одного скрипта в portal-checks-generator (пример: [https://yasm.yandex-team.ru/template/alert/portal.geohelper/](https://yasm.yandex-team.ru/template/alert/portal.geohelper/) и [https://github.yandex-team.ru/morda/admin/blob/master/portal-checks-generator/geohelper.py](https://github.yandex-team.ru/morda/admin/blob/master/portal-checks-generator/geohelper.py)).
* Если проверки могут покрыть несколько сервисов, то они выделяются в отдельный шаблон, которй потом вызывается из других шаблонов через [`<% include "..." %>`](https://wiki.yandex-team.ru/golovan/userdocs/templatium/alerts/?from=%1Fgolovan%2Ftemplatium%2Fnapisanie-shablonov-alertov%2F#reusage) (пример: [https://yasm.yandex-team.ru/template/alert/portal.infra.template](https://yasm.yandex-team.ru/template/alert/portal.infra.template)).
* В каждой проверке обязательно должен присутствовать тег срочности (см. [Оповещения](https://testing.docs.yandex-team.ru/portal/monitoring_notifications))
* В каждой проверке должна быть указана ссылка на инструкцию; также желательно добавить ссылку на чат смежников, если эта проверка на стыке Морды и другого сервиса.
