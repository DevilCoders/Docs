## Что это
Проект по переводу загрузки Дзена на server side в веб-мордах.

## Какие проблемы решает, какая цель
Исторически Дзен на Морде загружался с клиента. Такой вид загрузки вынуждал пользователя ожидать выполнения нескольких последовательных сетевых запросов до получения контента Дзена.
Решено было перенести формирование ответа Дзена на сервер с последующей выдачей его после ответа морды.

## Каким методом решается задача
Перевести бекенд и фронтенд морды на работу под аппхостом, сделать общий граф морда+дзен, где аппхост отдаёт клиенту ответ морды, затем ответ дзена.
