# Введение

Документ содержит описание операций [смены языка пользователя](concepts/operations-lang.md), [записи значений в куку my](concepts/operations-cookiemy.md) и [получения Geo-саджеста](concepts/operations-geo.md), которые можно осуществлять с помощью АPI Tune.

Целевой аудиторией данного документа являются разработчики интерфейсов.
