# Конфигурация вебпака для сборки морды
Здесь лежит минимальная морда, на которой можно отладить конфигурацию вебпака.
В этой морде есть примеры использования:
- страницы, использующие одинаковый набор уровней
- страницы, использующие разный набор уровней
- переобределение блоков на разных уровнях
- инлайн стилей, скриптов, svg и png файлов
- клиентсайд шаблонизация
- локализация
- скины
- страничные бандлы статики
- блочные файлы статики

Большинство из этих фич нужно поддержать в конфиге вебпака,
делаем это в соответствующих сабтасках https://st.yandex-team.ru/HOME-51622,
затем наносим добро в тюн, затем в морду.

##Как запустить на девинстансе

1. Склонировать
2. В `configurator/configurator.conf` создать секцию про свою дев машину, аналогично v109
3. `make conf all`, он установит прекоммит хук, положит конфиг nginx, оставит вебпак в --watch режиме и запустит нодовый сервер
4. Открыть в браузере `https://next-<инстанс>.wdevx.yandex.ru`

##Как запустить локально
1. Склонировать
3. `make all`, он установит прекоммит хук, оставит вебпак в --watch режиме и запустит нодовый сервер
4. Открыть в браузере `http://localhost:3000`
