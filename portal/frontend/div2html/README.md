### div2html

Умеет рисовать divjson / divjson2 в *статичную* хтмл.
Не рассчитано на использование в продакшене и для кликов пользователем, только для статичных картинок

`index.html` умеет рисовать divjson2. Для работы ожидает `?url=<url>`

`dialog.html` умеет рисовать диалоги алисы с divjson/divjson2. Для работы ожидает `?url=<url>`

Например:
`dialog.html?url=div1dialogs/example.json`

Нужно, чтобы файл по `?url` отдавался с CORS-заголовками. Файл с диска не заработает
