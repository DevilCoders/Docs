# direct-ad-adapter
Репозиторий адаптера рекламных данных для переезда на единый ответ БК

имеет cli интерфейс для запуска вне морды, для морды будет использоваться только функция адаптера

запуск в cli:

bin/direct-ad-adapter file.json


или npx из яндексового https://npm.yandex-team.ru:

npx direct-ad-adapter file.json
