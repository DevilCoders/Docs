# Hermione-mute-unchanged

Плагин для скипа тестов, для которых в локальном гите нет изменений.

## options

- **enabled** Состояние плагина
- **ref** Верхняя граница. По-умолчанию `HEAD`
- **prevRef** Нижняя граница. По-умолчанию `HEAD^`
- **match** Объект для сопоставления тестов и изменённых файлов.

    Ключами являются маски на полное название теста,
    Значениями - массивы масок на файлы.

    Например, следующий объект
    ```js
    {
        "units common block**": [
            "common/blocks/**",
            "common/blocks-touch/**"
        ]
    }
    ```
    заскипает тесты `units common blocks smth` и `units common block-touch smth2`, если среди изменений не окажется файлов из `common/blocks` и `common/blocks-touch`.


Опции для arc'а
- **arc** Флаг того, что нужно использовать arc, не git
- **pathRoot** Корень проекта в дереве arc'а. По-умолчанию пустая строка.

    В диффе все файлы будут иметь пути относительно корня. Файлы, не лежащие внутри этого корня, будут проигнорированы.
