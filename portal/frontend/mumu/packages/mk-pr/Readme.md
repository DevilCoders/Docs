# Make-pr

Создание ветки с коммитом и пулл-реквеста из этой ветки.

## commit(files, params)
 - `files` список файлов, изменения в которых нужно добавить в коммит. Файлы локально уже должны иметь нужные изменения
 - `params` объект, содержащий:
    - `message` сообщение для коммита
    - `baseHash` хэш коммита, от которого нужно отвести ветку
    - `branchName` имя создаваемой ветки
    - `owner` владелец github репозитория (пользователь или организация)
    - `repo` название github репозитория

## commitAndPR(files, params)
 - `files` список файлов, изменения в которых нужно добавить в коммит. Файлы локально должныуже должны иметь нужные изменения
 - `params` объект, содержащий все поля для commit'а и дополнительно:
    - `baseBranch` название ветки, в которую будет вливаться PR
    - `title` название пулл-реквеста
    - `body` опсиание пулл-реквеста (не обязательный параметр)
