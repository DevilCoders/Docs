## Get started
```bash
# Install the dependencies...
npm ci

# Run app development build & server on localhost:5000
npm run dev
```

## Building in production mode

To create an optimised version of the app:

```bash
npm run build
```

## Build mini-suggest
```bash
npm run build:mini-suggest
```

## Dev configuration variables
To change the dev configuration variables, create a .env.dev file and replace the necessary variables
https://github.com/Andarist/rollup-plugin-dotenv
