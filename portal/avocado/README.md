# Проект Авокадо

Авокадо это набор технологий, призванных заменить морду на перле.
Во многом, авокадо это набор сервисов для обслуживания запросов аппхоста, который обрабатывает запросы от пользователей для разных морд

### Директории

На данный момент директории в проекте делятся на 2 типа:
1. Сервисы (morda-go, avocado, map-addr)
2. Библиотеки (libs)
