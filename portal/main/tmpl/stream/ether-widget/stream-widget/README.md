# Блок stream-widget

Блок `stream-widget` используется для создания виджета Эфира на сторонних сайтах

Блок вызывается только в соответствующем хендлере на странице с виджетом, которая будет вставлена на сайт в `<iframe>`

При вызове блока в данных можно передать параметр `isVideo: Boolean`, который управляет форматом виджета:
- `true` - в виджет будет вставлен видео плеер
- `false` - в виджете будет превью, при клике на которое пользователь попадет на страницу просмотра в Эфире
