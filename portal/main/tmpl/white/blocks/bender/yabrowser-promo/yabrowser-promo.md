# Блок yabrowser-promo

Управляется из БК, данные приходят как обычный баннер (подобно тизеру).
Данные возвращает req.ads.banner('link_bro').

Примерный вид:

    "link_bro": {
      "id": "",
      "img": {
        "height": "60",
        "url": "https://yabs.yandex.ru/count/2HG1__________yFUGe0",
        "width": "60"
      },
      "linknext": "=uxt--PKAcm5jZegrOsz00gMhzmO5YBtn8GS7feG9iG5y0m00",
      "text1": "Браузер с удобным поиском",
      "text2": "",
      "text3": "",
      "url": "https://yabs.yandex.ru/count/2HyNZ1__________yFUGe0"
    }

Отсюда используются только поля

*   **img.url** — содержит изображение иконки Яндекс.Браузера;
*   **img.height** — высота;
*   **img.widgth** — ширина;
*   **linknext** — добавляется бекендом в стат-пиксель;
*   **text1** — содержит текст, который нужно завернуть в ссылку;
*   **url** — адрес ссылки.

Если в **img** ничего не пришло, то иконка отрисовывана не будет.

Размеры иконки пересчитываются до 18 пкс по высоте.
Внимание! Поле **text1** может прийти в виде **Удобный [Яндекс.Браузер]**.
В этом случае поведение меняться не должно, т.к эта возможность использовалась в rapido, т.е. **Удобный Яндекс.Браузер()**.
