# Страница с сервисами Яндекса

### Новые сервисы

- Создать ключи вида ${namedBlock}.${service_name}.title и ${namedBlock}.${service_name}.desc в [Танкер](https://tanker.yandex-team.ru/?project=home&branch=master&keyset=allServices) в кейсете allServices
- Открыть [MADM](https://madm-v82d2.wdevx.yandex-team.ru) и в экспорте services_v12_2 завести записи для всех необходимых доменов
- Добавить подготовленные иконки в папку blocks/common-all/services-all/services-all.inline
- make lang && make restart
