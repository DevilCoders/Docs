## Балун "сделать стартовой" ##

Показывает оранжевый балун на iOS для iPad'ов. Имеет ограничение на yandexuid (минимум 3 дня), количество и время показов, 
которые сохраняются в localStorage. Показывается только в сафари, также имеются проверки на clid'ы (чтобы не показывать тем,
кто уже сделал домашней). При нажатии на крестик записывает в localStorage максимальное количество показов, чтобы балун 
больше не отображался.
