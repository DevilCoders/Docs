# [Блок] player-container

## Описание

Контейнер для плеера.
Высота контейнера зависит от ширины. Пропорция 16x9.
Нужно задать ширину, высота подстроится.

Поверх плеера могут быть заглушки, кликабельный элемент overlay.

Блок нужен для того, чтобы без перемещения плеера по DOM (видео в iframe двигать не прерывая проигрывание нельзя).
Внутри верстка для разных вариантов положения.

Модификаторы overlay-position_title, ad-state_playing передвигают оверлей только поверх текста (чтобы можно было кликать в рекламу или плеер)
