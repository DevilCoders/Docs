# tap-formatter-html-template-morda
Кастомный tt2 шаблон для TAP::Formatter::HTML

Собирается в deb пакет, заливается в `common`.

## Установка
`sudo apt-get install yandex-libtap-formatter-html-perl-template-morda libtap-formatter-html-perl`

## Использование
`TAP_FORMATTER_HTML_TEMPLATE=morda.tt2 prove --formatter=TAP::Formatter::HTML ...`
