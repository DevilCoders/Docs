# admin-www
Сервис администрирования вертикальных сервисов.

## С чего начать разработку?
> Пожалуйста, не забудь про секреты! `npx @vertis/secrets-to-dotenv` в корне монорепы
1. Выполняем `npm i` (установит все зависимости и соберет проект)
2. Запускаем `npm run dev` для разработки
3. ...
4. PROFIT?

## Разработка

### Команды
* `npm run dev` watch изменений, вывод логов и запуск node
* `npm run build` сборка клиентского приложения в /build
* `npm start` запуск node (используется для продакшна)

### Проверка изменений при разработке
- При необходимости проверить работоспособность деплоя можно использовать специально созданные в тестинге сервисы Шивы. 
Сервисы имеют названия, начинающиеся с `shiva-test`, запускать их деплой можно с версиями `0.1.*`. 

- Проверку также можно произвести, если задеплоить ветку в прод: 
  ```
  LAYER=PROD, VERSION={your_version}, BRANCH={YOUR_BRANCH-123}
  ```
  Можно будет осуществлять деплой с указанными выше сервисами и версиями. 
  Ваша версия будет находиться по адресу: `https://branch-YOUR_BRANCH-123.admin.vertis.yandex-team.ru`
