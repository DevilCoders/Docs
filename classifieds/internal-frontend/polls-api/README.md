# polls-api
API для сервиса опросов Вертикалей

## С чего начать разработку?
1. Выполняем `npm i`
2. Запускаем `npm run dev`
3. ...
4. PROFIT?

### Команды
* `npm run dev` запуск node в режиме дебага
* `npm start` запуск node (прод режим)
