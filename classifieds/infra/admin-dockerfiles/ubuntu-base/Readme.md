Базовые образы для использования в Вертикалях. В них настроено время и Яндексовые репозитории

### Использование
~~~dockerfile
FROM registry.yandex.net/vertis-base/ubuntu:focal
# OR 
FROM registry.yandex.net/vertis-base/ubuntu:xenial
~~~
