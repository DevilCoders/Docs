## Обновление карты железных дорог для рисования маршрута поезда на странице нитки

### Обновление:
- Собрать бинарник.
```
ya make
```

- Запустить бинарник. \
 Будут загружены карты из YT.\
 Геометрия будет переведена в формат mif.
```
my_yql_token=<> file_path=<> ./pathfinder_railway_map
```

- Сделать zip файл с mid и mif файлами. \
Положить zip в админку через интерфейс админки или руками.\
Интерфейс админки: https://service.admin.rasp.yandex-team.ru/upload_railway_file
