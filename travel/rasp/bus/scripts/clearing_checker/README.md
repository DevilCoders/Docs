# Search and clearing of uncleared orders

## Sandbox usage

### 1. Build

Run YA_MAKE_2 task.
Example: https://sandbox.yandex-team.ru/task/535525578/view

## 2. Run

Run RUN_SCRIPT task with the resource id of the BUILD_OUTPUT from the previous step.
Example: https://sandbox.yandex-team.ru/task/537853311/view
