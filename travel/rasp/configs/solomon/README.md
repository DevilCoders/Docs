### Конфигурация мониторингов и алертов

В `alerts.yaml` лежат настройки алертов для всех основных сервисов Расписаний.
Подробно про работу с ними можно почитать тут https://wiki.yandex-team.ru/travel/rasp/dev/solomon/

Новый и предпочтительный способ настройки мониторингов - [solomon-tool](https://docs.yandex-team.ru/travel/ops/tools/solomon_tool)
