### Общее описание `Notifier`

#### Зона ответственности 

Запланировать отправку уведомлений определенного типа, в определенное время, с определенным контекстом. 

#### Интерфейс

Реализован поверх grpc в виде [набора методов для разных видов уведомлений](./components.md#API).

#### Домен

Уведомление - акт отправки Сообщения определенному Контакту в определенное время.
Контакт - описывает некий контакт, которому мы шлем уведомления. Например: email/phone/tg/pigon.
Сообщение - определяет layout, перечень возможных блоков, ограничено подмножеством возможных контактов.
Рассылка - определяет связь м/у контактами и типом сообщений.
