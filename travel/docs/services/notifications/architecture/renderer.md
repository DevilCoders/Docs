### Renderer

#### Зона ответственности 

Вернуть отображенный html по спецификации письма. Где спецификация письма включает тип письма и список блоков с данными, необходимыми для их отображения.

#### Интерфейс

Построен поверх http(s). Один метод - render, принимающий на вход спецификацию письма и возвращающий html.
Спецификацию можно найти [здесь](https://github.yandex-team.ru/data-ui/ya-travel-spec/blob/master/renderer/)

#### Архитектура

Реализован в travel-front, деплоится как отдельное приложение на отдельный кластер.
