## user-search-intents (also known as Tetris)

Объединение информации о пользователе со всех вертикалей Яндекс.Путешествий для улучшения взаимодействия с сервисами.

https://wiki.yandex-team.ru/users/slowpoke/komod/komod-logic-temp/

### О сервисе

Тетрис - сервис, предоставляющий пользователям подсказки на основе их истории заказов и поисков.

#### Поверхности

- Колдунщик Авиа
- Колдунщик Отелей
- Портал

#### Основные сценарии

- Пользователь сделал поиск в колдуне, ушел, при возвращении видит свой последний поиск, а так же еще несколько ближайших поисков
- Пользователь сделал поиск на портале, ушел, при возвращении видит свой последний поиск, а так же еще несколько ближайших поисков
- Пользователь поискал Отель, затем перешел к поиску Авиа билетов, сразу видит поиск на нужное направление и нужные даты
- Пользователь поискал Авиа билеты, затем перешел к поиску Отелей, сразу видит поиск на нужное направление и нужные даты
