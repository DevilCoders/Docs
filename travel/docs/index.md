# Кластер документации разработки Яндекс.Путешествий

## Что это?

В этом кластере содержится документация для разработки Яндекс.Путешествий.
Документация создается, наполняется, изменяется, поддерживается силами разработки для разработки.

В документации три основных отдела:
- Организационное. О процессах.
- Разработка и Эксплуатация. О компонентах.
- Концептуальное. Технические документы свободной формы.
