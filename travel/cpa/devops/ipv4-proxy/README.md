# ipv4-proxy

Прокси с ipv4-адресами из фиксированных подсетей для хождения к партнерам.

Сервис находится в Deploy: https://deploy.yandex-team.ru/projects/avia-partners-proxy

## Редактирование конфига

Конфиг нужно править автоматикой, но нужно внимательно отнестись к полученным правкам,
возможно конфиг поправили руками.
1. Собрать https://a.yandex-team.ru/arc/trunk/arcadia/travel/cpa/tools/
2. Запустить ``./tools proxy_config --output-fn ../devops/ipv4-proxy/docker/nginx/50-proxy.conf``
3. Возможно что-то отредактированно руками,
нужно понять, какой вариант правильный для каждого отдельно взятого партнера

## Сборка

Докер-образ собирается автоматически при каждом коммите.

Собранные ресурсы можно найти тут: https://sandbox.yandex-team.ru/resources?type=AVIA_PARTNERS_NGINX_PROXY_DOCKER&created=7_days

## Выкатка в deploy

После сборки ресурса нужно зайти в задачу, собравшую ресурс и зарелизить в нужное окружения.

Дальше согласно релизному правилу в Deploy произойдет выкатка.
