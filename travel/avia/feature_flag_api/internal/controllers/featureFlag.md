## FeatureFlag
Ручка для получения всех включеных функциональных флагу по коду сервиса.

```
GET /feature-flag?service-code={}
```
`service-code` - код сервиса. Все доступные коды сервисов можно посмотреть [здесь](https://admin.avia.yandex-team.ru/feature_flag_app/service)


### Формат ответа
```json
{"flags":["flag1", "flag2"],"lastUpdate":"2018-02-01T19:39:57+05:00"}
```

`flags` - список активных флагов, список может быть пустым.
`lastUpdate` - когда последний раз данные были обновлены.

Статус: 200

### Частные случаи
#### Если вы запросите функциональные флаги для сервиса, которово нет, то в качестве ответа получите ошибку
```json
{"message":"Unknown services"}
```

Статус: 400

#### Если вы запросите функциональные флаги во время перезагрузки сервисе, то получите:
```json
{"message":"Index is not ready"}
```
Статус: 503
