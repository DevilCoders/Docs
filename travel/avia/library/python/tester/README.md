tester
======

Общие фунции для запуска тестов


Старт тестирования
------------------
 1. Скопировть в корень проекта содержимое tester/example/
 2. Определить в local_settings константу TEST_DB_NAME, получить права на эту базу в mysql.
 2. Написать тесты, и расположить их в submodule/tests или просто в папке tests
 3. Запустить с помощью ``$ ./run_tests.py``, (возможно tester.runner будет избавлятся от лишней логики, поэтому лучше его не конфигурировать дополнительно).
 4. Либо Активировать окружение source ./common/virtualenv/bin/activate и запустить с помощью ``$ py.test``
