Возможные коды правил для тарифов:

(перечислены в коде тут: https://a.yandex-team.ru/arc_vcs/travel/avia/fare_families/internal/services/fare_families/data_structs/fare_families/fare_families.go#L43)

baggage - условия тарифа по включённому в стоимость билета багажу

carry_on - условия перевозки ручной клади

changing_carriage - условия обмена билета до вылета*

changing_carriage_no_show - условия обмена билета после вылета*

disclosure_url - ссылка на условия тарифа на сайте а/к

miles - ловия начисления квалификационных миль/баллов лояльности по тарифу

open_return_date - возможно ли купить по данному тарифу билет с открытой датой

refundable - условия добровольного (невынужденного) возврата билета до вылета*

refundable_no_show - условия добровольного (невынужденного) возврата билета после вылета*

seat_selection_check_in - условия выбора места в салоне во время онйлайн-регистрации на рейс по данному тарифу


\* примечание: понятие "до вылета" может отличаться для разных а/к, работу с этим понятием, возможно, будем улучшать
