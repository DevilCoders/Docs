# Описание ручки
Ручка возвращает авиакомпании и их синонимы для каждого языка в nominative падеже.
Падеж остался для сохранения обратной совместимости.

Эта ручка не прекэширует результат. Таким образом при каждом запросе вытаскивается список синонимов из базы.


## Формат запроса:
```
curl -X GET <API-HOST>/airlines/synonyms
```
