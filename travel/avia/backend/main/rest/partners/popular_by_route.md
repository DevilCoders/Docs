# Описание ручки
Ручка возвращает самые популярных партнеров и их 'очки' на данном направлении.


## Формат запроса:
```
/rest/partners/popular_by_route/<national_version>/<lang>/<from_point_key>/<to_point_key>
```
*national_version* - национальная версия

*lang* - язык интерфейса пользователя

*from_point_key* - откуда */[cs]\d+/*

*to_point_key* - куда */[cs]\d+/*

## Формат ответа:

```
/rest/airlines/popular/ru/ru/10
{
    "status": "ok",
    "data": [
        {
            "score": 0.90,
            "partner_code": "ozon"
        },
        {
            "score": 0.6,
            "partner_code": "s_seven"
        }
    ]
}
```


## Особенности:
Если одна из точке не существует или направление незначимо, то популярность партнера берется из популярности по нацверсии.

Очки партнера - это процент переходов, который занимает партнер на данном направлении/нацверсии.
Сумма очков по направлению нацверсии не меньше 0.95 процентов.
