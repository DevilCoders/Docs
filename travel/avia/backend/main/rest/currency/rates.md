# Описание ручки
Ручка возвращает список коэффициентов, с помощью которых можно конвертировать валюту в базовую валюту национальной версии

Базовая валюта - эта валюта, в которую нужно приводить все другие валюты для сравнения с другими ценами.


## Формат запроса:
```
'/rest/currencies/rates/<national_version>/<lang>'
```
*national_version* - национальная версия
*lang* - язык интерфейса пользователя

## Формат ответа:

```
https://<host>/rest/currencies/rates/ru/ru
{
  "status": "ok",
  "data": {
        "base_currency_id": 1,
        "rates": [
            {
                "currency_id": 8,
                "rate": 76.4847
            },
            {
                "currency_id": 1,
                "rate": 1
            },
            {
                "currency_id": 3,
                "rate": 68.5032
            }
        ]
    }
}
```


## Особенности:
Коэффициент для базовой валюты равен 1.

Курсы валют обновляются раз в 1 час.
