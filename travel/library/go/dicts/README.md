## Пакет для работы со справочниками

Структура пакета:

  * [base](./base) - содержит:
    * инструменты для работы с сериализованным в байты представлением справочников
    * структуру Repository реализующую базовый функционал расширений ([пример](./examples/custom_repository))
  * [extensions](./extensions) - набор расширений для репозиториев
  * [repository](./repository) (deprecated) - содержит репозитории для работы со справочниками
  * [examples](./examples) - пример работы
