# Утилита валидации json, который генерится для всех городов и типов писем

Для запуска - нужно указать правильные параметры и пароли в подходящих переменных окружения среды,
установить ( https://wiki.yandex-team.ru/passport/tvm2/tvm-daemon ) и запустить tvmtool:

```
./tvmtool --port 9005
```

и можно будет вот так запустить проверку текущего состояния исходников для всех городов на все типы писем:


```
SAMPLE_ORDER_ID=b4526782-9230-496a-8361-ccce10d4f148 ./cmd/localblocks/localblocks | grep -v registered
```

SAMPLE_ORDER_ID - это ID заказа, по примеру которого (число взрослых/детей) будет посылаться запрос в апи для получения
офферов отелей для каждого города; как правило, можно взять из тестовой базы любой заказ с датой заселения в недалёком будущем
