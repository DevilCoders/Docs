# Информеры

Информеры - это упрощенного вида аналоги страниц поиска и станции, которые встраиваются, с помощью iframe, на сторонние сайты.

У информеров своя отдельная сборка, которая запускается вместе со сборкой проекта под старые десктопные браузеры. Чтобы запустить проект вместе с информерами нужно воспользоваться командами `npm run local-base-desktop` (будет запущена только десктопная версия проекта) или `npm run local-base` (десктопная и мобильная версия). Информеры направления находятся по адресам `https://l.rasp.yandex.ru:3000/informers/search` и `https://l.rasp.yandex.ru:3000/informers/v2/search`. Информеры станции находятся по адресам `https://l.rasp.yandex.ru:3000/informers/station` и `https://l.rasp.yandex.ru:3000/informers/v2/station`.
