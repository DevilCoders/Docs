## CrowdTestProxy

Приложение для проксирования с `travel.crowdtest.yandex.ru` до `travel-test.yandex.ru`.
Используется при тестировании асессорами.

Также позволяет тестировать стенды с помощью проксирования с `https://travel-farm.crowdtest.yandex.ru/` через куку `farmId` до `https://ya-travel_{farmId}.travel.farm.yandex.ru/`.

Более подробно [в репозитории приложения](https://a.yandex-team.ru/arcadia/travel/frontend/crowdtest-proxy).
