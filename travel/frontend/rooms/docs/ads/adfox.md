### Что это?

Система управления рекламой.
В нашем случае это не реклама а альтернатива нашим баннерам, в которых мы рассказываем о наших крутых акциях.

[Документация adFox](https://yandex.ru/support/adfox-sites/)

### Не хочу ничего знать, мне просто нужно добавить баннер на страницу!

#### На стороне AdFox

Для выполнения этих действий нужен доступ в админку AdFox.

1. Завести в админке AdFox раздел и площадку под баннер
2. Завести рекламную кампанию / выбрать существующую
3. В рекламной кампании создать баннер / выбрать существующий баннер
4. Выбрать площадки для размещения баннера и сохранить изменения

#### На стороне проекта

1. Подключить компонент `AdFoxBanner` на странице
2. Передать соответствующие площадке параметры
    - `type` - соответствует типу баннера установленному для площадки (если был выбран тип баннера, которого нет в `EAdFoxBannerType`, то его нужно [получить](#%D0%BA%D0%B0%D0%BA-%D0%BF%D0%BE%D0%BB%D1%83%D1%87%D0%B8%D1%82%D1%8C-%D1%82%D0%B8%D0%BF-%D0%B1%D0%B0%D0%BD%D0%BD%D0%B5%D1%80%D0%B0) и добавить в enum)
    - `position` - соответствует позиции площадки (верх, середина, низ страницы)

#### Пример

Код:

```jsx
<AdFoxBanner
    type={EAdFoxBannerType.Inline}
    position={EAdFoxBannerPosition.Top}
/>
```

### В чем профит?

-   Управление баннерами без перевыкатки фронта
-   Верстка баннеров отдаётся на аутсорс

### Есть ли минусы?

-   Баннеры могут быть заблокированы блокировщиками рекламы
-   Необходимо понимать как появления/исчезновение баннера влияет на контент страницы. Есть две стратегии:
    -   резервировать место под баннер. Плюс - ничего не скачет, минус - если баннер не отобразили, то у нас будет дыра
    -   не резервировать место. Плюс - отключение/блокировка баннера не влияет на отображение, минус - при подгрузке баннера сдвигается контент, который находится под ним.

### Как получить тип баннера?

Необходимо зайти в раздел AdFox'а "сайт", и перейти на получение кода вставки для сервиса. Выбрать нужный тип баннера и сформировать код вставки. В коде вставки баннеру соответствует параметр `p2`
