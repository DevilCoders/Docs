# Как подменить базовый URL до апи бекенда.

Для того чтобы подменить базовый URL до нужной апишки на другой, необходимо поднять
отдельный инстанс приложения в тестовой ферме и задать нужный URL в переменных
окружения.

Для этого:

-   переходим на https://travel.farm.yandex-team.ru/create.
-   указываем имя проекта travel-ui/ya-travel
-   указываем имя ветки, инстанс которой необходимо поднять
    -   ветка `master` всегда совпадает с версией на проде
    -   ветка `dev` включает в себя все последние изменения
-   название папки оставляем пустым
-   в поле добавочное имя вставляем свой логин и, при желании, любой суффикс
    -   например: `hazam` или `hazam_v2`
-   нажимаем кнопку **Добавить переменную**
-   прописываем имя переменной в формате `BASE_URL_{имя сервиса}`, например, `BASE_URL_aviaBookingOrdersApi`.
    -   список имён используемых сервисов указан в [конфиге](https://a.yandex-team.ru/arcadia/travel/frontend/portal/server/configs/production.js) в ключах объекта в поле `servicesAPI`.
-   в качестве значения переменной подставляем нужный URL
-   нажимаем кнопку **Создать** и ждём пока пройдёт деплой

Список поднятых инстансов доступен [здесь](https://travel.farm.yandex-team.ru/project/ya-travel)
Ваш инстанс будет иметь имя `ya-travel_{ветка}_{добавочное имя}`
