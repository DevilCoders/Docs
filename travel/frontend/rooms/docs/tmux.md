# Tmux

tmux — это консольная утилита-мультиплексор,
предоставляющая пользователю доступ к нескольким терминалам в рамках одного экрана.
tmux может быть отключен от экрана: в этом случае он продолжит исполняться в фоновом режиме;
имеется возможность вновь подключиться к tmux, находящемуся в фоне.

[wikipedia.org](https://en.wikipedia.org/wiki/Tmux)

#### Установка

Установите tmux на дев-машине:

```
sudo apt-get install tmux
```

#### Workflow

Сессии tmux не зависят от корневого терминала.
Поэтому Webpack дев-сервер требуется запустить в сессии tmux,
для этого:

1. Попробовать подключится к существующей сессии, если ее нет создать новую:

```
tmux attach || tmux new
```

2. После этого вы попадаете в полноценную консоль,
   в которой запускаете Webpack дев-сервер.

3. Для отключения от сессии используйте **(раскладка должна быть английская)**:

```
Ctrl+b d  // Ctrl и b вместе, потом d
```
