# serviceProvider

serviceProvider это абстракция над сервисами `server/services`.

serviceProvider - это изоморфная обертка над сервисами. Она позволяет использовать одинаковый интерфейс для запросов с клиента и с сервера.

В зависимости от окружения будет использован для запроса нодовый сервис, либо же провайдер для браузера.

## Проблематика

-   Для ноды есть единая архитектура написания ручек, связанные запросы к апи объединяются в сервисы. Для клиента такой абстракции сейчас нет и запросы делаются разными способами.

-   Иногда в экшенах требуется делать запросы как с клиента так и с сервера в зависимости от окружения.

## service

Сервисы хранятся в папке `server/services` и реализуют бизнес логику, которая потом экспозится в контроллерах. Сервисы могут также сложить простой оберткой для запросов к апи.

## browserProvider

Реализует логику запросов к контроллерам, которые в свою очередь уже вызовут на ноде нужный сервис с теми же параметрами.
