# Статическая страница города для SEO-оптимизации

## методы

-   `getCityStaticPage`: `GET /api/hotels_portal/v1/city_static_page`

## Требования к бэкенду

1. Бэкэнд на запрос статической страницы города отдает заполненный шаблон или уникальную страницу города в соответствии со [Спецификацией](response.ts).

## Требования к фронтенду

1. Фронтенд рендерит статическую страницу на основе полученной структуры
