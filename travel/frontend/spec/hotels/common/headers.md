# Общие заголовки

-   `X-Ya-User-Gid` - число, кука yandex_gid
-   `X-Ya-ICookie` - строка. icookie, из ответа паспорта
-   `X-Ya-User-Device` - строка, десктоп или тач по разметке фронтенда, варианты: desktop, touch
-   `X-Ya-ExpBoxes` - строка, из ответа UAAS, там это заголовок `X-Yandex-ExpBoxes`
-   `X-Ya-User-IsStaff` - имеет ли пользователь привязанный к стаффу аккаунт. Значения: true/false
-   `X-Ya-User-IsPlus` - является ли пользователь плюсовиком. Значения: true/false
-   `X-Ya-White-Label-Partner-Id` - перечисление EWhiteLabelPartnerId, идентификатор партнера по программе White Label
