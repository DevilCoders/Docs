Ручка персонального саджеста Отелей (будем дергать в форме Отелей и на морде Авиа для формирования отельного пресета)

### Какие данные нужны в ответе:

###### Для синего саджеста (саджест кросссайла):

-   название города
-   geoId города ('213', без 'c')
-   checkinDate
-   chekoutDate
-   количество гостей (данные о гостях)
-   settlementId города в который укплен билет ('c213')

###### Для желтых саджестов (саджесты истории поисков):

-   название города
-   geoId города
-   checkinDate
-   chekoutDate
-   количество гостей (данные о гостях)
