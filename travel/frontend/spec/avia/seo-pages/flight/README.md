# Страница рейса авиа для SEO-оптимизации

## методы

-   `getFlightPage`: `GET http://api-gateway.testing.avia.yandex.net/v1/landing/flight/{IATA}/{NUMBER}/{NATIONAL_VERSION}/{LANGUAGE}/`

## Требования к бэкенду

1. Бэкэнд на запрос страницы направления отдает заполненный шаблон или уникальную страницу направления в соответствии со [Спецификацией](response.ts).

## Требования к фронтенду

1. Фронтенд рендерит статическую страницу на основе полученной структуры
