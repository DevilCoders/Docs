# Персонализация поиска пользователя

## методы

-   `getPersonalSearchVariants`: `GET http://api-gateway.testing.avia.yandex.net/v1/personal-search/`

## Требования к бэкенду

1. Бэкэнд на запрос персонализированных вариантов поиска отдает список вариантов в соответствии со [спецификацией](response.ts).

## Требования к фронтенду

1. Фронтенд рендерит компоненты на главной Авиа и добавляет в саджест варианты поиска
