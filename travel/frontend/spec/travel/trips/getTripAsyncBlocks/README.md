# getTripAsyncBlocks

Загрузка блоков, которые загружаются асинхронно,
после получения и показа поездки

В ответе метода getTrips лежат все блоки, в том числе и асинхронные,
но они не заполнены (`ITripEmptyAsyncBlock`), фронт находит все незаполненные асинхронные блоки
и делает запрос за ними в метод getTripAsyncBlocks.
После получения ответа обновляет блоки на странице.
