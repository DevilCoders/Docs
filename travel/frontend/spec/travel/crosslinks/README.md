## Перелинковки, Рецепты

Перелинковки и рецепты являются сео компонентами,
и так же показывают наиболее популярные направления в соответствующей вертикали.

## Методы

-   ~~`crossLinks` :: `GET /api/common/v1/crosslinks?type=<avia|trains|buses|hotels>`~~ Метод для получения перелинковок и рецептов

## Описание данных блока перелинковок/рецептов

API возвращает два массива блоков `ICrossLinkBlock` и `IRecipeBlock` , в которых, каждый блок является отдельной структурной еденицой на странице.
