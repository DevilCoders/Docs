# Статус-коды используемые в api портала

Чтобы взаимодействие с разными апи было предсказуемым, мы договорились на следующие статус коды для штатных и нештатных ситуаций, которые клиенты смогут обрабатывать заранее заложенным механизмом.

## Коды

| Код   | Есть расшифровка ошибки | Пояснение                                                                                  |
| ----- | ----------------------- | ------------------------------------------------------------------------------------------ |
| `2xx` |                         | Все хорошо (ожидаемо)                                                                      |
| `400` | Да                      | Запрос физически неверен (Передаем строку вместо числа, не передали обязательный параметр) |
| `422` | Да                      | Запрос логически неверен (несуществующий отель, перелет между одним и тем же аэропортом)   |
| `5xx` | Да                      | Все плохо (неожиданно)                                                                     |
