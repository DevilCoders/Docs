# City pages renderer

Render static pages for citys from tempates ([testing](https://tanker.yandex-team.ru/?project=travel-backend&branch=master&keyset=travel-api.hotels.city-page.testing), [prod](https://tanker.yandex-team.ru/?project=travel-backend&branch=master&keyset=travel-api.hotels.city-page.prod)) and data collected with [data miner](../data_miner)

## Build & run

```shell script
ya make bin && ./bin/region_pages_renderer
```
