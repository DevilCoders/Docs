### Скрипт для опроса API Google PageSpeed

Процесс выкачивает с PageSpeed информацию о скорости загрузки наших страниц и страниц конкурентов, а затем сохраняет её на yt.

Авторы, пользователи и owner-ы скрипта - аналитики (в частности @alyasevenyuk). Мы предоставляем инфраструктуру в виде сборки скрипта и его регулярного запуска в sbplanner-е.
