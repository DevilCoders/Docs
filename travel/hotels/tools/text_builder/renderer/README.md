# Hotel text renderer

Render hotel description with templates (
[testing](https://tanker.yandex-team.ru/project/travel-backend/keyset/travel-api.hotels.description.testing?branch=master),
[prod](https://tanker.yandex-team.ru/project/travel-backend/keyset/travel-api.hotels.description.prod?branch=master)
) and hotel features collected with [data miner](../data_miner)

## Build & run

```shell script
ya make bin && ./bin/hotel_text_renderer
```
