### Построитель BoY-пермарумов

[Проект](https://wiki.yandex-team.ru/travel/hotels/projects/awesome-rooms/)

Процесс получает данные из фидов некоторых BoY-партнёров и строит из них пермарумы.

#### BoY permaroom features

В папке ./resources лежат маппинги партнёрских признаков номеров во внутренние идентификаторы (общие для разных партнёров).
Маппинги были построены вручную:
* Названия в основном взяты от Expedia, а затем добавлены от travelline те, которых не было у экспедии
* Сопоставление приблизительное, для реального использования его нужно провалидировать
* Обновлять сопоставления нужно вручную
* Часть признаков партнёров не смаплены ни в какие внутренние идентификаторы (тогда id для них в табличке отсутствует)
