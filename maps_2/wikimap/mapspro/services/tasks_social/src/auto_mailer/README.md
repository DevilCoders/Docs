### Автоматизация рассылок для сообщества

Исходные данные для рассылок агрегируются регулярными процессами на YT (https://a.yandex-team.ru/arc/trunk/arcadia/maps/wikimap/stat/community/).

Данный процесс загружает результат агрегации по известному пути в YT и формирует кандидатов на отправку писем. Самой отправкой занимается соответствующий воркер (https://a.yandex-team.ru/arc/trunk/arcadia/maps/wikimap/mapspro/services/tasks_feedback/src/notifications_sender). Для передачи сформированных кандидатов на отпраку, они отгружаются в SQS очередь, которую notifications_sender разбирает.

Сообщения должны отправляться с определенной периодичностью. Поэтому для каждого пользователя проверяется, что для него нет записей в таблице sent_notifications за последний период с соответствующим типом.
