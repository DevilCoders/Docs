Инструмент для просмотра набора кластеров.

Позволяется просматривать изображения с объектами на них, переключаться между кластерами и объектами в них.

На вход принимаем:
1. ```--images``` - json файл с изображениями

2. ```--objects``` - json файл с объектами на избражениях

3. ```--clusters``` - json файл с кластерами объектов

Управление:
1. ```'n'``` - переход к следующему кластеру

2. ```'p'``` - переход к предыдущему кластеру

3. ```'d'``` - переход к следующему объекту

4. ```'a'``` - переход к предыдущему объекту

5. ```'Esc'``` - зевершение программы
