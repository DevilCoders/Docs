Утилита оценивает качество матчинга объектов.

На вход подаётся файл с детекциями и два файла с матчами (форматы можно найти [тут](https://a.yandex-team.ru/arc/trunk/arcadia/maps/wikimap/mapspro/services/mrc/eye/experiments/signs_map/readme.md)).

### Аргументы командной строки

1. ```--objects``` - файл с объектами

2. ```--min-bbox-size``` - минимальный размер объекта по каждой из осей

3. ```--gt-matches``` - файл с эталонной разметкой

4. ```--test-matches``` - файл с тестовой разметкой

5. ```--confidence-threshold``` - опциональный параметр. Если задан, то пытаемся прочитать confidence для каждого
матча и в позитивы отправляем матчи с уверенностью больше порога
