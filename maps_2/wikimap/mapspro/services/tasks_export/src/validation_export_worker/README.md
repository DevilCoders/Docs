# Пакетный экспорт

Это мета-воркер, который запускает другие задачи и следит за их выполнением.

Воркер запускет:
* экспорт
* валидацию

В конфиге `services.xml` в секции `/config/services/tasks/validation_export` перечислен список регионов,
в которых нужно запускать валидацию, и id пресета с проверками валидации для каждого региона.

После завершения экспорта воркер скачивает список id объектов, экспортированных по каждому региону.

После завершения валидаций воркер проверяет для каждого региона, были ли найдены фатальные ошибки в этом регионе.
Если в регионе не было фатальных ошибок, то он помечается, как протестированный.

В конце создается тикет в Стартреке в очереди **MAPSCONTENT** с результатами работы.
