Usage: cnn_semseg_dataset_rasterizer --input_list <value> --ann_map <value> --output_width <n> --output_height <n> --output_ann_dat <value> <options...>

Параметры:
  --input_list               текстовый файл со списком пар: 
                               <файл изображения, файл с многоугольниками разметки>
  --ann_map                  текстовый файл с картой: 
                               <имя класса, цифровая метка, тип генерируемой разметки>
                               тип разметки:
                                   0 - (finest) т.е. все пиксели, попадающие в контур объекта, отмечаются соотвествующей меткой
                                   1 - (coarse) меткой помечается только круг с центром в центре контура объекта и определенным радиусом (см. параметр ann_coarse_radius)
                                   2 - (wire) меткой отмечаются ребра полигона, соответствующего объекту
                                   3 - (vertices) меткой отмечаются вершины полигона, соответствующего объекту
  --output_width <n>         ширина результирующего изображения
  --output_height <n>        высота результирующего изображения

  --output_img_dat <value>   путь к выходному файлу куда будут сложены картинки (может не быть, тогда сгенерируется только разметка)
  --output_ann_dat <value>   путь к выходному файлу с разметкой

Дополнительные параметры:
  --ann_coarse_radius <n>           радиус круга для разметки типа coarse
  --img_eq_hist                     применить к изображению процедуру эквализации гистограммы
  --refine_markup_classes <value>   имена классов через запятую к которым надо применить дополнительное уточнение разметки
  --ann_vertex_size                 размер квадрата для разметки типа vertices
  --ann_wire_thickness              ширина ребра для разметки типа wire
  --check_not_empty                 сохраняются только ячейки изображения на которых присутствует хоть какая-то разметка


Пояснение
  Изображения режутся на куски, размер которых определяется параметром командной строки и могут не совпадать с размером изображений в исходных файлах. Если на целые куски изображение не режется, то остатки игнорируем.
