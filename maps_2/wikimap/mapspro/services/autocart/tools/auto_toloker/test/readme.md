# Инструмент для оценки качества автоматического валидатора зданий

Датасет, используемый для оценки качества, должен иметь следующий формат:
* **image** - папка со спутниковыми снимками
* **maks** - папка с масками зданий
* **list.txt** - CSV файл с именами изображений и ответами из Толоки

Для оценки качества сформирован набор данных с этим форматом, который выложен на sandbox:
https://sandbox.yandex-team.ru/resource/818563958/view

Для его скачивания можно воспользоваться командой:
> ./prepare_dataset.sh

После исполнения скрипта появится папка ./data со всеми необходимыми данными

Для запуска тестирования необходимо собрать проект:
> ya make -r --checkout

И запустить его:
> ./test --dataset_path ./data
