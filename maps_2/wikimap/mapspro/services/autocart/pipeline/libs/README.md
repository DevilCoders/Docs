# Библиотека для управления процессом

[assessors](https://a.yandex-team.ru/arc/trunk/arcadia/maps/wikimap/mapspro/services/autocart/pipeline/libs/assessors "assessors")

Библиотека для работы с асессорами: подсчет статистики, выгрузка ответов и т.д.

[assignments](https://a.yandex-team.ru/arc/trunk/arcadia/maps/wikimap/mapspro/services/autocart/pipeline/libs/assignments "assignments")

Библиотека для агрегации ответов толокеров и асессоров

[config](https://a.yandex-team.ru/arc/trunk/arcadia/maps/wikimap/mapspro/services/autocart/pipeline/libs/config "config")

Библиотека для работы с конфигурациями процесса

[detection](https://a.yandex-team.ru/arc/trunk/arcadia/maps/wikimap/mapspro/services/autocart/pipeline/libs/detection "detection")

Библиотека с детектором зданий на карте

[factory](https://a.yandex-team.ru/arc/trunk/arcadia/maps/wikimap/mapspro/services/autocart/pipeline/libs/factory "factory")

Библиотека для работы со спутниковой фабрикой: загрузка новых спутниковых релизов и из геометрий.

[hitman_client](https://a.yandex-team.ru/arc/trunk/arcadia/maps/wikimap/mapspro/services/autocart/pipeline/libs/hitman_client "hitman_client")

Простой клиент для работы с Хитманом: запуск процессов, проверка состояний процессов и т.д.

[hitman_processes](https://a.yandex-team.ru/arc/trunk/arcadia/maps/wikimap/mapspro/services/autocart/pipeline/libs/hitman_processes "hitman_processes")

Библиотека с Хитман процессами, которые используются в этом проекте: детектирование, валидации, заливка на карту.

[mpro](https://a.yandex-team.ru/arc/trunk/arcadia/maps/wikimap/mapspro/services/autocart/pipeline/libs/mpro "mpro")

Библиотека для работы с базой данных MPRO

[nmaps](https://a.yandex-team.ru/arc/trunk/arcadia/maps/wikimap/mapspro/services/autocart/pipeline/libs/nmaps "nmaps")

Библиотека для работы с Народной Картой: подготовка данных к заливке, заливка на карту.

[objects](https://a.yandex-team.ru/arc/trunk/arcadia/maps/wikimap/mapspro/services/autocart/pipeline/libs/objects "objects")

Библиотека, описывающая основные объекты, с которыми работает процесс детектирования: дома, дороги, регионы и т.д.

[storage](https://a.yandex-team.ru/arc/trunk/arcadia/maps/wikimap/mapspro/services/autocart/pipeline/libs/storage "storage")

Библиотека, реализующая логику работу с YT хранилищем

[toloka](https://a.yandex-team.ru/arc/trunk/arcadia/maps/wikimap/mapspro/services/autocart/pipeline/libs/toloka "toloka")

Ответы толокеров: Принять, Отклонить и т.д.

[wiki](https://a.yandex-team.ru/arc/trunk/arcadia/maps/wikimap/mapspro/services/autocart/pipeline/libs/wiki "wiki")

Библиотека для работы с wiki таблицами. Нужна для работы со списком асессоров

[yt_utils](https://a.yandex-team.ru/arc/trunk/arcadia/maps/wikimap/mapspro/services/autocart/pipeline/libs/yt_utils "yt_utils")

Вспомогательные функции и обертки для работы с YT
