## Назначение
Общая библиотека для поддержки фоновых задач

Фоновые задачи (воркеры):
  * cron - запускаются по расписанию и заверщают работу
  * daemon - висят в памяти, делают полезную работу с малым интервалом времени
  * grinder - daemon-воркеры, использующие [maps/tools/grinder](https://a.yandex-team.ru/arc/trunk/arcadia/maps/tools/grinder)
  * pubsub - daemon-воркеры, использующие [maps/wikimap/mapspro/libs/pubsub](https://a.yandex-team.ru/arc/trunk/arcadia/maps/wikimap/mapspro/libs/pubsub)
