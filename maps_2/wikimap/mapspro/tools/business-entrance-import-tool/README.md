# Утилита для импорта входов в организации из Справочника в Народную карту

Алгоритм работы:
1. Из таблицы [nyak-mapping](https://yt.yandex-team.ru/hahn/navigation?path=//home/altay/db/export/current-state/snapshot/nyak_mapping) на YT загружаются все входы в организации из Справочника.
2. Отбрасываются входы, относящиеся к организациям, расположенным в торговых и бизнес-центрах.
3. Отбрасываются входы, относящиеся к организациям вне заданных областей
4. Входы, расположенные на расстоянии не более 0.5 м, считаются одинаковыми (и, таким образом, расшариваются между несколькими организациями).
5. Для организаций, уже имеющих привязанные входы в НК, никакие действие не выполняются.
6. Мерж с уже существующими в НК подъездами не производится.
7. На выходе готовится JSON-файл для инструмента импорта, создающий недостающие входы и их связи с организацями в НК.

Входные параметры:
* area-ids - список ID площадных объектов (из НК), для организаций в пределах которых надо выполнить импорт
* config - полный путь с именем к файлу services.xml
* result-dir - полный путь каталога, в котором будет создан выходной файл poi_entrance.json для инструмента импорта
