Программа добаляет геометрию к объектам категорий image_overlay image_overlay_public

На первом этапе считываются все ревизии таких объектов где ещё нет геометрии.
Дальше для каждых уникальных атрибутов этих ревизий генерируется геометрия из первой
точки массива координат привязки, остальные делаются относительными.
В финальной части прочитанные ревизии удаляются и вставляются дополненными геометрии.

После завершения конвертации необходимо синхронизировать view для всех объектов этих категорий,
чтобы точка с геометрией попадала в выборки при подвижке.
