# Железнодорожный переезд

**Железнодорожный переезд** - отдельная категория маневра, контролирующая одно или два ребра, входящих в узел.

При простановке ж/д переезда автоматически включается контроль над всеми входящими ребрами, удовлетворяющими условиям
создания (с учетом направления движения). В режиме редактирования можно откликать одно из двух ребер, но хотя бы одно
должно остаться. Входящие в ж/д переезд участки отмечены зеленой стрелкой, направленной в узел.

Ж/д переезд создается из основной панели создания (группа "Дорожная инфраструктура") или из карточки узла.
Создать ж/д переезд можно путем разрезанием ребра или проставлением в имеющийся узел. Ж/д переезды создаются только
в узлах с двумя ребрами (любыми).

В узлах, удовлетворяющих условиям создания ж/д переезда, есть блоки добавления этого объекта. Если узел не удовлетворяет
условиям создания, этого блока нет. При попытке проставить ж/д переезд в узел, не удовлетворяющий условиям создания,
появляется ошибка `ERR_INVALID_POINT_FOR_COND_RAILWAY_CROSSING_SETUP`

Ж/д переезд отображается в хотспот-лейбле узла, в котором он расположен. В узле может быть только один ж/д переезд.
При попытке проставить ж/д переезд в узел, где он уже есть, при сохранении появляется ошибка `ERR_SLAVES_COUNT`

При перерисовке в режиме редактирования можно выбрать только имеющийся узел. Создать новый путем разрезания нельзя.

Правки ж/д переездов попадают в модерацию в категорию "Условия движения".

### Тикеты

**Разработка**: NMAPS-15250

**Кейсы**: ручных кейсов нет
