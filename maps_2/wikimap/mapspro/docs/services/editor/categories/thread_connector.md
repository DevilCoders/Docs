# Коннектор ниток

{% note info %}

**Коннектор ниток** – путь между двумя нитками общественного транспорта, по которому едет ОТ между высадкой и посадкой.

{% endnote %}

**Коннектор** - это сущность:
* со связями с двумя нитками: `src_thread` ("От нитки) и `dst_thread` ("До нитки")
* (опционально) с агрегированной геометрией участков дорог
* с атрибутом-флагом прогнозируемости `is_predictable` (стоит ОТ на этом коннекторе или проезжает сразу на следующую нитку)

title коннектора формируется из названий ниток. У участков дорог, входящих в геометрию коннектора, проставляется сервисный атрибут `rd_el:is_part_of_bus_thread_connector`

Если ОТ высаживает и сажает людей сразу на одной остановке, не делая при этом никакой круг через кольцо/парк,
то геометрия у коннектора пустая.

Коннектор создается из карточки нитки. В карточке просмотра нитки отображаются только коннекторы "От нитки".

Возможность добавить коннекторы есть у всех видов ниток. Коннектор возможен только между нитками одного маршрута. У одной нитки может быть
несколько коннекторов к другим: ОТ заканчивает движение по нитке и может затем поехать по одной из "обратных",
в зависимости от расписания, указаний диспетчера и пр.
