# Искусственная неровность

{% note info %}

**Искусственная неровность** (она же "лежачий полицейский") - специально устроенное возвышение на проезжей части
для принудительного ограничения скорости до 40 км/ч и менее.

{% endnote %}

**Искусственная неровность** - отдельная категория маневра, контролирующая одно или два ребра, входящих в узел.

При простановке неровности автоматически включается контроль над всеми входящими ребрами, удовлетворяющими условиям
создания (с учетом направления движения). В режиме редактирования можно откликать одно из двух ребер (это случай, когда
неровность сделана только для половины проезжей части - в одном направлении движения), но хотя бы одно должно остаться.
Входящие в неровность участки отмечены зеленой стрелкой, направленной в узел.

Искусственная неровность создается из основной панели создания (группа "Дорожная инфраструктура") или из карточки узла.
Создать неровность можно путем разрезанием ребра или проставлением в имеющийся узел. Неровности создаются только в узлах,
в которые входит одно или два ребра с автомобильной доступностью. Грузовая/такси/ОТ доступность для простановки
неровности не важна.

В узлах, удовлетворяющих условиям создания неровности, есть блоки добавления этого объекта. Если узел не удовлетворяет
условиям создания, этого блока нет. При попытке проставить неровность в узел, не удовлетворяющий условиям создания,
появляется ошибка `ERR_INVALID_POINT_FOR_COND_SPEED_BUMP_SETUP`

Неровность отображается в хотспот-лейбле узла, в котором она расположена. В узле может быть только одна искусственная
неровность. При попытке проставить неровность в узел, где она уже есть, при сохранении появляется ошибка `ERR_SLAVES_COUNT`

При перерисовке в режиме редактирования можно выбрать только имеющийся узел. Создать новый путем разрезания нельзя.

Правки неровностей попадают в модерацию в категорию "Условия движения".
