### Алерты DWH
Сейчас данные собираются по метрикам из другого проекта в Соломоне -- Balance Reports:\
https://solomon.yandex-team.ru/admin/projects/balance-reports \
Алерты отправляются в Telegram в чат мониторингов FinTools, а затем звонит ЖЖ.\
После добавления функции для нового алерта надо собрать проект в корне `arcadia/paysys/sre/tools/monitorings` и запустить:
```
make run -- --project dwh --environment stable --config dwh
```
Окно алерта (параметр window_secs в функциях MAAC) стоит делать достаточно большим, т.к. метрики не всегда успевают доехать.

#### Алерты на свежесть MV
Алерт с `warn` порогом в 24 часа и `crit` порогом в 36 часов.
Смотрит на даты последнего обновления MV, выбирает из них самую старую и сравнивает с порогом.

#### Алерты на расхождения количества строк
Алерт сравнивает общее количество строк в MV в двух точках во времени. `Warn` порог -- расхождение более 10%, `crit` -- более 25%.
Сравниваются периоды:
 - 1 день назад и 2 дня назад;
 - 1 день назад и 4 недели назад.
