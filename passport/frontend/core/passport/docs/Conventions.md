Конвенции касаются всего свежего кода: всё новое и редактируемое надо писать по конвенциям.

* Биндить js только на классы с префиксом "js-"
* Вешать css только на классы с префиксом из id блока: {.id}_my_element в яти и в стилусе
     ```
     some-block-id
         &_my_element
             <styles here>

         &_other_element
             <other styles>
     ```

* Имена js-файлов, которые экспортируют конструктор писать с большой буквы: /some/folder/MyConstructor
* Стилобатные skin и kind писать всегда в начале css-блока, чтобы изменения в стилобате не перетирали возможные переопределения
  Приоритет: skin или kind, потом миксины, потом css — тогда изменения миксинов не заденут кастомные переопределения, а изменения стилобата не заденут миксины.
