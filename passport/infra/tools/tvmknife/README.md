tvmknife
============

Это консольный инструмент для отдладки и тестирования сервисов, в которых внедрён TVM.

Сборка
---------
1) debian-пакет

   Можно [руками](https://sandbox.yandex-team.ru/scheduler/11566). Можно через `ci changelog` + `ci release --no-issue --no-conductor`

2) [ya tool tvmknife](https://sandbox.yandex-team.ru/scheduler/13113)

   Номер sandbox-таска надо закомитить [сюда](https://a.yandex-team.ru/arc/trunk/arcadia/build/ya.conf.json?rev=6411423#L5333-5345)
