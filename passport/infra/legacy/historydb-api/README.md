# HistoryDB API V.2

## Особенности сборки на buildfarm

Сборка настроена так, что пакет собирается только на агенте `precise-buildfarm-s2.passport.dev.yandex.net`.
Кеши `sbt`/`ivy` лежат в папке `/home/buildfarm/historydb/`. Сборка идет от имени пользователя с id 1234,
поэтому нужно убедиться, что любой пользователь может зайти в эту папку (и есть права на запись).
В т.ч. надо проверить что есть права на исполнение хомяка `buildfarm` (`chmod a+x /home/buildfarm`).

Кроме того, необходимо перед сборкой целевого jar-файла установить пакет libticket-parser2-java
