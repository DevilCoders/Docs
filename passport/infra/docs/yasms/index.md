# YaSMS — сервис отправки SMS

YaSMS предоставляет внутренний API, который позволяет [отправлять SMS](reference/sendsms.md) от Яндекса на произвольные телефонные номера.

{% note warning %}

Не предоставляйте пользователям возможность отправки произвольного текста на произвольные номера телефонов. YaSMS не предусматривает троттлинга сообщений или антифрода. Вся работа по безопасности должна быть произведена на стороне сервиса, использующего YaSMS.

{% endnote %}



## Регистрация потребителей и доступ к YaSMS {#access}

Каждый потребитель API должен быть зарегистрирован в YaSMS: все запросы от одного сервиса получают единый идентификатор отправителя, <q>sender</q>. Идентификатор необходим, чтобы различать сервисы, отправляющие запросы с одного IP-адреса.

Чтобы зарегистрировать отправителя или изменить уже существующего (например, запросить новые гранты), заполните заявку в [форме](https://forms.yandex-team.ru/surveys/17153/).

### Адреса и TVM-приложения {#addresses-and-tvm}

API доступен по следующим адресам:

- тестовый API (с данными [тестового Паспорта](https://passport-test.yandex.ru)) — [https://phone-passport-test.yandex.ru/](https://phone-passport-test.yandex.ru/).
- боевой API (с данными [боевого Паспорта](https://passport.yandex.ru)) — [https://sms.passport.yandex.ru](https://sms.passport.yandex.ru);

Для доступа к YaSMS необходимо использовать [TVM](https://wiki.yandex-team.ru/passport/tvm2/). Идентификаторы нужных TVM-приложений:

- тестовый YaSMS — 2000834;
- боевой YaSMS — 2000836.
