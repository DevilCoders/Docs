# Общие сведения

Черный ящик — это внутренний HTTP-сервис, позволяющий публичным сервисам Яндекса проверять авторизационные данные пользователей. Также Черный ящик дает возможность получать информацию о пользователях из базы данных Паспорта.

Черный ящик разработан для обеспечения единой политики авторизации на сервисах Яндекса. Черный ящик используется серверами, обслуживающими сервисы Яндекса, и скрыт от внешних пользователей.

Подробнее об авторизации и авторизационных данных читайте в разделе [Куки системы авторизации](https://docs.yandex-team.ru/authdevguide/concepts/Auth_About) документа [Яндекс.Паспорт. Обзор системы авторизации](https://docs.yandex-team.ru/authdevguide/). Данные о пользователях, которые хранятся Паспортом, описаны в разделе [Часто используемые данные системы авторизации](https://docs.yandex-team.ru/authdevguide/concepts/DB_About) того же документа.
