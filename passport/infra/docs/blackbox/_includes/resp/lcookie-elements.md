### Элементы ответа

#### uid
UID, записанный в куке.

#### timestamp
UNIX-время выставления куки.

#### login
Значение логина, записанное в куке. Может быть синтезированным для аккаунтов, не имеющих логина.

#### display_login
Человекочитаемое значение логина, записанное в куке. Для аккаунтов с синтезированным логином поле пустое.
