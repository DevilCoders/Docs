Цикл работы
================
1) удалить все файлы в каталоге для патронов кроме файла мета-файла. В нём хранится id последней полученной задачи
2) сходить в shooter за задачей
3) если есть задача и её ещё не выполняли, то приступить
4) запомнить id задачи, чтобы не пытаться её выполнить позже ещё раз. Так сделано из-за того, что нет возможности перетереть из-за SGID на той стороне
5) собрать трафик через goreplay
6) сделать chown ленты для своего пользователя
7) сжать запросы через gzip
8) отправить запросы через scp
9) отправить meta-файлик через scp
10) если на п5-9 случилась ошибка, попытаться отправить meta-файлик через scp с этой ошибкой
