YSA
==============

Проект собирает факторы про запросы от разных клиентов и пишет их в лог для дальнейшего использования в ML.

Есть два режима работы:
1) пиксель
2) миррор

Добавление нового секрета
-------------
При добавлении нового секрета в конфиги yav, пакет необходимо катить через кондуктор с галкой
'Не перезапускать сервисы'.

Сборка
-------------
Через `ci changelog` + `ci release --no-issue`

-------------
