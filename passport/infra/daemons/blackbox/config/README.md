Сборка механизированная
=============
Предполагается, что в PATH прописан путь к инструменту `passport/backend/ci/bin/ci` и он сконфигурирован через `ci config`.

В этом каталоге надо сделать:
1) сформировать changelog: руками или через `ci changelog`
2) `ci release --no-issue`

После этого будут созданы таски:
1) в Аркануме - чтобы влить version up
2) в Sandbox'е - чтобы собрать пакет
3) в Кондукторе

Сборка ручная
=============
В [Sandbox](https://sandbox.yandex-team.ru/scheduler/20182/view). В результате получится debian-пакет.

Таски в Кондукторе делать нужно самому.
