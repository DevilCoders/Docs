# Takeout

Сервис предоставляет возможность выгрузить человеку данные, хранимые о себе нами, в рамках закона GDPR.

 1. Пользователь нажимает кнопку "Скачать данные о себе".
 2. Данные собираются в фоновом режиме, асинхронно.
 3. Пользователь получает отбивку, что данные готовы.
 4. Скачивает данные.
