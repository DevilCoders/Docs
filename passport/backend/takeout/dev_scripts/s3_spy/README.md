# s3-spy
Инструмент для просмотра текущих выгрузок по префиксу s3.

Префикс имеет вид `uid/extract_id/service_name`. Можно указать только часть префикса. Можно указать более одного префикса.

## Опции
`-ou`,  `--only-unfinished` - показать выгрузки из сервисов без флажка `DONE`
`-of` - `--only-finished` - показать только те выгрузки из сервисов, у которых есть флажок `DONE`
`--short` - короткий формат выгрузки, `awk`-friendly выгрузка

За остальными в `--help`
