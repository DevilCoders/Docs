# touch
Инструмент для проставления redis-флажков в выгрузки по префиксу.

Префикс имеет вид `uid:extract_id:service_name`. Можно указать только часть префикса. Можно указать более одного префикса.

## Опции
`--done`, `--started`, `--ordered` - какой флажок поставить. Если хочется пометить выгрузку как успешно завершённую, то `--done`.

Если надо снять флажок, то дополнительно надо передать параметр `--unset`.

За остальными в `--help`
