# Тасклеты Паспорта

Задачи для использования в различных флоу [аркадийного CI](https://docs.yandex-team.ru/ci/).

## Как добавить новый?

См. [документацию](https://docs.yandex-team.ru/ci/job-tasklet). При любых изменениях задачи не забудь [прописать](https://docs.yandex-team.ru/ci/job-tasklet#yaml) новую версию ресурса с задачей в [Registry](https://a.yandex-team.ru/arc/trunk/arcadia/ci/registry/projects/passport/).


## Как тестировать и отлаживать?

В директориях многих тасклетов лежат файлы `run_test.sh` и `test_data.json`. Запуск скрипта позволяет сэмулировать выполнение тасклета на указанных входных данных.

N.B. После запуска скрипта может потребоваться вручную почистить результаты (например, закрыть созданный тикет). Подробности см. в stdout скрипта.
