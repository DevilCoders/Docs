# nginx-config-generator
Шаблонизатор для конфигов nginx


Как подключить к своему проекту:
1) Добавляем nginx_config_generator в PEERDIR своего проекта
2) Кладём в ресурсы (в директорию ```templates```) своего проекта Jinja-темплейты для конфигов. Опционально, добавляем свои макросы и фильтры.
3) Добавляем в проект генераторы настроек для конфигов (унаследованные от ```BaseConfig```)
3) Вызываем в postinst своего дебиан-пакета функцию ```build_configs```, а в prerm - функцию ```delete_configs``` (оборачиваем их в python-скрипты, передавая функциям список генераторов настроек из прошлого пункта)
4) (опционально) Фиксируем тестами состояние продакшн-конфигов nginx (см. модуль ```test```)

Также пример использования можно посмотреть в тестах данного пакета.
