## Конфигуратор мониторинга
Настраивает агрегации в Джаглере и алерты в Соломоне на основании простых конфигов.

Использовать нужно только при создании и изменении конфигурации.

### Использование
1. Установить пакет на дев-машину
```apt install yandex-passport-monconf```
2. Сделать конфигурацию для нужного сервиса на основании файлов config_example, положить все файлы конфигурации в отдельную директорию, например monitoring. Документация находится в примерах конфигов.
3. Запустить passport-monconf --config-dir ...путь к директории...
