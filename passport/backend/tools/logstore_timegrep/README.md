## logstore-timegrep

Достаёт из лога события, лежащие в указанном временном интервале. Пример использования:
```
logstore-f1:/storage/production/var/log/yandex/passport-api$ ~/logstore-timegrep debug.log.2022042211 "2022-04-22 11:21:00" "2022-04-22 11:25:00" | fgrep some-text
```

**Важно**: так как логи на логсторе не полностью упорядочены по времени, лучше брать временной интервал с запасом (скажем, по 1-2 минуты с каждой стороны)
