
URL к интерфейсу смены паролей на внешнем портале.

Этот параметр связан с предыдущим. Если пользователям запрещено менять пароли через интерфейс Яндекса, в данном параметре следует указать адрес интерфейса для смены паролей. Такой интерфейс должен быть реализован владельцем домена на его портале.

В зависимости от комбинации данного и предыдущего параметров Паспорт формирует ссылку для смены пароля в диалоге персональных данных. Возможны следующие сценарии:

- смена паролей запрещена и URL задан: ссылка для смены паролей указывает на данный URL;
- смена паролей запрещена, URL не задан: ссылка для смены паролей не выводится, и сменить пароль невозможно;
- смена паролей разрешена: ссылка для смены паролей указывает на интерфейс Яндекса независимо от заданного URL.
