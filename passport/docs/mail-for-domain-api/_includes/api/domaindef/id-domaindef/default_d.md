
Идентификатор или логин пользователя ПДД. Используется в сочетании с параметром `default_uid`:

- Если параметр `default` не передан, Паспорт использует UID из значения параметра `default_uid`.
    
- Если в параметре `default` указан UID существующего пользователя ПДД, то это значение используется в качестве значения параметра `default_uid` (вне зависимости от того, был он передан или нет).
    
- Если в параметре `default` указан логин, Паспорт будет искать пользователя с таким логином в домене с идентификатором, указанным в параметре `domain_id`. UID найденного пользователя используется в качестве значения параметра `default_uid`. Если пользователя с заданным логином нет в данном домене, Паспорт ответит ошибкой.
    
- Если в параметре `default` указана пустая строка, в качестве значения параметра `default_uid` используется 0.
