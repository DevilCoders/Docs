
Состояние DNS-записи типа mail exchange (MX): 0 — MX-запись не указывает на почтовый сервер Яндекса, 1 — MX-запись настроена правильно.

{% note info %}

При `mx` = 0 администратору почты периодически отправляются уведомления о необходимости указать почтовый сервер Яндекса в MX-записи домена. Также в этом случае считается, что почтовые ящики домена расположены за пределами Яндекса, и письма от пользователей Яндекса на адреса домена отправляются через внешние почтовые серверы.

{% endnote %}
