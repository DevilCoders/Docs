# Релизы документа

В разделе содержится информация о выпущенных релизах документа. Для каждого релиза указан номер, дата выпуска, краткое содержание изменений и дополнений, вошедших в релиз.

#### Релиз 1.2 от 5.11.2015 г.

Добавлен параметр `default` для метода [editDomain](../api/editdomain.md).

#### Релиз 1 от 27.10.2010 г.

Новый документ содержит следующие сведения:

- краткое описание Почты для доменов (ПДД);
- описание информационных объектов Паспорта `host`, `domain`, `account`, `subscription`.
- описание методов, с помощью которых Почта манипулирует информационными объектами Паспорта: `addHost`, `listHost`, `editHost`, `deleteHost`, `addDomain`, `listDomain`, `editDomain`, `domainAlias`, `deleteDomain`, `mdDomain`, `addAccount`, `listAccount`, `editAccount`, `addAlias`, `deleteAlias`, `deleteAccount`, `subscriptions`.
