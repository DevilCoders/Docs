# Общие сведения о Почте для доменов

Сервис Почта может принимать и обрабатывать письма, адресованные не только в собственные домены Яндекса, но и в сторонние домены. Данная возможность называется Почтой для доменов (ПДД) и предоставляется после подключения домена к Почте на странице [https://pdd.yandex.ru](https://pdd.yandex.ru/).

Для подключенного домена Яндекс становится почтовым хостингом. Письма на адреса домена помещаются в почтовые ящики Яндекса, а пользователям домена доступны все возможности сервиса Почта.

## Условия подключения домена

Для подключения домена к Почте необходимо выполнение следующих условий.

1. Подтверждение прав на домен. Для этого пользователю предлагается создать DNS-запись указанного типа и содержания либо поместить на сайт файл с указанным именем и содержанием.
1. Настройка DNS-записи типа mail exchange (MX). В MX-записи указывают почтовый сервер Яндекса. В результате такой настройки почтовые серверы Интернета передают письма, адресованные в домен, на указанный почтовый сервер.

После выполнения этих условий пользователь становится администратором почты домена и может настраивать параметры ПДД через веб-интерфейс.

## Аккаунты на доменах

Администратор почты домена создает аккаунты для пользователей этого домена. С каждым аккаунтом связан почтовый ящик, который создается автоматически. Логины аккаунтов и адреса почтовых ящиков совпадают и имеют вид `user@domain`.

Администратор почты может выбрать аккаунт, в почтовый ящик которого перенаправляются письма неизвестным адресатам домена (catch-all).

## Синонимы

К Почте можно подключать дополнительные домены в качестве синонимов (alias) основного домена. Письма на адреса доменов-синонимов поступают в почтовые ящики пользователей основного домена.

Существует возможность создавать синонимы аккаунтов. Такие синонимы являются альтернативными именами пользователей в домене. Например, для аккаунта `petr.petrov@okna.ru` можно создать синоним `petrov`. Письма на адрес `petrov@okna.ru` помещаются в почтовый ящик аккаунта `petr.petrov@okna.ru`.
