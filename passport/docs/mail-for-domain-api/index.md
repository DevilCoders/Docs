# Разделы документа

В документе приведены общие сведения о Почте для доменов (ПДД) и описан интерфейс взаимодействия сервисов Яндекс.Почта и Яндекс.Паспорт.

## Основные разделы документации {documentation-content}

### [Введение](concepts/about.md) {#pdd-about}

В разделе содержится общие сведения о Почте для доменов (ПДД), приведен и описан синтаксис запросов.

### [Общие сведения о Почте для доменов](concepts/pdd-overview.md) {#pdd-overiew}

В разделе описаны общие особенности Почты для доменов: особенности подключения домена к почте, возможность создания аккаунтов для пользователей домена, возможность подключить синонимы.

### [Интерфейс взаимодействия](concepts/api-overview.md) {#pdd-api-overview}

В разделе подробно описана информационная модель взаимодействия Яндекс.Почты с Яндекс.Паспортом, а также методы в составе информационных объектов `host` (почтовая база), `domain` (домен), `account` (аккаунт) и `subscription` (подписка).

### [Релизы документа](concepts/releases.md) {#releases}

В разделе содержится информация о выпущенных релизах документа. Для каждого релиза указан номер, дата выпуска, краткое содержание изменений и дополнений, вошедших в релиз.
