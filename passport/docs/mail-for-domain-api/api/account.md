# Аккаунты

В разделе описаны методы Паспорта для создания, редактирования, просмотра и удаления объектов account.

Услуги Почты для доменов предоставляются зарегистрированным пользователям Яндекса. Аккаунты таких пользователей называются аккаунтами на доменах, поскольку аккаунты связаны с подключенными доменами.

Аккаунты на доменах отличаются от обычных аккаунтов форматом логина. Формат описан ниже.

1. Логины совпадают с адресами электронной почты, то есть имеют формат `user@domain`, где `user` — имя пользователя в домене `domain`.
1. Имена пользователей уникальны в пределах домена и могут состоять из минимального количества символов, например из одного.
1. Логины могут содержать любые символы, допустимые в адресах электронной почты. В частности, в логинах может присутствовать символ подчеркивания, что не допускается для обычных аккаунтов.
1. Точка и дефис в логинах считаются различными символами, тогда как в обычных аккаунтах они равнозначны.
