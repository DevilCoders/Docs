# Сценарии в API Паспорта

API Паспорта представляет собой набор атомарных операций, необходимых для регистрации, авторизации и работы с пользовательскими данными. Эти операции классифицированы в [описании на Вики](http://wiki.yandex-team.ru/passport/python/api) в соответствии с типами решаемых задач: создание учетной записи, авторизация, показ капчи и т. д.

{% note info %}

Если вам необходимо только получать данные о пользователях, воспользуйтесь API Черного ящика, который описан в документе [Программный интерфейс сервиса Яндекс.blackbox](https://doc.yandex-team.ru/blackbox/concepts/about.xml).

{% endnote %}


Работу с десятками атомарных операций API облегчают сценарии — совокупности методов, необходимых для решения комплексных задач. Сейчас сценарии реализованы для различных видов регистрации учетной записи.

## Поддерживаемые сценарии

Паспорт поддерживает гранты на сценарии, которые выдаются аналогично грантам на отдельные методы. О настройке доступа к API читайте раздел [Доступ к API Паспорта](location.md).

В документе описаны только [сценарии регистрации](registration.md) (другие сценарии и методы планируется описать в будущем).
