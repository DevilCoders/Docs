# Социализм

Социализм — это компонент системы авторизации, который позволяет пользователям Яндекса авторизоваться с помощью учетных записей в некоторых социальных сетях, а сервисам — использовать универсальный интерфейс для обращения к соцсетям.

Социализм построен на использовании социальных профилей — связей между аккаунтами Яндекса и социальных сетей. С помощью социального профиля можно аутентифицировать пользователя или сохранить информацию о соответствующем профилю аккаунте соцсети.

Социализму посвящен документ [Яндекс.Паспорт. Обзор Социализма](https://doc.yandex-team.ru/Passport/social/concepts/about.html).

Социализм состоит из нескольких модулей:

![social](../_assets/social.png "Социализм")


## Веб-интерфейс {#web}

Включает в себя страницы, на которых обрабатывается социальная часть авторизации (страница `social.yandex.<tld>`, диалог, в котором можно подтвердить создание нового аккаунта и т. п.).


## Брокер {#broker}

Полученная информация сохраняется в базе данных и используется для авторизации пользователя на Яндексе.
