# Структура документа

Документ состоит из следующих разделов:

1. [Обзор режимов Паспорта](requests.md) — сведения об организации и классификации программных интерфейсов Паспорта.
1. [Доступ к режимам Паспорта](location.md) — адреса для вызова API и инструкции по получению
 необходимых грантов.
1. [Публичные режимы](../reference/register.md) — описания общедоступных режимов Паспорта.
1. [Внутренние режимы](../reference/admreg.md) — описания режимов Паспорта, доступных только из сетей Яндекса.

О выпущенных версиях документа читайте в разделе [Релизы](releases.md).
