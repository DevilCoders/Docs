# Введение

**Документ содержит** описание режимов Яндекс.Паспорта — серверных вызовов и веб-страниц с пользовательским интерфейсом. В документе приведен формат взаимодействия с API и информация о доступе к режимам.

**Целевая аудитория документа** — разработчики веб-сервисов Яндекса, которым необходимо регистрировать и авторизовать пользователей, а также изменять данные учетных записей.

**Документ не содержит** информацию об архитектуре Яндекс.Паспорта.

Пожелания и замечания к документу направляйте в Jira: [https://jira.yandex-team.ru/browse/DOC-1806](https://jira.yandex-team.ru/browse/DOC-1806).

**Документ, рекомендуемый для предварительного ознакомления**:

- [Обзор системы авторизации](https://docs.yandex-team.ru/authdevguide/) — содержит описание понятий и процессов, на которых основана работа системы авторизации для веб-сервисов.
