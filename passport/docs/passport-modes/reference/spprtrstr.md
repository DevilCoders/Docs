# spprtrstr

Предоставляет веб-интерфейс для восстановления доступа к учетной записи через службу поддержки (в отличие от автоматического восстановления доступа в режиме [restore](restore.md)).


## Формат запроса {#request}

```httpget
https://passport.yandex.ru/passport?
mode=spprtrstr
```
