# passport

Предоставляет обзорную страницу c данными учетной записи. Также содержит ссылки на все доступные страницы настроек и управления данными учетной записи:

- изменение персональных данных (режим [changereg](changereg.md));
- управление адресами электронной почты ([https://validator.yandex.ru/](https://validator.yandex.ru/));
- управление телефонными номерами (интерфейс описан в разделе [{#T}](https://docs.yandex-team.ru/yasms/) документа [Руководство разработчика сервиса, использующего SMS](https://docs.yandex-team.ru/yasms/));
- изменение пароля (режим [changepass](changepass.md));
- удаление учетной записи (режим `delete`).

{% note info %}

Режим `passport` вызывается по умолчанию, если в запросе к Паспорту не указан параметр `mode`.

{% endnote %}



## Формат запроса {#request}

```httpget
https://passport.yandex.ru/passport?
mode=passport
```
