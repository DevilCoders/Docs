Полученный токен можно сохранить в приложении и использовать для запросов к API до истечения [времени его жизни](../../../../../oauth/concepts/ya-oauth-intro.md). Токен должен быть доступен только вашему приложению, поэтому не рекомендуется сохранять его в куках браузера, открытых конфигурационных файлах и т. п.
