# Получение токенов внутри Яндекса

{% note warning %}

Перечисленные ниже способы не предназначены для широкого использования. Если вы уверены, что вам нужен один из них, напишите на рассылку [oauth@](mailto:oauth@yandex-team.ru) и расскажите о вашем случае.

{% endnote %}

Для сервисов и приложений Яндекса доступны дополнительные способы получения OAuth-токенов:

- [В обмен на куку Session_id](internal-tokens/sessionid.md).
    
- [В обмен на логин и пароль](internal-tokens/login-password.md).
    
- [В обмен на OAuth-токен](internal-tokens/xtoken.md) с правом **Выдача X-Token**. Этот метод используется только для мобильных приложений, посредством [Account Manager](https://wiki.yandex-team.ru/yandexmobile/accountmanager).
    
  По умолчанию право на получение X-токенов скрыто для всех пользователей Яндекс.OAuth, запросить доступ к нему можно в рассылке [oauth@](mailto:oauth@yandex-team.ru). После этого право можно будет выбрать на странице настроек приложения.
    
- [Без аутентификации пользователя](client-credentials.md). Таким образом можно получить только обезличенный токен, для авторизации внутренних серверных приложений Яндекса.
