Иcпользуйте один из вариантов, который вам больше подходит. Если вы хотите дополнительно уменьшить или увеличить высоту, вы можете это сделать с условием сохранения пропорций.

## Ширина кнопки {#width}

Максимальная ширина кнопки не ограничена. При уменьшении ширины следите за тем, чтобы боковые отступы были не меньше ширины иконки.

## Правила {#rules}

В таблице ниже приведены элементы, которые можно или нельзя изменять в кнопке.

Элемент | Разрешение | Рекомендации
----- | ----- | -----
Скругления | Да | Меняйте скругления под стиль вашего сайта или приложения.
Размеры | Да | Можно выбрать размер из предложенных, можно растягивать пропорционально.
Цвет | Нет | Не меняйте цвет кнопки, используйте варианты, которые представлены в гайде.
Обводка | Нет | Не меняйте толщину обводки, используйте варианты, которые представлены в гайде.
Содержимое кнопки | Нет | Не меняйте текст, иконку и отступы между элементами кнопки.
