# Терминология

#### _Авторизационная кука_

Кука, которую Паспорт выставляет пользователю при каждой авторизации.

#### _Авторизация_

Предоставление пользователю прав доступа к сервису.

Формально Паспорт предоставляет пользователям права только на использование функций Паспорта. Сервисы Яндекса должны проверять выставленные Паспортом куки и предоставлять права, соответствующие учетной записи.

#### _Валидная авторизационная кука_

Авторизационная кука считается валидной, если она успешно проходит проверку системы авторизации: срок жизни куки не истек, и содержащаяся в ней подпись подлинная.

#### _Глобальное разлогинивание_

Отзыв всех кук и токенов, дающих доступ к учетной записи.

Невалидными становятся все авторизационные куки и OAuth-токены, подписанные до глобального разлогинивания.

#### _Дорегистрация_

Задание логина и/или пароля для учетной записи, у которой эти атрибуты отсутствуют. Дорегистрировать можно лайт-аккаунт, а также учетную запись, созданную в ходе социальной регистрациии.

#### _Залогиновый сервис_

Сервис, который предоставляет дополнительные возможности авторизованным (залогиненным) пользователям.

#### _Лайт-аккаунт_

Учетная запись, зарегистрированная на Моем Круге по упрощенной схеме, с указанием только электронного адреса и пароля.

Паспорт больше не поддерживает регистрацию лайт-аккаунтов. При авторизации с таким аккаунтом пользователь должен будет [дорегистрироваться](https://doc.yandex-team.ru/Passport/passport-modes/reference/light2full.html).

#### _Сессия_

Промежуток времени, в течение которого авторизационная кука остается валидной.

Сессия начинается при выставлении авторизационной куки. Сессия заканчивается при явном разлогинивании пользователя или окончании срока жизни куки.

#### _Социальная авторизация_

Авторизация пользователя с помощью учетной записи внешнего социального сервиса (Facebook, Twitter и т. д.). Связь с внешними сервисами обеспечивает Социализация, описанная в соответствующем кластере Вики: [http://wiki.yandex-team.ru/social](http://wiki.yandex-team.ru/social).

#### _Социальная регистрация_

Создание учетной записи при первой [социальной авторизации](#socauth-dd) с внешней учетной записью. Подробнее обработка социальной авторизации описана в разделе [Социальный профиль](accounts-attributes.md#social-profile).

#### _Учетная запись_

Учетная запись (аккаунт) — это уникальный набор данных, соответствующих определенному пользователю Яндекса.
