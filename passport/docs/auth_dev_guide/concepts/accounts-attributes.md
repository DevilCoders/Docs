# Свойства учетных записей

Помимо обычных настроек и сведений о пользователе к аккаунту привязываются дополнительные данные:

- [карма](#karma);
- [социальный профиль](#social-profile).

## Карма {#karma}

Карма — это признак того, что учетная запись используется для рассылки спама или была зарегистрирована автоматически.

Конкретное значение кармы проставляется Фродобороной при регистрации аккаунта. Карма может быть изменена службой поддержки или с помощью режима Паспорта [`admkarma`](https://doc.yandex-team.ru/Passport/passport-modes/reference/admkarma.html).

{% note info %}

Карма не влияет на показ капчи пользователю при проверке логина и пароля в Паспорте. Капча — это мера борьбы с роботами, заполняющими формы интерфейса.

{% endnote %}


## Социальный профиль {#social-profile}

Социальный профиль — это информация об учётной записи пользователя, зарегистрированной в какой-либо социальной сети (провайдере профиля). Социальный профиль создается для каждой внешней учетной записи, с помощью которой пользователь регистрируется или авторизуется. Также профиль создается в процессе привязки аккаунта на странице [https://social.yandex.ru/](https://social.yandex.ru/).

Социальный профиль может содержать:

- Данные об учетной записи, предоставленные провайдером. В их число могут входить, например, идентификатор и имя учетной записи на сайте провайдера, а также имя, фамилия и пол пользователя.

- Токены, разрешающие действия от имени пользователя на сайте провайдера.
- Список сервисов Яндекса, получивших разрешение использовать данные профиля (например, в Поиске людей).


Если пользователь авторизуется с помощью соцсети, и используемая учетная запись соцсети Паспорту неизвестна, пользователю предлагается два варианта:

1. Привязать социальный профиль к существующей учетной записи. После этого пользователь сможет авторизоваться как с логином и паролем, так и социальным профилем.
1. Создать новую учетную запись. Аккаунт создается на основе социального профиля. Доступ к аккаунту обеспечивается исключительно социальной авторизацией (Авторизация пользователя с помощью учетной записи внешнего социального сервиса: Facebook, Twitter и т. д. Связь с внешними сервисами обеспечивает Социализация, описанная в соответствующем кластере Вики: [http://wiki.yandex-team.ru/social](http://wiki.yandex-team.ru/social)), пока не заданы логин и пароль. Для аккаунтов, созданных таким образом, логин и пароль можно задать в режиме [`postregistration`](https://doc.yandex-team.ru/Passport/passport-modes/reference/postregistration.html).


Данные, на хранение которых рассчитаны социальные профили, описаны в вики: [http://wiki.yandex-team.ru/social/entities](http://wiki.yandex-team.ru/social/entities).
