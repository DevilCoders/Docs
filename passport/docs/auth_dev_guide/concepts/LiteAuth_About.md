# Лайт-авторизация

{% note info %}

Раздел посвящен устаревшему понятию. Сервисам Яндекса больше не нужно различать пользователей, авторизованных с лайт-аккаунтами и обычными аккаунтами.

{% endnote %}


Пока Мой Круг был сервисом Яндекса, Паспорт регистрировал пользователей Моего Круга как лайт-аккаунты: авторизация на Моем Круге была недостаточно защищена для бесшовного входа на Яндекс. Сервисы сами могли принять решение о том, как обслуживать пользователей с лайт-авторизацией.

С тех пор как Мой Круг перестал быть сервисом Яндекса, регистрировать новые лайт-аккаунты стало невозможно. Пользователям с уже существующими лайт-аккаунтами Паспорт предлагает дорегистрироваться при входе на Яндекс: придумать логин и пароль и создать полноценный аккаунт.
