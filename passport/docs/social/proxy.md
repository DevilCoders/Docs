# Прокси

Прокси предоставляет унифицированный интерфейс для вызова API поддерживаемых провайдеров. Полный набор функций социальных сетей, доступных посредством Прокси, приведен в [вики-документации API](https://wiki.yandex-team.ru/social/proxy/v2).

В каждом вызове Прокси необходимо указывать:

* Идентификатор провайдера или конкретного приложения соцсети. Если указан только провайдер, Прокси использует приложение по умолчанию.

    Вся необходимая информация о провайдерах и приложениях [приведена на Вики](https://wiki.yandex-team.ru/social/providers).

* OAuth-токен, который дает нужные права. Если такого токена нет в Хранилище, нужно [отправить пользователя в Брокер](broker.md#data).

Адреса хостов Прокси для различных окружений перечислены в разделе [Хосты Социализма](location.md).
