# Введение

**Документ содержит** описание Социализма. Эта система помогает использовать аккаунты соцсетей, которые принадлежат пользователям Яндекса: получать данные об аккаунте, совершать действия от лица пользователя, авторизовывать его через социальную сеть. В документе рассмотрены задачи, решаемые Социализмом, а также компоненты системы.

**Целевая аудитория документа** — менеджеры и разработчики сервисов Яндекса, которые решают задачи интеграции с социальными сетями.

**Документ не содержит** общей информации о поддерживаемых соцсетях.

Пожелания и замечания к документу оставляйте в задаче [https://st.yandex-team.ru/DOC-2496](https://st.yandex-team.ru/DOC-2496).

## Структура документа

Документ состоит из следующих разделов:

1. [Общие сведения](system.md): описание компонентов и ключевых понятий Социализма.

1. [Хосты Социализма](location.md): адреса серверов системы, на которые следует отправлять запросы.

1. [Брокер](broker.md): описание компонента Социализма, который запрашивает информацию из соцсетей и привязывает полученные данные к учетным записям Яндекса. Также описан процесс взаимодействия сервисов Яндекса с Брокером.

1. [Прокси](proxy.md): описание компонента Социализма, который унифицирует интерфейс обращения к функциям соцсетей.

1. [Хранилище](db.md): описание компонента Социализма, который обеспечивает хранение данных с информацией об аккаунтах соцсетей.

О выпущенных версиях документа читайте в разделе [Релизы](releases.md).
