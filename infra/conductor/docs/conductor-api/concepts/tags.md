# Теги

Запросы позволяют получить список тегов и подробную информацию по любому из них.

#### Список тегов

```
{{ host }}/{{ api-ver }}/tags
```

#### Параметры определенного тега

```
{{ host }}/{{ api-ver }}/tags/<id тега>
```

## Связанные объекты {#section_d1z_vcs_gbb}
