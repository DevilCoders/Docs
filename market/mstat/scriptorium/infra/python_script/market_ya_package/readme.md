# Скрипт MARKET YA PACKAGE

Скрипт создает Sandbox-задачу по сборке пакета нашего сервиса из любой ветки Аркадии.
Скрипт предназначен для независимой сборки и деплоя в тестинг при отладке сервисов.


## Запуск

`python3 market_ya_package.py`


## Аргументы

Аргументы для создания Sandbox-задачи передаются в stdin в интерактивном режиме.

Свой SandboxAPI-токен можно получить [здесь](https://oauth.yandex-team.ru/authorize?response_type=token&client_id=968e0e6d85d647feb327d893a42cf26f&optional_scope=sandbox:api).
