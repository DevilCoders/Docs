# python_app

Шаблон приложения на python для разворачивания в RTC (RunTime Cloud).
Используются Flask + gunicorn.

## Development

### Tests
Запустить тесты:
```
ya make -rt
```

### Локальный запуск
```
./run --port 2606
```
