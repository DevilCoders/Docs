**Папка с отчетностью Претензионной группы СД**

Скрипты и описания(список указан в порядке исполнения):
* JUDGE_DREDD_transactions_auto - скрипт собирает статусы из тикетов в очереди CLAIMBLUE
* JUDGE_DREDD_TTL - скрипт собирает итоговую таблицу со всеми статусами, из таблиц JUDGE_DREDD_transactions_auto и JUDGE_DREDD_transactions
* JUDGE_DREDD_compensation - скрипт собирает таблицу на уровне единиц в заказах для дальнейшего использования в процессе автокомпенсаций
* JUDGE_DREDD_transactions - ручной скрипт (не запускается автоматически), нужен для добавления транзакций в таблицу JUDGE_DREDD_transactions
