Библиотека для подготовки файла настроек синих модификаторов

# Модификаторы цены доставки для оферов Беру
В зависимости от:
* региона пользователя
* веса посылки (крупногабаритный товар/посылка)
* цены посылки

меняет цену доставки.

# Тариф
В одной группе (modification) участвует от одного до нескольких тарифов.

Для каждого тарифа обязательно указывается цена доставки (user_price_rur).
Так же может быть задано ограничение (сверху, не включая):
* тариф для КГТ (large_size)
* цена посылки (price_rur_to)

Проверяется, что ограничения идут в порядке увеличения цены/веса.
Потому что в репорте будет принято первое удовлетворяющее ограничение.

В регионе должен быть единственный тариф без ограничений.

# Регионы
В разных группах регионы не должны пересекаться.

Должна быть одна единственная группа без указания регионов.
Она будет применяться на все неизвестные регионы
