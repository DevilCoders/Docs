# Библиотека для конвертации динамиков в универсальные правила

Библиотека принимает на вход файл динамика, вычитывает из него правила, создает на их основе правила в универсальном формате и передает их на вход дамперу.

Файлы динамиков, с которыми может работать библиотека:

Динамик АБО - market-sku-filters.pbuf. Правила из динамика АБО работают по следующему принципу:
- Если есть модель, то блокируем все оферы этой модели всех перечисленных поставщиков. Поля MarketSku и ShopSku игнорируются.
- Если есть запись, содержащая только модель без поставщиков, то блокируется вся модель. И контент тоже
- МСКУ блокируется аналогично модели (ShopSku игнорируется)
- Если нет модели и МСКУ, то блокируются товары поставщиков.
- Если есть поставщик и ShopSku, то блокируются этот товар поставщика
