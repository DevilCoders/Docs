# shop_category_ids
Библиотека для чтения flatbuffers файла `dsbs_shop_cat_ids.fb`.  
В файле для каждого DSBS офера содаржатся магазинные (оригинальные) category_id.  
Это нужно для вычисления опций доставки DSBS магазинов.  

1. [Про DSBS](https://wiki.yandex-team.ru/market/development/combinator/nordstream/dsbs/)
