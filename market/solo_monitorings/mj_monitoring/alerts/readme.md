## Стандартные алетры MJ

### Hardware

#### CPU usage is high
Высокая загрузка процеccора. Необходимо проверить распределение нагрузки

#### Использование памяти больше 80%
Проверьте, достаточно ли памяти выделено под heap. Нет ли утечек памяти.
В критическом случае может помочь рестарт контейнера


#### Свободного места на диске меньше 20%
Необходимо проверить использование места и настройку ротации логов.


### Endpoints
#### Закончились свободные потоки Jetty
Слишком много подключений к сервису. Возможно, запросы исполняются слишком долго

#### Время выполнения HTTP запросов больше 1с
Запросы исполняются слишком долго. Необходимо найти, с чем это связано

#### Ошибки по выполнении HTTP запросов
Получены ответы со статусом 500. Проверьте, почему произошли ошибки
