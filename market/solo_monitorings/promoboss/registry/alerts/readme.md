# Алерты Promoboss при работе с базой данных

## Hikaricp connections pool
### Hikaricp connections timeout
Слишком долгое ожидание свободного соединения к базе данных.

Проверьте, закрываются ли соединения и возвращаются ли они обратно в пул.

Проверьте время исполнения запросов.

Возможно, слишком большая нагрузка на сервис.

## Spring repository
### Время выполнения запроса через Spring repository больше 1с
Слишком долгое выполнения запроса к базе данных.

Проверьте запросы на оптимальность. Возможно, стоит добавить индексы.

### Ошибки выполнения запроса через Spring Repository
Ошибки при выполнении запросов к базе данных.

Проверьте правильность написания запросов и сервисов.

Добавьте Exception Handling
