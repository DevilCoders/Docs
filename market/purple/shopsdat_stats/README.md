# Схемы работы с поставщиками
1. fulfillment 1p - товар наш, храним у себя
2. fulfillment 3p - FBB (fulfillment by beru), товар магазина, храним у себя
3. click_and_collect - Витрина, товар магазина и забрать его можно только в магазине (таблетки, алкоголь)
4. dropshipping - Товар магазина, не храним у себя и не возим к себе (Холодильник)
5. cross docking - Товар магазина под заказ, привозим на склад только под заказ
