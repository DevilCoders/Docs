# Stalker

Тулза для сбора инфы с синих репортов.

В режиме run_web делает следующее.
Для специального офера
1) обновляет цену в datacamp, в качестве значения текущее время
2) собирает цену со всех бекендов и отдает через http

Результаты можно посмотреть здесь:
1. [testing](http://mi-blue01ht.market.yandex.net/manushkin/stalker.py)
2. [production](http://mi-blue01h.market.yandex.net/manushkin/stalker.py)
