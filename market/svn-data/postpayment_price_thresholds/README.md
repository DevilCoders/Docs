# Ограничения по максимальной стоимости заказа для службы доставки
Данные находятся в файле [postpayment_price_thresholds.cfg](https://a.yandex-team.ru/arc/trunk/arcadia/market/svn_data/postpayment_price_limits/postpayment_price_thresholds.cfg).

# Формат записи в файле postpayment_price_thresholds.cfg
```
[<название службы доставки в LMS>]
service_id=<ID соответствующего партнёра в LMS>
max_price=<максимально допустимая стоимость посылки>
```

Источник первой версии файла - [сводная таблица с ограничениями партнёров](https://wiki.yandex-team.ru/market/supply-chain/infoSD/).
Дальнейшие изменения (добавление партнёров, изменения порогов) нужно вносить вручную.

# Как работает
1. Файл с ограничениями приезжает под репорт.
2. Если при оформлении заказа стоимость посылки превышает порог, указанный для соответствующей службы доставки в этом файле, мы не позволяем оформить заказ с оплатой при получении.
