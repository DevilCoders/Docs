# Динамические тарифы доставки
Для улучшения unit экономики маркета добавляются динамические тарифы доставки.
В зависимости от признаков: FMCG, КГТ, Регион отправки и получения, Тип доставки - получаем коэффициенты для формулы.
Переменными в формуле являются: Вес, Объем, Кол-во товаров и Сумма мультизаказа.
Формула хардкодится в репорте. Критерии поиска и коэффициенты в файле [models.tsv](https://a.yandex-team.ru/arc/trunk/arcadia/market/svn-data/dynamic_delivery_tarrifs/models.tsv).
Стандартные значения коэффициентов, а также дополнительные параметры с трешхолдами хранятся в [dynamic_delivery_tariffs_constants.json](https://a.yandex-team.ru/arc/trunk/arcadia/market/svn-data/dynamic_delivery_tarrifs/dynamic_delivery_tariffs_constants.json).

Программа получает на вход
* input_files.json, в котором указаны имена двух файлов, описанных выше.
Выход:
* dynamic_delivery_tarrifs_models.pbuf.sn (результирующий артефакт)

Для обновления тестовых моделей, если изменится бизнес логика, можно пофильтровать основной файлик данным regex'ом `^((?!\t[0-1]\t[0-2]\t[0-2]\t).)*$`
