# Бесплатная доставка DSBS магазинов
Для некоторых партнеров крайне важна бесплатная доставка до клиента.
Информация о данных партнерах хранится в free_dsbs_shops.json.

Программа получает на вход:
* free_dsbs_shops.json, в котором указан список магазинов.
Выход программы:
* файл проверен на сооветствие структуры/данных ожидаемым репортом.
