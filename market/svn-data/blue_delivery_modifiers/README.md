# Модификаторы цены доставки для оферов Беру
В зависимости от:
* региона пользователя
* веса посылки
* цены посылки

меняет цену доставки.

# Конвертация данных
Есть шаблон modifiers_template.jsoт, в котором перечислены группы(modifiers).
Внутри каждой группы есть:
* тарифы (tarifs)
* указан файл, в котором перечислен список регионов (regions_file_name). Файлы должны располагаться рядом с шаблоном

Остальные поля не учитываются.
Например, можно оставить комментарий в поле comment.

На выходе получаем проверенный файл с группами с подставленными регионами.

# Тариф
В одной группе участвует от одного до нескольких тарифов.

Для каждого тарифа обязательно указывается цена доставки (user_price_rur).
Так же может быть задано ограничение (сверху, не включая):
* вес посылки (weight_kg_to)
* цена посылки (price_rur_to)

Проверяется, что ограничения идут в порядке увеличения цены/веса.
Потому что в репорте будет принято первое удовлетворяющее ограничение.

В регионе должен быть единственный тариф без ограничений.

# Регионы
В разных группах регионы не должны пересекаться.

Файлы регионов должен быть в формате csv.
Разделитель столбцов - символ табцляции.
В каждой строке первый столбец должен содержать номер региона.
В остальных столбцах может быть задано название региона (для отладки).
