# TensorFlow object detection profiler

*Внимание, данный проект не предназначен для сборки в tier0*
Проект, использует **внешнюю** зависимость (tensorflow2), которую необходимо настроить через venv на Jetson AGX Xavier для запуска проекта. Данный профайлер рассчитан на использование и тестирование только с PLT Detector графом, у которого имеются следуюшие выходы:

```python
output_final_names = [
    'pltlabels/probs',
    'pltlabels/bboxes',
    'pltlabels/classes',
]
```

Для запуска скрипта необходимо предварительно подготовить `*.pb` файлы графов, которые интересно померить на производительность.

Вызов скрипта осуществляется следующим образом:

```bash
python profile_tfinf.py
```

Предварительно необходимо произвести настройку `config.yaml` файла, который отвечает за проведение профайлинга.
В файле указываются параметры запуска и graph файлы, которые необходимо запускать.
