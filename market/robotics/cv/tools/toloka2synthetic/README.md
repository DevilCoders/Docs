# Toloka to Synthetic dataset

Утилита позволяет генерировать датасет из существующей разметки в Толоке.

Для работы необходимо иметь набор данных в виде существующей разметки, а также набор фонов для генерации.
Утилита кэширует кропы из существующей разметки и размещает их на случайно выбранном фоне. Аугментация разметки приводится в конфиге.

Подробное описание приведено в тикете: https://st.yandex-team.ru/YMBOT-347

## Предварительные условия
Перед началом работы со скриптом необходимо заполнить `~/.aws/credentials` кредами от S3.
```text
[default]
aws_access_key_id = <AWS_ACCESS_KEY>
aws_secret_access_key = <AWS_SECRET_ACCESS_KEY>
```

В скрипте используется Python `boto3` библиотека, которая подхватит эти параметры автоматически

## Пример запуска
```bash
./toloka2synthetic --config config.yaml
```

Данная утилита работает с S3 хранилищем. Для получения токенов и корректной работы рекомендуется ознакомиться с [инструкцией](https://wiki.yandex-team.ru/roboticsmarket/manuals/s3-manual/)
