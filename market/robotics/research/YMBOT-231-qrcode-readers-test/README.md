# YMBOT-231 Experiment

Experiment on reading QR codes from PLT Labels. 

More info is available in the Wiki: https://wiki.yandex-team.ru/roboticsmarket/cv/research/experiments/ymbot-231/ 

## Prepare
Before launch, download [plt-qr-minidataset.tsv](https://wiki.yandex-team.ru/roboticsmarket/cv/research/experiments/ymbot-231/.files/plt-qr-minidataset.tsv) and put it to data folder
