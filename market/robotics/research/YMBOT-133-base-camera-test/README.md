# YMBOT-133 Experiment

Experiment on reading barcodes and QRcodes with  [IMX219 sensor](https://github.com/rellimmot/Sony-IMX219-Raspberry-Pi-V2-CMOS/blob/master/RASPBERRY%20PI%20CAMERA%20V2%20DATASHEET%20IMX219PQH5_7.0.0_Datasheet_XXX.PDF) (Raspberri Pi Camera v.2) with different lightning condition. For image decoding, Yandex REST-API service was used.

Full page of experiment is located on [Wiki page](https://wiki.yandex-team.ru/roboticsmarket/cv/research/experiments/experimentimx290/).
