#Реарры на мапи

###Для чего этот пакет?
В этом пакете собираются все реары, используемые в мапи.
Каждый реар должен лежать в отдельном файле с соответсующим названием. Файл также
должен содержать краткое описание функционала реарра и ссылки на эксперимент в абшнице и/или проект, с которым связан этот реарр.

Названия файлов должны иметь следующий вид: ```<НаименованиеФункционалаRearr> ```

###Почему так?
Если скидывать всё в один класс, то все, кто будут вливать реарры, будут об него спотыкаться, будут мерж-конфликты и тд.
Если же хранить каждый реарр в отдельном файле, конфликтов будет минмиум и удаление реарра не станет болью. Также такое решение
позволяет исчерпывающе описывать функиционал реарра и быстро находить его.
