Парсер репорта для нужд байбокса. Для каждой msku из входной таблицы ходит в place sku_offers и парсит BUYBOX_DEBUG_LOG.
Если во входных rearr флагах указан market_blue_buybox_by_gmv_ue=1; то парсит "новый" формат формат дебаг лога,
если нет парсит старый.

Пример использования есть в папке integral_tests/test.sh, пример входных данных в data/input.tsv
