# Расчет цен для честных скидок
Сперва обновляет исключени по категориям и СКЮ
Для этого парсит тикеты и добавляет данные из экселек в таблицы на YT

Запускает два скрипта:
update_offers и calc_max_prices

update_offers - вспомогательный скрипт, нужен для того, чтобы не ворочать много таблиц market-offers
Сейчас для расчета честных скидок нужна история цен за 180 дней, они хранятся в таблице history_prices
update_offers добавляет в эту таблицу новый день и убирает первый день из нее

calc_max_prices - рассчитывает максимально допустимые цены (зачеркнутую и обычную) для выставления скидки
Подробности и схема в тикете https://st.yandex-team.ru/MADV-144
Самые важные столбцы в выходной таблице это max_price - максимально разрешенная цена для скидки
и max_old_price - максимально разрешенная зачеркнутая цена.
При этом дельта между ними не проверяется.
