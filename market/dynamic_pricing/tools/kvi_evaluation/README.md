## Процесс определения списка KVI-товаров
**KVI (Key value indicator) товары** - самые ходовые товары, цену которых все примерно знают и на основе которых покупатели принимают решение - дорогой магазин или дешевый; эти товары формируют ценовой имидж магазина.

#### Как определяются KVI-товары
https://wiki.yandex-team.ru/users/efedosyuk/Principy-formirovanija-KVI-tovarov/

#### Как часто формируется спиcок
20го числа каждого месяца - чтобы не попадать на праздничные дни.
