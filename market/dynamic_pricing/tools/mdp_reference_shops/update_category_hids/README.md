# Обновление списка референсных магазинов.

## Как пользоваться?
`../../../../ya make` - билдим

`./update_categiry_hids --help` - смотрим как запускать

## Зачем пользоваться?
Этот скрипт предназначен для обновления таблицы рефересных магазинов.

## Че можно поменять/добавить?
Где-то в джанке есть скрипт который дергает домены для конкретных категорий, его тоже можно куда-то перенести.
