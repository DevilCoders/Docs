###Как собрать

1. Из директории модуля запустить:

    `ya ide idea --group-modules tree --omit-test-data --iml-in-project-root --local -r ~/IdeaProjects/config-converter`


2. Исполняемый класс Parser.java


###Как пользоваться

1. Обновить локальный транк аркадии для получения актуальных конфигов (при запуске из транка):

    `arc pull trunk`


2. Установить переменную окружения PATH_TO_ARCADIA, например:

    `PATH_TO_ARCADIA=/Users/dallas1337/Arc/arcadia`


3. По умолчанию парсер генерирует 4 дампа для наборов правил, наборов границ, правил, и границ.
Для того, чтобы парсер соединил все в один дамп, нужно указать следующий параметр:

    `onefile`


4. Запустить приложение, оно сгенерирует sql дамп(ы) в корневой директории проекта
