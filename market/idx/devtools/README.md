# market/idx/devtools

Инструменты для разработчиков и менеджеров индексатора.
Директория предназначена для общего кода службы, который не используется в продакшене.

## Содержание
- [analyze-scripts](https://a.yandex-team.ru/arc/trunk/arcadia/market/idx/devtools/analyze-scripts/) - скрипты сбора метрик
- [configure-logbroker](https://a.yandex-team.ru/arc/trunk/arcadia/market/idx/devtools/configure-logbroker/README.md) - создание топиков и читателей оптом
- [dukalis](https://a.yandex-team.ru/arc/trunk/arcadia/market/idx/devtools/dukalis/) - инструмент для диагностики и поиска данных
- [dutygen](https://a.yandex-team.ru/arc/trunk/arcadia/market/idx/devtools/dutygen/README.md) - генератор дежурств индексатора
- [generation_timeline](https://a.yandex-team.ru/arc/trunk/arcadia/market/idx/devtools/generation_timeline/) - визуализация этапов сборки поколения
- [hypothese_analyse](https://a.yandex-team.ru/arc/trunk/arcadia/market/idx/devtools/hypothese_analyse/README.md) - проверка гипотезы оптимизации сборки поколения, построение графа зависимостей
- [logopener](https://a.yandex-team.ru/arc/trunk/arcadia/market/idx/devtools/logopener/) - сбор логов сервисов индексатора
- [nanny-revert](https://a.yandex-team.ru/arc/trunk/arcadia/market/idx/devtools/nanny-revert/README.md) - откат на старые снепшоты няни вручную
- [planeshift-balalaika](https://a.yandex-team.ru/arc/trunk/arcadia/market/idx/devtools/planeshift-balalaika/) - настройка балалайки
- [qinstall](https://a.yandex-team.ru/arc/trunk/arcadia/market/idx/devtools/qinstall/) - инсталляция пакетов вручную
- [release](https://a.yandex-team.ru/arc/trunk/arcadia/market/idx/devtools/release/cupload/) - загрузка пакетов в репозиторий кондуктора
- [regional_delivery_serializer](https://a.yandex-team.ru/arc/trunk/arcadia/market/idx/devtools/regional_delivery_serializer/) - создание файла regional_delivery.gz из оригинального regional_delivery.mmap
- [quick_pipelines_monitorings](https://a.yandex-team.ru/arc/trunk/arcadia/market/idx/devtools/quick_pipelines_monitorings/) - создание мониторингов saas hub
- [whosfat](https://a.yandex-team.ru/arc/trunk/arcadia/market/idx/devtools/whosfat/README.md) - расчет размера полей внутри блобов ытевых таблиц
- [whosfat_column](https://a.yandex-team.ru/arc/trunk/arcadia/market/idx/devtools/whosfat_column/README.md) - расчет размера каждой колонки ытевой таблицы
- [yql_proto_scheme](https://a.yandex-team.ru/arc/trunk/arcadia/market/idx/devtools/yql_proto_scheme/README.md) - генерация YQL парсера по proto схеме
