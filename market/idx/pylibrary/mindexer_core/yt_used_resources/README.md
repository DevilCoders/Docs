Питонячий код помогающий следить за расходом ресурсов ытя по запуску крона.

quota.py отвечает за диск, чанки, ноды, таблеты.

operations.py отвечает за расход операций: cpu, time, memory (можно добавлять любые доступные системные статистики из ытя, https://yt.yandex-team.ru/docs/problems/jobstatistics#sistemnye-statistiki)
