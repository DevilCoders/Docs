# Инструменты для нагрузочного тестирования Датакемпа

## feedgen
Генератор фидов-патронов.

Есть встроенный бредогенератор (простейшая цепь Маркова) для текстовых полей.
Для его использования надо взять выборку обычных xml-фидов и запустить скрипт make_bred.py.

## push_parsing_task
Постановка заданий на парсинг синтетических фидов.

## upload_feed
Загружает фиды в MDS.
