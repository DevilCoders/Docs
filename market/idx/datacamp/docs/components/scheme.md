## Компоненты { #components }
* piper — микросервис, отвещающий за обработку сообщений из logbroker. Читает данные из различных источников и транзакционно складывает их в YT
* picrobot - микросервис, отвечающий за скачивание картинок и загрузку их в аватарницу.
* stroller - микросервис, отвещающий за обработку http запросов: получение и обновление оффера по идентификаторам, поиск
* scanner — микросервис, отвечающий за получение данных для офферов из YT таблиц
* miner — микросервис обогащения офферов маркетными знаниями, валидации и подготовки офферов
* routines — микросервис для запуска периодических задач офферного хранилища, управления хранилищем
* parser — микросервис, отвечающий за парсинг фидов
* dispatcher - микросервис для рассылки данных об изменении оферов подписчикам
