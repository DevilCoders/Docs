# Routines
Микросервис для запуска кроновых задач оферного хранилища и его администрирования.

## Админка хранилища
[Подробнее на отдельной странице](admin.md)

## Запуск кроновых задач
На данный момент часть кроновых задач запускается в няне, их список можно посмотреть в [routines/bin/main.py](https://a.yandex-team.ru/arc/trunk/arcadia/market/idx/datacamp/routines/bin/main.py?rev=r8604646#L151-164). Остальная часть запускается в sandbox, подробнее про запуск задач череез sandbox [{#T}](sb-routines.md).

## Мониторинги работоспособности хранилища
На хостах няни запускаются специальные мониторинги, которые проверяют что хранилище парсит/майнит системные офферы.
