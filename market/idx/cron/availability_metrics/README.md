## Тулза для рассчета метрик доступности

### Описание
Агрегирует данные с разных типов графиков и рассчитывает по ним единую метрику доступности.
Данная метрика далее попадает в YT табличку [//home/market/production/indexer/stats/availability](https://yt.yandex-team.ru/arnold/navigation?path=//home/market/production/indexer/stats/availability)

Результирующие графики можно смотреть на [дашборде стабильности datacamp](https://datalens.yandex-team.ru/sxsn29208df6l-stabilnost-datakempa)
