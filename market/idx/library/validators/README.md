## Цель
Данная библиотека задумана с цельа собрасть в себе различные валидаторы, которые используются в коде индексатора маркета.

В качестве примера можно привести `offer_id` он должен валидироваться в следующих местах:
- `feedparser`
- `qparser`
- `saas_hub`
- `miner`
- 'datacamp controller'
Что бы избежать дублирования кода вся валидация вынесена в эту библиотеку.
