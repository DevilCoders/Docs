## Утилита для быстрой отладки новых дивных карточек

Пример использования
```
./card_renderer -f sample_card_data/choice_gallery.json
```


### Формат передаваемого файла
```
{
  "card_name": "<название карточки>",
  "form": {"name": "...", "slots": [...]}, # форма, которую басс возвращает винсу
  "context": <any> # опционально поле; данные, которые басс возвращет вместе с карточкой
}
```

Примеры файлов лежат в sample_card_data. Дополнение этой папки только приветсвуется!
