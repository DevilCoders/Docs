Утилита для кастомизации всякого разного внутри контейнера. Запускается автоматически с правами рута из prepare скрипта.

Можно использовать для настройки окружения с целью более комфортной работы. К примеру, она меняет настройки htop так чтобы его было удобнее использовать для экспресс-диагностики состояния контейнера.
