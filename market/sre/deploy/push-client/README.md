# Ресурс-обертка для бинарника push-client-а #

## Мотивация ##

Больше деталей можно найти тут: MARKETSRETODO-454

Этот ресурс призван решить проблему непреднамеренного обновления пуш-клиента на весь Маркет, безконтрольного и безжалостного. 

Создав этот ресурс, мы решаем две задачи:
1. Можно безопасно сделать тикетную интеграцию, а значит автоматизацию.
2. Переезжать на правильную версию пуш-клиента по нашему желанию, а не по воле разработчиков пуш-клиента.
