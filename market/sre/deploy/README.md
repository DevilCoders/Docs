# Репозиторий с конфигурацией слоев для Yandex Deploy #

## Структура каталога ##

`porto` - содержит башячку для формирования слоев для подов в Деплое.

`sre-binaries` - каталог, в котором лежит pkg.json, который билдит и собирает в один ресурс разные SRE бинари, типа dpreparer и prostater.

`sre-configs` - каталог содержит общие настройки, скрипты и конфиги, которые должны разъезжаться на все, или почти все контейнеры Маркета. Там же находится `pkg.json`, собирающий общий слой.

`features` - каталог содержит "фичи", то есть набор темплейтов и манифестов к dpreparer. 

Мы рекомендуем отдельно использовать ресурс из `sre-configs/pkg.json` и отдельно создавать каталоги для своих подразделений и служб, в которых подключать фичи из каталога `features`.

Можно также дописывать свои "фичи". Только не забывайте добавлять описание.
