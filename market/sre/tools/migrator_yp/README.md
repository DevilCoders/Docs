Migrate to YP
===========================================================================

Используется только на период миграции в YP. Текущий статус: В разработке.
Команда:

```
./scripts/migrate_to_yp.sh testing_market_clickdaemon_vla
```

Пытается сделать все и сразу:
- посчитать ресурсы;
- переключить движок на YP
- выделить подсет и эндпоинтсет
- сформировать снапшот и активировать
