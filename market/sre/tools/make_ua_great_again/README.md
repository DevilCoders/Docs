## Make UA Great Again
https://wiki.yandex-team.ru/users/kfour/sxema-ukrainy-dlja-servisov/

Пакет содержит утилиту для генерации if-up и if-down скриптов по настройке туннелей, маршрутов и адресов для обхода блокировок в Украине
После установки пакета утилита генерирует файлы /etc/network/if-{up,down}.d/make-ua-great-again.
