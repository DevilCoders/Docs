Проверятор корректности реквизитов tvm
======================================

# Зачем нужна

Утилите можно скормить tvm id клиента, tmv id сервера и секрет и убедиться, что сгенерился тикет. Таким образом мы можем проверить корректность настроек tvm-авторизации. Для целей настоящего дебага лучше использовать tvmknife: https://wiki.yandex-team.ru/passport/tvm2/debug/. Этот же тул правильнее применять в разных скриптах автоматизации на баше.

# Как пользоваться

Можно так:

`./tvm-checker --tvm-client-id 123 --tvm-server-id 456 --tvm-secret-file path/to/file/where-secret.txt`


А можно вот так:

`./tvm-checker -s 123 -d 456 -f path/to/file/where-secret.txt`

И еще пример:

`./tvm-checker -s 123 -d 456
Enter secret: 
`

Не забудьте также посмотреть всю справку, передав аргумент `-h`:
`
./tvm-checker -h
`

В случае не успеха, утилита выведет трейсдамп и код-ответа, отличный от 0.
