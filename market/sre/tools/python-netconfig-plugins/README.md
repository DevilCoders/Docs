## yandex-market-netconfig-plugins

* grey-lo-vips: create fdee:fdee::/32 address 'aliases' for every ipv6 VIP-address. Enabled with conductor tag `ipvs_grey_lo_vips`.
* use_local_table: always route local VS-traffic to lo-interfaces. Enabled with conductor tag `ipvs_use_local_table`.
* make-ua-great-again: exclude ip rules to table 666 for addresses for Ukraine.
* wh_additional_interfaces: additional interfaces and network settings for warehouses
