## report-pinger2graphite

Отправляет агрегированные данные из выхлопа report-pingerа в графит.
Если пингер работает на хосте локально - данные будут созданы под узлом текущего хоста, иначе – под балансером:
* marketindexer.tst.vs.market.yandex.net – для тестинга
* marketindexer.http.yandex.net – для production
