monrunner
====================

Обертка для запуска monrun от root без необходимости иметь sudo

Общее описание и принцип работы
-------------------------------

Исполняемый файл /usr/bin/monrunner после инсталляции получает SUID и GUID биты.
При его запуске, из-под пользователя root выполняется команда /usr/bin/monrun,
с передачей ей всех тех аргументов, что получил на вход monrunner.
Код возврата приложения будет совпадать с кодом возврата monrun, если его удастся запустить.
