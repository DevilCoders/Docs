# Репозиторий с Pillar для Salt Маркета

Здесь живут все данные для самогенерируемых salt'ом конфигов.

# Использование yav
В pillar можно добавлять данные из yav, для этого надо воспользоваться вот таким синтаксисом jinja:
```jinja2
{{ salt["yav.get"]("sec-01fz11y8qebaj4q78hnqnds9n2[auth]") }}
```
Добавлять можно прям в yaml файл как в [Примере](docker/init.sls)
