market-redis-formula
==

Используется в Маркете для установки и конфигурирования redis и опционально twemproxy.

----------

Доступные стейты
==
* redis
	 
	 Конфигурация через pillar необязательна, можно просто накатить на машину и получить redis, слушающий 127.0.0.1 и ::1, ограниченный 5Гб памяти и не персистентный. Если нужна конфигурация - берём example.pillar за образец и развлекаемся.
* twemproxy

	Требует конфигурации через pillar, без неё притащит дефолтный конфиг, который будет пытаться ходить в redis на локальном хосте. См example.pillar и официальную документацию на https://github.com/twitter/twemproxy
