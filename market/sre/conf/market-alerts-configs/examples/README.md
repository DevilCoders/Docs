# Примеры
В данном каталоге размещены файлы с примерами конфигов.
Вы можете выбирать для себя подходящий и на примере его создавать собственные.

Вы также можете создавать здесь примеры для собственных конфигов, чтобы в будущем просто копировать их.
Только, просьба, создавать их в отдельных каталогах и максимально подробно документировать.
