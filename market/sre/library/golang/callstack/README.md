# callstack

Библиотека для сбора информации оо стэке запуска - цепочке всех родительских процессов от PID=1 до текущего процесса.
Идея взята из iptruler (функция fn\_fill\_callstack()).
С помощью полученного списка мы можем выяснить, кто является запускателем данного процесса (конкрентый человек), даже если действия делаются под sudo.
