# Маркет для бизнеса 

- [Документация](https://wiki.yandex-team.ru/market/market4business/) Маркета для бизнеса.
- [ABC-сервис](https://abc.yandex-team.ru/services/b2b_market/)

## Компоненты

- [Бабуся](clients) - сервис хранения информации о покупателях юридических лицах. Так же содержит логику генерации 
  счета на оплату и хранения юридических документов по заказу.
- [АПИ до Бабуси](clients-api)
- OpenAPI [описание](clients-yaml) сервисов, куда ходит Бабуся и др. сервисы, но которые не содержат собственного 
  описания.
- [Утилиты](bin) для автоматизации работы разработчика.
