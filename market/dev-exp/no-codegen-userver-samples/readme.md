## Примеры использования userver без кодогенерации

В этой директории находятся сервисы-примеры по использованию userver в Аркадии
без использования кодогенерации, включая примеры написания тестов и поднятия
различных баз.

Если вы используете кодогенерацию, то
[смотрите соответствующие примеры](https://wiki.yandex-team.ru/taxi/backend/userver/tutorial/).
