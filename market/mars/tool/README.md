# Утилиты для MaRS-а

Чтобы воспользоваться утилитами, нужно сначала собрать папку _market/mars/tool_.

Данная папка содержит функции, которые не должны быть включены в итоговую сборку.

## dsp_generator

Данная утилита позволяет сгенерировать тело для запроса в ручку __dsp/offline__:

### Пример использования

Справка:
```
./tool/dsp_generator/gen.py --help
```

Пример генерации профиля с моделями 1, 2 и 123:
```
./tool/dsp_generator/gen.py -c 1 -c 2 -c 123
```
