## Описание
Данная директория и ее поддиректории содержат описания протобуфов используемых в маркетном индексаторе и смежных компонентах.

## Документация
Имеется документация протобуфов в markdown формате:
https://a.yandex-team.ru/arc/trunk/arcadia/market/proto/docs/proto-docs.md

Документацию генерится из кометариев в proto файлах, но ее необходимо перегенерить вручную.
Что бы обновить `proto-docs.md` нужно явно вызвать команду для ее обновления и закоммитить изменения 
```
market/proto:$ make docs
market/proto:$ ya tool svn commit docs -m "понятное описание коммита с тикетом"
```

Для гернерации документации в данный момент используется плагин для `protoc`:
https://github.com/pseudomuto/protoc-gen-doc

На странице плагина имеются [примеры](https://github.com/pseudomuto/protoc-gen-doc#writing-documentation) описания документации.

Для того что бы начать генерить документацию на дополнительный proto файл.
Нужно добавить его после опции `--input` в docs/generator/ya.make и в секцию `IN` там же.
