# @yandex-market/seo

Помогает в поисковой оптимизации (SEO) Яндекс.Маркета.

## API

Подробное описание интерфейса есть в JSDoc каждой функции. Здесь краткий обзор задач, решаемых библиотечкой.

### `getSlug`

Перерабатывает название страницы в кусочек человекопонятного URL'а (ЧПУ) aka slug.

```js
getSlug('fließend flüssig'); //=> 'fliessend-flussig'
```

Для имени компании `яндекс` сделано исключение. Имя компании транслитерируется в `yandex`, а не в `iandeks`.

### `compileTitle`

Собирает параметрический SEO заголовок, используя рецепты, полученные от Репорта, либо из фильтров по правилам, полученным из Букера.

### `getMetaPagerUrls`

Получает URL'ы для мета-пагинации: ссылки на «предыдущую» и «следующую» страницы.

### `transliterate`

Транслитерирует кириллические символы по стандарту `Doc 9303` и переводит их в нижний регистр

## Как сделать Pull Request?

После того, как вы внесли изменения, убедитесь, что написали на них юнит тесты. Запустите проверки кода.

```sh
veendor install
npm test
```

Убедитесь, что вы увеличили номер версии в `package.json` и задокументировали ваши изменения в `CHANGELOG.md`.

Для запуска линтера нужна node>8
```sh
npm run test:lint
```
