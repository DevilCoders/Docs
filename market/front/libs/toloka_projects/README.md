# toloka_projects

Сейчас 2 проекта:
- отображение категорий и тематик (тематика - набор категорий) 
- отображение товарной выдачи

в обоих логика может быть как обычных запрсов на бекенд,
так и запросов с контекстом (сделаны для рекомендательной морды https://st.yandex-team.ru/MARKETFRONT-29162#60671e8f2e469c397d919430) 

#### Подход работы в текущих проектах:
через ручки капи (https://api.market.yandex.ru) запрашиваем нужные нам данные (категории/товары)
и вставляем в итнтерфейс разметки Толоки

на итоговых файлах не стоит оставлять комментариев, выводов в консоль и тестовых проходов (в сборке авт. отсеивание идет комментов)
в толоку загружаем из папки сборки dist

Дока по доступным классам жц интерфейса Толоки:
https://wiki.yandex-team.ru/assessment/toloka/techdocs/frontend/sdk-referrence/
