## Visitors для сниппета оффера в табе Цен

Тут собраны visitors (посетители) для сграбливания данных по сниппету оффера в табе Цен.

Сниппеты можно посмотреть тут: https://market.yandex.ru/product--smartfon-apple-iphone-11-64gb/558171067/offers

### Про visitors из этот папки

* `main.js` - получение информации по сниппетам из списка офферов

### Как использовать

Опция для использования при cli-вызове data-grabber:

* `main.js`: `--visitor=/productOffers/main`
