## Visitors для блока с юридической информацией на страницах КМ и выдачи.

Тут собраны visitors (посетители) для сграбливания данных по блоку юр. информации.

Блок можно увидеть на странице https://market.yandex.ru/product--smartfon-apple-iphone-11-128gb/558168089

### Про visitors из этот папки

* `main.js` - получение информации по всему блоку юр. информации.

### Как использовать

Опция для использования при cli-вызове data-grabber:

* `main.js`: `--visitor=/pageJurInfo/main`
