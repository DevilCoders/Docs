## Visitors для блока с информацией о магазине на станице информации о магазине.

Тут собраны visitors (посетители) для сграбливания данных со страницы информации о магазине.

Блок можно увидеть на странице https://market.yandex.ru/shop-info?shopIds=37758

### Про visitors из этот папки

* `main.js` - получение информации по всему блоку информации о магазине.

### Как использовать

Опция для использования при cli-вызове data-grabber:

* `main.js`: `--visitor=/shopInfo/main`
