WebdriverIO Commands for Ginny
====

Общие кроссервисные [команды](http://webdriver.io/guide/usage/customcommands.html) для [Ginny](https://github.yandex-team.ru/market/ginn).

Описние команд находится в JSDoc соответствующих файлов.

## API
### Exports
```js
const commands = require('@yandex-market/webdriver-io-commands');
```

___

#### `{string} get targetDir`

Возвращает абсолютный путь к папке команд.
