# b2b-core

Общая библиотека для работы с резолверами и Stoutом в b2b интерфейсах маркета. Используется в ПИ, 3pl и микрофронтах ПИ.

# Разработка
Зависимости уставнавливаем командой `npm ci`
Для проверки в рамках партнерки можно сделать `npm build` и подложить в `node_modules` партнерки

# Публикация
1. Коммитим и пушим
2. Задаем версию `npm version minor|patch` (В зависимости от того, это мелкий фикс или новая фича)
3. Проходим все ревью. Будет обязательно ревью от @kykint
4. Мержим PR в транк
5. Публикация запустится автоматически ([здесь](https://a.yandex-team.ru/projects/marketpartnerfront/ci/releases/timeline?dir=market%2Ffront%2Flibs%2Fb2b-core&id=publish-release))

Перед публикацией запустятся все проверки и сборка.
