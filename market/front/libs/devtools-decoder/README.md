# Devtools Decoder

Chromium расширение - Декодер для расшифровки base64 строки из qr кода в девтулзах.

# Разработка и создание упакованного расширения

- Скачайте репозиторий на локальную машину.
- Запустите скрипт `npm run build`
- Зайдите в chrome или yandex browser на страницу `chrome://extensions`

## Разработка
На странице расширений выбираем пункт `Загрузить распакованное расширение`
и выбираем папку `build`, в качестве источника.
При внесении изменений обновляем расширение на той же странице.

## Упаковка
На странице расширений выбираем пункт `Упаковать расширение`
и выбираем папку `build`, в качестве источника.

# Использование 

Открыть в браузере страницу `chrome://extensions`
Перетащить в него файл `build.crx` который получается после упаковки расширения.
