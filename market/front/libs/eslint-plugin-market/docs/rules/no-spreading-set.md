# no-spreading-set

Запрещает спредить Set

> Почему?
Сломается при сборке вебпаком при текущей конфигурации.

Стандартные для Set методы и свойства (has, add, length etc) использовать можно

## Примеры

Примеры **неправильного** кода:

```js
[...new Set(array)]
```

Примеры **правильного** кода:

```js
import {uniq} from 'ambar';
uniq(array);
```
