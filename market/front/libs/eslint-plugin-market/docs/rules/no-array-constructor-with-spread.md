# no-array-constructor-with-spread

Запрещает спредить в массив пустой массив, созданный через Array

> Почему?
Сломается при сборке вебпаком при текущей конфигурации.

## Примеры

Примеры **неправильного** кода:

```js
[...Array(5)]
[...new Array(5)]
[...Array(5).keys()]
[...new Array(5).keys()]
```

Примеры **правильного** кода:

```js
new Array(5).fill(null).map((n, i) => i)
```
