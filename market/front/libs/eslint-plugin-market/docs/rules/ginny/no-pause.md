# ginny/no-pause

Запрет на использование this.browser.pause();

> Почему?

Пауза плохо сказывается на времени выполнения тестов. Необходимо писать корректный wait или указывать причину, почему необходимо использовать pause.

## Примеры

Примеры **неправильного** кода:

```js
makeCase({
    test() {
        // ...
        await this.browser.pause(600);
        // ...
    },
});
```
