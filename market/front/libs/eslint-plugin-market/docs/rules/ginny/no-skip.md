# ginny/no-skip

Запрещено использовать метод skip в контексте hermione при использовании hermione-suite-manager.
Стоит включать только в директории сьютов, т.к. селектор этого правила очень "общий" и сработает для любого метода skip.

> Почему?

В связи с появлением нового процесса скипа автотестов. Должен быть один инструмен, привязанный к процессу. [Задача](https://st.yandex-team.ru/MARKETFRONTECH-808)

## Примеры

Примеры **неправильного** кода:

```js
module.exports = makeSuite({
    story: {
        'по умолчанию': {
            'this.skip запрещен': makeCase({
                test() {
                    this.skip('причина скипа');
                }
            })
        }
    }
});
```
