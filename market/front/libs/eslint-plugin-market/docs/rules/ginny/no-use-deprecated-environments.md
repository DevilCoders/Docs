# ginny/no-use-deprecated-environments

Запрещено использовать дьяволские окружения

> Почему?

Потому что в данных окружениях тесты либо не прогоняются, либо ведут себя неправильно.

## Примеры

Примеры **неправильного** кода с конфигом `['prestable']`:

```js
module.exports = makeSuite({
    environment: 'prestable',
    story: {
        'по умолчанию': {
            // ...
        }
    }
});
```
