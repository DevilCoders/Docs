# epics-expressions

Проверяет, что в эпиках происходит обязательная фильтрация экшенов перед другими действиями. Обращает внимание только на вызовы вида `actions?$?.pipe`.

По умолчанию `ofType` и `filter`.

В опциях правила можно указать список обязательных методов: `'epics-expressions': ['error', 'ofType', 'filter']`

> Почему?
По мотивам инцидента MARKETFRONT-45054

## Примеры

Примеры **неправильного** кода:

```js
action$.pipe(merge(...){...})
```

Примеры **правильного** кода:

```js
action$.pipe(ofType('SOME_ACTION_TYPE'), merge(...){...})
```
