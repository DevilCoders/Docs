Заголовки

```jsx
<section>
    <Heading size={1}>Heading 1</Heading>
    <Heading size={2}>Heading 2</Heading>
    <Heading size={3}>Heading 3</Heading>
    <Heading size={4}>Heading 4</Heading>
    <Heading size={5}>Heading 5</Heading>

    <br />
    Примеры жирных заголовков:

    <Heading bold size={2}>Bold Heading 2</Heading>
    <Heading bold size={5}>Bold Heading 5</Heading>

    <br />
    Примеры заголовков с отступами:
    <Heading margin="bottom" size={2}>Heading 2</Heading>
    <Heading margin="bottom" size={5}>Heading 5</Heading>
</section>
```
