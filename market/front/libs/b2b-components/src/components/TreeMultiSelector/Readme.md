Дерево с выбором элементов. Если нужно просто выбрать элементы из дерева то можно использовать
компонент TreeSelector, этот компонент нужно для того чтобы разложить элементы дерева по
нескольким группам. Один элемент можен быть выбран только в одной группе, при этом
родительский элемент можен быть в одной группе, а дочерний - в другой.
