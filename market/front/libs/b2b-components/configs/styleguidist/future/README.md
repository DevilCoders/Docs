# Разрабатываемые b2b-компоненты

В секции `future` представлены компоненты, которые еще не реализованы.

## Как добавить компонент

1) В папке `configs/styleguidist/future` необходимо создать папку с названием Компонента (с большой буквы, в PascalCase)
2) В папке Компонента нужно создать как минимум 2 файла: `index.js` и `Readme.md`
3) Макеты компонентов кладутся в ту же директорию Компонента, желательно в `svg`
4) В `index.js` нужно "зареквайрить" макеты добавленного компонента, а также его различные состояния, которые можно определить через "props", например:

    **Checkbox/index.js**

    ```jsx
    import React from 'react';

    // кладем в папку Checkbox макет дефолтного чекбокса
    import checkbox from './checkbox.svg';

    // кладем в папку Checkbox макет выбранного чекбокса
    import checkboxChecked from './checkboxChecked.svg';

    // экспортируем компонент, который в зависимости от свойства `checked` отображается по-разному
    export default props => {
        // если `checked` указан, то покажем макет с выбранным чекбоксом
        if (props.checked) {
            // для использования макета необходимо указать нужный макет как свойство `src` у `img`
            return <img src={checkboxChecked} />;
        }

        // по умолчанию показываем дефолтный
        return <img src={checkbox} />;
    };
    ```
    Макеты компонентов желательно хранить в `svg`
5) В `Readme.md` необходимо описать, что это за компонент и как он используется (различые его состояния). Для того, чтобы отобразить компонент, его нужно описать в секции с синтаксисом `js`. Например:

    ```md
        ```js
        // ComponentName - название Компонента (совпадает с названием созданной директории)
        <ComponentName />
        ```
    ```

    Пример описания для компонента Checkbox:

    **Checkbox/Readme.md**

    ```md
        Флажок, флаговая кнопка, чекбокс (от англ. check box), галочка — элемент графического пользовательского интерфейса, позволяющий пользователю управлять параметром с двумя состояниями — ☑ включено и ☐ выключено.

        ```js
        <Checkbox />
        ```

        Выбранный чекбокс (checked):

        ```js
        <Checkbox checked />
        ```
    ```
