## 0.30.0 (14 June 2022)

### Features

- добавлен трансформер для эпиков (опустошает модуль эпиков - полезно в серверной части)
- все необходимые сущности для mandrel-контекста создаются автоматически
- Result теперь содержит методы: get/setClient/Backend

### Refactoring

У Mandrel-слоя удалены метода init/getUser/Route/Request/Response.
Вместо этого используется `initContext`

Теперь у любого слоя может быть 4 специальных свойства:
- `methods` - клиент/бекенд методы
- `client` - методы у которых будет выполнена только клиентская часть
- `backend` - методы у которых будет выполнена только бекенд-часть
- `worker` - прямой доступ к методам воркера
