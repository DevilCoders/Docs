## Удаляет комментарии дубликаты от роботов из тикетов.

Минимальный набор переменных окружения для запуска шедулера:

- `STARTREK_TOKEN` – токен робота в Стартреке, от лица которого будем удалять комментарии.

- `TRACKER_FILTER` – фильтр Стартрека по которому ищем список тикетов. Например, `Queue: MARKETFRONT and Status: Open`.

- `ROBOTS_NAMES` – никнеймы роботов через точку с запятой `;`, комментарии которых будем удалять. Например, `robot-metatron;robot-market-infra`.

- `FEATURES` – уникальные признаки комментариев через точку с запятой `;`. По признакам будем определять "уникальность" комментария. Например, `testing-touch;testing-desktop`.
