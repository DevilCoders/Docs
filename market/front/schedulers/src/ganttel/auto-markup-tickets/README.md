## Расставляет теги контуров 

Минимальный набор переменных окружения для запуска шедулера:

- `STARTREK_TOKEN` – токен робота в Стартреке, от лица которого будем ставить теги.

- `PLANNER_TOKEN` – токен в планнере для получения данных контуров

- `TEMPLATE_QUEUE_ACCESS_ID` - ID шаблона с запросом предоставления роботу доступа к очереди

- `TEMPLATE_FILTER_EDITING_REQUEST_ID` - ID шаблона с запросом редактирования контурного фильтра
