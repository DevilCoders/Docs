# Поиск российских топонимов

Доступна по адресу `https://{host}/api/v1?name=findLocation`

Обращается к https://wiki.yandex-team.ru/suggest/geo/ для получения результатов.
В качестве параметров принимает подстроку для поиска и, опционально, требуемое количество результатов в выдаче.

```typescript
type Params = {
    query: string;
    count?: number;
};
```

В ответе будет коллекция топонимов (село, аул, город, жилой комплекс, садовое товарищество). Тип описывающий единицу коллекции

```typescript
type NormalizedToponym = {
    id: RegionId;
    title: string;
    subtitle: string;
    type: string;
    size: number;
    position: string;
};
```
