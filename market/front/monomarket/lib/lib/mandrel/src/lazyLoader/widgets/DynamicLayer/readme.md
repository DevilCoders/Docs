# Виджет DynamicLayer

Ленивые виджеты не рисуются на сервере с некоторых пор.

При этом FakeRoot - статичный, и содержит ряд обвязок, которые не надо перерисовать на клиенте, 
и вообще он нарезается на части, чтобы обработать батчинг виджетов. 

Чтобы на клиенте нарисовать ленивый виджет, добавляем прослойку - DynamicLayer 
- Проставляет рисуемому виджету `bare: true`
- Скрывает виджет на сервере в отображении
- На клиенте отрисует виджет
- Но, по параметру вообще ничего не делает и рисует всё статично
