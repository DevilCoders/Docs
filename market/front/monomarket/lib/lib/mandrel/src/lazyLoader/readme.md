## WidgetLoaders

Папку надо переименовать из `lazyLoader` в `widgetLoaders`

### Типы загрузчиков

- Бразуер - браузер пользователя
- Хост - node.js приложение фронта, с которым взаимодействует браузер
- Ремоут - node.js приложение фронта, с которым взаимодействует Хост.
  Браузер не ходит напрямую в ремоут из-за CSRF.


|Тип                   | Где лежит контроллер виджета | Вызов верстки виджета     | Клиентский код виджета|
|--------------------  |:----------------------------:|:-------------------------:|:---------------------:|
| Виджет без лоадера   | Хост                         | При первой отрисовке (SSR)| В бандле страницы     |
| LazyLoader           | Хост                         | С клиента по событию      | В бандле страницы     |
| RemoteLazyLoader     | Ремоут                       | С клиента по событию      | Приходит отдельно     |
| RemoteLoader         | Ремоут                       | При первой отрисовке (SSR)| Приходит отдельно     |
| BundledLazyLoader (?)| Хост                         | С клиента по событию      | Приходит отдельно     |
