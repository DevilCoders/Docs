# 2.0.0

**PR:** [#552](https://github.yandex-team.ru/market/monomarket/pull/552)
**Author:** @maslov-rn **Date:** 2021-12-08

Расширение luster для запуска http-сервера, предназначенного для сбора метрик приложения.
На http://localhost:1447/api/metrics доступны метрики node.js:

- eventLoopLag (min, max, p50, p90, p99, mean), взял реализацию как в prom-client
- стандартные node.js метрики из https://github.yandex-team.ru/data-ui/monlib-nodejs (process.uptime, health, ...)

Сбор осуществляется с запущенных воркеров с аггрегацией значений метрик

### Usage

Для запуска, необходимо в luster.conf.js (в секцию extensions) добавить:

```
...
    extensions: {
        ...
        '@yandex-market/luster-metrics': {
            socket: process.env.LUSTER_METRICS_SOCK,
            workers: 2,
        },
    },
...
```

### Migration

Не требуется
