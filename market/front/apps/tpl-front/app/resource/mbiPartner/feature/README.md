# Методы программы

* Получение данных о программе [getFeatureInfo][1];
* Включение программы [enableFeature][2]
* Отключение программы [disableFeature][3]
* Снятие скрытия программы для конфигурирования [showFeature][4]
* Включение магазина в прод (successFeature)
* Получение данные о времени наложения и снятия cutoff'а (closedCutoff)

[3]: https://wiki.yandex-team.ru/MBI/NewDesign/components/market-payment/disableFeature/
