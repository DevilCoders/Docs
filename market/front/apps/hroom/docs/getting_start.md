## Начало работы
```
git clone https://github.yandex-team.ru/market/hroom.git
npm i

# создать в корне проекта файл .env
# получить все необходимые токены и прописать их в .env-файл

# Запустить сборку в режиме отслеживания изменений
npm run dev

# создать все необходимые таблицы в базе данных
npm run exec -- sync_db

# выкачать данные из wikis в локальную базу
npm run exec -- load_data

# Запустить сервер бота
npm run server
```

## Работа с конфигами
Дефолтные конфиги описаны в файле configs/default.
В корне проекта в файле .env описываются параметры индивидуальные для рабочего окружения, а также можно переопределять дефолтные значения.
Для переопределения вложенных дефолтных значений необходимо использовать "__".
Например:
```bash
db__password="custom_passord"
```
