# BCM 

Ведется активная работа по переносу ресурсов.

Актуальная документация и соглашения про BCM в белом - https://wiki.yandex-team.ru/market/frontend/BCM-в-Белом/

## Философия BCM
Основная идея - использовать удобный, легко-читаемый интерфейс для реализации логики работы с различными бекендами. В данном случае за основу берётся концпеция разделения всей логики на 3 типа сущностей: Бекенды, Клиенты и Модели.
 
Подробнее можно прочитать здесь https://github.yandex-team.ru/market/bcm2#backendclientmodel
