# Виджет WindowScrollTracker

Предназначен для отправки информации о скроллинге страницы в метрику.
Эмитит событие при начале скроллинга, при изменении направления и при окончании.

## Опции

scrollDebounceTimeout: number - таймаут (мс) для проверки изменения скролла
directionThrottleTimeout: number - таймаут (мс) для проверки изменения направления
