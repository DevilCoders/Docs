# Progress

Полоса прогресса

Горизонтальная полоса прогресса золотого цвета
Может принимать кастомные классы

Значение value от 1 до 100 (означает прогресс)
className будет присвоен корневому элементу

* Необязательные параметр valueClassName присвоится цветной части прогресс-бара
* Необязательный параметр theme, отвечает за то, будет ли анимирован прогресс-бар ("animated"),
по умолчанию -- нет ("simple")
* Необязательный параметр color определяет цвет закрашиваемой области. В простой теме по дефолту используется желто-оранжевый цвет, в анимированной - всегда градация от красного к зеленому.
* Необязательный параметр height отвечает за толщину прогресс-бара, по умолчанию - 3px

Пример использования:
```jsx harmony
const Progress = require('@self/project/src/components/Progress').default;

const className = 'test-class';
const value = '88';

 <Progress className={className} value={value} />
```
