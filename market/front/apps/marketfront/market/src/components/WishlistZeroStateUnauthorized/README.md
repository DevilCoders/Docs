## WishlistZeroStateUnauthorized
Компонент пустого состояния для списка отложенных товаров неаутентифицированного пользователя.

## Пример использования
```jsx
const WishlistZeroStateUnauthorized  = require('@self/platform/components/WishlistZeroStateUnauthorized');

<WishlistZeroStateUnauthorized domainZone="ru" returnUrl="https://any/url"/>
```
