# Звезды рейтинга у магазина

# ColorSet

Кружочки с цветами продукта

Пример использования:
```jsx
const ShopRatingStars = require('@self/project/src/components/ShopRatingStars').default;

<ShopRatingStars
    rate={3}
/>
```

С хинтом:
```jsx
const ShopRatingStars = require('@self/project/src/components/ShopRatingStars').default;

<ShopRatingStars
    rate={4}
    hint="На основании отзывов за прошлое ревью"
/>
```
