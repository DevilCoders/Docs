# HighlightedText

Общий тайтл для карточки сниппета

Пример использования:
```jsx
const HighlightedText = require('@self/project/src/components/HighlightedText').default;
const {text} = require('@self/project/src/components/HighlightedText/__fixtures__');

<HighlightedText
    partClassName="part"
    text={text}
/>
```
