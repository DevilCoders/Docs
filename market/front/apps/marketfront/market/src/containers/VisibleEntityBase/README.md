# VisibleEntityBase

Универсальный маппер компонентов / контейнеров / слотов виджетов на основе VisibleEntity.
Создает контейнер со всем необходимым и легко расширяется.

## Параметры

- lookup — Маппинг требуемых сущностей из VisibleEntity на нужные компоненты
- customMapper — Свой редаксовый маппер для извлечение нужных пропсов из стейта (опционально)
- Wrapper — Враппер для оборачивания в него всех таргетных компонентов (опционально)

## Как пользоваться

Заимпортить, вызвать из модуля makeVisibleEntityContainer (дефолтный экспорт) с нужными параметрами и
заэкспортить.
 
При создании React элемента прокидывать пропсу visibleEntityId. Если контейнер не нашел в стейте
такого visibleEntity, свернется без лишних дивов. Если не нашел требуемый компонент в маппинге (lookup), залогирует
ошибку и тоже свернется.

    Пример: market/platform.touch/containers/SearchResult/index.js
