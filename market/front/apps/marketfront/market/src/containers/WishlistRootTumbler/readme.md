# Root tumbler

Данный контейнер идеологически - копия соседа `WishlistTumbler`. Прибей его при первой возможности
(ну или соседа прибей). А еще тут тач не поддержан... И не должен быть поддержан.
Контейнеры не должны различаться между платформами.

### Мотивация
Новая выдача костылем использует старые снипеты (потому что левитановские обещают вот-вот).

А вот wishlist (эпики, редюсеры, коллекции) пытаемся использовать свежий
