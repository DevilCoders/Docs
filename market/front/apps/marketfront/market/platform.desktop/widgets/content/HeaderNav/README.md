# Виджет HeaderNav

## Описание

Виджет HeaderNav представляет из себя меню (дропдаун), находящийся в шапке страницы. По клику на аватарку распахивается с анимацией, имеет дополнительное состояние с панелью переключения между аккаунтами.

* После первого открытия ходит с клиента в mail.yandex.ru за количеством писем
* Так как position: absolute, фон сделан в виде :after, высота подлоджки зависит от содержимого (responsive)
* Аватарка не дублируется: при анимации отъезжает тот же компонент, что показан в дефолтном состоянии. Поэтому меню внутри маунтится при маунте всего компонента, скрывается за счет CSS
* Распахнутое меню прячется при клике вне или **любом** скролле
