# Контентный виджет страницы подписки/отписки

* Так как страницу подписки/отписки с живыми данными открыть непросто, существует опция `?mock=1`, тело мока лежит в файле `./mock.js`.
* Виджет отвечает как за подписку, так и за отписку. Семантически страницы сильно связаны, верстка в них очень похожа.
* При клике на кнопку "Отправить" ajax-запроса не происходит, только выстрел в метрику.
