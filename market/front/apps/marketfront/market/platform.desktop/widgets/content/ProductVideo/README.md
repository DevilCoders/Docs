# Виджет видеообзоры

Этот виджет используется на карточке товара. Имеет три темы:
* **mini** -- отображает одно (первое) видео и дает ссылку на остальные
* **expanded** -- копия обертки плеера YouTube: слева видео, справа список рекомендаций
* **boxed** -- как expanded, но имеет заголовок, подложку, а также серый цвет ссылки под плеером

Под собой использует другой виджет -- попап ComplainVideoPopup для отправки жалоб
