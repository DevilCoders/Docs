# Виджет для отображения карусели на точки входа на сравнения
 
## Описание директорий

* `containers/` -- контейнер для сниппета сравнения.
* `components/` -- компоненты используемые в виджете
    * `BooleanCharacteristic/` -- отображение булевых значений с иконками, немного отличается от аналогичной в большом автосравнении.
    * `ColorCharacteristic/` -- отображение кружочков с цветами, немного отличается от аналогичной в большом автосравнении.
    * `NumberCharacteristic/` -- отображение числовых параметров в виде графиков, немного отличается от аналогичной в большом автосравнении.
    * `CompareButton/` -- компонент для отображения кнопки на переход в большое автосравнение.
    * `SimpleCharacteristic/` -- отображение текстовы характеристик, немного отличается от аналогичной в большом автосравнении.
    * `ComparisonList/` -- Компонент для отображения списка характеристик.
    * `Header/` -- Компонент для отображения шапки сравнений с картинками и названиями товаров.
    * `Snippet/` -- Компонент для сниппета точки входа на автосравнение.
