# Виджет ResultsPaged

Обеспечивает отрисовку частей выдачи.

Главная задача этого виджета: при любом обновлении выдачи в рантайме уничтожить все виджеты SearchPartition, кроме главного (main).
То есть, по клику "Показать еще" будут уничтожены все виджеты кроме главного, а главная партиция дорисует себе недостающие сниппеты.
