# Попап "Пожаловаться" для виджета ProductVideo

Попап вызывается непосредственно из виджета ProductVideo. Имеет четыре состояния:
* Дефолт
* Выведено предупреждение, что ни одна из опций не выбрана, а попытка отправить жалобу совершена
* Выведено сообщение об успехе отправки
* Выведено сообщение об ошибке
