Это тест для новой формы жалобы (КМ/оффер/магазин) на реакте. По мере реактизации страниц, где вызывается виджет
platform.desktop/widgets/parts/ComplainFormPopup нужно применять этот тест сьют  
Разница в функцинале в том, что в новой форме есть кнопка "назад" (возвращает на шаг назад), 
а выбор причины жалобы происходит не по клику на ссылку,
а выбором радиобаттона и далее нажимаем кнопку "пожаловаться".

product.js - тестирует жалобу на КМ
shopAndOffers - тестирует жалобу на Магазин/Оффер
