## Visitors для выдачи.

Тут собраны visitors (посетители) для сграбливания данных с выдачи и ее сниппетов (с учетом разных вариантов отображения).

Страница выдачи Маркета: https://market.yandex.ru/catalog--topory/57713/list?hid=7757500

### Про visitors из этот папки

* `list-block-grid.js` - получение информации по всему блоку гридовой выдачи.
* `list-block-list.js` - получение информации по всему блоку листовой выдачи.
* `snippets-grid.js` - получение информации по сниппетам гридовой выдачи.
* `snippets-list.js` - получение информации по сниппетам листовой выдачи.
