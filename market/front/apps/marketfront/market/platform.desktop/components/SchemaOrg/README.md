# Компоненты для разметки с помощью schema.org

Offer
Пример использования:
```jsx
const Offer = require('@self/platform/components/SchemaOrg/Offer').default;

<Offer
    price="1337"
    currency="RUR"
/>
```

Rating
Пример использования:
```jsx
const Rating = require('@self/project/src/components/SchemaOrg/Rating').default;

<Rating
    ratingValue="4"
    worstRating="1"
    bestRating="5"
/>
```
