# Резолвер addPostVote

## Описание

Добавляет лайк или дизлайк посту

## Параметры

| Parameter | Type | Description | Default |
|---------------|----------|------------------|------------------|
| `postId` | number | Айди поста | - |
| `vote` | number | -1 - dislike, 1 - like | - |
