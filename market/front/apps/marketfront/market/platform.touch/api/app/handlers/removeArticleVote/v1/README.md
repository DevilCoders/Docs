# Резолвер removeArticleVote

## Описание

Резолвер, который позволяет удалить лайк/дизлайк на журнальную статью. Необходима авторизация.
В случае успеха отвечает пустым массивом.
Обязательные параметры:
- _articleId_ - айдишник статьи;
