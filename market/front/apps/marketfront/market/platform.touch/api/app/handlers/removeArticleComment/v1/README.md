# Резолвер removeArticleComment

## Описание

Резолвер, который позволяет удалить комментарий на журнальную статью. Необходима авторизация.
В случае успеха отвечает пустым ответом.

Обязательные параметры:
- _id_ - айдишник комментария;
- _articleId_ - айдишник статьи;

Необязательные параметры:
- _levelRootId_ - айдишник сущности, которая является корнем уровня, нужно для "схлопывания" комментариев в уровни. Если значение не передано, то рутом является исходная статья;
- _borderId_ - параметр, показывающий какой комментарий на уровне сейчас отображается последним, используется только если нужно после удаления подгрузить другие комментарии, чтобы не было "пустоты";

_Сущность user была переименована в publicUser. Для обеспечения обратной совместимости отправляются 2 коллекции - в старом и новом формате. В новом коде предпочтительно использовать новое имя сущности и коллекции, т.е publicUser. По возможности в старом коде также желательно выполнить миграцию._
