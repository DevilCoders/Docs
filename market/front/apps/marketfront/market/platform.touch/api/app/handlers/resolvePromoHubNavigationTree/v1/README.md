
### Хэндлер *resolvePromoHubNavigationTree*

Позволяет получить дерево навигации. По умолчанию возвращает корневой узел с промохаба (nid : 61522 )

- Парметры
    - `nid: integer` - не обязательный параметр, указывает на корневой узел дерева. Если не передан, то корнем является узел **61522**
