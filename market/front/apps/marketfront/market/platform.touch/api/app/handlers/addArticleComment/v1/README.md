# Резолвер addArticleComment

## Описание

Резолвер, который создает комментарий на журнальную статью. Необходима авторизация.
Можно отвечать на другие комментарии, таким образом создавать "дерево" комментариев. Комментарий, на который ответ, определяется параметром parentId.
Возращает созданный комментарий.
Обязательные параметры:
- _articleId_ - айдишник статьи;
- _text_ - текст комментария;
Необязательные параметры:
- _levelRootId_ - айдишник сущности, которая является корнем уровня, нужно для "схлопывания" комментариев в уровни. Если значение не передано, то рутом является исходная статья, и создаваемый комментарий будет первого уровня;
- _parentId_ - айдишник комментария, на который "ответ";
- _isReply_ - является ли создаваемый комментариев ответом на другой комментарий;
- _source_ - место создания комментария, например `feed`, нужно для разбивки на персовых графика;

_Сущность user была переименована в publicUser. Для обеспечения обратной совместимости отправляются 2 коллекции - в старом и новом формате. В новом коде предпочтительно использовать новое имя сущности и коллекции, т.е publicUser. По возможности в старом коде также желательно выполнить миграцию._
