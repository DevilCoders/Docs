# Виджет Избранного для добавления/удаления сущности

## Описание
Независимый виджет с разными view, но одинаковой механикой: toggle сущности (модель, оффер) в Избранном.
Самостоятельно устанавливает нужный статус при инициализации.


## Особенности
Для проверки статуса использует под капотом балковые резолверы,
поэтому для каждой сущности допустимо инициализировать свой инстанс.
