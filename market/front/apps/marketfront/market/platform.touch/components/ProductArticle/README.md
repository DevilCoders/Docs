Отвечает за отображение статьи из внешнего источника.

```jsx
<ProductArticle name="Название статьи" resource="ixbt.com" url="#" />
```

### Параметры

#### Свойство `name`

Отображаемоке название статьи

#### Свойство `resource`

Источник статьи для иконки. Загружается отсюда: `//favicon.yandex.net/favicon/{resource}`

#### Свойство `url`

Ссылка на статью
