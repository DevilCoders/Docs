### Описание

Компонент для отображения списка того, что нравится, основа для компонента `ReasonsToBuy`

### Примеры

```jsx
<div>
    <ReasonsToBuyList list={['довольно-таки неплохо', 'очень приятно']} />
</div>
```

#### Свойство `prefix`

```jsx
<div>
    <ReasonsToBuyList prefix="Покупателям не нравится" list={['разврат', 'наркотики']} />
</div>
```
