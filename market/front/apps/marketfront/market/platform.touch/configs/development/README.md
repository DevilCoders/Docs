###Конфигурация development окружения Маркета
-----------------------------

####Переменные окружения, с группировкой по конфигам:

  + `luster.conf.js`
    - `NODE_DEBUG_PORT` Номер порта, на котором будет поднят debugger

  + `stout.js`
    - `NODE_PORT` Номер tcp порта(или путь на файловой системе для UNIX Socket) где будет ожидать соединения наше приложение

  + `node.js`
    - `SKIP_TVM` Пропустить инициализацию TVM. Требуется для корректного импорта роутов в автотестах.

  + `ginny.conf.js`
    - `GINNY_CONFIG_PATH` Путь к конфигурационному файлу [Ginny](https://github.yandex-team.ru/market/ginny).
