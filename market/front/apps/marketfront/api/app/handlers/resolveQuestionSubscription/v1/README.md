# Резолвер resolveQuestionSubscription

## Описание

Резолвер, который позволяет получить значение подписки юзера на новые ответы на вопрос.
Если подписки нет, то возвращается пустой ответ.
Необходима авторизация.

Обязательный параметр:
- _questionId_ - айдишник вопроса;
