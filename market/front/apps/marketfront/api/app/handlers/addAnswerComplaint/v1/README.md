# Резолвер addAnswerComplaint

## Описание

Резолвер, который создает жалобу на ответ.
Обязательные параметры:
- _answerId_ - айдишник ответа;
- _reasonId_ - айдишник причины жалобы, возможные значеня:
  - 1 - Другое
  - 2 - Спам или реклама
  - 3 - Оскорбтельное высказывание
Необязательные параметры:
- _text_ - сообщение жалобы, ограничение 2000 символов;
