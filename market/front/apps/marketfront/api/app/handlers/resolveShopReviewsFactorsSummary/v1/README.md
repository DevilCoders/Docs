# Резолвер resolveShopReviewsFactorsSummary

## Описание

Резолвер, возвращающий сущность shopReviewFactor:
1. факторы на магазин - оценки на разные факторы магазина
2. % рекомендаций - сколько % людей рекомендуют этот магазин
Необходимо задать параметр - `shopId` или `businessId`
