### Описание

Здесь находится конвертация в старый формат `showPlace`.
Новая сущность называется `offerShowPlace`, отличается переименованием поля `offer` -> `offerId`.

А так же в коллекции `VisibleEntity` для шоуплейсов `referenceEntity` имеет значение `'showPlace'` и поле с айдишником называется `showPlaceId`.

Этот костыль надо будет закопать в рамках [BLUEMARKET-12490](https://st.yandex-team.ru/BLUEMARKET-12490), когда потребители фапи перейдут на новые форматы.
