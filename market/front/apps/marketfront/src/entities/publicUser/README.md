# Сущность user

Это декларация сущности publicUser, которая представляет блэкбоксовского юзера, **НЕ ТЕКУЩЕГО** юзера.
Эти юзеры используются как внешние юзеры, авторы контента (отзывы, вопросы и тд).
Текущий юзер описан в мандреле, его можно получить из контекста с помощью мандрелевского метода `getStoutUser` или проектного резолвера `resolveCurrentUser`.
