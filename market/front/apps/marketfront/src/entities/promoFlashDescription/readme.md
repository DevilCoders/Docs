#### Сущность promoFlashDescription

Представляет собой сериализацию описания акции из `Loyalty`.

#### Формат

```javascript
const promoFlashDescription = {
    // shopPromoId акции из YT
    shopPromoId: 'flash_autoland_01',
    feedId: 0,
    promoKey: '3vsqZd9WQ4otZDP2mYU-tg',
    description: '',
    // Ссылка на страницу с правилами
    url: 'https://pokupki.market.yandex.ru/special/blue-flash?shopPromoId=flash_autoland_01',
    promoId: 95738,
    // статус акции
    status: 'ACTIVE',
    // Дата начала акции
    startDate: '01-11-2020 00:00:00',
    // Дата завершения акции
    endDate: '21-03-2021 00:00:00',
}
```
