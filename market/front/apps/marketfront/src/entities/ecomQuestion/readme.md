Формат по вопросам как происходит:

На бекенде есть дерево вопросов, по которому проводим пользователя, и пушим ответы в ручку front_events. 
 
Этапы запрсов
1) запрашиваем вопрос
2) на события показа вопроса и ответа на него пушим в front_events
3) запрашиваем след. вопрос, спрашивая ту же ручку что в 1м пункте запрашивали (в нее докидывая previous_answer)

Такое разделение запрсоов (во 2м пункте в front_events) нужно для скорости прорастания ответа в таблицы по которым рекомендации можем учитывать (за 1-2 секунды ответ прорастает примерно).
