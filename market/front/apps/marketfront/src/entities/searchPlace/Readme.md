# Кастомизация выдачи

`searchPlace` -- искусственный параметр, цель которого уметь кастомизировать выдачу.

- Позволяет создавать несколько выдач (групп поисковых виджетов) на одной странице
- Позволяет синхронизировать запросы за конкретной выдачей

Изначально документировался, как "место расположения выдачи", отсюда и возникло название.

## Требования

- searchPlace не про параметры из поисковой строки
- searchPlace нужно определять асинхронно
- searchPlace нельзя комбинировать: на выдаче не могут одновременно жить два серч плейса
- searchPlace про продуктовые фичи
