#### Сущность promoBundleDescription

Представляет собой сериализацию описания акции из `Loyalty`.

#### Формат

```javascript
const promoBundleDescription = {
    // shopPromoId акции из YT
    id: 'altuninf_test5555',
    // текстовое описание
    description: '',
    startDate: '01.07.2019',
    endDate: '31.07.2022',
    // для акций m=n тут лежит n
    quantityInBundle: 2,
};
```
