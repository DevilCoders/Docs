# Сущность businessOrderDocument

## Используется только для m2b заказов (черный маркет)

collection: businessOrderDocument

Коллекция заполняется данными из бэка бабуси(marketb2bclients) из ручек с документами

Сваггер бубуси http://marketb2bclients.tst.vs.market.yandex.net/swagger-ui/index.html модель DocumentDto

- Формат документов отличается от обычных чекаутерных orderRecepit.
- businessOrderDocument используется только для m2b заказов!
