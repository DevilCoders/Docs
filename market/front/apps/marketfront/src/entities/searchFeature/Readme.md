# Кастомизация выдачи

На фронте существует несколько кастомных выдач, которые отличаются особым образом.
Цель этой фичи -- собрать воедино всю логику таких выдач.

В будущем можно легко унести все конфиги в CMS, так функционал можно будет легко редактировать из GUI.

## Правила по организации кода

- Нельзя кастомизировать виджеты выдачи, основываясь на контексте выдачи изнутри виджета:
виджет не должен знать, что он находится на странице хаба отзывов. Вместо этого он должен знать,
как он может быть кастомизирован снаружи пропсами.
- Каждая фича состоит из условия (CONDITION), cms-конфига настройки (CMS_CONFIG) и json-лэйаута (CMS_LAYOUT).
Пожалуйста, не импортируйте их из фич, а только из общих мап (типо `{[searchFeature]: CMS_CONFIG}`).
