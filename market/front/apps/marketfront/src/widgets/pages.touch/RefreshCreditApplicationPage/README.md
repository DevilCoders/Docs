Из-за невозможности быстро развернуть отдельный сервис для MARKETFRONT-72243, было принято решение создавать отдельную страницу маркета, а затем перенести ее в отдельный сервис по задаче MARKETFRONT-86121.

# Описание

Данная страница необходима для организации работы с привязкой номера телефона через страницу паспорта
при работе Кредитного Брокера в desktop/top. Страница паспорта открывается в отдельном окне браузера, после
привязки номера или ухода со страницы паспорта, путем клика по кнопке "Назад" формы привязки номера, будет
редирект на данную страницу, которая пошлет postMessage сигнализирующий о том, что работа формы паспорта окончена, и пользователя нужно продолжить вести по флоу брокера, Кредитный Брокер ловит данное сообщение, закрывает окно с паспортом и продолжает свой флоу
