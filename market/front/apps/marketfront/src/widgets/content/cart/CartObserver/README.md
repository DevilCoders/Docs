# CartObserver

Виджет, нужен для работы с такими событиями на странице в корзине, реакция на которые должна происходить вне зависимости
от набора виджетов, пришедших из CMS.

Идеальным решением было бы не создавать этот виджет и делать всю работу в виджете-layout-е, но
он статический, туда нельзя добавить эпики.
