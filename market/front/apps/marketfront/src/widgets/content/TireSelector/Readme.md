# Подборщик шин

- апиаризованный аналог устаревшего видждета из десктопа
- исторически под результатами хотели показывать кнопку, которая бы открывала попап с инструкцией про шины, но эта задача потерялась
- выглядит, как четыре селектора (контролы) и сгруппированные результаты
- результаты можно выбирать, это будет менять фильтры выдачи
