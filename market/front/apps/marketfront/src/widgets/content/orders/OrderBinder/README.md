## OrderBinder - Виджет привязки заказа (заказов)

* Привязывает заказы по bindKey от неавторизованного пользователя к авторизованному;
* Резолверы виджета берут данные из кеша;
* Реальный первый вызов в странице заказов, до фетча заказов(чтобы список заказов был сформирован вместе с привязанными заказами);

Во вьюшке отображает статус об успешной/неуспешной привязке заказов:
* Успешный статус выводится если хотя бы один из заказов привязался;
* Сообщение об ошибке формируется в зависимости от статусов ошибок от бэкендов, либо статус берется из конкретного заказа.
