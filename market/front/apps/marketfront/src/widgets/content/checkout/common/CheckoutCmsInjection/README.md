# CheckoutCmsInjection

Виджет, позволяющий вставить настраиваемый через cms контент в некоторые места чекаута.

**Важно!** Виджет сам получает _resolveFullCheckoutPurchaseInformation_,
дочерние виджеты этого не делают. Связано с тем, что сложно прокидывать необходимые
параметры через CMS.

Этот виджет всегда должен сам выставлять все коллекции,
которые могут потребоваться дочерним виджетам.

Дочерние виджеты должны выставлять коллекции-заглушки в контроллере и
соответствующие редьюсеры.

## Места в чекауте

На данный момент есть четыре места:
- Под заголовком финальной страницы чекаута
- Под селектором типа доставки, тип доставки - курьер
- Под селектором типа доставки, тип доставки - ПВЗ
- Под селектором типа доставки, тип доставки - почта
- Под селектором типа доставки, тип доставки - цифровая доставка
- Под селектором типа доставки, тип доставки - Курьером по вызову (@expFlag Доставка Лавкой, BLUEMARKET-11274)
