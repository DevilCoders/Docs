# CheckoutParcels

## Что это?

Виджет сгрупированных посылок.

- Логика слияния/разбивки корзин по опциям заказа
    > Причем только визуальная разбивка, не влияющая на сами данные

## Что внутри?

На основе ответа актуализации отрисовывает слоты для сгруппированных посылок.

_Прим._ это происходит не совсем очевидно, из-за фронтовой консолидации посылок.
За аксиому принимаем, что сгруппированных посылок не может быть больше чем корзин нам вернулось из актуализации,
соответственно в контролллере создаем слот под каждую "возможную" группу. Но во вьюхе рендерим только "активные" группы посылок.

### CheckoutParcel

Виджет для посылки/корзины.

### FailureInfoPopup

Компонент для отображения попапа с _протухшими_ товарами после чекаута (попытки купить товары).
