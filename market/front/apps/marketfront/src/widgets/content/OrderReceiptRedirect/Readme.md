### Виджет редиректа на чек заказа

Виджет редиректит на страницу pdf чека по идентификатору заказа.
Виджет подключается в OrderReceiptRedirectPage, куда можно попасть по ссылке "Посмотреть чек" после оплаты заказа.

#### Механизм работы
По orderId выбирается превый чек с типом INCOME
- Если чек найден - редирект на страницу чека (требуется id чека)
- Если чек не найдем - ошибка
  - Не найден не один чек по заказу (ReceiptsNotFoundError)
  - Чеки найдены, но не найден ни один INCOME чек (ReceiptIncomeNotFoundError)
