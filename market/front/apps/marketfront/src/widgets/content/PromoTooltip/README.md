# Промо тултипы в шапке

Стараемся собрать все тултипы сервиса в этот виджет. Сейчас тут все тултипы промоирующие Плюс.
Т.к. в файлах еще есть импорты из Маркета, то флоу покупок не проходит, поэтому где-то оставлены файлы .market.js
Важно! На одной странице в рантайме должен быть только 1 тултип Плюса!

## Как добавить новый тултип
1. Добавь новый виджет в папку /tooltips по аналогии с существующими
   - используй общие функции из папки /common
   - данные "тултип уже показан" нужно обновлять в контроллере/эпике этого тултипа
1. Добавь логику подключения нового тултипа в контроллер в этой папке
   - этот котнтроллер однозначно из контекста определяет какой тултип должен быть показан на текущей странице
   - проверка "показывали ли уже тултип" тоже делается внутри этого контроллера
1. Добавь информацию в этот ридми
