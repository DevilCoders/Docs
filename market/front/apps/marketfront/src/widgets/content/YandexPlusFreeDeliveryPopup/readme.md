## Виджет "Доставка с Плюсом беслпатная"

###Что содержит
Это попап, он содержит информацию о бесплатной доставке по Плюсу в зависимости от региона пользователя.
Внутри виджета только логика самого попапа, все "вызовы" попапа лежат снаружи виджета.

###Где и как используется
Используется КМ Покупок и Маркета. 
Для НЕПлюсовиков, у которых при подключении Плюса доставка на оффер будет бесплатной, отображается ссылка. По ссылке открывается этото попап.

###Как подключить на страницу
1. Добавить слот с этим виджетом в нужный контроллер
2. Добавить в вьюху `@self/root/containers/FreeDeliveryWithPlusLink`
   - внутри уже вставляется слот виджета-попапа
   - отображается ссылка с прокинутым окликом для показа попапа
