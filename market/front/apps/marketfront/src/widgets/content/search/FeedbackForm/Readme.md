# Форма качества поиска

Виджет располагается под выдачей. По дефолту выглядит как две кнопки "Да"/"Нет" и ссылка "Искать 'текст' в Яндексе".

При клике "Да"/"Нет" отправляет запросы в ручку `api:search-feedback`. При клике на кнопку "Нет" показывает поле для ввода комментария, нажатие кнопки "Отправить" отправляет новый POST-запрос. Поле для ввода имеет валидацию.
