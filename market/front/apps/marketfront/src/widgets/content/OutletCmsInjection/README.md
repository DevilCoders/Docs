# OutletCmsInjection

Виджет, позволяющий вставить настраиваемый через CMS контент в некоторые места страницы c информацией о ПВЗ (`/outlets/<outletId>`)

## Места (place)

На данный момент есть одно место:
- Под контентом блока с условиями доставки для ПВЗ

См. `OUTLET_CMS_INJECTION_PLACE` в `'@self/root/src/entities/cms/constants/pages/outletCmsInjection';`

См. `./mapping.js`
