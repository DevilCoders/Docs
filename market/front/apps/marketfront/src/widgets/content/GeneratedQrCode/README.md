# GeneratedQrCode - Генерируемый QR-код

Отображает QR-код, формируемый яндекс.диском по урлу https://disk.yandex.net/qr/?clean=1&text={link}format=svg.

Где link - на шифруемая ссылка

## Даные

+ **size** - размер QR-кода в px;
+ **link** - Обычныая ссылка в формате строки: https://www.yandex.ru;
+ **distributionLinkProps** - данные из adjust или appmetrika для формирования ссылки на приложение;
+ **qrCodeText** - Текст который будет отображаться справа от QR-кода, по дефолту "*Наведите камеру на QR-код, чтобы скачать*";
