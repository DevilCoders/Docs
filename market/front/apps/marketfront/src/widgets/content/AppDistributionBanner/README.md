# AppDistributionBanner - Баннер дистрибуции приложений

# Виджеты

## AppDistributionBannerLoader - загрузчик баннера

Зачем? Страница спасибо за заказ не cms'сизированна и нет возможности вставить виджет AppDistributionBanner через cms.
Но в планах есть задача на перевод страницы на cms. Времено делаем Loader который будет грузить специальную разметку cms для баннера.

## AppDistributionBanner - виджет баннер с дистрибуцией мобильного приложения

# Данные

+ **qrCodeInfo** - Параметры для отображения QR-код ссылки. Поддерживает:
    - обычную ссылку (https://www.yandex.ru);
    - adjust ссылку (дока - tbd)
    - appmetrika ссылку (дока - tbd)
+ **distributionLinkInfo** - Параметры для отображения кнопки-ссылки для скачивания/открытия приложения. Поддерживает:
    - adjust ссылку (дока - tbd)
    - appmetrika ссылку (дока - tbd)
+ **title** - Строка, не обязательный параметр. Заголовок отображаемый в баннере.
+ **subtitle** - Строка, не обязательный параметр. Подзаголовок отображаемый в баннере. Строка
+ **bannerPicture** - Изображение(иконка), которая будет отображается в начале баннера.

**Важно, для корректного отображения баннера должно быть заданно либо qrCodeInfo либо distributionLinkInfo!**
