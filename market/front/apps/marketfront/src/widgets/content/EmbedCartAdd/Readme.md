# EmbedCartAdd

Виджет страницы добавления в корзину.

Показывается на внешних сервисах, например на серпе, при добавлении в корзину из колдунщика.

Подробнее на [вики](https://wiki.yandex-team.ru/market/pokupka/projects/users-2019/kontur-core/projects/cart-checkout-as-service/)

По умолчанию отсылает `postMessage` в родительскую страницу и ждет в ответ сообщение c `{event: 'addToCart'}`;
При получении такого валидирует `origin` на пренадлежность Яндексу, при успехе сохраняет
его в данные виджета и все остальные сообщения шлёт уже только этому `origin`-у.

Для дебага изпользуй параметр `embedDebug=1`.
