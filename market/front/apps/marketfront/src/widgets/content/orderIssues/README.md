# Попапы нотификаций по заказам

Виджет OrderIssueNotifications группирует попапы по заказам и управляет тем, какие из пришедших нотификаций будут
показаны и в каком порядке. Он добавлен в корневой виджет каждой платформы.

Попапы, которые тут есть:
- VerifyCancellationPopup — нотификации по отмене заказов по инициативе пользователя со стороны магазина
- CancellationRejectionPopup — нотификации по отказу в отмене магазином
- DeletedOrderItemsPopup — нотификации по удалению товаров из заказа
