# Виджет, который откладывает исполнение других виджетов

- После того как страница была загружена, каждый инстанс виджета `IdleLoader` сообщает, что готов быть загружен
- Запросы о загрузке балкуются как обычный поход за ленивыми виджетами, затем внутри каждого инстанса создается новый виджет

Зачем? Например, для попапов. Это MVP, не стоит относиться всерьез.
