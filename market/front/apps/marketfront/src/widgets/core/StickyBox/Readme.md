**StickyBox**

Под капотом — компонент StickyBox (`@self/root/src/components/StickyBox`).

Виджет для использования на CMS страницах функционала StickyBox для сторонних виджетов (например, ScrollBox).
Подключается добавлением в декларацию пропсы `props.stickyBox: {...stickyBoxProps}`.
Параметры можно посмотреть в описании типов у виджета и компонента.

Включается в контроллере CmsLayout, подробнее можно посмотреть там.
