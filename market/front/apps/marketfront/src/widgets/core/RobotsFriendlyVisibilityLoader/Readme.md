# Ленивые виджеты для роботов

Обертка вокруг VisibilityLoader для обработки роботом

## Синопсис

Исторически мы игнорируем кто к нам пришел: робот или человек, и всегда отдаем ленивый контент лениво для роботов.
Таким образом они его не видят.

Если мы хотим показать роботу виджет нелениво, эта обертка обработает такой кейс и вернет исходный виджет без обертки.

## Развитие

Если эта обертка покажет себя успешно, нужно сделать ее частью виджета VisibilityLoader.
