#Виджет проброса текстового запроса при переходе с рекламы на яндекс.маркет
// чтобы при перезагрузке страницы этот текст остался

https://st.yandex-team.ru/BLUEMARKET-9368

##Потребители
* поиск

##Описание работы
На страницах
* Категории
* КМ

На сервере получает квери параметр поискового запроса, шифрует его
и ставит в query 'q=...'

получение из реферера:
    - в случае перехода с рекламы из параметра 'q'
    - в случае перехода с маркета из параметра 'text'
при релоаде (понимает что релоад по referrer hostname='beru.ru')
    - смотрит на наличие query параметра

и на клиенте эпиком пробрасывает их в текущий url при помощи replaceState
чтобы при обновлении страницы они остались
