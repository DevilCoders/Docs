**Fink**

Буквально – "доносчик".
Полезен, чтобы добавить событие рендера страницы в метрику.

*Возможность для улучшения:*
Сделать возможность "честного" cia-visible родителя, растянув Fink по его размерам. Иначе событие может не триггериться,
потому что самого доносчика не видно. Сам Fink отправить на задний план
1. Убедиться, что финк будет реагировать в таком формате на visibilitySensor
2. Придумать флаг, который будет говорить финку занимать все пространство контента (например, следить за всем виджетом)
