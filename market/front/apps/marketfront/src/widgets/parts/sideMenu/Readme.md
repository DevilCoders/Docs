# Виджет бокового меню тача

Выполнен таким образом, чтобы быть максимально легким.

- Виджет `Entrypoint` подключает кнопку для активации меню
- Виджет `Index` рисует белую подложку и паранжу
- Виджет `Content` рисует контент меню

## Ленивая загрузка

- Виджет `Content` подключается лениво.
- Для роботов вообще меню не готовим.
- В отличие от десктопа готовим шторку заранее, делая запрос в api:lazy-render с клиента.
