# DistributionInCut

Виджет дистрибуционной врезки. Может использоваться как самостоятельный виджет на CMS-зированных страницах, или вставляться через **DisctributionLoader**, так же его основе можно создавать новые компоненты дистрибуции имеющие различную кастомную логику показа например **SmartBanner** или **Баннер с подскролом**.
Данный виджет имеет два различных вью
+ **MiniInCut** маленькая врезка, контент отображается в строку, содержит картинку, текстовую часть, части которой отображаются в столбец, коммуникативная часть для скачивания/перехода в приложение.
+ **HammerInCut** большая врезка, весь контент отображается в колонке, содержит картинку, текст и кнопку для скачивания/перехода в приложение.

# Данные

+ **qrCodeInfo** - Параметры для отображения QR-код ссылки. Поддерживает:
    - обычную ссылку (https://www.yandex.ru);
    - adjust ссылку
    - appmetrika ссылку (_желательно не использовать, так как планируется полный переход на **adjust** добавлено для обратной совместимости с текущим функционалом_)
+ **distributionLinkInfo** - Параметры для отображения ссылки дистрибуции
Поддерживает:
    - adjust ссылку
    - appmetrika ссылку (_желательно не использовать, так как планируется полный переход на **adjust** добавлено для обратной совместимости с текущим функционалом_)
    - обычную ссылку (https://www.yandex.ru);
    - идентификатор special страницы
+ **rulesLink** - Параметры задающие ссылку на правила акции.
+ **title** - Строка, не обязательный параметр. Заголовок отображаемый в баннере.
+ **subtitle** - Строка, не обязательный параметр. Подзаголовок отображаемый в баннере.
+ **message** - Html строка, не обязательный параметр. Содержит произвольную разметку информационного сообщения.
+ **bannerPicture** - Изображение(иконка), которая будет отображается в начале баннера. Поддерживает кастомизацию размера.
+ **isForceVisible** - Показ коммуникации при установленном приложении у пользователя: если данный флаг установлен, то виджет будет отображаться пользователю даже если у него установлено приложение.
+ **vieType** - Задает тип вью для виджета. На данный момент поддерживается:
    - MiniInCut
    - HammerInCut
+ **zoneName** - Имя зоны для метрики
+ **hasCloseButton** - Флаг регулирующий отображение кнопки скрытия виджета.
+ **isCloseAfterDistributionLinkClick** - Флаг позволяющий скрывать виджет после клика по ссылке дистрибуции.

**Важно: дял отображения MiniInCut должен быть задан хотя бы один из параметров qrCodeInfo или distributionLinkInfo. Для отображения  HammerInCut должен быть задан distributionLinkInfo**
