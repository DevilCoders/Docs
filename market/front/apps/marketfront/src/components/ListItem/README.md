Список элементов с чертой между ними

#### Пример

```jsx
<div style={{display: 'flex'}}>
    <div>
        <ListItem>
            ListItem1
        </ListItem>
        <ListItem>
            ListItem2
        </ListItem>
        <ListItem>
            ListItem3
        </ListItem>
    </div>
</div>
```
