### Адаптивная обертка для формы

## Описание

Обертка формы, адаптирует положение нижней границы внутреннего контейнера 
под высоту системной клавиатуры с учетом положения родителя относительно
нижней части окна браузера.

## Дескриптор

В основном это касается устройств под управлением ios, в частности браузер safari, 
где в отличие от chrome окно браузера не меняет своего размера, а сдвигает body 
"как есть" вверх на высоту системной клавиатуры, при этом все фиксированные элементы
становятся статичными. Помимо всего прочего, компонент отключает еще одну особенность 
ios safari это эластичный скролл у body из-за, которого может ехать верстка формы с
фиксированными компонентами.

## Параметры

### isShown

Параметр, для определения видимости родителя.

### isAnyChildrenGetFocus

Параметр, для определения момента получения фокуса у одного из полей формы.
Должен передаваться true в момент, когда одино из полей формы получило фокус.

### parentBottomOffset

Параметр, положения нижней границы родителя, вычитается из высоты клавиатуры.

### browserName и osFamily

Параметры, для определения рабочей платформы.
