### Добавляет div обертку с тенью и margin(опционально) 
#### Параметры

##### marginType === 'bottom'
отступ только снизу 12px
```jsx
<Separator marginType='bottom'>
```

##### marginType === 'top'
отступ только сверху 12px
```jsx
<Separator marginType='top'>
```

##### marginType === 'none' (по дефолту)
без отступов
```jsx
<Separator marginType='none'>
```

##### shadow === true (по дефолту), false - без тени
без отступов
```jsx
<Separator shadow=true>
```
