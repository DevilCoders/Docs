### Иконка дополнительной информации

```jsx
<div>
    <InfoOpener />
    {' '}
    <InfoOpener isActive={true} />
    {' '}
    <InfoOpener type="info" />
    {' '}
    <InfoOpener type="answer" />
</div>
```

#### По умолчанию
```jsx
<InfoOpener />
```

#### Активная иконка
```jsx
<InfoOpener isActive={true} />
```

#### Иконка с информацией
```jsx
<InfoOpener type="info" />
```

#### Иконка с вопросом
```jsx
<InfoOpener type="answer" />
```
