### Тултип появляющийся один раз

Сохраняет в куках факт своего появления, после чего не появится.
Для каждого такого тултипа можно задать свой идентификатор, по которому он будет запоминать отображался он или нет.


### Песочница
```jsx harmony
<TemporaryTooltip placeIdent="Стоит удалить куку, чтобы что-то увидеть">
    <TemporaryTooltip.Content>Some text or elements</TemporaryTooltip.Content>
    <TemporaryTooltip.Hint>Hint Content</TemporaryTooltip.Hint>
</TemporaryTooltip>

```
