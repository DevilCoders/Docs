# Clickable

Компонент, который делает элемент доступным с клавиатуры и умеет обрабатывать события клавиатуры (Enter/Space), вызывая функцию из `onClick`.

Виды отображения:
* кнопкой (по умолчанию)
* остальными элементами (если передали параметр `as`)

Пример использования:
```javascript
const Clickable = require('@self/root/src/components/Clickable').default;

<div>
    <p>Попробуйте пронавигироваться до элементов с помощью клавиатуры и нажать клавишу Space или Enter</p>
    <p>Дефолтное отображение кнопкой</p>
    <Clickable
        onClick={() => alert('Событие')}
    />
    <p>Отображение как div</p>
    <Clickable
        as="div"
        onClick={() => alert('Событие')}
    />
</div>
```
