Резолверы для работы со списками избранного. 

Помимо общего списка избранного, есть пользовательские списки

**addUserWishlist** добавляет пользовательский список

**addUserWishlistItems** добавляет новые элементы в пользовательские списки

**addWishlistItem** добавляет элемент в общий список

**deleteUserWishlist** удаляет пользовательский список

**deleteUserWishlistItems** удаляет элементы из пользовательского списка

**deleteWishlistItem** удаляет элемент из общего списка

**resolveUserWishlist** получает свойства пользовательского списка

**resolveUserWishlistItems** получает элементы пользовательского списка

**resolveUserWishlistItemsCount** получает количество элементов в пользовательском списке

**resolveUserWishlists** получает пользовательские списки (их свойства)

**resolveUserWishlistsCount** получает количество пользовательских списков

**resolveWishlist** получает элементы общего списка

**resolveWishlistCount** получает количество элементов общего списка

**updateUserWishlist** изменяет свойства пользовательского списка
