## Метрика виджетов cms

- названия директорий соответствуют названиям виджетов
- в директориях с виджетами располагаются файлы с тестами на метрику виджетов/компонентов
 
#### Структура файлов и их назначение

```$xslt
- indexPage.js             // Агригирует все тесты на виджеты для 1 страницы
                           // подготавливает данные для тестов (кадавр моки, установка селекторов, специфичные параметры для метрики)
                           // задаёт через meta ссылки на issue, т.к. тесты могут переиспользоваться
- scrollBox                // Директория с тестами метрики на виджет scrollbox
  - scrollBox.js           // Тесты метрики на виджет scrollbox
  - snippetInScrollBox.js  // Тесты метрики на компонент snippet, расположенный внутри виджета scrollbox
  - index.js               // Композиция тестов виджета и сниппета
                          
- schemas.js               // схемы для валидации метрик гарсонов

- ...                      // тесты на другие виджеты
```
 
- избегаем кастомных параметров схемы метрики, вроде заголовка виджета:
  
  `{title:String}` вместо `{title: "Стоит приглядеться"}`
