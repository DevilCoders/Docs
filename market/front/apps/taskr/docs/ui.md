# Taskr UI


![ui](taskr-deploy.png)


- `deploy` – очередь с задачами на деплой в *production*, требуют ручного подтверждения
- `<name>:check` - очередь задач на периодическую проверку актуальности данных для `<name>`
  > когда будет следующая проверка можно увидеть в табе `DELAYED`
- `<name>` – очередь задач на обновление данных в `testing`
  - задачи в этой очереди создаются и выполняются автоматически на основе `<name>:check`

> `<name>:check` и `<name>` могут содержать не более 50 последних завершенных или упавших задач

## Деплой

1. Перейти в очередь `deploy`
2. Таб `DELAYED`
3. Выбрать джобу, которую нужно задеплоить (например, `partner-tanker` для обновления Танкера в production)
4. Нажать на кнопку `Promote`
   ![promote](taskr-promote.png)
