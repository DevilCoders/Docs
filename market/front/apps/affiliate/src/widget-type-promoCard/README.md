# widget-type-promoCard

Виджет в виде карточки с информацией.
Его задача - рассказывать о партнёрских инструментах.
Появился в рамках задачи: https://st.yandex-team.ru/MARKETAFFILIATE-3653.

Договоренности с продактом по поводу этого виджета:
* В этом виджете не планируется ничего, кроме статических
текста-изображений-ссылок, ходить за данными никуда не потребуется; единственный план на далекое
будущее — ротация нескольких карточек с разным статическим контентом
* Эту карточку не потребуется показывать в виджете статистики заказов — там самостоятельная промо-карточка
* Нет продуктовых причин делать этот виджет частью виджета статистики
* В будущем было бы круто сделать так, чтобы у продакта была бы возможность менять/добавлять
карточки вне релизов виджетов, через CMS/Танкер/Бункер
