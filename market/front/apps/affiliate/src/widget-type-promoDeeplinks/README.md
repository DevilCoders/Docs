# widget-type-promoDeeplinks

Виджет в виде каталога публичных акций на Маркете.

Мы хотим в широком смысле растить GMV партнёрской сети, а он ощутимо завязан на разного рода партнёрах-агрегаторах промокодов, пишущих об акциях. Хотим дать им удобный инструмент для производства своих материалов, считаем, что в результате материалов станет больше у уже подключённых партнёров-промокодников, будет выше конверсия из триала в договор у новых промокодников, а также использование офферов из инструмента сделает материалы партнёров-контентников конверсионнее.
К тому же, разумеется, мы хотим экономить ресурсы на ручной сбор акций и рассылку их партнёрам.

Появился в рамках задачи [MARKETAFFILIATE-5529](https://st.yandex-team.ru/MARKETAFFILIATE-5529).
