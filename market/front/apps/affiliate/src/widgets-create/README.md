# widgets-create

Показывающий код. Содержит необходимые модули для встраивания виджетов на сайты партнёров.

В состав пакета входит так называемая "Виджетная крутилка", код для сбора метрики на сайте партнёра и т.д.

Показывающий код взаимодействует с виджетом через его интерфейс, а также отвечает за предоставление объекту виджета API функций, необходимых для взаимодействия с окружением.

P.S.:
В данном пакете не должно быть кода, отвечающего за реализацию конкретного виджета.
