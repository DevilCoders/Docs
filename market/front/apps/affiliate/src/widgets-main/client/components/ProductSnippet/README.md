# ProductSnippet

Компонент-шаблон для отображения данных о продаваемом товаре.

Компонент является абстрактным и используется как предустановленный шаблон разметки внутри других компонентов (например, Offer или Model).

### Type: N1

Особенности:
- Ячейка слайдера

```
╔═══════════════════════════════╗
║ shopName                      ║
╠═══════════════╦═══════════════╣
║ price         ║        button ║
╚═══════════════╩═══════════════╝
```

### Type: N2

Особенности:
- Рейтинг перемещается в отдельный столбец при большой ширине строки

```
╔══════════════╦═══════════════════════════════════╦═══════════════════╦═══════════╦════════╦═══════════╦═══════════════╗
║              ║  shopName                         ║                   ║           ║        ║           ║               ║
║ ?warningIcon ╠═══════════════════════════════════╣ ?rating ?opinions ║ ?delivery ║ ?color ║ ?discount ║ ?price|button ║
║              ║  ?shopRating ?shopReviews         ║                   ║           ║        ║           ║               ║
╚══════════════╩═══════════════════════════════════╩═══════════════════╩═══════════╩════════╩═══════════╩═══════════════╝
```
