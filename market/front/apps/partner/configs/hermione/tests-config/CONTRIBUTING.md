# Добавление скипа

открываешь файл [skipped.js](https://github.yandex-team.ru/market/partner-hermione-tests-config/blob/master/skipped.js)

нажимаешь на карандаш
![step1](./source/step1.png)

вносишь изменения, соблюдая структуру из [README.md](./README.md)

В описании изменений пишешь:

(номер тикета): краткое описание

`MPARKETPARTNER-1337: скип каких-то тестов`

выбираешь опцию "Create a new branch" и заполняешь название ветки

feature/(номер тикета)-краткое-описание

`feature/MARKETPARNTER-1337-skip-checkout`

![step2](./source/step2.png)

дальше нажимаешь `propose file change`

тебя редиректнет на страницу "Open a pull request"

там нажимаешь `create pull request`
