# Методы страницы юридической информации

* Получение юридической информации о магазине [GET /organizationInfo][1];
* Сохранение юридической информации о магазине [POST /organizationInfo][2];
* Получение типов организации по региону [GET /organizationInfo/types][3];
* Получение юридической информации из базы СПАРК [GET /organizationInfo/spark][4];
