# Message methods

* [getMessageThemes] - Получить список доступных тематик уведомлений
* [getMessageHeaders] - Получить информацию для пейджинга по сообщениям и темы сообщений на данной странице
* [getMessage] - Получить сообщение
* [updateLastReadDate] - Обновить в базе id последнего прочитанного сообщения
