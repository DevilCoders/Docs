# ImageUploader (загрузчик изображений)

## Общее описание

Загрузчик изображений.

## Пропсы

| Наименование                 | Тип                       | Описание                                              | Обязательность |
| ---------------------------- | ------------------------- | ----------------------------------------------------- | -------------- |
| imageUrl                     | строка                    | ссылка на текущую картинку                            | -              |
| onChange                     | (url?: string) => void;   | колбэк, который дернется, при смене ссылки            | -              |
| onError                      | (error?: unknown) => void | колбэк, который дернется, при ошибке загрузки         | -              |
| uploadApiRoute               | строка                    | имя ручки (страницы)                                  | +              |
| uploadApiRouteParams         | Record<string, unknown>   | дополнительные параметры, которые передадутся в ручку | -              |
| paddingX                     | число                     | внутренний отступ по горизонтали                      | -              |
| paddingY                     | число                     | внутренний отступ по вертикали                        | -              |
| uploadButtonText             | ReactNode                 | текст кнопки загрузки                                 | -              |
| deleteButtonText             | ReactNode                 | текст кнопки удаления                                 | -              |
| requiredImageDimensionsLabel | ReactNode                 | текст требуемого разрешения изображения               | -              |

## Используемые бэкенды

Используемый бэкенд передается в поле uploadApiRoute
