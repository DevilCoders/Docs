# SummaryPromotionContract (Договор на продвижение)

<img src="https://media.github.yandex-team.ru/user/11807/files/ff2e6680-e018-11eb-86bc-703c5049ef28" width="300">

## Общее описание

Виджет, предупреждающий о необходимости подать документы для получения вознаграждения по [продвижению Маркета](https://yandex.ru/support/marketplace/accounting/payment.html).

Здесь появится информация о вознаграждении за скидки, которые партнёр предоставляет по МаркетБонусам и бонусам «Спасибо» от Сбера, если товары оплатят бонусами.

| Права               | Поля из AppState | Модели   | Фичи кокона |
| ------------------- | ---------------- | -------- | ----------- |
| Доступен всем ролям | `campaignId`     | FBX, DBS | -           |

### Состояния

У виджета бывает четыре состояния:

-   `PENDING`: От партнёра требуется прислать закрывающие документы до 15 числа текущего месяца.
-   `CLOSED`: Партнёр прислал документы вовремя в текущем месяце
-   `OBSOLETE`: Партнёр не успел прислать документы вовремя
-   `--`: За месяц партнёр не оказывл услуги продвижения Маркета, и отображается сообщение о том, что "здесь что-то будет".
