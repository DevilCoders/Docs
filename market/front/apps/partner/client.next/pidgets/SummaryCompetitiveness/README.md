# SummaryCompetitiveness (Конкурентность цен)

<img src="https://jing.yandex-team.ru/files/ekhmelev/Screenshot%20at%20Jun%2023%2013-07-05.png" width="500">

## Общее описание

Виджет отображает индекс конкурентности цен магазина. Используется на единой сводке магазинов, на единой сводке бизнеса, в отчете "Цены конкурентов"

| Права               | Поля из AppState | Модели  | Фичи кокона             |
| ------------------- | ---------------- | ------- | ----------------------- |
| Доступен всем ролям | -                | FBX/DBS | -                       |

## Используемые бэкенды
