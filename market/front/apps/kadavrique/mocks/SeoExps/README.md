# SeoExps

## Данные в стейте

```js
{
    "seoExps": {
        "modifications": ModificationsValue
    }
}
```
где `ModificationsValue` - массив, который содержит эксперименты

## Метод modifications
Возвращает массив экспериментов, которые запущены для этой платформы, сервиса, коллекции и его айди
