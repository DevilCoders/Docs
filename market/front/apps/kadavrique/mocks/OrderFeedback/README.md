# OrderFeedback

Бекенд отзывов на заказы

Основные сущности:
- Отзыв на заказ
- Вопросы, которые можно задать про конкретный заказ
  - Набор вопросов зависит от оценки, которую выставил пользователь
  - Набор вопросов зависит от самого заказа (ПВЗ или курьер, предоплата или постоплата, и т.д.)
- Сценарии, по которым мы ведем пользователя после получения ответов
