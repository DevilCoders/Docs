# Carter

Данные в стейте:

```
{
    "Carter.items": [
        CartItem,
        ...
    ]
}
```
где `CartItem` - элемент корзины.
На обязательность его полей завязки нет.

Все методы работают с `Carter.items`, при отдаче данные обогащаются необходимыми полями (id, creationTime и т.д.)
При модификации (например, изменение количества или удаление) модифицируются данные стейта.

**Принятые допущения:**
 * нет никакой работы с пользователем, считаем, что в рамках одной сессии одна корзина
 * вся работа происходит только со списком `'BASKET'`
 * нет обработки ошибок и вообще ошибочных ответов, нужно будет дописать при необходимости
