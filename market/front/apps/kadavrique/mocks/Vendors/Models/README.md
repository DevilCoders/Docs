# Модели вендора
Ключ в стейте `vendorModels`.

**Структура стейта**
```js
    // Список моделей
    [
        {
            id: 1429703292,
            title: 'Cпортивный ремень',
        }
    ]
```

#### POST `/vendors/<vendorId>/models`

Возвращает список моделей с пагинацией.

##### URL параметры
- page `<number>` - номер страницы
- pageSize `<number>` - количество элементов на странице

##### Ответ
- result `<Object>` - результат работы ручки
    - item
        - items `<Object[]>`- список моделей
        - pager `Object` - объект пейджера
- errors `<Array[Object]>` - список ошибок
