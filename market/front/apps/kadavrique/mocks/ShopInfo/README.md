# ShopInfo

Данные в стейте:

```
{
    "ShopInfo.collections": {
        outletLegalInfo: {
            [OUTLET_ID]: {

            },
        },
        supplierInfo: {
            [SUPPLIER_ID]: {

            },
        },
    }
}
```

Никакие поля не являются обязательными.

Если юр.инфо по запрошенным идентифкаторам нет, то сгенерируются случайные данные по схеме.

Если есть, то существующая в стейте информация дополнится случайными данными по схеме


**Принятые допущения:**
* не предусмотрены ошибочные ответы
