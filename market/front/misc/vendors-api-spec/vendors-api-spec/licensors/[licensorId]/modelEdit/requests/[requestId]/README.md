# Ресурс /licensors/[licensorId]/modelEdit/requests/[requestId]

## GET /licensors/[licensorId]/modelEdit/requests/[requestId]

Получение заявки на создание/изменение модели. Работает аналогично [методу для вендора](/vendors/%5BvendorId%5D/modelEdit/requests/%5BrequestId%5D#get-vendorsvendoridmodeleditrequestsrequestid)

## PUT /licensors/[licensorId]/modelEdit/requests/[requestId]

Изменение заявки на создание/изменение модели по идентификатору заявки. Работает аналогично [методу для вендора](/vendors/%5BvendorId%5D/modelEdit/requests/%5BrequestId%5D#put-vendorsvendoridmodeleditrequestsrequestid)

### Примечания
Изменение бренда и категории в заявке на создание/редактирование модели ограничено теми брендами на которые имеет права лицензиар, и в теми категориями, в которых лицензиар имеет права на эти бренды.

## DELETE /licensors/[licensorId]/modelEdit/requests/[requestId]

Удаление заявки на создание/изменение модели по идентификатору заявки. Работает аналогично [методу для вендора](/vendors/%5BvendorId%5D/modelEdit/requests/%5BrequestId%5D#delete-vendorsvendoridmodeleditrequestsrequestid)
