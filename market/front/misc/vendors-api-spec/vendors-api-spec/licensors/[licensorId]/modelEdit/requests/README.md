# Ресурс /licensors/[licensorId]/modelEdit/requests

## POST /licensors/[licensorId]/modelEdit/requests

Создание заявки на создание/изменение модели. Работает аналогично [методу для вендора](/vendors/%5BvendorId%5D/modelEdit/requests#post-vendorsvendoridmodeleditrequests)

### Примечание
Создание заявки на создание/редактирование модели ограничено теми брендами на которые имеет права лицензиар, и в теми категориями, в которых лицензиар имеет права на эти бренды.

## GET /licensors/[licensorId]/modelEdit/requests

Получение списка заявок на создание/изменение модели. Список отсортирован (сначала свежие заявки). Работает аналогично [методу для вендора](/vendors/%5BvendorId%5D/modelEdit/requests#get-vendorsvendoridmodeleditrequests).
