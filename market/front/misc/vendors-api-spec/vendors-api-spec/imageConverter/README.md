## (!DEPERECATED!) Группа ручек

Конвертер изображений в BASE64 может использоваться фронтом для подготовки
юзерских данных к финальной отправке в виде обычного JSON'а.

## Поддерживаемые форматы загрузки:
 - [~~URL~~](/imageConverter/url)
 - [~~multipart~~](/imageConverter/multipart)

## (!DEPERECATED!)
Вся логика конвертации внешних изображений в base64 задепрекейчена!
Вместо этого должны использоваться ручки [хранилища файлов](/files/[serviceKey])
и передача файлов по ID.
