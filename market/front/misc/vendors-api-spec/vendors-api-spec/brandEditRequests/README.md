# Ресурс /brandEditRequests

## GET /brandEditRequests

Получение списка заявок на редактирование бренда

### Доступы
  - `vnd:moderation:read`

### Параметры
  - **uid** `<UID>` - обязательный параметр для аутентификации
  - status `<Array[String]>` - статусы заявок (если не указано ни одного - присылаются все)
  - text `<String>` - текст для фильтрации заявок (отбираются карточки, в которых либо название вендора регистронезависимо **содержит** этот текст, либо номер заявки **совпадает** с этим текстом)
  - [Параметры пагинации](/_entities/Pager.md#Параметры)

### Примечания
  - Параметр `text` сравнивается с полем `vendorTitle` в каждой заявке, регистр не имеет значения.

### Ответ
  <[CollectionResponse](/_entities/responses/CollectionResponse.md) [[BrandEditRequest](/_entities/brand-editor/BrandEditRequest.md)] >

  Присланные заявки в списке не содержат данных бренда (старых или новых) и статусов полей.
