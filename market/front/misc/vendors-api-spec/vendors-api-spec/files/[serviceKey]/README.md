## Группа ручек работы с файл-стораджем

Вендора хранят раличные файлы различных хранилищах, в зависимости от "услуги" (сервиса),
для которого загружаются эти файлы.

#### Возможные значения `[serviceKey]`:
- `documents` - документы оферты
- `brand-editor` - логотипы брендов для редактора
- `model-editor-images` - картинки моделей для редактора моделей
- `model-editor-feed` - загруженные вендором файлы с данными о моделях

#### Примечания
Файл должен загружаться с ключом именно того сервиса, в котором он потом будут использоваться.
Если загрузить файл в хранилище сервиса "А", и попытаться использовать айдишку в сервисе "Б" -
работоспособность не гарантируется (скорее всего, файл не будет найден).
