# Типы сущностей в API

##### Эта папка содержит описание некоторых типов данных и их свойств, используемых в API вендоров.

Довольно часто различные ручки могут принимать или возвращать логически-одинаковые типы данных,
возможно, с минимальными различиями в наличии отдельных полей.

Из-за этого гораздо удобнее описать типы этих сущностей отдельно, а в описании ручки уточнить любые
тонкости в её использовании.
