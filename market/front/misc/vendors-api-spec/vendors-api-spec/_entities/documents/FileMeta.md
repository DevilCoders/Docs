# Сущность "FileMeta"

Мета-информация о файле, загруженном на сервер

### Поля:
  - **id** `<String>` - идентификатор файла
  - **name** `<String>` - имя файла (под которым он был загружен на сервер)
  - **uploadedAt** [`<DateTime>`] - дата загрузки файла на сервер 
  - **size** `<Number>` - размер файла в байтах
  - **publicLink** [`<HtmlLink>`] - публичная ссылка на файл
   (Ссылка содержит полный публично-доступный URL фйла.
   В качестве текста ссылки всегда будет стоять содержимое поля `name`)

### Примеры:
```
{
  "uploadedAt": "2017-050-22T13:32:01Z",
  "name": "account_details_*7410.pdf",
  "id": "5c155ed6-b050-4981-b263-9a2cf0c7109b",
  "size": 49333
}
```
