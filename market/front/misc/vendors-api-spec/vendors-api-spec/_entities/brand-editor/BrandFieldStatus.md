# Сущность "BrandFieldStatus"

Данные о статусе отдельного поля в заявке на редактирование бренда.

### Поля:
  - status `<Boolean>` - одобрил ли модератор изменения по данному полю
  - comment `<String>` - комментарий модератора

### Примечания
  - Отсутствие поля `status` означает **отсутствие вынесенного решения**. Это и не "одобрил", и не "отверг".
  Значит модератор либо ещё не просмотрел предлагаемые изменения, либо не вынес конкретного решения.

### Примеры:
```
{
  "status": true
}
{
  "status": true,
  "comment": "Хороший логотип"
}
{
  "status": false,
  "comment": "Плохой логотип"
}
{
  "comment": "Не могу разобрать, хороший логотип, или плохой"
}
```
