# Сущность "Category"

Одна из категорий товаров на Маркете

### Поля:
  - id `<Integer>` - идентификатор категории
  - path `<Array[String]>` - краткий "путь" категории (не более 4-х последних элементов)
  - ~~name~~ `<String>` - _(deprecated)_ элементы `path` соединённые через слеш
  - outputType `<`[CategoryOutputType](/_entities/CategoryOutputType.md)`>` - тип выдачи категории
  - grouped `<Boolean>` - групповая категория или нет
  - canCreateSku `<Boolean>` - можно ли создавать SKU в категории

### Примечания
  - Поле `name` оставлено для обратной совместимости, его значение гарантированно равно элементам поля `path` склеенным по слешу. В дальнейшем поле будет удалено.
  - Коллекция `path` содержит не более 4-х последних элементов из пути категории (см. [VNDFRONT-306](https://st.yandex-team.ru/VNDFRONT-306))

### Примеры:
```
{
  "id": 42,
  "path": ["Все товары", "Бытовая техника для дома", "Пищевые репликаторы"]
  "name": "Все товары/Бытовая техника для дома/Пищевые репликаторы"
}
```
```
{
  "id": 115,
  "path": ["Все товары", "Безопасность и оборона", "Для дома и офиса", "Бытовые детекторы пожирателей"]
  "name": "Все товары/Безопасность и оборона/Для дома и офиса/Бытовые детекторы пожирателей"
}
```
```
{
  "id": 3654,
  "path": ["Безопасность и оборона", "Для детей", "Портативные излучатели", "Дисней"]
  "name": "Безопасность и оборона/Для детей/Портативные излучатели/Дисней"
}
```
