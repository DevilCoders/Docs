# Сущность "SkuMode"

Тип параметров моделей. На уровне модели остаются только параметры с SkuMode=NONE.

### Возможные значения:
  - `"DEFINING"` - определяющий параметр (в одной модели не может быть два SKU с одинаковым набором таких параметров)
  - `"INFORMATIONAL"` - информационный параметр (нет ограничений на уникальность значений)
