# Сущность "FeedValidationReport"

Отчёт по валидации VendorYML файлов

### Поля:
  - metrics `[`[`<FeedReportMetric>`](/_entities/model-editor/report/FeedReportMetric.md)`]` - общие метрики по всем переданным файлам
  - files `[`[`<FeedReportFile>`](/_entities/model-editor/report/FeedReportFile.md)`]` - отчёты по каждому из переданных файлов
  - createdTimestamp [`<DateTime>`](/entries/DateTime) - время создания отчёта
