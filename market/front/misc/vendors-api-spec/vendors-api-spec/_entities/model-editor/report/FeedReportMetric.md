# Сущность "FeedReportMetric"

Метрика отчёта по переданным VendorYML файлам

### Поля:
  - value `<Double>` - значение метрики
  - message [`<FeedReportMessage>`](/_entities/model-editor/report/FeedReportMessage.md) - сообщение с описанием метрики
  - unit `<String>` - единица измерения
  - formattedValue `<String>` - строковое представление значения
