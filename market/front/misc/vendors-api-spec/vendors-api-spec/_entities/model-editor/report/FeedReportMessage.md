# Сущность "FeedReportMessage"

Текстовое сообщение для отчёта по переданным VendorYML файлам

### Поля:
 - text `<String>` - текст сообщения
 - code `<String>` - код сообщения
 - textArguments `[<String>]` - текстовые аргументы
 - internalMessage `<String>` - служебное сообщение
 - row `<Integer>` - номер строчки в файле, к которой относится сообщение
 - column `<Integer>` - номер столбца в строке, к которому относится сообщение
