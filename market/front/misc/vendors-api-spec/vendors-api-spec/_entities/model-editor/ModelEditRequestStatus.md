# Сущность "ModelEditRequestStatus"

Статус заявки на создание/изменение модели

### Значения:
  - "STALLED_DRAFT" - устаревший черновик заявки (не менялся более 30 дней)
  - "DRAFT" - черновик заявки
  - "NEW" - заявка не отправлена 
  - "ACCEPTED" - заявка принята к исполнению
  - "PROCESSING" - заявка исполняется
  - "COMPLETED" - заявка исполнена
  - "REJECTED" - заявка отклонена
  - "NEED_INFO" - необходима дополнительная информация
  - "NEED_FIX" - необходимы исправления в заявке
  - "REJECTED_DUPLICATED" - заявка не принята (дубликат)
  - "REJECTED_UNKNOWN_CATEGORY" - заявка не принята (передана неизвестная категория)
  - "REJECTED_UNKNOWN_BRAND" - заявка не принята (передан неизвестный бренд)
