# Сущность "ModelImage"

Картинка модели 

### Поля:
- **url** `<String>` - ссылка на оригинальную картинку во внутреннем хранилище
- sourceUrl `<String>` - исходная ссылка на оригинальную картинку
- fileId `<String>` - идентификатор файла в MDS
- skuIds `[<Number>]` - идентификаторы SKU, привязанных к картинке (если отсутствует, то картинка относится к модели в целом)
