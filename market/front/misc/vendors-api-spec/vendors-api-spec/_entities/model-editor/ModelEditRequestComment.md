# Сущность "ModelEditRequestComment"

Комментарий к заявке на редактироване модели 

### Поля:
  - **text** `<String>` - текст комментария
  - **timestamp** `<Timestamp>` - время написания комментария
  - **direction** `<Number>` - направление комментария (0 - от вендора, 1 - от оператора)
  - author `<Number>` - идентификатор автора комментария
