# Статус применения ставки на модель

Статус применения ставки на модель

### Возможные значения:
  - `UNKNOWN` - статус применения ставки неизвестен
  - `IN_PROGRESS` - ставка пока не применена, ожидается применение (либо ставка еще не передана в подсистему индексации, либо еще не получен ответ)
  - `PARTLY_APPLIED` - ставка передана в подсистему индексации и частично применена
  - `APPLIED` - ставка применена
  - `ERROR` - ошибка применения ставки
