# Прогноз кликов и показам по категории вендора с учетом популярности его товаров

### Поля:
  - **id** `<Number>` - идентификатор категории
  - showsCountPerPosition `<Map<Number, Number>>` - ожидаемое количество показов по позициям
  - clicksCountPerPosition `<Map<Number, Number>>` - ожидаемое количество кликов по позициям
  - ordersCountPerPosition `<Map<Number, Number>>` - ожидаемое количество заказов по позициям
