# Код ошибки применения ставки на модель

Код ошибки применения ставки на модель

### Возможные значения:
  - `UNKNOWN` - неизвестная ошибка применения ставки
  - `NOT_ALLOWED` - ставка неприменима
  - `NOT_FOUND` - отсутствует товарное предложение
  - `WRONG_BID_VALUE` - некорректное значение ставки
