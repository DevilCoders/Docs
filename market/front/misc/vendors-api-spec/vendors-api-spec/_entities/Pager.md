# Сущность "Pager"

Параметры пагинации ответов

### Параметры
  - page `<Integer>` - номер страницы
  - pageSize `<Integer>` - размер страницы (число элементов на странице)

### Поля
  - **total** `<Integer>` - общее число элементов
  - **pages** `<Integer>` - число страниц
  - **page** `<Integer>` - текущая страница

### Примечания
- Если не переданы параметры `page` и `pageSize` - возвращается список без пагинации
  - Пейджер в выдаче при этом выглядит так: `pager: { total: <Integer>, pages: 1, page: 1 }`

### Примеры
```
{
  "total": 6,
  "pages": 3,
  "page": 2
}
```
```
{
  "total": 6,
  "pages": 1,
  "page": 1
}
```
