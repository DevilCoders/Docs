# Сущность "ModelQuestionComment"

Объект комментария к ответу на вопрос по товару

### Поля
  - **id** `<Number>` - идентификатор комментария
  - **text** `<String>` - текст комментария
  - **updateTime** [`<DateTime>`](/_entities/DateTime) - время последнего изменения комментария
  - **canEdit** `<Boolean>` - признак возможности редактирования/удаления комментария
  - **postedByVendor** `<Boolean>` - признак того, что комментарий оставлен вендором
  - user [`<User>`](/_entities/User) - автор комментария
