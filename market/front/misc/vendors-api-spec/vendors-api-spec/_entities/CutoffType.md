# Сущность "CutoffType"

Тип отключения, навешенного на кампанию вендора.

### Возможные значения:
  - `"CLIENT"` (Отключено по решению клиента)
  - `"ADMIN"` (Отключено по решению менеджера)
  - `"FINANCE"` (Отключено по нехватке средств)
  - `"OFFER_DOCUMENTS"` (Отключено по истечению сроков активности документов оферты)
  - `"POSTPONED"` (Размещение отложено)
