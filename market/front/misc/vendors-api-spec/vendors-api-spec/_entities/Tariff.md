# Сущность "Tariff"

Информация о тарифе платной услуги

### Поля:
  - **id** `<Number>` - идентификатор тарифа
  - **name** `<String>` - название тарифа
  - starts `<DateTime>` - время начала работы тарифа для вендора (не задано для текущего) - *тариф сейчас отдаётся либо один (текущий), либо два - текущий и следующий*
  - **details** `<Object>` - описание тарифа
    - **minShops** `<Number>` - минимальное кол-во магазинов, предусмотренных тарифом (только для услуги "Рекомендованные магазины")
    - maxShops `<Number>` - максимальное кол-во магазинов, предусмотренных тарифом (только для услуги "Рекомендованные магазины")
    - **cost** `<Number>` - стоимость тарифа в центах
    - **periodType** `<String>` -  тип тарификации. "day" - ежедневная, "month" - ежемесячная
