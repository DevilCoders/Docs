# Ресурс /tariffs/recommended

## GET /tariffs/recommended

Получение списка тарифов услуги "Рекомендованные магазины"

### Параметры
  - **uid** `<UID>` - сейчас необязательный параметр из-за [бага](https://st.yandex-team.ru/VNDMARKET-53)
  - vendorId `<Integer>` - идентификатор вендора. Если параметр заполнен, метод возвращает только разрешенные для вендора тарифы. Для контрактных вендоров - с ежемесячной тарификацией, для принявших оферту - с ежедневной тарификацией.

### Ответ
  - **meta** `<Object>`
  - result `<Object>`
    - **item** `<Object>`
      - **tariffs** `<Array[`[Tariff](/_entities/Tariff.md)`]>` - список доступных тарифов услуги "Рекомендованные магазины"
  - errors `<Error[]>`

### Ошибки
#### [Стандартные](https://github.yandex-team.ru/market/vendors-api-spec#Стандартные-ошибки)
