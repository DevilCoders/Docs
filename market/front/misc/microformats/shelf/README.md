# Shelf

Подложка с группой кликабельных элементов с картинкой и текстом.

## Параметры
- **align** `<enum>` – положение баннеров – обязательный параметр
    - `left` – слева
    - `right` – справа
- **picture** [`<picture>`](https://github.yandex-team.ru/market/microformats/tree/master/picture) – фоновое изображение – обязательный параметр
- **formulas** [`<formula[]>`](https://github.yandex-team.ru/market/microformats/tree/master/formula) – список формул – обязательный параметр
- name `<string>` – название
