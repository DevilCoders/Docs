# Микроформат блока filter
Блок содержит в себе информацию о фильтре
В контексте оффера блок содержит информацию о параметрах для данного оффера.

## Описание параметров
`subType` — это подтип, используется в фильтре по цвету (subType="color")  
`found` — это количество результатов, которые найдутся если чектуть этот фильтр с учетом уже примененных фильтров  
`initialFound` — это количество результатов, которые найдутся если чектуть этот фильтр **без** учета уже примененных фильтров  
`hasBoolNo` — наличие hasBoolNo в фильтре означает что должно быть значение "Неважно". При зажатой галке (у булевых фильтров) отправляется в репорт значение 1, при отжатой - не отправляется ничего
