## Разработка

- [ ] Сгенерировать свои [SSH ключи](https://wiki.yandex-team.ru/diy/macos/ssh/#generacijaparykljuchejj), [документация для linux](https://wiki.yandex-team.ru/diy/linux/ssh/#generacijaparykljuchejj)
- [ ] Положить свои SSH ключи на [staff](https://staff.yandex-team.ru)
- [ ] Подложить свои SSH ключи на [github](https://github.yandex-team.ru/settings/ssh)
- [ ] Создать [~/.ssh/config](https://paste.yandex-team.ru/4355338)
- [ ] [Настроить в ssh Agent Forwarding](https://wiki.yandex-team.ru/diy/macos/ssh/#ispolzovanie)
- [ ] Настроить любимый редактор / IDE:
    - VS Code
    - webstorm [лицензия](https://wiki.yandex-team.ru/jetbrains/LicenseServers/)
- [ ] Указать логин и почту в git config выполнив 2 команды
    - `git config --global user.name "$(whoami)"`
    - `git config --global user.email "$(whoami)@yandex-team.ru"`
- [ ] Убедиться что состоишь в github организации [market](https://github.yandex-team.ru/market), если нет, то попросить руководителя добавить
- [ ] Ознакомиться с файловой структурой проекта [wiki](https://wiki.yandex-team.ru/users/tzota/market/tutorials/pi-partnernode-file-structure/)
- [ ] Дизайн система - [levitan](https://levitan.yandex-team.ru/)
