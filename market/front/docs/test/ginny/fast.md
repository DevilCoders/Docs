### Как не ждать запуска тестов по 5 минут

1) Уточняем директорию конктерного теста в конфиге, меняя общий паттерн на конкретную страницу.
Например, `cart`.

```js
hermione: {
    sets: {
        common: {
          files: 'spec/hermione/test-suites/tops/pages/n-page-cart/*.hermione.js',
        },
    },
}
```

В таком случае не будет строится граф всех зависимостей и вызываться `require` всех модулей.

2) Для удобного запуска конкретного теста пользуется опцией `fgrep`.
Например, для теста: `Кредиты в корзине`

```js
hermione: {
    system: {
        mochaOpts: {
            frgrep: 'Кредиты в корзине'
        }
    }
}
```

3) Для удобного дебага лучше выставить `timeout` побольше:
```js
hermione: {
    system: {
        mochaOpts: {
            timeout: 60000 * 10
        }
    }
}
```
