# Testament

[Testament](https://a.yandex-team.ru/arc_vcs/market/front/libs/testament) - платформа для написания виджетных тестов (интеграционных тестов для виджетов) в Яндекс.Маркете
Testament позволяет писать интеграционные виджетные тесты, как обычные unit-тесты на [jest](https://jestjs.io/docs/).

## Цели Testament

- Максимальное (в идеале 100%) покрытие виджетов интеграционными тестами. **Каждому виджету по тесту!**
- Перенос нагрузки с "тяжелых" hermione тестов на более легкие Testament тесты
- Движение к зеленому транку в общем стеке тестирования. Тесты testament быстрее и стабильнее чем hermione.
