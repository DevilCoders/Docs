## Что такое выдача

Выдача — система виджетов, решающая задачу поиска товаров на Маркете. Пример: [https://market.yandex.ru/catalog--mobilnye-telefony/54726/list?hid=91491&text=iphone&was_redir=1&rs=eJwzYgpgBAABcwCG&rt=10&nid=54726&cpa=1](https://market.yandex.ru/catalog--mobilnye-telefony/54726/list?hid=91491&text=iphone&was_redir=1&rs=eJwzYgpgBAABcwCG&rt=10&nid=54726&cpa=1).

Страница выдачи — одна из основных страниц фронта Маркета. На выдачу можно попасть, используя поисковую строку в шапке.
