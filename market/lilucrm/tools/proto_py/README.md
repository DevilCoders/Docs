# Инструмент для сборки proto-файлов в python модули

Простая утилита, предназначенная для сборки произвольного proto-файла и его зависимостей в модуль (модули) пригодный для использования в скриптах на python 2.7. Для работы требует установленного *protoc*. 

Подробности на [wiki](https://wiki.yandex-team.ru/users/apershukov/Protopy/)
