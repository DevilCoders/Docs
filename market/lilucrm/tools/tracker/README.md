# Утилита для работы со startrek

## Перед использованием
- Получить oauth-токен для доступа к трекеру, перейдя по [ссылке](https://oauth.yandex-team.ru/authorize?response_type=token&client_id=5f671d781aca402ab7460fde4050267b).
- Собрать утилиту, выполнив `ya make` в её директории.
- После сборки в той же директории появится бинарь `st`, готовый к запуску. Проверить его дотупность можно, выполнив команду `./st -h`

## Что умеет

### Построить диаграмму Ганта
Построить интерактивную диаграмму Ганта для подзадач определенной корневой задачи. В качестве корневой задачи
можно указать сразу несколько тикетов. Утилита рекурсивно пройдет по всем подзадачам указанных тикетов и добавит их на
диаграмму с учетом зависимостей. При этом тикеты у которых есть хотя бы одна подзадача будут проигнорированы.
Между задачами не должно быть циклических зависимостей.

Пример:
```bash
st gantt --token xxx LILUCRM-5776 LILUCRM-5725
```

Если нужно сохранить html диаграммы в файл:
```bash
st gantt --token xxx --html LILUCRM-5776 LILUCRM-5725 > gantt.html
```
