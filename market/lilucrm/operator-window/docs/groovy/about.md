# Groovy

В платформу встроен язык [Groovy](https://groovy-lang.org/) для возможности скриптования бизнес-логики приложения. Это позволяет в рантайме без релиза приложения модифицириовать бизнес-логику приложения, автоматизировать бизнес-процессы не прибегая к участию разработчика.

Платформа предоставляет следующее АПИ:
- [Модификация](bcp.md) бизнес-объектов
- [Выборки](query.md) бизнес-объектов
- [Работа](tx.md) с транзакциями
- [Работа](conf.md) с конфигурацией приложения
- [Работа](wf.md) с жизненным циклом бизнес-объектов
- [Работа](mail.md) с почтой
