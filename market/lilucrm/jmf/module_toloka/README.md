# Модуль интеграции с Толокой / Янгом
Толока — это открытая краудсорсинговая платформа, которая помогает быстро и эффективно собирать и размечать большое количество данных для машинного обучения и других бизнес-задач

Янг — это внутренняя инсталляция Толоки, предназначенная для работы асессоров, саппортов, модераторов и любых других специалистов по контенту. Все эти люди, в отличие от толокеров, связаны с компанией теми или иными контрактами, поэтому от них можно ожидать более высокого качества, регулярной работы на проектах и соблюдения NDA.
