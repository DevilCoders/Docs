# Базовые метаклассы и логика вокруг них

1) [Описывает](src/main/resources/metadata/module_default_metadata.xml) основные метаклассы.

**catalogItem** - базовый метакласс справочников. Справочник - бизнес-объекты, количество которых ограничено, набор элементов
меняется редко. Такие ограничения позволяют упростить пользовательский интерфейс для выбора элементов справочников, 
кешировать элементы справочников. Для обеспечения этой логики вводится новый
[тип](../catalog_items/src/main/java/ru/yandex/market/jmf/catalog/items/attribute/catalogitem/CatalogItemType.java) атрибутов "Элемент справочника".
