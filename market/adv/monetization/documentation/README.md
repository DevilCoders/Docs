## PUML
Это удобный способ изложить свои мысли в виде диаграмм, которые нашли свое широкое применение в разработке ПО. Puml-файл
с описанием процессов работы сервиса нужно оставлять в папке ``documentation/puml``, а на вики выкладывать только
изображение. Благодаря такому подходу, у нас всегда будет актуальная документация и описание процессов.

Как настроить:

1. Установить плагин [PlantUml Integration](https://plugins.jetbrains.com/plugin/7017-plantuml-integration).
2. Установить [Graphviz](https://www.graphviz.org/download/) для отображения диаграмм.
3. [Документация](https://plantuml.com) по тому, как можно заполнять формы.
4. Just do it.
