Компонент для работы со стэком из sticky элементов. Вероятно, весьма ограниченный, неправильный и для частной задачи.

Основная идея такая: есть уровни, в рамках них есть компоненты. Проблема в том, что компонент сверху может занимать разный размер по высоте. Так что просто sticky + фиксированный top не работает. Вот тут высчитывается этот самый top.

Ограничения:
- не учитывает sticky родителей и т.п., там надо думать что и как честно делать
- не перерисовывает, если компонент обновился без скролла (при скролле перерисует)
