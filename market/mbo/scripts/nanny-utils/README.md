Утилитка для массового редактирования nanny-сервисов.

Редактирования см. в `recipes`.

Как пользоваться:
```
python3 recipes/inject_async_profiler.py service1 service2 service3
```

Должны быть установлены `requests` (`pip3 install requests`).

Не активирует снапшоты (пока что) - надо зайти и активировать.
