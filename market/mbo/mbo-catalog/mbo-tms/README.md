# mbo-tms
Сервис, который крутит регулярные таски по шедулеру.
Характер тасок различный от генерации поколений и выгрузок, до дампа различных данных из оракла в другие бд, типа yt, s3 и т.д.

## Запуск на aida
Для запуска на aide необходимо использовать скрипт scripts/dev/run-mbo-tms.py

## Запуск тасок на aida/различных серверах
- Заходим по ssh
- выполняем telnet localhost 83
- выполняем команду tms-run <Название> (Example: tms-run tms-run offersWithReplicationExecutor)

Отследить статус можно с помощью фронта https://mbo-testing.market.yandex-team.ru/mbo-tms/manage-tasks.xml БТ))) так как используется база бт

## Локальный запуск интеграционных тестов
Для запуска интеграционных тестов используется команда:
```bash
ya make -ttt
```
Для запуска тестов необходима предварительная настройка etcd, описание лежит в корневом [README.md](../README.md#локальный-запуск-автотестов-через-ya-make)
