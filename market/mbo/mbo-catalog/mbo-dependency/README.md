# mbo-dependency
Спец модуль для того, чтобы везде не прописывать одни и те же зависимости, был создан для ускорения переезда из github в arc.
В ya.make присутствует вот эта важная строка INCLUDE(${ARCADIA_ROOT}/market/pricelabs/core/fontconfig.inc) она необходима для того, чтобы была возможность запускать тесты в яндексовом CI, так как он не имеет шрифтов и эти шрифты необходимо для него положить, если вдруг какой-то тест будет ломаться из-за NPE и эта ошибка будет бросаться в FontConfiguration.getVersion, добавьте к себе эту строку и заведётся.
По какой-то причине, даже добавление сюда, не добавляет это к тестам (-_-).
