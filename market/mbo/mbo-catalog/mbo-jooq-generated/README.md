# mbo-jooq-generated
Модуль внутри которого генерируются сущности жука на основе бд.
В одном модуле как в гредле объединить это не возможно, поэтому был создан отдельный метод для этой операции.

## ya make
Для корректной генерации происходит запуск java приложения, оно уже преднастроено и генерирует ресурсы в ${BINDIR}/generated.
Без отдельного запуска java приложения реализовать этот билд было бы невозможно.
Файл в директории src/generated/java/.init необходим, чтобы после стягивания arc-а директория куда, генерирует приложение существовала.
Если директории не будет, всё сломается и не будет работать.
