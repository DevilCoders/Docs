## mbo-db-changesets
Временный модуль для обеспечения совместимости при запуске интеграционных и функциональных тестов пока не совершён полный переезд на tier-0.
После переезда данный модуль надо будет изменить и изменить конфигурацию интеграционных и функциональных тестов.
Модуль повторяет генерацию ресурсов описанную в gradle конфигурации liquibaseChangesets.
