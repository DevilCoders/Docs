# mbo-content-api
Сервис который должен был разгрузить exporter.
Основное текущее назначение, добавление, удаление, изменение, публикация и распубликация нод в навигационном дереве.

## Запуск на aida
Для запуска необходимо запустить скрипт scripts/dev/run-mbo-content-api.py из директории mbo-catalog
