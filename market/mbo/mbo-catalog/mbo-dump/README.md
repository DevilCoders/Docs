# mbo-dump
Библиотека с кучей экспортеров, которые позволяют дампить различные данные из mbo.
Уже написано много различных дамперов для дампов из одних систем хранения данных в другие.
В либе содержится огромное количество различных простейших поведенчиских абстракций, интерфейсов и базовых классов.
Абстракции в будущем лучше перевезти в jlibrary и написать конкретные либы для каждого случая, либо просто реализовать модули в рамках mbo.
