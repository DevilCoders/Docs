# mbo-http-exporter

Хранилище контентных знаний, аналогично выгрузке, позволяет получать штучные данные, как в выгрузке, по http запросам.

## Ya make
В генерации скриптов, шинанда -XX:+UseShenandoahGC отключена

## Локальный старт
Запускаем MboHttpExporterMain в идее и следуем инструкциям.
