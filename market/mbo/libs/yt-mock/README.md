yt-mock
--
Моки классов YT для юнит тестрования. В будущем планируется закопать в пользу локального YT.

#### Почему не использовать рецепты?
- Не работают из коробки на маках и виндах
- Не позволяют дебажить код

#### Почему не докер?
- На момент реализации моков это решения еще не было/было нестабильное.

**В поиске инициативы закопать моки и сделать нормальное решение по кнопке.**
