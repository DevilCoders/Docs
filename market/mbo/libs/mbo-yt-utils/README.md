YT-utils
=====

## Index
Частичный перенос логики работы из хранилища моделей MBO.

Библиотека позволяет работать с yt-таблицами как с индексами к основной таблице.
* `AbstractYtMultipleIndexWriter` – позволяет делать соотношение основной строки к n строкам в индексе
* `AbstractYtSingleIndexWriter` – позволяет использовать соотношение 1<->1 индекса к основной таблмце

Запись в таблицы-индексы происходит в одной транзакции с записью в основную таблицу.

Для чтения необходима отнаследовать классы:
* `YtIndexQuery` – для построения запроса к конкретному индексу и проверки его доступоности
в зависимости от переданного SearchFilter
* `YtIndexReader` – отвечают за конвертацию исходного запроса к конкретному индексу
(основная логика уже есть в `AbstractYtIndexReader`)

Пример в тестах.
