# mbo-ui

## Начало работы

Для начала работы сгенерируйте проект с помощью команды

```shell script
ya ide idea --iml-in-project-root --project-root ~/arc/idea-projects/market/mbo/libs/mbo-ui
```

и откройте папку проекта в idea.
Данная подходит для структуры каталогов arc, описанной в [wiki](https://wiki.yandex-team.ru/users/sulim/howto/arc-directory-structure/).
Если вы используете другую структуру, замените `~/arc/idea-projects/market/mbo/libs/mbo-ui` на нужную папку.

## Continuous delivery pipeline

Данная библиотека собирается пайплайном [MBO libraries arcadia](https://tsum.yandex-team.ru/pipe/projects/mbo/delivery-dashboard/mbo-libs-arc).
