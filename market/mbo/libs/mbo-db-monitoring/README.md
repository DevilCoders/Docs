mbo-db-monitoring
---

Небольшая либа по сбрасыванию состояния мониторинга в БД.
Полезно использовать, когда мониторинг может сработать только у одного сервиса.
И чтобы не потерять событие (из-за рестарта, отключения ДЦ), событие записывается в БД, откуда его считывают и другие сервисы.

Либа написана по аналогии мониторингов market-java-application (ComplexMonitoring, MonitoringUnit).
