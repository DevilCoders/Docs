Что это такое?
==============

Телеграмм-бот, который умеет писать в чаты о зависших тикетах.

Пример:
```markdown
MBO bot, [27.03.20 14:42]
Подзависли в ревью - 1:
- MBO-23595 (https://st.yandex-team.ru/MBO-23595) @AlButusov Комментарий "Отправить на доработку руководителю" в заданиях, ревьюверы: @yurimalinov (3d)

Тестируются - 1:
- MBO-24544 (https://st.yandex-team.ru/MBO-24544) @anmalysh Проверить новый функционал заливки маркетинговых описаний (2d)

Протестировано, пора смержить - 1:
- MBO-24413 (https://st.yandex-team.ru/MBO-24413) @aleksei_sakharov MDM Рассчет КГТ по данным МДМ, ревьюверы: @dmserebr (7d)

Смержены и никак не выложатся - 2:
- MBO-24371 (https://st.yandex-team.ru/MBO-24371) @stalhk Отдавать метрики MBO в prometheus формате через micrometer (3d)
- MBO-24610 (https://st.yandex-team.ru/MBO-24610) @sergeiermakov_yandex Фронт ревьит фронт :) (3d)
```

Как подключить себе
===================

1. @market_mbo_bot - добавить его в чат и написать ему что-нибудь (с призывом)
2. дёрнуть https://mbo-gwt-a-aida-36641.user.derkoat.yandex.ru/api/messages - там будет сообщение, нужно выцепить id чата
3. дёрнуть https://mbo-gwt-a-aida-36641.user.derkoat.yandex.ru/api/send-message?chatId=${chatId}&message=Hi there! для проверки связи (важно, для групп chatId должен быть отрицательным)
4. https://github.yandex-team.ru/yuramalinov/mbo-info/blob/master/mbo-info-app/src/main/resources/config.json - добавить сюда свой конфиг, что хотите отслеживать, с какими sla (c правильным id чата)
5. написать @yuramalinov чтобы перевыложил (делается руками, см. ниже)
