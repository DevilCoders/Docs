# dev_uploader

Утилита командной строки для загрузки market-application на dev машинки и их дальнейший запуск.

## Начало работы

Перед началом работы рекомендуется создать файл `~/.dev_uploader.yaml` и заполнить его по шаблону

```yaml
base_port: 12345 # Ваш порт из таблицы https://wiki.yandex-team.ru/market/development/ports/ranges/

```
