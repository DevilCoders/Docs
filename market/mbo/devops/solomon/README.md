# solomon

Конфигурация графиков, дашбордов и алертов в [Solomon](solomon.yandex-team.ru/) с помощью [solomon tool](https://a.yandex-team.ru/arc/trunk/arcadia/travel/devops/solomon).


## Начало работы

Для начала работы прочтите [инструкцию](https://a.yandex-team.ru/arc/trunk/arcadia/travel/devops/solomon/README.md) к основной тулзе и выполните пункты из раздела [quickstart](https://a.yandex-team.ru/arc/trunk/arcadia/travel/devops/solomon/README.md#quickstart).


## Работа с тулзой

Ко всем командам надо добавлять параметр. Либо произвести настройку, как указано в [quickstart](https://a.yandex-team.ru/arc/trunk/arcadia/travel/devops/solomon/README.md#quickstart).

```
--configs-path=market/mbo/devops/solomon
```
