Market proto ui
--

https://market-proto-ui.vs.market.yandex.net/

Что это такое?
--------------
Это - простой интерфейс, в котором можно посмотреть proto-ручки, их адреса и sample-запросы в них.
Используйте для знакомства с API/проверки.

Ограничения:
- формирует GET запросы => большой запрос не засунуть
- тут есть не все ручки: открываем `ProtoServicesConfig`, и добавляем нужное по вкусу

Как выложить новую версию
-----------------------
- Создаем ПР
- Проходим ревью и мержим в мастер
- ЦУМ автоматически соберет и выложит новую версию с помощью пайплайна:  
   https://tsum.yandex-team.ru/pipe/projects/mbo/delivery-dashboard/market-proto-ui

Как проверить изменения
-----------------------
- самое простое - `./gradlew run`
- он не шибко быстрый, если фронтенд один раз собран - можно `./gradlew run -X compileFront`
- сам `MarketProtoUiApp` можно просто запускать в IDEA без дополнительных приседаний - он запустится, т.к. пропертей ему не нужно
И делаем PR!
