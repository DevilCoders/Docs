Билд CKEditor 5 для Market CMS
========================================

Это - кастомный билд CKEditor, необходимый для работы с реакт-компонентом. Он подключает в себя библиотеку [typograf](https://github.com/typograf/typograf), с помощью которой можно нормализовывать текст.

Этот билд в дальнейшем можно экстендить, навешивая кастомные библиотеки и логику. В этом помогут следующие страницы:
[Подробное описание архитектуры редактора](https://ckeditor.com/docs/ckeditor5/latest/framework/guides/architecture/intro.html)
[Пример подключения плагина к редактору](https://ckeditor.com/docs/ckeditor5/latest/framework/guides/creating-simple-plugin.html)
