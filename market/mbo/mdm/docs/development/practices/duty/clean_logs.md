# Очистка логов на нодах

У нас есть мониторинг на исчерпание места внутри нод. Чаще всего забивается папка с логами, а самый толстый лог -  `mbo-mdm.log`.

Чтобы почистить этот файл, надо:
- Открыть nanny
- Скопировать ссылку для root ssh
- сделать `cd ./logs/mbo-mdm/`
- выполнить команду `truncate -s 100M ./mbo-mdm.log`
- проверить, что сервис продолжает писать в файл командой `tail ./mbo-mdm.log -f`
