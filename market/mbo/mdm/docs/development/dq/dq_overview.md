**Data quality** (DQ) - это централизованный инструмент для работы с качеством данных.
Контроль, очистка, обогащение, мониторинг и профилирование данных силами дата стюартов с минимальным участием разработки.

В МДМ было принято решение по использованию готового продукта для DQ от [Ataccama](https://www.ataccama.com/)
