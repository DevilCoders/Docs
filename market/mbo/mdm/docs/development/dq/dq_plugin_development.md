Attaccama ONE IDE позволяет расширят свой функционал за счет кастомных плагинов (собраный jar, положенный в classpath runtim-а)
Проект для компонентов чтения и записи в YT лежит в [аркадии](https://a.yandex-team.ru/arc/trunk/arcadia/market/mbo/mdm/mdm-dqc)

Релизится при помощи ЦУМ [пайплайна](https://tsum.yandex-team.ru/pipe/projects/mbo/delivery-dashboard/dq-plugins)
