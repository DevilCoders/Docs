# Микросервис lbdumper

## Abstract

Микросервис строился в рамках [задачи](https://st.yandex-team.ru/MARKETINDEXER-32302): по поставке поофферных данных об автостратегиях в репорт через индексатор.

Назначение: чтение данных из топика logbroker, конвертация и последующая запись в динамические таблицы YT.
