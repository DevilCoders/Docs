==Промо. Новая механика (для партнеров/для клиентов)
**Использовать при: запуске новой механике для клиентов, новой маркетплейсной механике и новой партнерской механике**

**Ответственный:** запускающий менеджер 


**Ограничение по количеству акций в механике?** 
- нет ограничения ->  инфраструктура и стрельбы 
- ограничения есть и еще не стреляли этой механикой ->  инфраструктура и стрельбы 

**Ограничения по количеству офферов в акции?**
- нет ограничения ->  инфраструктура и стрельбы 
- ограничения есть и еще не стреляли таким объемом ->  инфраструктура и стрельбы 

Смотрим со стороны систем: 
1) лоялти (включая походы с фронта)
2) репорт (включая походы с фронта)
3) idx (сварит ли)
4) App (ios+ android)
5) web 

Пример: 
**Запуск маркетплейсных флешей**
Ограничение по количеству акций в механике? да, заводит маркет
Ограничения по количеству офферов в акции? нет -> инфраструктура и стрельбы 

Смотрим со стороны систем: 
1) лоялти -> прикапыватеся информация по офферам, выпиливаем
2) репорт (включая походы с фронта) -> нужны стрельбы по новой механике 
3) idx (сварит ли) -> нужно сварить поколение. Решили варить на 200 млн
4) App (ios+ android) ->  много таймеров могут повесить сервис, нужна оптимизация
5) web ->  много таймеров могут повесить сервис, нужна оптимизация
