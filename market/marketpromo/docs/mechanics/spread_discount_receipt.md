# Прогрессирующая скидка за чек (spread-discount-receipt, SPREAD_DISCOUNT_RECEIPT)

## Описание { #general }
Скидка увеличивается в зависимости от общей стоимости товаров, купленных по акции, чем больше чек, тем больше скидка.

### Способы заведения
1. Описание акции [через ки](../create/ciface.md), ассортимент [через трекер](../create/tracker.md)
FBY✅️ FBS✅️ DBS✅️ Экспресс✅️

### Места отображения
Все сниппеты и места выдачи (виджеты/выдачи/карточка/корзина)

### Настройки акции
1. Можно задать минимальную/максимальную стоимость товаров. Настраивается покатегорийно
2. Можно задать скидку в процентах или в рублях
3. Для акции нужно указать бюджет. Отслеживание бюджета производится через кубы.

### Точки коммуникации об акции:
- Товарный сниппет/департамент/корзина
- В корзине пишем выгоду в общей строке скидки
- В чеке на каждую позицию начисляем скидку, полученную по акции

### Возвраты:
Принимает риски и перерасчетов по товарам на возврат не делаем (процент возвратов в FMCG менее 1%).
Про антифрод думает отдельно, либо отправляем в акцию товары, которые обмену и возвраты не подлежат
