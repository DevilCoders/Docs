# Маркет бонус (купон, монетка) (coin, MARKET_COIN)

## Описание { #general }
В рамках маркетинговой активности или за полезные действия, пользователь может получить купон.
Он его может потратить на любой заказ, выполнив условия применения купона.

### Способы заведения
1. Описание акции [через ки](../create/ciface.md), 1Р/3Р ассортимент [через лоялти](../create/loyalty/coin.md)
FBY✅️ FBS✅️ DBS✅️ Экспресс✅️
2. Маркетинговые акции за наш счёт на 1Р/3Р ассортимент [через лоялти](../create/loyalty/coin.md)
FBY✅️ FBS✅️ DBS✅️ Экспресс✅️

### Купон на бесплатную доставку
