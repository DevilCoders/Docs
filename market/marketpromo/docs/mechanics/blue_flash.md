# Флеш (blue-flash, BLUE_FLASH)

## Описание { #general }

Разовые распродажи определенных товаров в ограниченный период времени с углубленной скидкой.

[Корневой продуктовый тикет](https://st.yandex-team.ru/BMP-1924)

### Настройки акции
1. Можно указывать точное время старта и окончания акции

### Как выставляются скидки по акции и какие есть проверки?
1. При загрузки акции, выставляется конечная цена (**цена продажи по акции**), по которой товар участвует в акции.
   На базе **цены продажи по акции** и **базовой ценой** на карточке автоматически рассчитается % скидки и будет меняться динамически, при изменении базовой цены.
   **Базовая цена** = old price = это цена из Аксапты, если на карточке товара есть перечеркнутая цена, то скидка считается от нее, если нет перечеркнутой, то скидка считается от текущей цены продажи.
2. Дополнительные [проверки на честность](../restrictions/index.md)

### Какие ограничения виджета с таймером?
1. Виджет таймера на главной странице может быть только один
2. Таймер запускается по Московскому времени на стартует по всем регионам, в соответствии со временем региона
3. Максимальное время таймера 48 часов

### Может ли быть две акции параллельно?
Да, акции запустятся, можно будет собрать лендинг под каждую из этих акций и перейти на лендинг с карточки модели по клику на таймер, но тогда виджет таймера на главной лучше не использовать, в виджете поддерживается только одна ссылкой на один акционный лендинг.

### Можно ли указать в одной акции товары с разных складов?
1. Если поле склада пустое, то акция действует на все склады
2. Если в поле указан склад, то акция будет действовать только для этого склада
https://st.yandex-team.ru/BLUEACTION-1447

### Как добавить таймер к виджету?
"ДОБАВИТЬ - НАСТРОЙКА ТАЙМЕРА ОБРАТНОГО ОТСЧЕТА" и задать время таймера, в момент старта акции виджет автоматически появится на главной и пойдет обратный отсчет, по истечению времени действия акции виджет скроется

### Какие есть возможности коммуникации механики?
1. Пуши - по истечению срока действия акции пуши сгорают
2. Таймеры на всем пути пользователя главная - выдача - карточка - корзина
3. Баннер в шапке с таймером
