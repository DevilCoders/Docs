# Кешбек (cashback, CASHBACK)

## Описание { #general }
Кешбек выдаётся за участие в программе лояльности.
За покупки на маркете, поездки в такси...
На маркете можно накопить или потратить кешбек.

### Способы заведения
1. Описание акции [через ки](../create/ciface.md), 1Р/3Р ассортимент [через лоялти](../create/loyalty/cashback.md)
FBY✅️ FBS✅️ DBS✅️ Экспресс✅️
2. Партнёр может самостоятельно завести акцию в ПИ
FBY✅️ FBS✅️ DBS✅️ Экспресс✅️
3. Маркетинговые акции за наш счёт на 1Р/3Р ассортимент [через лоялти](../create/loyalty/cashback.md)
FBY✅️ FBS✅️ DBS✅️ Экспресс✅️
