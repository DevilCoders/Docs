# Потенциальный ассортимент

nile скрипт, который берёт на вход:
1. Историю честных цен из индексатора
2. Акции из акционного хранилища
3. Оферы из статической выгрузки оферного хранилища

В результате формирует таблицу с потенциальным ассортиментом, который через сканнер накатывается на ОХ
