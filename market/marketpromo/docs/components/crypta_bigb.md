# Крипта/BigB

Общая информация о BigB - https://wiki.yandex-team.ru/bigb/

Сегмент из крипты в BigB данные выгружаются с задержкой минимум в 1 сутки

Сегмент используется на проекте Буст пвз через промокод https://st.yandex-team.ru/MADV-846
Сегмент крипты для буста ПВЗ - https://lab.crypta.yandex-team.ru/segments?segment=segment-b0459597

Для проверки пользователя в сегменте:
1. при походе из FAPI используем uuid
2. при походе из веба используем puid, если нет то yuid

{% note info %}

на данной момент проверка пользователя в сегменте при использование таких id качественно не протестирована

{% endnote %}

При переустановке приложения (актуально для тестировщиков, на обновление не распространяется) uuid пользователя меняется, нужно ждать пересчета сегмента, затем обновления экспорта в BigB, только тогда пользователей появится в сегменте, т.е. лаг около 2-3 суток
