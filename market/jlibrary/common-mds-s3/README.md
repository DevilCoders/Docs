# Common MDS S3

## Модули
* **[common-mds-s3-client](common-mds-s3-client)** - Компоненты для работы с MDS S3.
* **[common-mds-s3-spring](common-mds-s3-spring)** - Компоненты и конфигурации для интеграции библиотеки в Spring-проекты.
* **[common-mds-s3-tms](common-mds-s3-tms)** - Компоненты для интеграции с TMS-framework'ом.
