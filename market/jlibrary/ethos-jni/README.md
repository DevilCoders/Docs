Библиотека и пакет для использования/обучения текстового классификатора Ethos.

Собрать текстовый классификатор можно склонировав задачу в сендобке:
https://sandbox.yandex-team.ru/task/114801838/view
Сборка осуществляется из сорсов /arc/trunk/arcadia/kernel/ethos/lib/text_classifier/
После успешного завершения задачи, нужно перейти на вкладку ресурсы и выбрать ресурс с типом BUILD_OUTPUT.

Сборка JNI осуществляется через задачу:
https://sandbox.yandex-team.ru/task/114797976/view
Сборка осуществляется из сорсов /arc/trunk/arcadia/market/ir/ethosjni
