# generate-jooq

Generate-jooq предназначен для кодогенерации классов на основе БД схемы.
Эта библиотека публикуется в артефактори, поэтому она не содержит бинарей постгреса.
Для использования библиотеки в аркадии необходимо использовать generate-jooq-with-postgres.

См. [generate-jooq-with-postgres](../generate-jooq-with-postgres)
