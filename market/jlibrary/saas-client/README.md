# saas-client

Библиотека для взаимодействия с SaaS через indexer-proxy и search-proxy.
Работа с индексацией и работа с поиском разделены на два модуля — saas-indexer-client 
и saas-search-client соответственно.

[Library is released with CD pipeline](https://tsum.yandex-team.ru/pipe/projects/jlibrary/delivery-dashboard/Common-Husk)
