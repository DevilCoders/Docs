# О библиотеке
Библиотека предоставляет набор объектов и утилит для работы с динамическими таблицами

## Сборка и тестирование
[ya.make](https://wiki.yandex-team.ru/yatool/make/)

## Как пользоваться
Простая точка входа для старта - класс [YtDynamicTableClientFactory](src/main/java/ru/yandex/market/yt/client/YtDynamicTableClientFactory.java),
в нем собраны все основные запчасти для работы с библиотекой.
