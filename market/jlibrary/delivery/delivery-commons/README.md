# Delivery commons

### Описание

Репозиторий для модулей с небольшой функциональностью, которую можно переиспользовать
в нескольких модулях доставки

### Деплой

Задеплоить можно с помощью [таски в TeamCity](https://teamcity.yandex-team.ru/viewType.html?buildTypeId=Market_Delivery_DeliveryCommons_Publish)

В параметрах таски нужно выбрать модуль, который надо задеплоить

Если модуль новый, то его нужно добавить в параметр `mbuild.releaseModule` в TeamCity
