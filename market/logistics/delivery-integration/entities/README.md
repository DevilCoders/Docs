#Delivery Entities

###Что это

Библиотека с сущностями, которые могут понадобиться любому проекту, 
который работает со спецификацией Delivery Service Module (DSM API).
DSM - модуль, который работает со службами доставки. (Жизненный цикл заказ, трекинг статусов, 
выгрузка справочников и многое другое. подробнее смотри доку)

###Документация

На момент написания этого файла последняя версия спецификации (2.0) разрабатывается в этой задаче
https://st.yandex-team.ru/DELIVERY-2949

###Как использовать

Добавить в gradle `compile("ru.yandex.market.delivery:entities:0.+")`

###Как контрибьютить

Через Pull Request

Перед публикацией стабильной версии необходимо инкрементить версию, 
для публикации снапшота необходимо добавыить `-SNAPSHOT` к версии, 
ключевое слово превратиться в timestamp в момент публикации
