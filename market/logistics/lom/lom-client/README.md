# LOM client
## Публикация в artifactory
Для публикации конкретной ревизии в artifactory.yandex.net нужно запустить [релизный пайплайн](https://tsum.yandex-team.ru/pipe/projects/delivery/delivery-dashboard/logistics-lom-client)

Выбрать нужную ревизию и нажать кнопку ```"…" -> "Создать релиз"```

После успешного завершения пайплайна, клиент будет доступен как ```ru.yandex:lom-client:<номер ревизии в аркадии>```.

Например *ru.yandex:lom-client:5366154*
