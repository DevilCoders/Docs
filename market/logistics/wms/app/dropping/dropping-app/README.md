Сервис предназначен для обработки процесса дропинга (Взял посылку и положил на палету службы доставки).

Сборка: `ya make`

Тесты:  `ya make -tt`

Для локального запуска используйте опции
```
-Djava.library.path={HOME_DIR}/arcadia/bin
-Denv.config={HOME_DIR}/IdeaProjects/infor-sce/properties/local/local.properties
```

Доступно здесь: `http://localhost:7777/dropping`
