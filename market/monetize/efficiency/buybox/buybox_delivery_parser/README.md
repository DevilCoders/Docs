Парсер сроков доставки оффера из репорта. Получает на вход список msku, waremd5, идет в place offerInfo, и парсит выдачу.
Результаты записывает в yt таблицу.

Пример использования в папке integral_tests/test.sh, пример входных данных data/test.tsv.
