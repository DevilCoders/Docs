# Stapler
Фреймворк для построения приложеий с кодогенерацией графов в Nirvana.
Используя этот фреймворк можно писать кубики для исполнения произвольного кода и построения произвольных пайплайнов в Nirvana.
Код призван наносить пользу в процессе продакшенизации проекта команды replenishment и улучшению архитектуры приложения.

- [Презентация в wiki ](https://wiki.yandex-team.ru/users/arakhmatulin/sozdanie-grafov-s-pomoshhju-monetizationutils/)
- [Чуть подробнее в v1/readme.md](./v1/README.md)
