# Инструменты для облегчения работы с аркадией и арком
## ya-idea
Скрипт, который позволяет генерировать проект с рекомендуемыми параметрами.
Помимо генерации проекта скрипт отправляет статистику сборки.

### Дашборд со статистикой сборки
https://grafana.yandex-team.ru/d/jmKnZCkGk/mbi-build-stat?orgId=1&var-p=1&var-l=All&var-vcs=All&var-path=All&var-platform=All&var-version=All

### Команды скрипта
* `ya-idea` или `ya-idea build` генерирует проект, который находится в месте,
 где был запущен скрипт. В аргументы можно подать название для генерируемого проекта.
 Если не подавать аргументы - используется название, генерируемое по умолчанию.
* `ya-idea open` открыть сгенерированный проект.
В аргументы можно подать название сгенерированного проекта, который нужно открыть.
Если не подавать аргументы - используется название, генерируемое по умолчанию, для проекта,
в директории которого запускается команда.

### Установка скрипта:
#### Автоматическая установка
* Запустить скрипт в этой папке, который называется setup.sh: `bash ./setup.sh`
* Если ya-idea не появилась перезапустить bash с помощью `exec bash`.

#### Ручная установка
* Добавить в PATH или сделать alias на этот скрипт `<ARCADIA ROOT>/direct/bin/ya-idea`.
* Создать файл `~/.ya/direct-ya-idea.overrides.sh`.
* Добавить в этот файл строчку `source <ARCADIA ROOT>/market/mbi/mbi-tools/arcadia/ya-idea.overrides.sh`.
Так же в этом файле можно переопределить переменные и функции на свой вкус.

### Директория с проектами
По умолчанию IDEA проекты будут храниться в директории "~/IdeaProjects".
Для того чтобы поменять это поведение:
* Открываем файл `~/.ya/direct-ya-idea.overrides.sh`
* Добавляем в любое место строчку `YA_IDEA_PROJECT_ROOT="your-custom-path"`
* В конце пути должен быть символ `/`

### Полезности
Можно настроить автооткрытие папки с проектом в аркадии при запуске терминала в папке со сгенеренным проектом.
Это может быть полезно при запуске терминала из идеи.
Для настройки нужно добавить в свой `.bashrc` или `.zshrc` файл строчку
`source <ARCAIDA_ROOT>/market/mbi/mbi-tools/arcadia/autoopen-arcadia-term.sh`
