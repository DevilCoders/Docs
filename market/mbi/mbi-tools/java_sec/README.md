# Смысл

При разработке новых ручек mbi-partner в целях тестирования требуется добавлять права на новую ручку. 

Обычно разработчики генерируют файл partner_domain.sql и запускают его, что неправильно, т. к. это приводит к перезатиранию прав на параллельно созданные ручки другими разработчиками.

Поэтому нужно использовать данный скрипт для добавления прав на новые ручки mbi-partner в dev-окружении
