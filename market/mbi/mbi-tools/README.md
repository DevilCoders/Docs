
# Инструментарий для разработчиков MBI

# s3

Инструменты для работы с MDS поверх Amazon S3.

# log-tools

Залезает на logview и выгрепывает кусок лога из логов partner-api или mbi-partner по времени.

# mbi-cli

https://wiki.yandex-team.ru/mbi/mbi-cli/

# release

Инструменты для подготовки релизов

# java_sec

Скрипт для добавления прав на новые ручки mbi-partner в dev-окружении

# samovar

Скрипт для скачивания фидов из самоварных бакетов по mdsKeys
