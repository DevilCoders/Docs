#Клиент для сервиса аудита
Сервис аудита собирает логи из всех модулей MBI. Сбор логов идет через соответсвующий топик в логброкере
Клиент выполняет публикацию сообщений в топик. На текущий момент поддерживается один топик
- /mbi/test/b2b-audit
- /mbi/prod/b2b-audit

Для публикации сообщений в топик необходимо оформить доступ сервиса, который будет использовать этого клиента.
Для этого можно оформить тикет в очередь MBI с запросом на запись в  в нужный топик. В тикете указать tvm_id сервиса
