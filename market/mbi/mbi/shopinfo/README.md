### shopinfo

#### Параметры для локального запуска из IDEA
1. Classpath of module `shopinfo`
2. Main class `ru.yandex.common.util.application.XmlAppContextMain`
3. Working directory `$MODULE_WORKING_DIR$`
4. VM properties example
```properties
-Dbean.file=classpath:bean.xml
-Dconfigs.path=src/main/properties.d
-Denvironment=local
-Doracle.net.tns_admin=/your/homedir
```
