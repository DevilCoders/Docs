Конфигурация дэшборда компонента yandex-pers-grade в Grafana
==============================================================

Адрес основного дэшборда: https://grafana.yandex-team.ru/dashboard/db/pers-grade
Адрес дэшборда со статистикой по количеству отзывов из разных источников: https://grafana.yandex-team.ru/dashboard/db/pers-tms-grade-stats

### Выгрузка конфигурации дэшборда.
1. Зайти на дэшборд.
2. Нажать "Share dashboard" на верхней панели.
3. Нажать "Export".
4. Выбрать "Save to file".

### Загрузка конфигурации дэшборда.
1. Нажать на кнопку выбора дэшборда на верхней панели.
2. Нажать "Import".
3. Выбрать "Upload .json file".
4. В выпадашке для "market" выбрать "market".
