# Жизненный цикл объектов

Реализует логику жизненного цикла объектов: статусов объектов, переходов между статусами.

[WfService](src/main/java/ru/yandex/market/jmf/logic/wf/WfService.java) содержит конфигурации жизненных циклов метаклассов. В конфигурации
описываются возможные статусы, переходы между статусами, действиями на вход и выход из статуса.

В [bcp_wf.xml](src/main/resources/metadata/bcp_wf.xml) BCP-операции связанные с жизненным циклом.
