# Java Meta Framework

Основная идея фреймворка - это описывыние всех используемых бизнес-сущностей метаинформацией на основании которой будет строится модель
данных базы данных, пользовательский интерфейс, система прав и пр.

Предосылками к такой системе является требование иметь возможность добавлять атрибуты к бизнесс-объектам без перевыкладки приложения,
что должно увеличить скорость внесения изменении в бизнес-процессы и позволит часть функции по настройке системы выполнять владельцем
автоматизируемых процессов. Изменение вспомогательной бизнес-логики осуществляется с помощью скриптов на динамическом языке, что
так же не требует перевыкладки приложения.  

## Модули

1) *[metadata](metadata/README.md)* - реализация сервиса метафинформации описывающий бизнес-сущности в системе.
2) *[db_api](db_api/README.md)* - абстракция хранения бизнес-сущностей на основании описанной метаинформации.
3) *[db_hibernate](db_hibernate/README.md)* - реализация хранения бизнес-сущностей с использованием Hibernate ORM.
4) *[attributes](attributes/README.md)* - поддержка базовых типов атрибутов, таких как "Строка", "Число" и пр.
5) *[bcp](bcp/README.md)* - абстракция для выполнения операции изменения бизнес-объектов таких как создание объекта, его изменения и удаления.
6) *[logic_default](logic_default/README.md)* - описание базовых бизнес-сущностей и логики работы с ними.
7) *[logic_wf](logic_wf/README.md)* - логика работы с жизненным циклом объектов.
8) *[timing_engine](timing_engine/README.md)* - реализация работы с временными характеричтиками объектов. Добавляет справочники "Время обслуживания" 
и позволяет высчитывать регламентное время завершения работ. Другими словами это модуль для расчета SLA.
9) *[task_engine](task_engine/README.md)* - базовая реализация работы с задачами обрабатываемыми операторами колл-центра.
10) *[db_support](db_support/README.md)* - поддержка работы с базой данных.
11) *[script_engine](script_engine/README.md)* - поддержка динамических языков.
12) *[tx_support](tx_support/README.md)* - поддержка транзакции.
13) *[utils](utils/README.md)* - вспомогательные классы общего значения.
