# Sovetnik Redir

URL redirector for Sovetnik.

## Setup

```sh
nvm use && npm i
```

## Start server

```sh
npm start
```

## Testing

### Local

```sh
npm t
npm run test:coverage # with coverage
```

### CI

```sh
npm run test:ci
```

### Watch mode

```sh
npm run test:watch
```

## Linting

```sh
npm run lint
```

## Formatting

```sh
npm run format
```
