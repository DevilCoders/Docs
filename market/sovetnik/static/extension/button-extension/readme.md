Код расширения Советника для Chrome-like браузеров с кнопкой на панели

Расширение содержит инжект-скрипт, который билдится билдером инжект-скриптов (etc/builder/inject-scripts/chrome-injector.js)
Если собирается инжект-скрипт для расширения с кнопкой, все файлы из этой папки добавляются в нужное место.

Добавляться они могут в: 
1. Background-page
2. Content-script
3. Скрипт Советника (в общем Gruntfile есть таск internal-script, который билдит скрипт для этого расширения)
