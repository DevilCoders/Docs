# Review process

## Requirements

You need node `v14.17.1`.

## Guide

1. Unpack `opera_2210590_source.zip`

```console
unzip -q opera_2210590_source.zip
```

1. Go to to `source` directory

```console
cd opera_2210590_source
```

1. Run `review_compare.sh` script

```console
./review_compare.sh
```

If you haven't seen any errors after script completed then all corresponding files are the same.
