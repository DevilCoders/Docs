# Landings

## Setup

You should have node (see version in `engines` section in [package.json](package.json)).

```sh
nvm use && npm i
```

## Development

```sh
npm run dev:{landing-page-name}

# Example
npm run dev:welcome
```

## Build for production

```sh
npm run build:{landing-page-name}

# Example
npm run build:welcome
```

## Serve production build on your localhost

```sh
npm run serve:{landing-page-name}

# Example
npm run serve:welcome
```
