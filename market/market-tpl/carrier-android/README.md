# Магистральные перевозки
Android приложение магистральных перевозок.

## Сбор apk-файла для водителей
1. Открываем Android Studio.
2. Кликом на панели слева открываем окно "Build Variants".
3. Переключаем "Active Build Variant" модуля "Carrier.app" в состояние "release".
4. В меню Android Studio выбираем "Build" -> "Build Bundle(s) / APK(s)" -> "Build APK(s)".
5. Apk-файл находится по пути "app/build/outputs/apk/release/app-release.apk".
6. Проверить информацию о apk-файле (например, с помощью Finder), что дата создания/модификации файла актуальна.
