# Tariff2YT

Cервер, который скачивает тарифы из tarifficator'a и загружает их в YT.

## Usage
Для локального запуска потребуется поднять tvmtool демон, так как для работы с
tarifficator'ом нужны tvm ticket'ы

Для этого есть скрипт, который можно запустить так:
```
../../scripts/start_tvmtool/run_tvmtool.sh > /dev/null 2>&1 &
```

После этого в переменную окружения YT\_TOKEN следует прописать свой токен от YT.

Теперь можно спокойно запускать сервер c дефолтными входными значениями.
