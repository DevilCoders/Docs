# Рендеринг моделей с использованием gumoful шаблонов
### Как открыть проект
```$bash
ya make . --add-protobuf-result # для того, чтобы ide видела proto классы
ya ide clion && clion .
```

### Как запустить проект

Запуск нужно осуществлять только с linux машинки. Иначе Yt отругается на неподдерживаемый исполняемый файл.

```$bash
ya make -r
DEBUG_LOCAL_RUN=1 ./yt_renderer_reducer
```

### Как это работает

Запускается reduce операция по двум таблицам
* таблица к категориями
* таблица с моделями

Причем, таблица категорий исполняет роль внешней (справочной) таблицы и должна идти в списке таблиц первой.
