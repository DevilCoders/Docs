### Скрипты для графов наблюдения за экспериментами для колдунщиков Маркета

Здесь будут храниться все скрипты используемые в проекте в [nirvana](https://nirvana.yandex-team.ru/browse?selected=8301833)

Пока что добавил простейший дебаг для кубика [Python3 Json Process](https://nirvana.yandex-team.ru/browse?selected=7368808)
Параметры поддеррживаются не все, исполняемые код из параметра code находится в файле /python3_json_process/my_code

Для запуска сложите все свои input файлы из nirvana в input и укажите их отностиельный путь в команде из папки /python3_json_process
```bash ./run_python3json_process_cube.sh --in1 /input/in1.json --in2 /input/in2.json --in3 /input/in3.json  --scripts_archive /input/archive.tar```
Результаты по умлочанию будут лежать в папке /python3_json_process/output
