#### Либровые выжимки премиальной врезки.

Делали по аналогии с вебовскими выжимками органики

Аргументы:
* **date** - дата
* **pool** - yt пул
* **destination-table** - куда сложить
* **us-table** - таблица с юзер-сессиями (опционально для дебага)
* **features-table** -  market-feature-log (опционально для дебага)
* **cached-order-dict-path** - путь к закешированному словарю заказов (опционально для дебага)
