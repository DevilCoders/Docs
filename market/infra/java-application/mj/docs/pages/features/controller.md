# Генерация контроллера и swagger-ui
Для генерации контроллера достаточно описать [OpenApi спецификацию](../how_it_works/configuration/api_yaml.md) своего сервиса.

Для контроллера можно подключить [TVM](tvm.md).

На все или некоторые ручки можно повесить [rate limiter](resilience4j.md#Использование-rate-limiter-в-генерируемых-контроллерах).

На ручках контроллера будет работать трассировка.
