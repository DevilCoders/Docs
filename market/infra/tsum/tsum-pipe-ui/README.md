# tsum-pipe-ui
UI-часть пайплайна, написанная на реакте. Собирается webpack'ом в один бандл, который подключается в ``tsum-ui``.

## Как собрать
Для сборки используется ``gradle-node-plugin``, поэтому не обязательно иметь установленный npm.

Для первоначальной сборки достаточно вызвать ``./gradlew build``, находясь в корневой директории ЦУМа.

### Разработка
Во время активной разработки сборку приходится делать по много раз, что неудобно. Чтобы решить эту проблему, существует режим, в котором webpack следит за изменениями в директории ``src`` и в случае обнаружения таковых автоматически запускает пересборку.
 
Чтобы запустить webpack в режиме слежения за исходниками, достаточно выполнить:
```
cd tsum-pipe-ui
.gradle/npm/npm-*/bin/npm start
```
