# Что это
Набор утилит, которые планируется использовать при нагрузочном тестировании Здоровья Маркета. Предполагается, что будет возможность как локального запуска, так и в составе платформы нагрузочного тестирования, когда она будет готова.

# Как создать проект
Находясь в папке `loadtesting` выполнить команду:
```
ya ide golang . -P ~/GolandProjects/market-health-loadtesting
```

В результате в папке `~/GolandProjects/market-health-loadtesting` будет создан проект, который можно открыть в IDE GoLand.
