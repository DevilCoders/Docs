# Стилизация, CSS и Levitan B2B

По хорошему, почти все можно сделать при помощи прокидывания пропсов в компоненты Левитана. Тем не менее, можно кастомизировать компоненты при помощи CSS.

# Контейнеры и компоненты

Контейнеры при живом Reatom — это компоненты, которые импортируют хуки из reatom. Если хочется написать новый компонент или контейнер, стоит проверить, есть ли такой уже, и переиспользовать.
