## Какую роль играют библиотеки:
* **resource_updater** – содержит код создания конкретного инстанса IResourceUpdater, который будет использоваться в репорте для загрузки ресурсов.
* **traceable_resource_updater** – создает обертку над IResourceUpdater, которая позволяет логгировать в common.log информацию о загрузке ресурсов в репорте.
