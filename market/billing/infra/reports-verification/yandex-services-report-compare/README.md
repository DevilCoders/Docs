## Сверка отчётов об услугах маркетплейса с актами

Перед запуском скрипта нужно разложить отчёты и акты по соответствующим папкам

В качестве примера приведена папка `2022-01`

Структура каталогов
```
2022-01/act/ - сюда складываются акты в виде pdf файлов для каждого partner_id, например 959046.pdf
2022-01/report/ - сюда складываются отчёты в виде xlsx файлов для каждого business_id, например 675145.xslx
2022-01/business2partner.tsv - файл со связкой business_id -> partner_ids. 
```

Для сравнения нужно запустить скрипт `compare_reports.py` и передать ему имя каталога с разложенными данными.

Например `compare_reports.py 2022-01`


## Скачивание отчётов из MDS пачкой

Для массового скачивания отчётов из MDS по ссылкам, которые генерируются в `SHOPS_WEB.ASYNC_REPORTS.URL_TO_DOWNLOAD` можно использовать `s3_report_downloader.py`

Список ссылок из базы нужно положить в файл (например `urls`) и запустить скрипт, передав имя файла в качестве параметра.
