## Пример использования
Раскладываем файлы в папку, именуя как угодно. После этого составляем список файлов следующим образом:
```
report-1-old.xlsx report-1-new.xlsx
report-2-old.xlsx report-2-new.xlsx
...
report-n-old.xlsx report-n-new.xlsx
```

Запускаем скрипт:`python3 compare.py main()`
Передаем на вход список файлов в описанном выше формате. На выход в терминале выводится пара документов и результат
сравнения. `OK` или `FAIL` со статистикой по расхождению.
