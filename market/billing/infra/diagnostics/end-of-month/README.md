## списки таблиц/биллингов

* `bill_results_tables.txt` -- список таблиц с результатами биллинга. Используется для выборки "что не собрали в tlog"
* `bill-results-env-ingnore-in-oracle.txt` -- какие env с датами, до которых доработали биллинги, игнорируем в Oracle. Используется для выборки 'что не собрали в tlog'
* `bill-results-env-ingnore-in-pg.txt` -- какие env с датами, до которых доработали биллинги, игнорируем в PG. Используется для выборки 'что не собрали в tlog'
