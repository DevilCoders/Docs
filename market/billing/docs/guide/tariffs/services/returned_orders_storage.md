# Хранение невыкупленных заказов в сортировочном центре (услуга returned_orders_storage)

{% note info %}

Обычно используется для __отмены платного хранения возвратов__

{% endnote %}

## В договоре/ДС

* Стандартный вариант  
    ![alt Пример договора](../../../_assets/images/tariffs/services/returned_orders_storage/agreement_example_1.png)

     Текст договора выглядит обычно так:

    ```text
    1. Стороны пришли к соглашению в период с «01» июля 2021 года по «31» июля 2021 года включительно
    п.7.11 Приложения №3 к Договору принять в следующей редакции:
    «7.11. В случае отказа Потребителя от Заказа и при его возврате на сортировочный центр,
    хранение такого Заказа от 7 (семи) календарных дней будет тарифицироваться в соответствии
    с суммой, которая составляет 15 (пятнадцать) рублей в сутки за каждый Заказ, за исключением
    дня забора Заказа Заказчиком из сортировочного центра. В период с «01» июля 2021 года по
    "31" июля 2021 года, хранение Заказов, находящихся в ПЭК Бутовотдельно не
    тарифицируется и включено в стоимость услуг по п.2.1 Приложению №3»
    ```

## В админке Тарифницы

В рамках Меты нужно заводить отдельные поля в рамках следующего окна:  
![alt Добавление меты](../../../_assets/images/tariffs/services/returned_orders_storage/meta_addition.png)

При этом, обратите внимане на следующие вещи:

* Relative/absolute значение - процент или фикс.
* Количество дней хранения, после которых применяется тариф в рамках поля "storageDays"
* В рамках поля "scPartnerId" можно отдельно указывать идентификатор сорт. центра, __только__ на котором нужно отменить хранение. 

Финальная Мета выглядит так:  
![alt Результирующая мета](../../../_assets/images/tariffs/services/returned_orders_storage/final_meta.png)

### Как завести отмену сразу для множества партнеров

В рамках админки нужно перейти на какую-нибудь страницу, например, с актинвыми тарифами. Сверху над списокм тарифов присутствует некоторое количество кнопок, одной из которых является "Обнуление платного хранения для поставщиков".  
Выглядит это так:  
![alt Массовое обнуление платного хранения возвратов](../../../_assets/images/tariffs/services/returned_orders_storage/bulk_creation.png)

После этого нужно указать дату __начала__ обнуления (дата окончания автоматически будет проставлена как +1 месяц); ключ тикета, в рамках которого собирали подписанные ДС и отдельно указать список партнеров.  
Выглядит это так:  
![alt Массовое обнуление платного хранения возвратов. Заполнение данных.](../../../_assets/images/tariffs/services/returned_orders_storage/bulk_creation_adding_data.png)

После заполнения можно проверить список и нажать "Создать".  
![alt Массовое обнуление платного хранения возвратов. Проверка и сохранение.](../../../_assets/images/tariffs/services/returned_orders_storage/bulk_creation_check_and_save.png)
