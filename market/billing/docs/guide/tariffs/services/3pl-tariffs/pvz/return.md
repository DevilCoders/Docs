# Прием клиентского возврата

В [оферте (приложение №1, таблица 1.3)](https://yandex.ru/legal/pvz/) данная услуга записана так:
```
Прием Клиентских возвратов
```
Физический смысл - Маркет платит деньги пвз за то, что пвз принял возврат у клиента.

## В договоре/ДС

Случаев отдельных ДС на данные услугу не было (считаются по общему тарифу)
