# Сверки

## Что это и зачем оно нужно
Для прохождения аудита необходимо делать много промежуточных сверок между
разными компонентами Маркета, каждый год это отнимает много времени сотрудников.
Исходя из этого захотелось создать инструмент, который бы облегчил/ускорил
процесс. Сервис "market-checker" предоставляет возможность создавать,
изменять, запускать сверки и импорты данных, заводить тикеты по результатам проверок
с призывом ответственных за компонент лиц. Так же поддерживается политика
SOX.
