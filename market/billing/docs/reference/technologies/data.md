# Технологии работы с данными



## FDW (Foreign Data Wrapper) {#fdw}

Плагин в PG. Позволяет подключать внешние источники данных.
В частности, можно подключить оракловые таблицы и обращаться к ним как к постгресовым.

* Документация: <https://www.postgresql.org/docs/12/ddl-foreign-data.html>
