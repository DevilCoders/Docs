# Генерация отчётов для дешборда алгоритмов репленишмента
Здесь находятся графы построения источников для дешборда команды алгоритмов репленишмента. На сам дешборд можно посмотреть [здесь](https://datalens.yandex-team.ru/ihjv425ii0xyb-replenishment-dashboard).

## Структура проекта
- **forecast_metrics**: Граф расчёта источника по метрикам качества прогноза. Используется для [этих](https://datalens.yandex-team.ru/ihjv425ii0xyb-replenishment-dashboard?tab=7aG) и [этих](https://datalens.yandex-team.ru/ihjv425ii0xyb-replenishment-dashboard?tab=VQM) графиков.
