# Скрипты по SN (формирование входных данных и аналитика)

## baseline

Папка для скриптов, на основе данных репленишмента, формирующих бейзлайн.

Под бейзлайном понимаются входные данные для SN, при которых результатом оптимизации должен являться план при текущей конфигурации SN.

Для бейзлайна должен биться план-факт с каким то качеством, качество опредлеим позднее.

На основе бейзлайна формируются "гипотезы", т.е. возможные конфигурации SN, немного отличающиеся от бейзлайна

На основе разницы в плановых показателях между результатами запуска AlpacaSN поверх бейзлайна и гипотезы может формировться экшен план.

Для каждой гиптезы будет свой экшен план со своим процессом аппрува и воплощением в реальном мире.

### ss

В отличае от репленишмента в SN сейфти не растет со временем

### delivery_options

В отличае от репленишмента в SN допустимо брать лог параметры не только из прод систем по и проставлять дефолтные.

Так что формируем DO на основе исторических поставок и матрицы блокировок

Более того добавляем дефолтные DO для 3p поставщиков
