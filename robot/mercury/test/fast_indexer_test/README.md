# fast_indexer_test

Канонизируемый быстрый indexer_test без запуска локального YT и Меркури.

## Как обновить sample
1. Получить `messages.protobuf` в оригинальном тесте indexer_test, как описано в подразделе "Как отлаживать SaaS часть" его README.
2. Загрузить файлик в sandbox и полученный resource id прописать в `ya.make` текущего теста в секцию DATA, заменив предыдущий.
3. Переканонизировать текущий тест, запустив с опцией `-Z`.
