Биндинги и вспомогательные классы для обработки потока RT документов с переиспользованием кода Jupiter
* interface - интерфейсы и связывание
* rt - подокументная обработка в mercury с переиспользованием кода shards_prepare
* file - вызов shardmerge_utils с машин с rtyserver (непосредственно на базовых)
* merger - формат-зависимая обработка индексов при слиянии (merge)
* printer - инструменты для дампа rtdoc препаратов и extbuilder-специфичных форматов
