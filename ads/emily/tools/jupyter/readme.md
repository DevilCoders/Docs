# Jupyter in QYP

## Setup

### On server

```bash
logos/tools/jupyter$ make setup
```

## Usage

### QYP Remote

```bash
logos/tools/jupyter$ make server [PORT=1234]
```

### Localy

```bash
logos/tools/jupyter$ make tunnel HOST=<name>.sas.yp-c.yandex.net [PORT=1234]
```

**Open:** http://localhost:1234
