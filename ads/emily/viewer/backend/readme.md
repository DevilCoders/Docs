# ML Viewer - Backend

Директория бэкэнда

## Запуск

### Локально (Docker)

```bash
make run
```

### Локально на aiohttp dev сервер (Docker with autoreload)

Подробный дебаг и авторелоад при изменении кода

#### Установить зависимости

```bash
make install-requirements
```

#### Запустить сервер

```bash
make run-dev
```

**[localhost:8080/api/docs](http://localhost:8080/api/docs)**
